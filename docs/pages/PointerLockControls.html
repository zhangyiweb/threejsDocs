PointerLockControls

此类的实现基于 Pointer Lock API。PointerLockControls 非常适合第一人称 3D 游戏。

代码示例

const controls = new PointerLockControls( camera, document.body );
// add event listener to show/hide a UI (e.g. the game's menu)
controls.addEventListener( 'lock', function () {
	menu.style.display = 'none';
} );
controls.addEventListener( 'unlock', function () {
	menu.style.display = 'block';
} );

导入

PointerLockControls 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { PointerLockControls } from 'three/addons/controls/PointerLockControls.js';

构造函数

new PointerLockControls( camera : Camera, domElement : HTMLElement )

构造一个新的控制器实例。

camera

由控制器管理的相机。
domElement

用于事件监听的 HTML 元素。

默认值为 null。

属性

.isLocked : boolean (readonly)

控制器是否处于锁定状态。

默认值为 false。

.maxPolarAngle : number

相机俯仰角上限。范围以弧度为单位，为 [0, Math.PI]。

默认值为 Math.PI。

.minPolarAngle : number

相机俯仰角下限。范围以弧度为单位，为 [0, Math.PI]。

默认值为 0。

.pointerSpeed : number

指针移动对相机旋转影响的乘数。

默认值为 1。

方法

.getDirection( v : Vector3 ) : Vector3

返回相机的朝向。

v

用于存储方法结果的目标向量。
返回值： 归一化的方向向量。

.lock( unadjustedMovement : boolean )

激活指针锁定。

unadjustedMovement

禁用操作系统级别的鼠标加速调整，改为访问原始鼠标输入。设为 true 将禁用鼠标加速。

默认值为 false。

.moveForward( distance : number )

使相机平行于 xz 平面向前移动。假定 camera.up 为 y 轴向上。

distance

有符号距离。

.moveRight( distance : number )

使相机平行于 xz 平面向右移动。

distance

有符号距离。

.unlock()

退出指针锁定。

事件

.change

当用户移动鼠标时触发。

类型：
  • Object

.lock

当指针锁定状态为「已锁定」（即鼠标被捕获）时触发。

类型：
  • Object

.unlock

当指针锁定状态为「未锁定」（即鼠标不再被捕获）时触发。

类型：
  • Object
