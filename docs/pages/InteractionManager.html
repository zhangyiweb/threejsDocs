InteractionManager

独立于场景图管理 3D 对象的交互。

对于带有 HTMLTexture 的对象，管理器每帧计算 CSS matrix3d 变换，使底层 HTML 元素与其网格保持对齐。由于这些元素是画布的子元素，浏览器会原生地向它们派发指针事件。

代码示例

const interactions = new InteractionManager();
interactions.connect( renderer, camera );
// Objects live anywhere in the scene graph
scene.add( mesh );
// Register for interaction separately
interactions.add( mesh );
// In the animation loop
interactions.update();

导入

InteractionManager 是一个插件，必须显式导入，参见 Installation#Addons。

import { InteractionManager } from 'three/addons/interaction/InteractionManager.js';

构造函数

new InteractionManager()

属性

.camera : Camera

用于计算元素变换的相机。

默认值为 null。

.element : HTMLCanvasElement

画布元素。

默认值为 null。

.objects : Array.<Object3D>

已注册的交互对象。

方法

.add( …objects : Object3D ) : this

向管理器添加一个或多个对象。

objects

要添加的对象。

.connect( renderer : WebGPURenderer | WebGLRenderer, camera : Camera )

存储计算元素变换所需的渲染器和相机。

renderer

渲染器。
camera

相机。

.disconnect()

断开此管理器，清除渲染器和相机引用。

.remove( …objects : Object3D ) : this

从管理器中移除一个或多个对象。

objects

要移除的对象。

.update()

更新所有已注册对象的元素变换。在动画循环中每帧调用一次。
