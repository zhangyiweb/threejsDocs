Texture3DNode

此类 uniform 节点表示 3D 纹理。

构造函数

new Texture3DNode( value : Data3DTexture, uvNode : Node.<(vec2|vec3)>, levelNode : Node.<int> )

构造一个新的 3D 纹理节点。

value

3D 纹理。
uvNode

uv 节点。

默认值为 null。
levelNode

级别节点。

默认值为 null。

属性

.isTexture3DNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

方法

.generateOffset( builder : NodeBuilder, offsetNode : Node ) : string

生成偏移量代码片段。

builder

当前节点构建器。
offsetNode

要为其生成代码的偏移节点。
重写： TextureNode#generateOffset
返回值： 生成的代码片段。

.generateUV( builder : NodeBuilder, uvNode : Node ) : string

生成 uv 代码片段。

builder

当前节点构建器。
uvNode

要为其生成代码的 uv 节点。
重写： TextureNode#generateUV
返回值： 生成的代码片段。

.getDefaultUV() : Node.<vec3>

返回默认 uv 节点；在 3D 纹理场景下， 这是一个三维 uv 节点。

重写： TextureNode#getDefaultUV
返回值： 默认 uv 节点。

.getInputType( builder : NodeBuilder ) : string

重写默认实现，返回固定值 'texture3D'。

builder

当前节点构建器。
重写： TextureNode#getInputType
返回值： 输入类型。

.normal( uvNode : Node.<vec3> ) : Node.<vec3>

根据给定的 uv 计算法线。这些纹理坐标表示 3D 纹理内部的一个位置。与几何法线不同，此法线 表示 3D 纹理内部标量数据的斜率或梯度。

uvNode

定义 3D 纹理中位置的 uv 节点。
返回值： 表示数据中斜率/梯度的法线。

.setUpdateMatrix( value : boolean )

已用空实现重写，因为对于 3D 纹理会忽略 updateMatrix 标志。 uv 变换矩阵不会应用于 3D 纹理。

value

更新开关。
重写： TextureNode#setUpdateMatrix
