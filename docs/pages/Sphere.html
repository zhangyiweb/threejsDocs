Sphere

由中心点和半径定义的解析三维球体。该类主要用于 作为三维对象的包围球（Bounding Sphere）。

构造函数

new Sphere( center : Vector3, radius : number )

构造一个新的球体。

center

球体的中心

默认值为 (0,0,0)。
radius

球体的半径。

默认值为 -1。

属性

.center : Vector3

球体的中心

.isSphere : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.radius : number

球体的半径。

方法

.applyMatrix4( matrix : Matrix4 ) : Sphere

使用给定的 4x4 变换矩阵变换此球体。

matrix

变换矩阵。
返回值： 对此球体的引用。

.clampPoint( point : Vector3, target : Vector3 ) : Vector3

将点钳制在球体内。如果点在球外，则会将其 钳制到球体边缘上最近的点。已在球内的点 不受影响。

point

要钳制的点。
target

用于存储方法结果的目标向量。
返回值： 钳制后的点。

.clone() : Sphere

返回一个新球体，其值从本实例复制而来。

返回值： 本实例的克隆。

.containsPoint( point : Vector3 ) : boolean

如果此球体包含给定点（包含球体表面），则返回 true。

point

要检查的点。
返回值： 此球体是否包含给定点。

.copy( sphere : Sphere ) : Sphere

将给定球体的值复制到本实例。

sphere

要复制的球体。
返回值： 对此球体的引用。

.distanceToPoint( point : Vector3 ) : number

返回从球体边界到给定点的最近距离。 如果球体包含该点，则距离为负。

point

要计算距离的点。
返回值： 到该点的距离。

.equals( sphere : Sphere ) : boolean

如果此球体与给定球体相等，则返回 true。

sphere

要测试相等性的球体。
返回值： 此包围球是否与给定球体相等。

.expandByPoint( point : Vector3 ) : Sphere

扩展此球体的边界以包含给定点。

point

要包含的点。
返回值： 对此球体的引用。

.fromJSON( json : Object ) : Sphere

从序列化结构设置包围球。

json

用于设置球体的序列化 JSON。
返回值： 对此包围球的引用。

.getBoundingBox( target : Box3 ) : Box3

返回包围此球体的包围盒。

target

用于存储方法结果的目标包围盒。
返回值： 包围此球体的包围盒。

.intersectsBox( box : Box3 ) : boolean

如果此球体与给定包围盒相交，则返回 true。

box

要测试的包围盒。
返回值： 此球体是否与给定包围盒相交。

.intersectsPlane( plane : Plane ) : boolean

如果此球体与给定平面相交，则返回 true。

plane

要测试的平面。
返回值： 此球体是否与给定平面相交。

.intersectsSphere( sphere : Sphere ) : boolean

如果此球体与给定球体相交，则返回 true。

sphere

要测试的球体。
返回值： 此球体是否与给定球体相交。

.isEmpty() : boolean

如果球体为空（半径设为负数），则返回 true。

半径为 0 的球体仅包含其中心点， 不视为空。

返回值： 此球体是否为空。

.makeEmpty() : Sphere

使此球体为空，即在三维空间中不包围任何区域。

返回值： 对此球体的引用。

.set( center : Vector3, radius : number ) : Sphere

通过复制给定值来设置球体的分量。

center

中心。
radius

半径。
返回值： 对此球体的引用。

.setFromPoints( points : Array.<Vector3>, optionalCenter : Vector3 ) : Sphere

计算点列表的最小包围球。 如果给定了可选的中心点，则将其用作球体的中心。 否则，将计算包围这些点的轴对齐包围盒的中心。

points

三维空间中的点列表。
optionalCenter

球体的中心。
返回值： 对此球体的引用。

.toJSON() : Object

返回包围球的序列化结构。

返回值： 表示对象状态的序列化结构。

.translate( offset : Vector3 ) : Sphere

按给定偏移量平移球体的中心。

offset

偏移量。
返回值： 对此球体的引用。

.union( sphere : Sphere ) : Sphere

扩展此球体，使其同时包围原球体和给定球体。

sphere

要包含的球体。
返回值： 对此球体的引用。
