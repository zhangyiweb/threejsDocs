AxesHelper

用于以简单方式可视化三个坐标轴的对象。X 轴为红色，Y 轴为绿色，Z 轴为蓝色。

代码示例

const axesHelper = new THREE.AxesHelper( 5 );
scene.add( axesHelper );

构造函数

new AxesHelper( size : number )

构造一个新的坐标轴辅助对象。

size

表示坐标轴的线段尺寸。

默认值为 1.

方法

.dispose()

释放该实例分配的 GPU 相关资源。当应用中不再使用该实例时，应调用此方法。

.setColors( xAxisColor : number | Color | string, yAxisColor : number | Color | string, zAxisColor : number | Color | string ) : AxesHelper

定义坐标轴辅助对象的颜色。

xAxisColor

X 轴的颜色。
yAxisColor

Y 轴的颜色。
zAxisColor

Z 轴的颜色。
返回值： 对该坐标轴辅助对象的引用。
