TreeGenerator

生长程序化树木骨架——树干、枝条与细枝，各自扫掠成锥形 管——并烘焙进单个非索引 BufferGeometry（仅含 position 与 normal）， 以便实例化成森林。它只生成枝干；树叶需作为单独一层添加。

给定 seed 时分支是确定性的：递归扫掠铺设带平行传输坐标系的 轻柔弯曲管（因此不会扭转），按管道模型分叉（每个子枝远细于父枝）， 沿每根枝的上部以黄金角滚动散布子枝，并将它们向上拉向光源。 外扩的根部、非线性锥度与重力下垂共同塑造出树木特征。

参数通过流式构建器设置：每个默认值都有对应的 set<Param>() （setSeed、setLevels、setChildren 等），各自返回 this 以便链式调用。

每次 build() 返回由调用方拥有的全新独立网格，因此同一个 生成器可重新设参并反复构建，长出形态各异的林分：

代码示例

const generator = new TreeGenerator( material );
const oak = generator.setSeed( 1 ).setLevels( 4 ).build();
const pine = generator.setSeed( 2 ).setLevels( 5 ).build();

构造函数

new TreeGenerator()
