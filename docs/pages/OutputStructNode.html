OutputStructNode

此节点可用于在着色器程序中定义多个输出。

构造函数

new OutputStructNode( …members : Node )

构造一个新的 output struct 节点。构造函数可接受任意数量的节点作为成员。

members

节点参数列表。

属性

.isOutputStructNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.members : Array.<Node>

定义输出的节点数组。
