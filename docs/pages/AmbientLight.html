AmbientLight

该光源会均匀地照亮场景中的所有物体。

由于没有方向，无法用于投射阴影。

代码示例

const light = new THREE.AmbientLight( 0x404040 ); // soft white light
scene.add( light );

构造函数

new AmbientLight( color : number | Color | string, intensity : number )

构造一个新的环境光。

color

光源颜色。

默认值为 0xffffff。
intensity

光源强度。

默认值为 1。

属性

.isAmbientLight : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
