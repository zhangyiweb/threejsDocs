ToneMappingNode

此节点表示色调映射操作。

构造函数

new ToneMappingNode( toneMapping : number, exposureNode : Node, colorNode : Node )

构造一个新的色调映射节点。

toneMapping

色调映射类型。
exposureNode

色调映射曝光度。
colorNode

要处理的颜色节点。

默认值为 null。

属性

.colorNode : Node

表示要处理的颜色。

默认值为 null。

.exposureNode : Node

色调映射曝光度。

默认值为 null。

方法

.customCacheKey() : number

重写默认的 customCacheKey() 实现，将色调映射类型纳入缓存键。

重写： TempNode#customCacheKey
返回值： 哈希值。

.getToneMapping() : number

获取色调映射类型。

返回值： 色调映射类型。

.setToneMapping( value : number ) : ToneMappingNode

设置色调映射类型。

value

色调映射类型。
返回值： 对此节点的引用。
