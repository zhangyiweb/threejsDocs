LightProbeGenerator

用于创建 LightProbe 实例的工具类。

导入

LightProbeGenerator 是一个插件，必须显式导入，参见 Installation#Addons。

import { LightProbeGenerator } from 'three/addons/lights/LightProbeGenerator.js';

静态方法

.fromCubeRenderTarget( renderer : WebGPURenderer | WebGLRenderer, cubeRenderTarget : CubeRenderTarget | WebGLCubeRenderTarget ) : Promise.<LightProbe> (async)

根据给定的（辐射）环境贴图创建光照探针。 该方法期望环境贴图以立方体渲染目标的形式表示。

立方体渲染目标必须为 RGBA 格式，因此 cubeRenderTarget.texture.format 必须设置为 RGBAFormat。

renderer

渲染器。
cubeRenderTarget

环境贴图。
返回值： 解析为所创建光照探针的 Promise。

.fromCubeTexture( cubeTexture : CubeTexture ) : LightProbe

根据给定的（辐射）环境贴图创建光照探针。 该方法期望环境贴图以立方体贴图的形式表示。

cubeTexture

环境贴图。
返回值： 所创建的光照探针。
