Cache

一个简单的缓存系统，由 FileLoader 内部使用。 要为所有使用 FileLoader 的加载器启用缓存，请在应用中设置一次 THREE.Cache.enabled = true.。

属性

.enabled : boolean

是否启用缓存。

默认值为 false。

.files : Object.<string, Object>

保存已缓存文件的字典。

静态方法

.add( key : string, file : Object )

添加一条缓存条目，并用键引用该文件。如果该键已 持有文件，则会被覆盖。

key

用于引用已缓存文件的键。
file

要缓存的文件。

.clear()

移除缓存中的所有值。

.get( key : string ) : Object | undefined

获取给定键对应的缓存值。

key

用于引用已缓存文件的键。
返回值： 已缓存的文件。若键不存在则返回 undefined。

.remove( key : string )

移除与给定键关联的已缓存文件。

key

用于引用已缓存文件的键。
