CSS3DSprite

CSS3DObject 的特化版本，将 DOM 元素表示为精灵（sprite）。

导入

CSS3DSprite 是一个插件，必须显式导入，参见 Installation#Addons。

import { CSS3DSprite } from 'three/addons/renderers/CSS3DRenderer.js';

构造函数

new CSS3DSprite( element : HTMLElement )

构造一个新的 CSS3D 精灵对象。

element

DOM 元素。

属性

.isCSS3DSprite : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.rotation2D : number

精灵的旋转角度（单位：弧度）。

默认值为 0。
