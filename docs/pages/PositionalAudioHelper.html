PositionalAudioHelper

此辅助对象用于显示位置音频的方向锥体。

PositionalAudioHelper 必须作为位置音频的子对象添加。

代码示例

const positionalAudio = new THREE.PositionalAudio( listener );
positionalAudio.setDirectionalCone( 180, 230, 0.1 );
scene.add( positionalAudio );
const helper = new PositionalAudioHelper( positionalAudio );
positionalAudio.add( helper );

导入

PositionalAudioHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { PositionalAudioHelper } from 'three/addons/helpers/PositionalAudioHelper.js';

构造函数

new PositionalAudioHelper( audio : PositionalAudio, range : number, divisionsInnerAngle : number, divisionsOuterAngle : number )

构造一个新的位置音频辅助对象。

audio

要可视化的音频。
range

方向锥体的范围。

默认值为 1。
divisionsInnerAngle

方向锥体内侧部分的分割数。

默认值为 16。
divisionsOuterAngle

方向锥体外侧部分的分割数。

默认值为 2。

属性

.audio : PositionalAudio

要可视化的音频。

.divisionsInnerAngle : number

方向锥体内侧部分的分割数。

默认值为 16。

.divisionsOuterAngle : number

方向锥体外侧部分的分割数。

默认值为 2。

.range : number

方向锥体的范围。

默认值为 1。

方法

.dispose()

释放此实例分配的 GPU 相关资源。 当应用中不再使用此实例时，请调用此方法。

.update()

更新辅助对象。每当位置音频的方向锥体发生变化时，必须调用此方法。
