Wireframe

用于基于宽线创建线框的类。

此模块只能与 WebGLRenderer 一起使用。使用 WebGPURenderer 时， 请从 lines/webgpu/Wireframe.js 导入该类。

代码示例

const geometry = new THREE.IcosahedronGeometry();
const wireframeGeometry = new WireframeGeometry2( geo );
const wireframe = new Wireframe( wireframeGeometry, material );
scene.add( wireframe );

导入

Wireframe 是一个插件，必须显式导入，参见 Installation#Addons。

import { Wireframe } from 'three/addons/lines/Wireframe.js';

构造函数

new Wireframe( geometry : LineSegmentsGeometry, material : LineMaterial )

构造一个新的线框。

geometry

线几何体。
material

线材质。

属性

.isWireframe : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.computeLineDistances() : Wireframe

计算渲染虚线所需的距离值数组。 对于几何体中的每个顶点，该方法会计算从当前点 到线起点的累积长度。

返回值： 对此实例的引用。
