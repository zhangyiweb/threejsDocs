IESSpotLight

SpotLight 的 IES 版本。只能与 WebGPURenderer 一起使用。

构造函数

new IESSpotLight( color : number | Color | string, intensity : number, distance : number, angle : number, penumbra : number, decay : number )

构造一个新的 IES 聚光灯。

color

光源颜色。

默认值为 0xffffff。
intensity

光源强度，以坎德拉（cd）为单位。

默认值为 1。
distance

光源的最大作用范围。0 表示无限制。

默认值为 0。
angle

光线沿其方向的最大散射角度，上限为 Math.PI/2。

默认值为 Math.PI/3。
penumbra

聚光灯锥体因半影而衰减的百分比。取值范围为 [0,1]。

默认值为 0。
decay

光线沿距离衰减的程度。

默认值为 2。

属性

.iesMap : Texture

IES 贴图。它是一个查找表，存储归一化的衰减因子 （0.0 到 1.0），表示光源在特定角度下的强度。

默认值为 null。
