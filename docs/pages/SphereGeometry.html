SphereGeometry

用于生成球体几何体的类。

代码示例

const geometry = new THREE.SphereGeometry( 15, 32, 16 );
const material = new THREE.MeshBasicMaterial( { color: 0xffff00 } );
const sphere = new THREE.Mesh( geometry, material );
scene.add( sphere );

构造函数

new SphereGeometry( radius : number, widthSegments : number, heightSegments : number, phiStart : number, phiLength : number, thetaStart : number, thetaLength : number )

构造一个新的球体几何体。

radius

球体半径。

默认值为 1.
widthSegments

水平分段数。最小值为 3。

默认值为 32.
heightSegments

垂直分段数。最小值为 2。

默认值为 16.
phiStart

水平起始角度（弧度）。

默认值为 0.
phiLength

水平扫掠角大小。

默认值为 Math.PI*2.
thetaStart

垂直起始角度（弧度）。

默认值为 0.
thetaLength

垂直扫掠角大小。

默认值为 Math.PI.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

静态方法

.fromJSON( data : Object ) : SphereGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
