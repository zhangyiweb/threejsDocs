OverrideContextNode

一种专用上下文节点，用于在节点子图或流程中覆盖特定的目标节点。这允许在特定流程节点的编译过程中动态替换特定输入（例如法线和位置向量），而无需重建或复制源节点。

代码示例

// Method chaining example:
node.overrideNode( positionLocal, () => positionLocal.add( vec3( 1, 0, 0 ) ) );
// Context assignment example:
material.contextNode = overrideNode( positionLocal, () => positionLocal.add( vec3( 1, 0, 0 ) ) );

构造函数

new OverrideContextNode( overrideNodes : Map.<Node, function()>, flowNode : Node | null )

构造一个新的 override context 节点。

overrideNodes

将目标节点映射到各自覆盖回调函数的 Map。
flowNode

应修改其上下文的节点。

默认值为 null。

属性

.isOverrideContextNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.type : string (readonly)

返回节点的类型。

方法

.getFlowContextData() : Object

通过遍历层级结构从所有父上下文节点收集上下文数据，合并所有遇到的 OverrideContextNode 实例的 overrideNodes 映射。

重写： ContextNode#getFlowContextData
返回值： 收集到的上下文数据，包含合并后的 overrideNodes 映射。
