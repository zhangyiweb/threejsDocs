StackTrace

表示用于调试目的的堆栈跟踪的类。

构造函数

new StackTrace( stackMessage : Error | string | null )

通过捕获并过滤当前堆栈跟踪来创建 StackTrace 实例。

stackMessage

可选的堆栈跟踪，用于替代重新捕获。

默认值为 null。

属性

.isStackTrace : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.stack : Array.<{fn: string, file: string, line: number, column: number}>

堆栈跟踪。

方法

.getError( message : string ) : string

返回包含堆栈跟踪的完整错误消息。

message

错误消息。
返回值： 带有堆栈跟踪的完整错误消息。

.getLocation() : string

返回栈顶帧格式化后的位置字符串。

返回值： 格式化后的堆栈跟踪消息。
