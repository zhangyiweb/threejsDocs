WebGL

用于进行基本 WebGL 2 能力检测的工具模块。

导入

WebGL 是一个插件，必须显式导入，参见 Installation#Addons。

import WebGL from 'three/addons/capabilities/WebGL.js';

静态方法

.getWebGL2ErrorMessage() : HTMLDivElement

返回一个表示格式化错误信息的 div 元素，可在不支持 WebGL 2 时追加到网站中。

返回值： 表示不支持 WebGL 2 的格式化错误信息的 div 元素。

.isColorSpaceAvailable( colorSpace : string ) : boolean

若给定颜色空间可用则返回 true。此方法仅在支持 WebGL 2 时可用。

colorSpace

要检测的颜色空间。
返回值： 给定颜色空间是否可用。

.isWebGL2Available() : boolean

若 WebGL 2 可用则返回 true。

返回值： WebGL 2 是否可用。
