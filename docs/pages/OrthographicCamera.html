OrthographicCamera

使用正交投影的相机。

在此投影模式下，物体在渲染图像中的大小与其到相机的距离无关，始终保持恒定。这对于渲染 2D 场景和 UI 元素等非常有用。

代码示例

const camera = new THREE.OrthographicCamera( width / - 2, width / 2, height / 2, height / - 2, 1, 1000 );
scene.add( camera );

构造函数

new OrthographicCamera( left : number, right : number, top : number, bottom : number, near : number, far : number )

构造一个新的正交相机。

left

相机视锥体的左裁剪面。

默认值为 -1。
right

相机视锥体的右裁剪面。

默认值为 1。
top

相机视锥体的上裁剪面。

默认值为 1。
bottom

相机视锥体的下裁剪面。

默认值为 -1。
near

相机的近裁剪面。

默认值为 0.1。
far

相机的远裁剪面。

默认值为 2000。

属性

.bottom : number

相机视锥体的下裁剪面。

默认值为 -1。

.far : number

相机的远裁剪面。必须大于 OrthographicCamera#near 的当前值。

默认值为 2000。

.isOrthographicCamera : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.left : number

相机视锥体的左裁剪面。

默认值为 -1。

.near : number

相机的近裁剪面。有效范围大于 0 且小于 OrthographicCamera#far 的当前值。

请注意，与 PerspectiveCamera 不同，正交相机的近裁剪面可以使用 0 作为有效值。

默认值为 0.1。

.right : number

相机视锥体的右裁剪面。

默认值为 1。

.top : number

相机视锥体的上裁剪面。

默认值为 1。

.view : Object

表示视锥体窗口规格。不应直接编辑此属性，而应通过 PerspectiveCamera#setViewOffset 和 PerspectiveCamera#clearViewOffset 进行设置。

默认值为 null。

.zoom : number

相机的缩放系数。

默认值为 1。

方法

.clearViewOffset()

从投影矩阵中移除视图偏移。

.setViewOffset( fullWidth : number, fullHeight : number, x : number, y : number, width : number, height : number )

在更大的视锥体中设置偏移。这对于多窗口或多显示器/多机器配置非常有用。

fullWidth

多视图配置的完整宽度。
fullHeight

多视图配置的完整高度。
x

子相机的水平偏移。
y

子相机的垂直偏移。
width

子相机的宽度。
height

子相机的高度。
参见：
  • PerspectiveCamera#setViewOffset

.updateProjectionMatrix()

更新相机的投影矩阵。在更改任何相机属性后必须调用。
