WaterMesh

基础的平面反射水面效果。

注意：该类只能与 WebGPURenderer 一起使用。 使用 WebGLRenderer 时，请改用 Water。

参考：

  • Flat mirror for three.js
  • An implementation of water shader based on the flat mirror
  • Water shader explanations in WebGL

导入

WaterMesh 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { WaterMesh } from 'three/addons/objects/WaterMesh.js';

构造函数

new WaterMesh( geometry : BufferGeometry, options : WaterMesh~Options )

构造一个新的水面网格。

geometry

水面网格的几何体。
options

配置选项。

属性

.alpha : UniformNode.<float>

透明度值。

默认值为 1。

.distortionScale : UniformNode.<float>

扭曲缩放。

默认值为 20。

.isWaterMesh : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.resolutionScale : number

效果的分辨率缩放。

默认值为 0.5。

.size : UniformNode.<float>

尺寸值。

默认值为 1。

.sunColor : UniformNode.<color>

太阳颜色。

默认值为 0xffffff。

.sunDirection : UniformNode.<vec3>

太阳方向。

默认值为 (0.70707,0.70707,0.0)。

.waterColor : UniformNode.<color>

水面颜色。

默认值为 0x7f7f7f。

.waterNormals : TextureNode

水面的法线贴图。
