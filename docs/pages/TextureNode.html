TextureNode

此类 uniform 节点表示一张 2D 纹理。

构造函数

new TextureNode( value : Texture, uvNode : Node.<(vec2|vec3)>, levelNode : Node.<int>, biasNode : Node.<float> )

构造一个新的纹理节点。

value

纹理。

默认值为 EmptyTexture。
uvNode

UV 节点。

默认值为 null。
levelNode

层级节点。

默认值为 null。
biasNode

偏置节点。

默认值为 null。

属性

.biasNode : Node.<float>

表示在细节层级（LOD）计算期间要应用的偏置。

默认值为 null。

.compareNode : Node.<float>

表示纹理采样所对比的参考值。

默认值为 null。

.depthNode : Node.<int>

使用纹理数组时，深度节点定义要选择的层。

默认值为 null。

.gatherNode : Node.<int>

表示要 gather 的通道的可选索引常量。 该值必须在 [0, 3] 范围内，且为编译期常量。

默认值为 null。

.gradNode : Array.<Node.<vec2>>

定义后，将使用显式梯度对纹理进行采样。

默认值为 null。

.isTextureNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.levelNode : Node.<int>

表示应选择的 mip 层级。

默认值为 null。

.offsetNode : Node.<(ivec2|ivec3)>

表示在采样纹理之前应用于未归一化纹理坐标的可选纹素偏移。

默认值为 null。

.referenceNode : Node

引用节点。

默认值为 null。

.sampler : boolean

纹理值应采样还是直接获取（fetch）。

默认值为 true。

.updateMatrix : boolean

UV 变换矩阵是否应自动更新。若要更改此属性的值，请使用 setUpdateMatrix()。

默认值为 false。

.updateType : string

默认情况下不会执行 update() 方法。取决于是否应用了 UV 变换矩阵和/或 flipY，update() 会按对象执行。

默认值为 'none'。

重写： UniformNode#updateType

.uvNode : Node.<(vec2|vec3)>

表示纹理坐标。

默认值为 null。

.value : Texture

纹理值。

重写： UniformNode#value
