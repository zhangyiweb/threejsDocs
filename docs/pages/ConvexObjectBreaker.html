ConvexObjectBreaker

此类可用于将凸 Geometry 对象细分为多个碎片。

使用 prepareBreakableObject 函数准备可破碎的 Mesh 对象， 然后调用各种函数对对象进行细分（subdivideByImpact、cutByPlane）。 细分产生的子对象无需再次调用 prepareBreakableObject。

对象需满足以下条件：

  • Mesh 对象必须具有 buffer geometry 和 material。
  • 顶点法线必须为平面法线（未平滑）。
  • 几何体必须为凸体（库中不会检查）。可使用 ConvexGeometry 创建凸几何体。 BoxGeometry、SphereGeometry 及其他 凸基本体也可使用。

注意：此库会向对象的 userData 成员添加变量（参见 prepareBreakableObject 函数）。 与其他库一起使用时请谨慎，并阅读相关代码。

导入

ConvexObjectBreaker 是一个插件，必须显式导入，参见 Installation#Addons。

import { ConvexObjectBreaker } from 'three/addons/misc/ConvexObjectBreaker.js';

构造函数

new ConvexObjectBreaker( minSizeForBreak : number, smallDelta : number )

构造一个新的凸对象破碎器。

minSizeForBreak

碎片可继续破碎的最小尺寸。

默认值为 1.4。
smallDelta

判定某点属于某平面的最大距离。

默认值为 0.0001。

方法

.cutByPlane( object : Object3D, plane : Plane, output : Object ) : number

使用平面将给定的 3D 对象细分为多个碎片。

object

要细分的 3D 对象。
plane

用于切割 3D 对象的平面。
output

用于存储碎片的对象。
返回值： 碎片数量。

.prepareBreakableObject( object : Object3D, mass : number, velocity : Vector3, angularVelocity : Vector3, breakable : boolean )

所有应可破碎的 3D 对象都必须调用此方法。

object

3D 对象。必须具有凸几何体。
mass

3D 对象的质量（kg）。必须大于 0。
velocity

3D 对象的速度。
angularVelocity

3D 对象的角速度。
breakable

3D 对象是否可破碎。

.subdivideByImpact( object : Object3D, pointOfImpact : Vector3, normal : Vector3, maxRadialIterations : number, maxRandomIterations : number ) : Array.<Object3D>

通过撞击将给定的 3D 对象细分为多个碎片（即另一对象在某一表面点撞击给定 3D 对象）。

object

要细分的 3D 对象。
pointOfImpact

撞击点。
normal

撞击法线。
maxRadialIterations

径向切割的迭代次数。
maxRandomIterations

非径向切割的最大随机迭代次数。
返回值： 碎片数组。
