DataArrayTexture

直接从原始缓冲区数据创建纹理数组。

构造函数

new DataArrayTexture( data : TypedArray, width : number, height : number, depth : number )

构造一个新的数据数组纹理。

data

缓冲区数据。

默认值为 null。
width

纹理的宽度。

默认值为 1。
height

纹理的高度。

默认值为 1。
depth

纹理的深度。

默认值为 1。

属性

.flipY : boolean

若设为 true，纹理在上传到 GPU 时会沿垂直轴翻转。

已覆盖并默认设为 false。

默认值为 false。

重写： Texture#flipY

.generateMipmaps : boolean

是否为纹理生成 mipmap（若可能）。

已覆盖并默认设为 false。

默认值为 false。

重写： Texture#generateMipmaps

.image : Object

数据纹理的图像定义。

重写： Texture#image

.isDataArrayTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.layerUpdates : Set.<number>

需要在纹理中更新的所有层的集合。

.magFilter : NearestFilter | NearestMipmapNearestFilter | NearestMipmapLinearFilter | LinearFilter | LinearMipmapNearestFilter | LinearMipmapLinearFilter

当一个纹素覆盖多于一个像素时，纹理的采样方式。

已覆盖并默认设为 NearestFilter。

默认值为 NearestFilter。

重写： Texture#magFilter

.minFilter : NearestFilter | NearestMipmapNearestFilter | NearestMipmapLinearFilter | LinearFilter | LinearMipmapNearestFilter | LinearMipmapLinearFilter

当一个纹素覆盖少于一个像素时，纹理的采样方式。

已覆盖并默认设为 NearestFilter。

默认值为 NearestFilter。

重写： Texture#minFilter

.unpackAlignment : boolean

指定内存中每行像素起始位置的对齐要求。

已覆盖并默认设为 1。

默认值为 1。

重写： Texture#unpackAlignment

.wrapR : RepeatWrapping | ClampToEdgeWrapping | MirroredRepeatWrapping

定义纹理在深度方向上的包裹方式，对应 UVW 映射中的 W。

默认值为 ClampToEdgeWrapping。

方法

.addLayerUpdate( layerIndex : number )

标记纹理的特定层需要更新。 通常当 Texture#needsUpdate 设为 true 时， 整个数据纹理数组都会发送到 GPU。标记特定层 只会传输与数组中特定深度相关的 mipmap 子集， 这通常性能更好。

layerIndex

应当更新的层索引。

.clearLayerUpdates()

重置层更新注册表。
