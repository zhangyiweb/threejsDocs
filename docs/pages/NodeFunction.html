NodeFunction

节点函数的基类。必须为每种支持的原生着色器语言 实现一个派生模块。与其他 Node* 模块类似，此类仅在构建过程中相关， 不会在用户级代码中使用。

构造函数

new NodeFunction( type : string, inputs : Array.<NodeFunctionInput>, name : string, precision : string )

构造一个新的节点函数。

type

节点类型。此类型是节点函数的返回类型。
inputs

函数的输入。
name

函数的名称。

默认值为 ''。
precision

精度限定符。

默认值为 ''。

属性

.inputs : Array.<NodeFunctionInput>

函数的输入。

.name : string

uniform 的名称。

默认值为 ''。

.precision : string

精度限定符。

默认值为 ''。

.type : string

节点类型。此类型是节点函数的返回类型。

方法

.getCode( name : string ) : string (abstract)

此方法返回节点函数的原生代码。

name

函数的名称。
返回值： 着色器代码。
