SubBuildNode

此节点用于在节点系统中构建子构建（sub-build）。

构造函数

new SubBuildNode( node : Node, name : string, nodeType : string )

node

要在子构建中构建的节点。
name

子构建的名称。
nodeType

节点类型（如果已知）。

默认值为 null。

属性

.isSubBuildNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.name : string

子构建的名称。

重写： Node#name

.node : Node

要在子构建中构建的节点。
