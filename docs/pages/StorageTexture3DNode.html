StorageTexture3DNode

此特殊版本的纹理节点可用于 通过计算着色器向 3D 存储纹理写入数据。

构造函数

new StorageTexture3DNode( value : Storage3DTexture, uvNode : Node.<vec3>, storeNode : Node )

构造一个新的 3D 存储纹理节点。

value

3D 存储纹理。
uvNode

uv 节点。
storeNode

应存储到纹理中的值节点。

默认值为 null。

属性

.isStorageTexture3DNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

方法

.generateOffset( builder : NodeBuilder, offsetNode : Node ) : string

生成偏移量代码片段。

builder

当前节点构建器。
offsetNode

要为其生成代码的偏移节点。
重写： StorageTextureNode#generateOffset
返回值： 生成的代码片段。

.generateUV( builder : NodeBuilder, uvNode : Node ) : string

生成 uv 代码片段。

builder

当前节点构建器。
uvNode

要为其生成代码的 uv 节点。
重写： StorageTextureNode#generateUV
返回值： 生成的代码片段。

.getDefaultUV() : Node.<vec3>

返回默认 uv 节点；在 3D 纹理场景下， 这是一个三维 uv 节点。

重写： StorageTextureNode#getDefaultUV
返回值： 默认 uv 节点。

.setUpdateMatrix( value : boolean )

已用空实现重写，因为对于 3D 纹理会忽略 updateMatrix 标志。 uv 变换矩阵不会应用于 3D 纹理。

value

更新开关。
重写： StorageTextureNode#setUpdateMatrix
