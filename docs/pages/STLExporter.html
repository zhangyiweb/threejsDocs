STLExporter

STL 导出器。

STL 文件仅描述三维对象的表面几何，不包含颜色、纹理或其他常见模型属性。STL 格式同时规定了 ASCII 与二进制表示，其中二进制更为紧凑。STL 文件不含比例信息或索引，单位为任意单位。

代码示例

const exporter = new STLExporter();
const data = exporter.parse( mesh, { binary: true } );

导入

STLExporter 是一个插件，必须显式导入，参见 Installation#Addons。

import { STLExporter } from 'three/addons/exporters/STLExporter.js';

构造函数

new STLExporter()

方法

.parse( scene : Object3D, options : STLExporter~Options ) : string | ArrayBuffer

解析给定的 3D 对象并生成 STL 输出。

若该 3D 对象由多个子对象和几何体组成，它们会在文件中合并为单个网格。

scene

包含要编码网格的场景、网格或任何其他 3D 对象。
options

导出选项。
返回值： 导出的 STL。

类型定义

.Options

STLExporter 的导出选项。

binary
boolean

是以二进制格式还是 ASCII 格式导出。

默认值为 false。
