AssignNode

这些节点表示赋值操作，即将一个节点赋值给另一个节点。

构造函数

new AssignNode( targetNode : Node, sourceNode : Node )

构造一个新的赋值节点。

targetNode

目标节点。
sourceNode

源节点。

属性

.isAssignNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.sourceNode : Node

源节点。

.targetNode : Node

目标节点。

方法

.hasDependencies() : boolean

此节点是否在其他节点的上下文中被多次使用。此方法被覆盖， 因为它始终返回 false（赋值是唯一的）。

重写： TempNode#hasDependencies
返回值： 指示是否存在对其他节点的多个依赖的标志。始终为 false。

.needsSplitAssign( builder : NodeBuilder ) : boolean

将源赋值给目标时是否需要拆分。当目标和源数据类型的分量长度 不匹配时可能发生这种情况。

builder

当前的节点构建器。
返回值： 将源赋值给目标时是否需要拆分。
