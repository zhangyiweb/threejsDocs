VRButton

用于创建按钮的工具类，该按钮可基于 WebXR 启动沉浸式 VR 会话。可通过工厂方法创建按钮，然后将其添加到网站的 DOM 中。

代码示例

document.body.appendChild( VRButton.createButton( renderer ) );

导入

VRButton 是一个插件，必须显式导入，参见 Installation#Addons。

import { VRButton } from 'three/addons/webxr/VRButton.js';

属性

.xrSessionIsGranted : boolean

XR 会话是否已授权。

默认值为 false。

静态方法

.createButton( renderer : WebGLRenderer | WebGPURenderer, sessionInit : XRSessionInit ) : HTMLElement

构造一个新的 VR 按钮。

renderer

渲染器。
sessionInit

VR 会话的配置对象。
返回值： 按钮；若不支持 immersive-vr，则返回错误信息。

.registerSessionGrantedListener()

注册 sessiongranted 事件监听器。当会话被授权时，VRButton#xrSessionIsGranted 标志将变为 true。模块会自动调用此方法，因此应用层通常无需手动调用。
