SharpenNode

用于对比度自适应锐化（RCAS）的后处理节点。

参考：https://gpuopen.com/fidelityfx-superresolution/。

导入

SharpenNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { sharpen } from 'three/addons/tsl/display/SharpenNode.js';

构造函数

new SharpenNode( textureNode : TextureNode, sharpness : Node.<float>, denoise : Node.<bool> )

构造一个新的锐化节点。

textureNode

表示效果输入的纹理节点。
sharpness

锐化强度。0 = 最大锐化，2 = 无锐化。

默认值为 0.2。
denoise

是否在噪点区域减弱锐化。

默认值为 false。

属性

.denoise : Node.<bool>

是否在噪点区域减弱锐化。

默认值为 false。

.isSharpenNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.sharpness : Node.<float>

锐化强度。0 = 最大，2 = 无。

默认值为 0.2。

.textureNode : TextureNode

表示效果输入的纹理节点。

.updateBeforeType : string

updateBeforeType 被设置为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

方法

.dispose()

释放内部资源。当效果不再使用时应调用此方法。

重写： TempNode#dispose

.getTextureNode() : PassTextureNode

以纹理节点形式返回效果结果。

返回值： 表示效果结果的纹理节点。

.setSize( width : number, height : number )

设置效果的输出尺寸。

width

宽度（像素）。
height

高度（像素）。

.setup( builder : NodeBuilder ) : PassTextureNode

此方法用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

此方法用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
