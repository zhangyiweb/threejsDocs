BoxHelper

用于以图形方式显示对象周围世界轴对齐包围盒的辅助对象。实际包围盒由 Box3 处理，这只是用于调试的可视化辅助。当从其创建的对象发生变换时，可通过 BoxHelper#update 自动调整大小。注意：对象必须具有几何体才能正常工作，因此不适用于精灵（sprites）。

代码示例

const sphere = new THREE.SphereGeometry();
const object = new THREE.Mesh( sphere, new THREE.MeshBasicMaterial( 0xff0000 ) );
const box = new THREE.BoxHelper( object, 0xffff00 );
scene.add( box );

构造函数

new BoxHelper( object : Object3D, color : number | Color | string )

构造一个新的包围盒辅助对象。

object

要显示其世界轴对齐包围盒的三维对象。
color

包围盒的颜色。

默认值为 0xffff00.

属性

.object : Object3D

正在可视化的三维对象。

方法

.dispose()

释放该实例分配的 GPU 相关资源。当应用中不再使用该实例时，应调用此方法。

.setFromObject( object : Object3D ) : BoxHelper

为传入的对象更新线框包围盒。

object

要为其创建辅助对象的三维对象。
返回值： 对该实例的引用。

.update()

更新辅助对象的几何体，使其匹配对象（包括任何子对象）的尺寸。
