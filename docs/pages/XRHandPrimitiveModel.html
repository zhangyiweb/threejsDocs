XRHandPrimitiveModel

表示 XRHandModelFactory 根据所选配置文件可能生成的手部模型类型之一。XRHandPrimitiveModel 根据所选的 primitive 选项，使用球体或立方体基本体表示手部。

导入

XRHandPrimitiveModel 是一个插件，必须显式导入，参见 Installation#Addons。

import { XRHandPrimitiveModel } from 'three/addons/webxr/XRHandPrimitiveModel.js';

构造函数

new XRHandPrimitiveModel( handModel : XRHandModel, controller : Group, path : string, handedness : XRHandedness, options : XRHandPrimitiveModel~Options )

构造一个新的 XR 手部基本体模型。

handModel

手部模型。
controller

WebXR 控制器。
path

模型路径。
handedness

XR 输入源的左右手属性。
options

模型选项。

属性

.controller : Group

WebXR 控制器。

.envMap : Texture

模型的环境贴图。

默认值为 null。

.handModel : XRHandModel

手部模型。

方法

.updateMesh()

根据跟踪的 XR 关节数据更新网格。

类型定义

.Options

XRHandPrimitiveModel 的构造函数选项。

primitive
'box' | 'sphere'

基本体类型。
