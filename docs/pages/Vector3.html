Vector3

表示三维向量的类。三维向量是三个有序数字 （标记为 x、y 和 z），可用于表示多种事物，例如：

  • 三维空间中的点。
  • 三维空间中的方向与长度。在 three.js 中，长度始终是从 (0, 0, 0) 到 (x, y, z) 的欧几里得距离（直线距离）， 方向也是从 (0, 0, 0) 指向 (x, y, z)。
  • 任意有序的三元组数字。

三维向量还可用于表示其他事物，例如动量向量等， 但以上是 three.js 中最常见的用途。

遍历向量实例将按相应顺序产生其分量 (x, y, z)。

代码示例

const a = new THREE.Vector3( 0, 1, 0 );
//no arguments; will be initialised to (0, 0, 0)
const b = new THREE.Vector3( );
const d = a.distanceTo( b );
