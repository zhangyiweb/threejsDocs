ArrayElementNode

用于表示对类数组节点数据结构进行元素访问的基类。

构造函数

new ArrayElementNode( node : Node, indexNode : Node )

构造一个数组元素节点。

node

类数组节点。
indexNode

定义元素访问的索引节点。

属性

.indexNode : Node

定义元素访问的索引节点。

.isArrayElementNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.node : Node

类数组节点。

方法

.generateNodeType( builder : NodeBuilder ) : string

由于节点类型从类数组节点推断得出，因此覆盖了此方法。

builder

当前的节点构建器。
重写： Node#generateNodeType
返回值： 节点类型。

.getMemberType( builder : NodeBuilder, name : string ) : string

由于成员类型从类数组节点推断得出，因此覆盖了此方法。

builder

当前的节点构建器。
name

成员名称。
重写： Node#getMemberType
返回值： 成员类型。
