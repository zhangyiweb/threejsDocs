LatheGeometry

创建具有轴对称性的网格（如花瓶）。车削绕 Y 轴旋转。

代码示例

const points = [];
for ( let i = 0; i < 10; i ++ ) {
	points.push( new THREE.Vector2( Math.sin( i * 0.2 ) * 10 + 5, ( i - 5 ) * 2 ) );
}
const geometry = new THREE.LatheGeometry( points );
const material = new THREE.MeshBasicMaterial( { color: 0xffff00 } );
const lathe = new THREE.Mesh( geometry, material );
scene.add( lathe );

构造函数

new LatheGeometry( points : Array.<(Vector2|Vector3)>, segments : number, phiStart : number, phiLength : number )

构造一个新的车削几何体。

points

二维空间中的点数组。每个点的 x 坐标必须大于零。
segments

要生成的圆周分段数。

默认值为 12.
phiStart

起始角度（弧度）。

默认值为 0.
phiLength

车削截面的弧度范围（0 到 2π）。2π 为完整闭合车削，小于 2π 则为部分车削。

默认值为 Math.PI*2.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

静态方法

.fromJSON( data : Object ) : LatheGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
