ShapePath

该类用于将一系列路径转换为形状数组。特别用于字体与 SVG 相关场景。

构造函数

new ShapePath()

构造一个新的形状路径。

属性

.color : Color

形状的颜色。

.currentPath : Path

当前正在生成的路径。

默认值为 null。

.subPaths : Array.<Path>

已为该形状生成的路径。

默认值为 null。

.userData : Object

可用于存储关于该形状路径的自定义数据的对象。主要由 SVGLoader 用于存储样式信息。

方法

.bezierCurveTo( aCP1x : number, aCP1y : number, aCP2x : number, aCP2y : number, aX : number, aY : number ) : ShapePath

通过连接当前点与给定点，向路径添加一个 CubicBezierCurve 实例。

aCP1x

第一个控制点的 x 坐标。
aCP1y

第一个控制点的 y 坐标。
aCP2x

第二个控制点的 x 坐标。
aCP2y

第二个控制点的 y 坐标。
aX

终点的 x 坐标。
aY

终点的 y 坐标。
返回值： 对该形状路径的引用。

.lineTo( x : number, y : number ) : ShapePath

通过连接当前点与给定点，向路径添加一个 LineCurve 实例。

x

终点的 x 坐标。
y

终点的 y 坐标。
返回值： 对该形状路径的引用。

.moveTo( x : number, y : number ) : ShapePath

创建一条新路径，并将其当前点移动到给定点。

x

x 坐标。
y

y 坐标。
返回值： 对该形状路径的引用。

.quadraticCurveTo( aCPx : number, aCPy : number, aX : number, aY : number ) : ShapePath

通过连接当前点与给定点，向路径添加一个 QuadraticBezierCurve 实例。

aCPx

控制点的 x 坐标。
aCPy

控制点的 y 坐标。
aX

终点的 x 坐标。
aY

终点的 y 坐标。
返回值： 对该形状路径的引用。

.splineThru( pts : Array.<Vector2> ) : ShapePath

通过连接当前点与给定点列表，向路径添加一个 SplineCurve 实例。

pts

二维空间中的点数组。
返回值： 对该形状路径的引用。

.toShapes() : Array.<Shape>

将这些路径转换为形状数组。

返回值： 形状数组。
