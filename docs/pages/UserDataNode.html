UserDataNode

一种特殊的引用节点，可将 userData 字段中的值链接到节点对象。

由于 UserDataNode 继承自 ReferenceNode，当 rotation 用户数据字段发生变化时，节点值会自动更新。

代码示例

sprite.userData.rotation = 1; // stores individual rotation per sprite
const material = new THREE.SpriteNodeMaterial();
material.rotationNode = userData( 'rotation', 'float' );

构造函数

new UserDataNode( property : string, inputType : string, userData : Object )

构造一个新的用户数据节点。

property

应由该节点引用的属性名称。
inputType

引用的节点数据类型。
userData

对 userData 对象的引用。若未提供，则使用采用该节点材质的 3D 对象上的 userData 属性。

默认值为 null。

属性

.userData : Object

对 userData 对象的引用。若未提供，则使用 采用该节点材质的 3D 对象上的 userData 属性。

默认值为 null。

方法

.updateReference( state : NodeFrame | NodeBuilder ) : Object

重写以确保 ReferenceNode#reference 指向正确的 userData 字段。

state

当前要评估的状态。
重写： ReferenceNode#updateReference
返回值： 对 userData 字段的引用。
