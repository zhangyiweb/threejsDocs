NormalMapNode

此类可用于将法线贴图应用于材质。

代码示例

material.normalNode = normalMap( texture( normalTex ) );

构造函数

new NormalMapNode( node : Node.<vec3>, scaleNode : Node.<vec2> )

构造一个新的法线贴图节点。

node

表示法线贴图数据。
scaleNode

控制效果的强度。

默认值为 null。

属性

.node : Node.<vec3>

表示法线贴图数据。

.normalMapType : TangentSpaceNormalMap | ObjectSpaceNormalMap

法线贴图类型。

默认值为 TangentSpaceNormalMap。

.scaleNode : Node.<vec2>

控制效果的强度。

默认值为 null。

.unpackNormalMode : string

控制如何解包采样的法线贴图值。

默认值为 NoNormalPacking。
