Backend

大部分渲染相关逻辑实现于 Renderer 模块及相关管理组件中。但有时仍需执行特定于当前 3D 后端（WebGPU 或 WebGL 2）的命令。此抽象基类定义了封装所有后端相关逻辑的接口。各后端的派生类必须实现该接口。

构造函数

new Backend( parameters : Object ) (abstract)

构造一个新的后端。

parameters

包含后端参数的对象。

属性

.coordinateSystem : number (abstract, readonly)

后端的坐标系。

.data : WeakMap.<Object, Object>

此弱映射保存对象（如纹理、属性或渲染目标）的后端特定数据。

.domElement : HTMLCanvasElement | OffscreenCanvas

渲染器绘制目标画布元素的引用。

默认值为 null。

.hasTimestamp : boolean (readonly)

后端是否支持时间戳查询。

.parameters : Object

后端的参数。

.renderer : Renderer

渲染器的引用。

默认值为 null。

.timestampQueryPool : Object

时间戳查询池的引用。

.trackTimestamp : boolean

是否使用 Timestamp Query API 跟踪时间戳。

默认值为 false。

方法

返回值： 查询池。
