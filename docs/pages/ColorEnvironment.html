ColorEnvironment

此类表示一个具有均匀颜色的场景，可用作 PMREMGenerator#fromScene 的输入。生成的 PMREM 表示均匀环境光，可通过赋值给 Scene#environment 用于基于图像的光照（IBL）。

代码示例

const environment = new ColorEnvironment( 0x00ff00 );
const pmremGenerator = new THREE.PMREMGenerator( renderer );
const envMap = pmremGenerator.fromScene( environment ).texture;
scene.environment = envMap;

导入

ColorEnvironment 是一个插件，必须显式导入，参见 Installation#Addons。

import { ColorEnvironment } from 'three/addons/environments/ColorEnvironment.js';

构造函数

new ColorEnvironment( color : number | Color )

构造一个新的颜色环境。

color

环境的颜色。

默认值为 16777215。

方法

.dispose()

释放内部资源。当不再需要该环境时应调用此方法。
