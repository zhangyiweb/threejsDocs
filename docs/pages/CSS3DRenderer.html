CSS3DRenderer

此渲染器可通过 CSS3 transform 属性 对 DOM 元素应用分层 3D 变换。 若希望在不使用基于 canvas 的渲染的情况下为网站应用 3D 效果， CSS3DRenderer 尤其有用。也可用于将 DOM 元素与 WebGL 内容结合。

不过，存在一些重要限制：

  • 无法使用 three.js 的材质系统。
  • 也无法使用几何体。
  • 该渲染器仅支持 100% 的浏览器缩放和显示缩放。

因此 CSS3DRenderer 仅面向普通 DOM 元素。这些元素会被包装为特殊的 3D 对象（CSS3DObject 或 CSS3DSprite），然后添加到场景图中。

导入

CSS3DRenderer 是一个插件，必须显式导入，参见 Installation#Addons。

import { CSS3DRenderer } from 'three/addons/renderers/CSS3DRenderer.js';

构造函数

new CSS3DRenderer( parameters : CSS3DRenderer~Parameters )

构造一个新的 CSS3D 渲染器。

parameters

参数。

属性

.domElement : HTMLElement

渲染器将其子元素追加到的 DOM。

方法

.getSize() : Object

返回包含渲染器宽度和高度的对象。

返回值： 渲染器的尺寸。

.render( scene : Object3D, camera : Camera )

使用给定相机渲染给定场景。

scene

场景或其他类型的 3D 对象。
camera

相机。

.setSize( width : number, height : number )

将渲染器调整为给定的宽度和高度。

width

渲染器的宽度。
height

渲染器的高度。

类型定义

.Parameters

CSS3DRenderer 的构造函数参数。

element
HTMLElement

渲染器将其子元素追加到的 DOM 元素。 若未在此处传入，将创建一个新的 div 元素。
