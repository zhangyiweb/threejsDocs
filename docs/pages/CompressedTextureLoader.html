CompressedTextureLoader

用于加载压缩纹理格式（S3TC、ASTC 或 ETC）的抽象基类。 纹理在内部通过 FileLoader 加载。

派生类必须实现 parse() 方法，其中包含各自格式的 解析逻辑。

构造函数

new CompressedTextureLoader( manager : LoadingManager ) (abstract)

构造一个新的压缩纹理加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback ) : CompressedTexture

从指定 URL 开始加载，并将加载完成的压缩纹理 传递给 onLoad() 回调。该方法还会返回一个新的纹理对象，可直接 用于创建材质。若采用这种方式，当相应加载过程完成时， 纹理可能会出现在场景中。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load
返回值： 压缩纹理。

类型定义

.TexData

表示 parse() 方法的结果对象类型。

width
number

基础 mip 的宽度。
height
number

基础 mip 的高度。
isCubemap
boolean

数据是否表示立方体贴图。
mipmapCount
number

mipmap 数量。
mipmaps
Array.<{data:TypedArray, width:number, height:number}>

保存 mipmap 的数组。 每个条目包含对应级别的数据和尺寸。
format
number

纹理格式。
