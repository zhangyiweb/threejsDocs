GTAONode

用于对场景应用 Ground Truth Ambient Occlusion（GTAO）的后处理节点。

参考：Practical Real-Time Strategies for Accurate Indirect Occlusion。

代码示例

const renderPipeline = new THREE.RenderPipeline( renderer );
const scenePass = pass( scene, camera );
scenePass.setMRT( mrt( {
	output: output,
	normal: normalView
} ) );
const scenePassColor = scenePass.getTextureNode( 'output' );
const scenePassNormal = scenePass.getTextureNode( 'normal' );
const scenePassDepth = scenePass.getTextureNode( 'depth' );
const aoPass = ao( scenePassDepth, scenePassNormal, camera );
const aoPassOutput = aoPass.getTextureNode();
renderPipeline.outputNode = scenePassColor.mul( vec4( vec3( aoPassOutput.r ), 1 ) );

导入

GTAONode 是一个插件，必须显式导入，参见 Installation#Addons。

import { ao } from 'three/addons/tsl/display/GTAONode.js';

构造函数

new GTAONode( depthNode : Node.<float>, normalNode : Node.<vec3>, camera : Camera )

构造一个新的 GTAO 节点。

depthNode

表示场景深度的节点。
normalNode

表示场景法线的节点。
camera

用于渲染场景的相机。

属性

.depthNode : Node.<float>

表示场景深度的节点。

.distanceExponent : UniformNode.<float>

用于微调遮蔽效果的另一选项。建议取值范围为 [1,2]，用于衰减 AO。

.distanceFallOff : UniformNode.<float>

环境光遮蔽的距离衰减值。数值越低，AO 效果越明显。取值应在 [0,1] 范围内。

.normalNode : Node.<vec3>

表示场景法线的节点。若构造函数未传入法线（例如 MRT 不可用），可在着色器中从深度值自动重建法线。

.radius : UniformNode.<float>

环境光遮蔽的半径。

.resolution : UniformNode.<vec2>

效果的分辨率。可通过 resolutionScale 缩放。

.resolutionScale : number

分辨率缩放。默认以全分辨率渲染以获得最佳质量，但对大多数场景而言 0.5 已足够。

默认值为 1。

.samples : UniformNode.<float>

计算 AO 时使用的采样数。数值越高质量越好，但运行时开销也越大。

.scale : UniformNode.<float>

环境光遮蔽的缩放。

.thickness : UniformNode.<float>

环境光遮蔽的厚度。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.FRAME，因为该节点会在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

.useTemporalFiltering : boolean

是否使用时域滤波。设为 true 时需要配合使用 TRAANode，有助于降低噪声，但也会引入典型的 TAA 伪影，例如拖影和时域不稳定。

若设为 false，可能需要通过 DenoiseNode 手动降噪。

默认值为 false。

方法

.dispose()

释放内部资源。当不再需要该效果时应调用此方法。

重写： TempNode#dispose

.getTextureNode() : PassTextureNode

将效果结果作为纹理节点返回。

返回值： 表示效果结果的纹理节点。

.setSize( width : number, height : number )

设置效果的尺寸。

width

效果的宽度。
height

效果的高度。

.setup( builder : NodeBuilder ) : PassTextureNode

用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
