BilateralBlurNode

用于创建双边模糊效果的后处理节点。

双边模糊在平滑图像的同时保留锐利边缘。与对所有像素均匀模糊的标准高斯模糊不同， 双边模糊会分析相邻像素的强度/颜色。若相邻像素与中心像素差异过大（表示存在边缘）， 则将其排除在模糊过程之外。

参考：https://en.wikipedia.org/wiki/Bilateral_filter

导入

BilateralBlurNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { bilateralBlur } from 'three/addons/tsl/display/BilateralBlurNode.js';

构造函数

new BilateralBlurNode( textureNode : TextureNode, directionNode : Node.<(vec2|float)>, sigma : number, sigmaColor : number )

构造一个新的双边模糊节点。

textureNode

表示效果输入的纹理节点。
directionNode

定义模糊的方向和半径。

默认值为 null。
sigma

控制模糊滤波器的空间核。数值越大，模糊半径越宽。

默认值为 4。
sigmaColor

控制强度核。数值越大，允许更大颜色差异的像素一起模糊。

默认值为 0.1。

属性

.directionNode : Node.<(vec2|float)>

定义模糊的方向和半径。

.resolutionScale : number

分辨率缩放比例。

默认值为 1。

.sigma : number

控制模糊滤波器的空间核。数值越大，模糊半径越宽。

.sigmaColor : number

控制颜色/强度核。数值越大，允许更大颜色差异的像素一起模糊。 数值越小，边缘保留越严格。

.textureNode : TextureNode

表示效果输入的纹理节点。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

方法

.dispose()

释放内部资源。当不再需要该效果时应调用此方法。

重写： TempNode#dispose

.getTextureNode() : PassTextureNode

以纹理节点的形式返回效果结果。

返回值： 表示效果结果的纹理节点。

.setSize( width : number, height : number )

设置效果尺寸。

width

效果的宽度。
height

效果的高度。

.setup( builder : NodeBuilder ) : PassTextureNode

用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
