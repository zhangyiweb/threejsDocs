Cylindrical

该类可用于将三维空间中的点表示为 柱坐标。

构造函数

new Cylindrical( radius : number, theta : number, y : number )

构造一个新的柱坐标。

radius

原点到 x-z 平面上一点的距离。

默认值为 1。
theta

在 x-z 平面上从正 z 轴起逆时针测量的角度，单位为弧度。

默认值为 0。
y

相对 x-z 平面的高度。

默认值为 0。

属性

.radius : number

原点到 x-z 平面上一点的距离。

默认值为 1。

.theta : number

在 x-z 平面上从正 z 轴起逆时针测量的角度，单位为弧度。

默认值为 0。

.y : number

相对 x-z 平面的高度。

默认值为 0。

方法

.clone() : Cylindrical

返回一个新柱坐标，其值从本实例复制而来。

返回值： 此实例的克隆。

.copy( other : Cylindrical ) : Cylindrical

将给定柱坐标的值复制到本实例。

other

要复制的柱坐标。
返回值： 对此柱坐标的引用。

.set( radius : number, theta : number, y : number ) : Cylindrical

通过复制给定值来设置柱坐标各分量。

radius

半径。
theta

theta 角度。
y

高度值。
返回值： 对此柱坐标的引用。

.setFromCartesianCoords( x : number, y : number, z : number ) : Cylindrical

根据给定的笛卡尔坐标设置柱坐标各分量。

x

x 值。
y

y 值。
z

z 值。
返回值： 对此柱坐标的引用。

.setFromVector3( v : Vector3 ) : Cylindrical

根据给定向量设置柱坐标各分量，该向量假定保存的是 笛卡尔坐标。

v

要设置的向量。
返回值： 对此柱坐标的引用。
