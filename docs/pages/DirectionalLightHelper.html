DirectionalLightHelper

用于辅助可视化 DirectionalLight 对场景影响的辅助对象。由表示灯光位置与方向的平面和线段组成。

当平行光或其目标发生变换，或灯光属性更改时，需要调用相应辅助对象的 update() 方法。

代码示例

const light = new THREE.DirectionalLight( 0xFFFFFF );
scene.add( light );
const helper = new THREE.DirectionalLightHelper( light, 5 );
scene.add( helper );

构造函数

new DirectionalLightHelper( light : DirectionalLight, size : number, color : number | Color | string )

构造一个新的平行光辅助对象。

light

要可视化的灯光。
size

平面的尺寸。

默认值为 1.
color

辅助对象的颜色。若未设置，辅助对象将采用灯光的颜色。

属性

.color : number | Color | string

构造函数中传入的颜色参数。若未设置，辅助对象将采用灯光的颜色。

.light : DirectionalLight

正在可视化的灯光。

.lightPlane : Line

包含显示平行光位置的线段。

.targetLine : Line

表示平行光目标方向的线段。

方法

.dispose()

释放该实例分配的 GPU 相关资源。当应用中不再使用该实例时，应调用此方法。

.update()

更新辅助对象，使其与正在可视化的灯光的位置和方向相匹配。
