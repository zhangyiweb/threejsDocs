GLSLNodeBuilder

面向 GLSL 的节点构建器。

此模块从节点材质生成 GLSL 着色器代码，并生成相应的绑定与顶点缓冲定义。 这些数据随后由渲染器用于为渲染对象创建渲染与计算管线。

构造函数

new GLSLNodeBuilder( object : Object3D, renderer : Renderer )

构造一个新的 GLSL 节点构建器。

object

3D 对象。
renderer

渲染器。

属性

.builtins : Object.<string, Array.<string>>

一个字典，为每个着色器阶段保存所用内置变量的数组。

.extensions : Object.<string, Map.<string, Object>>

一个字典，为每个着色器阶段保存所用扩展的 Map。

.transforms : Array.<Object.<string, (AttributeNode|string)>>

一个数组，保存用于定义 Transform Feedback 上下文中 varying 与 attribute 数据的对象。

.uniformGroups : Object.<string, Object.<string, NodeUniformsGroup>>

一个字典，为每个着色器阶段（'vertex'、'fragment'、'compute'） 再保存一个按组（'render'、'frame'、'object'）管理 UBO 的字典。
