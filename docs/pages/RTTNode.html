RTTNode

RTTNode 接收另一个节点，并使用 QuadMesh 将其渲染到纹理（RTT）中。 此模块在后处理场景中尤为重要，某些节点的效果需要 纹理输入。基于本模块的辅助函数 convertToTexture() 可让节点系统在需要时自动确保纹理输入。

构造函数

new RTTNode( node : Node, width : number, height : number, options : Object )

构造一个新的 RTT 节点。

node

用于渲染纹理的节点。
width

内部渲染目标的宽度。若未指定宽度，渲染目标会自动调整大小。

默认值为 null。
height

内部渲染目标的高度。

默认值为 null。
options

内部渲染目标的选项。

默认值为 {type:HalfFloatType}。

属性

.autoResize : boolean (readonly)

内部渲染目标是否应自动调整大小。

默认值为 true。

.autoUpdate : boolean

纹理是否应自动更新。

默认值为 true。

.height : number

内部渲染目标的高度。

默认值为 null。

.isRTTNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.node : Node

用于渲染纹理的节点。

.renderTarget : RenderTarget

渲染目标。

.textureNeedsUpdate : boolean

纹理是否需要更新。

默认值为 true。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.RENDER，因为节点会在 其 RTTNode#updateBefore 方法中每帧渲染时更新纹理。

默认值为 'render'。

重写： TextureNode#updateBeforeType

.width : number

内部渲染目标的宽度。 若未指定宽度，渲染目标会自动调整大小。

默认值为 null。

方法

.getResolutionScale() : number

获取分辨率缩放比例。

返回值： 分辨率缩放比例。

.setResolutionScale( resolutionScale : number ) : RTTNode

设置分辨率缩放比例。 分辨率缩放比例是一个与渲染器宽度和高度相乘的因子。

resolutionScale

要设置的分辨率缩放比例。1 表示全分辨率。
返回值： 对此节点的引用。

.setSize( width : number, height : number )

设置内部渲染目标的大小。

width

要设置的宽度。
height

要设置的高度。
