KeyframeTrack

表示一个按时间排列的关键帧序列，由时间列表和对应的值列表组成， 用于对对象的特定属性进行动画。

构造函数

new KeyframeTrack( name : string, times : Array.<number>, values : Array.<(number|string|boolean)>, interpolation : InterpolateLinear | InterpolateDiscrete | InterpolateSmooth | InterpolateBezier )

构造一个新的关键帧轨道。

name

关键帧轨道名称。
times

关键帧时间列表。
values

关键帧值列表。
interpolation

插值类型。

属性

.DefaultInterpolation : InterpolateLinear | InterpolateDiscrete | InterpolateSmooth | InterpolateBezier

该关键帧轨道的默认插值类型。

默认值为 InterpolateLinear。

.TimeBufferType : TypedArray | Array

该关键帧轨道的时间缓冲区类型。

默认值为 Float32Array.constructor。

.ValueBufferType : TypedArray | Array

该关键帧轨道的值缓冲区类型。

默认值为 Float32Array.constructor。

.ValueTypeName : string

值类型名称。

默认值为 ''。

.name : string

轨道名称可引用变形目标、骨骼或动画对象中的其他值。请参阅 PropertyBinding#parseTrackName 了解可用于属性绑定的字符串格式。

.times : Float32Array

关键帧时间。

.values : Float32Array

关键帧值。

方法

.InterpolantFactoryMethodBezier( result : TypedArray ) : BezierInterpolant

创建新贝塞尔插值器的工厂方法。

贝塞尔插值器需要在创建插值器前通过轨道的 settings 属性设置切线数据，settings 应包含：

  • inTangents：Float32Array，每个关键帧每个分量的 [时间, 值] 对
  • outTangents：Float32Array，每个关键帧每个分量的 [时间, 值] 对
result

结果缓冲区。
返回值： 新的插值器。

.InterpolantFactoryMethodDiscrete( result : TypedArray ) : DiscreteInterpolant

创建新离散插值器的工厂方法。

result

结果缓冲区。
返回值： 新的插值器。

.InterpolantFactoryMethodLinear( result : TypedArray ) : LinearInterpolant

创建新线性插值器的工厂方法。

result

结果缓冲区。
返回值： 新的插值器。

.InterpolantFactoryMethodSmooth( result : TypedArray ) : CubicInterpolant

创建新平滑插值器的工厂方法。

result

结果缓冲区。
返回值： 新的插值器。

.clone() : KeyframeTrack

返回一个复制了当前实例值的新关键帧轨道。

返回值： 当前实例的克隆。

.getInterpolation() : InterpolateLinear | InterpolateDiscrete | InterpolateSmooth | InterpolateBezier

返回当前插值类型。

返回值： 插值类型。

.getValueSize() : number

返回值大小。

返回值： 值大小。

.optimize() : KeyframeTrack

通过移除等效的连续关键帧（在变形目标序列中很常见）来优化该关键帧轨道。

返回值： 对该关键帧轨道的引用。

.scale( timeScale : number ) : KeyframeTrack

按因子缩放所有关键帧时间（用于帧与秒之间的转换）。

timeScale

时间缩放因子。
返回值： 对该关键帧轨道的引用。

.setInterpolation( interpolation : InterpolateLinear | InterpolateDiscrete | InterpolateSmooth | InterpolateBezier ) : KeyframeTrack

定义该关键帧轨道的插值方法。

interpolation

插值类型。
返回值： 对该关键帧轨道的引用。

.shift( timeOffset : number ) : KeyframeTrack

将所有关键帧在时间上向前或向后移动。

timeOffset

时间值的偏移量。
返回值： 对该关键帧轨道的引用。

.trim( startTime : number, endTime : number ) : KeyframeTrack

移除指定时间范围之外的关键帧，但不改变该范围内任何值。

注意：该方法不会将关键帧平移到轨道起始时间，因为对插值关键帧这样做会改变其值。

startTime

起始时间。
endTime

结束时间。
返回值： 对该关键帧轨道的引用。

.validate() : boolean

对关键帧轨道执行最小验证。若值有效，则返回 true。

返回值： 关键帧是否有效。

静态方法

.toJSON( track : KeyframeTrack ) : Object

将关键帧轨道转换为 JSON。

track

要序列化的关键帧轨道。
返回值： 序列化后的关键帧轨道 JSON。
