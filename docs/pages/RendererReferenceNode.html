RendererReferenceNode

此节点是一种特殊类型的引用节点，用于 将渲染器属性与节点值关联。

当更改 renderer.toneMappingExposure 时，exposureNode 的节点值会 自动更新。

代码示例

const exposureNode = rendererReference( 'toneMappingExposure', 'float', renderer );

构造函数

new RendererReferenceNode( property : string, inputType : string, renderer : Renderer )

构造一个新的渲染器引用节点。

property

节点所引用属性的名称。
inputType

用于表示属性值的 uniform 类型。
renderer

属性所属的渲染器。未设置渲染器时， 节点引用当前状态的渲染器。

默认值为 null。

属性

.renderer : Renderer

属性所属的渲染器。未设置渲染器时， 节点引用当前状态的渲染器。

默认值为 null。

方法

.updateReference( state : NodeFrame | NodeBuilder ) : Object

根据给定状态更新引用。仅当未设置 RendererReferenceNode#renderer 时才会评估状态。

state

当前状态。
重写： ReferenceBaseNode#updateReference
返回值： 更新后的引用。
