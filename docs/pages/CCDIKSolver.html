CCDIKSolver

此类使用 CCD 算法 求解逆向运动学（IK）问题。

CCDIKSolver 设计用于与 SkinnedMesh 实例配合使用。

导入

CCDIKSolver 是一个插件，必须显式导入，参见 Installation#Addons。

import { CCDIKSolver } from 'three/addons/animation/CCDIKSolver.js';

构造函数

new CCDIKSolver( mesh : SkinnedMesh, iks : Array.<CCDIKSolver~IK> )

mesh

蒙皮网格。
iks

IK 对象数组。

默认值为 []。

属性

.iks : Array.<CCDIKSolver~IK>

IK 对象数组。

.mesh : SkinnedMesh

蒙皮网格。

方法

.createHelper( sphereSize : number ) : CCDIKHelper

创建用于可视化 CCDIK 的辅助对象。

sphereSize

球体大小。
返回值： 创建的辅助对象。

.update( globalBlendFactor : number ) : CCDIKSolver

通过求解 CCD 算法更新所有 IK 骨骼。

globalBlendFactor

若 IK 链未定义自身的 blendFactor，则应用此混合因子。

默认值为 1.0。
返回值： 对此实例的引用。

.updateOne( ik : CCDIKSolver~IK, overrideBlend : number ) : CCDIKSolver

通过求解 CCD 算法更新单个 IK 骨骼。

ik

要更新的 IK。
overrideBlend

若 IK 对象未定义 blendFactor，则使用此值。

默认值为 1.0。
返回值： 对此实例的引用。

类型定义

.BoneLink

此类型表示骨骼链接。

index
number

链接骨骼的索引，对应 Skeleton.bones 数组中的骨骼。
limitation
number

旋转轴。
rotationMin
number

旋转下限。
rotationMax
number

旋转上限。
enabled
boolean

该链接是否启用。

默认值为 true。

.IK

此类型表示 IK 配置对象。

target
number

目标骨骼索引，对应 Skeleton.bones 数组中的骨骼。
effector
number

效应器骨骼索引，对应 Skeleton.bones 数组中的骨骼。
links
Array.<CCDIKSolver~BoneLink>

骨骼链接数组。
iteration
number

计算迭代次数。数值越小速度越快，但精度越低。

默认值为 1。
minAngle
number

单步最小旋转角（弧度）。
maxAngle
number

单步最大旋转角（弧度）。
blendFactor
number

混合因子。
