BVHLoader

BVH 格式的加载器。

导入 BVH 文件并输出单个 Skeleton 和 AnimationClip。 目前该加载器仅支持包含单个根节点的 BVH 文件。

代码示例

const loader = new BVHLoader();
const result = await loader.loadAsync( 'models/bvh/pirouette.bvh' );
// visualize skeleton
const skeletonHelper = new THREE.SkeletonHelper( result.skeleton.bones[ 0 ] );
scene.add( result.skeleton.bones[ 0 ] );
scene.add( skeletonHelper );
// play animation clip
mixer = new THREE.AnimationMixer( result.skeleton.bones[ 0 ] );
mixer.clipAction( result.clip ).play();

导入

BVHLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { BVHLoader } from 'three/addons/loaders/BVHLoader.js';

构造函数

new BVHLoader( manager : LoadingManager )

构造一个新的 BVH 加载器。

manager

加载管理器。

属性

.animateBonePositions : boolean

是否对骨骼位置进行动画。

默认值为 true。

.animateBoneRotations : boolean

是否对骨骼旋转进行动画。

默认值为 true。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 BVH 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( text : string ) : Object

解析给定的 BVH 数据并返回结果数据。

text

作为字符串的原始 BVH 数据。
重写： Loader#parse
返回值： 表示已解析资源的对象。
