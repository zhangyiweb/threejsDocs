DragControls

此类可用于提供拖放交互。

代码示例

const controls = new DragControls( objects, camera, renderer.domElement );
// add event listener to highlight dragged objects
controls.addEventListener( 'dragstart', function ( event ) {
	event.object.material.emissive.set( 0xaaaaaa );
} );
controls.addEventListener( 'dragend', function ( event ) {
	event.object.material.emissive.set( 0x000000 );
} );

导入

DragControls 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { DragControls } from 'three/addons/controls/DragControls.js';

构造函数

new DragControls( objects : Array.<Object3D>, camera : Camera, domElement : HTMLElement )

构造一个新的控制器实例。

objects

可拖拽的 3D 对象数组。
camera

所渲染场景的相机。
domElement

用于事件监听的 HTML DOM 元素。

默认值为 null。

属性

.objects : Array.<Object3D>

可拖拽的 3D 对象数组。

.raycaster : Raycaster

用于检测 3D 对象的射线投射器。

.recursive : boolean

可拖拽对象的子对象是否可独立于其父对象被拖拽。

默认值为 true。

.rotateSpeed : number

在 rotate 模式下拖拽时对象的旋转速度。数值越高，旋转越快。

默认值为 1。

.transformGroup : boolean

此选项仅在 objects 数组包含单个可拖拽组对象时生效。若设为 true，控制器不会变换单个对象，而是变换整个组。

默认值为 false。

事件

.drag

当用户拖拽 3D 对象时触发。

类型：
  • Object

.dragend

当用户完成拖拽 3D 对象时触发。

类型：
  • Object

.hoveroff

当指针移出 3D 对象时触发。

类型：
  • Object

.hoveron

当指针移入 3D 对象或其某个子对象时触发。

类型：
  • Object
