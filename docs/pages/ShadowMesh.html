ShadowMesh

一种跟随场景中投射阴影网格移动、但被限制在单一平面上的阴影网格。 该技术可作为传统阴影贴图的高性能替代方案。不过，它有以下严重限制：

  • 阴影只能投射到平面上。
  • 不支持软阴影。

代码示例

const cubeShadow = new ShadowMesh( cube );
scene.add( cubeShadow );

导入

ShadowMesh 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { ShadowMesh } from 'three/addons/objects/ShadowMesh.js';

构造函数

new ShadowMesh( mesh : Mesh )

构造一个新的阴影网格。

mesh

投射阴影的参考网格。

属性

.frustumCulled : boolean

已重写，默认禁用视锥体裁剪。

默认值为 false。

重写： Mesh#frustumCulled

.isShadowMesh : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.matrixAutoUpdate : boolean

已重写，默认禁用自动矩阵更新。局部矩阵在 ShadowMesh#update 中手动计算。

默认值为 false。

重写： Mesh#matrixAutoUpdate

.meshMatrix : Matrix4

表示参考网格的世界矩阵。

方法

.update( plane : Plane, lightPosition4D : Vector4 )

更新阴影网格，使其跟随投射阴影的参考网格。

plane

阴影网格投影到的平面。
lightPosition4D

光源位置。
