MeshPhysicalMaterial

MeshStandardMaterial 的扩展，提供更高级的基于物理的渲染属性：

  • 各向异性（Anisotropy）：能够表现拉丝金属等材料中可见的各向异性特性。
  • 清漆层（Clearcoat）：某些材料——如车漆、碳纤维和潮湿表面——需要在可能不规则或粗糙的另一层之上再覆盖一层清晰的反射层。Clearcoat 可近似该效果，而无需单独的透明表面。
  • 虹彩（Iridescence）：允许渲染随观察角度与光照角度变化而产生色相变化的效果。可见于肥皂泡、油膜，或许多昆虫的翅膀。
  • 基于物理的透明度：Material#opacity 的一个局限是高度透明的材料反射较弱。基于物理的透射为玻璃等薄而透明的表面提供了更真实的选项。
  • 高级反射率：为非金属材料提供更灵活的反射率。
  • 光泽（Sheen）：可用于表现布料与织物材料。

由于这些复杂的着色特性，MeshPhysicalMaterial 每个像素的性能开销高于其他 three.js 材质。大多数效果默认关闭，启用时会增加开销。为获得最佳效果，使用此材质时请始终指定环境贴图。

构造函数

new MeshPhysicalMaterial( parameters : Object )

构造一个新的 mesh physical 材质。

parameters

包含一个或多个定义材质外观属性的对象。材质的任何属性（包括继承材质的任何属性）都可在此传入。颜色值可以是 Color#set 接受的任意类型。

属性

.anisotropy : number

各向异性强度，范围为 0.0 到 1.0。

默认值为 0。

.anisotropyMap : Texture

红色与绿色通道表示切线、副切线空间中范围为 [-1, 1] 的各向异性方向，可由 anisotropyRotation 旋转。蓝色通道包含范围为 [0, 1] 的强度，会与 anisotropy 相乘。

anisotropyMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.anisotropyRotation : number

切线、副切线空间中各向异性的旋转角度，以弧度为单位，从切线方向逆时针测量。当存在 anisotropyMap 时，此属性会对纹理中的向量提供额外旋转。

默认值为 1。

.attenuationColor : Color

白光在到达衰减距离时因吸收而转变成的颜色。

默认值为 (1,1,1)。

.attenuationDistance : number

介质密度，表示为光在介质中与粒子发生相互作用前平均传播的距离。该值以世界空间单位给出，且必须大于零。

默认值为 Infinity。

.clearcoat : number

表示清漆层的强度，范围为 0.0 到 1.0。使用与清漆相关的属性可启用在基础层之上具有薄半透明层的多层材质。

默认值为 0。

.clearcoatMap : Texture

该纹理的红色通道会与 clearcoat 相乘， 以便逐像素控制涂层强度。

clearcoatMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.clearcoatNormalMap : Texture

可用于为清漆层启用独立的法线。

clearcoatNormalMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.clearcoatNormalScale : Vector2

clearcoatNormalMap 对清漆层的影响程度，范围为 (0,0) 到 (1,1)。

默认值为 (1,1)。

.clearcoatRoughness : number

清漆层的粗糙度，范围为 0.0 到 1.0。

默认值为 0。

.clearcoatRoughnessMap : Texture

该纹理的绿色通道会与 clearcoatRoughness 相乘，以便逐像素控制涂层粗糙度。

clearcoatRoughnessMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.dispersion : number

定义透过相对清晰体积时颜色角分离（色散）的强度。任何大于等于零的值都有效，现实中的典型范围为 [0, 1]。此属性仅可用于透射物体。

默认值为 0。

.ior : number

非金属材料的折射率，范围为 1.0 到 2.333。

默认值为 1.5。

.iridescence : number

虹彩层的强度，根据表面与观察者之间的夹角模拟 RGB 色移，范围为 0.0 到 1.0。

默认值为 0。

.iridescenceIOR : number

虹彩 RGB 色移效果的强度，以折射率表示。 介于 1.0 到 2.333 之间。

默认值为 1.3。

.iridescenceMap : Texture

该纹理的红色通道会与 iridescence 相乘，以便逐像素控制虹彩。

iridescenceMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.iridescenceThicknessMap : Texture

定义虹彩层厚度的纹理，存储在绿色通道中。 厚度的最小与最大值由 iridescenceThicknessRange 数组定义：

  • 绿色通道为 0.0 时，厚度等于数组的第一个元素。
  • 绿色通道为 1.0 时，厚度等于数组的第二个元素。
  • 中间的值会在数组元素之间线性插值。

iridescenceThicknessMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.iridescenceThicknessRange : Array.<number, number>

恰好包含 2 个元素的数组，分别指定虹彩层的最小与最大厚度。 虹彩层厚度对效果的影响，与 thickness 对 ior 的影响类似。

默认值为 [100,400]。

.isMeshPhysicalMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.reflectivity : number

反射程度，范围为 0.0 到 1.0。默认值为 0.5，对应折射率为 1.5。

这用于模拟非金属材料的反射率。当 metalness 为 1.0 时无效。

默认值为 0.5。

.sheen : number

光泽层的强度，范围为 0.0 到 1.0。

默认值为 0。

.sheenColor : Color

光泽色调。

默认值为 (0,0,0)。

.sheenColorMap : Texture

该纹理的 RGB 通道会与 sheenColor 相乘，以便逐像素控制光泽色调。

sheenColorMap 表示颜色数据，纹理必须指定 Texture#colorSpace。大多数 sheenColorMap 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。

.sheenRoughness : number

光泽层的粗糙度，范围为 0.0 到 1.0。

默认值为 1。

.sheenRoughnessMap : Texture

该纹理的 alpha 通道会与 sheenRoughness 相乘，以便逐像素控制光泽粗糙度。

sheenRoughnessMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.specularColor : Color

仅对非金属，为法线入射处的镜面反射着色。

默认值为 (1,1,1)。

.specularColorMap : Texture

该纹理的 RGB 通道会与 specularColor 相乘， 以便逐像素控制高光颜色。

specularColorMap 表示颜色数据，纹理必须指定 Texture#colorSpace。大多数 specularColorMap 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。

.specularIntensity : number

仅对非金属，用于缩放镜面反射量的浮点数。 设为零时，模型实际上等同于 Lambertian。范围为 0.0 到 1.0。

默认值为 1。

.specularIntensityMap : Texture

该纹理的 alpha 通道会与 specularIntensity 相乘， 以便逐像素控制高光强度。

specularIntensityMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.thickness : number

表面下方体积的厚度。该值以网格的坐标空间给出。若值为 0，则材质为薄壁；否则材质为体积边界。

默认值为 0。

.thicknessMap : Texture

定义厚度的纹理，存储在绿色通道中。会与 thickness 相乘。

thicknessMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.transmission : number

透射（或光学透明度）程度，范围为 0.0 到 1.0。

薄的、透明或半透明的塑料或玻璃材料，即使完全透射，仍在很大程度上保持反射性。可用 transmission 属性来模拟这些材料。

当透射非零时，应将 opacity 设为 1。

默认值为 0。

.transmissionMap : Texture

该纹理的红色通道会与 transmission 相乘，以便逐像素控制光学透明度。

transmissionMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。
