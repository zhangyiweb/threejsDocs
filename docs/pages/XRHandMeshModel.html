XRHandMeshModel

表示 XRHandModelFactory 根据所选配置文件可能生成的手部模型类型之一。XRHandMeshModel 表示使用自定义资源的手部模型。

导入

XRHandMeshModel 是一个插件，必须显式导入，参见 Installation#Addons。

import { XRHandMeshModel } from 'three/addons/webxr/XRHandMeshModel.js';

构造函数

new XRHandMeshModel( handModel : XRHandModel, controller : Group, path : string, handedness : XRHandedness, loader : Loader, onLoad : function, customCache : Object )

构造一个新的 XR 手部网格模型。

handModel

手部模型。
controller

WebXR 控制器。
path

模型路径。
handedness

XR 输入源的左右手属性。
loader

加载器。若未提供，将使用 GLTFLoader 实例来加载模型。

默认值为 null。
onLoad

控制器模型加载完成后执行的回调。

默认值为 null。
customCache

可选的共享缓存对象，用于在多个实例间存储和复用已加载的资源。

默认值为 null。

属性

.bones : Array.<Bone>

表示手部骨骼的骨骼数组。

.controller : Group

WebXR 控制器。

.handModel : XRHandModel

手部模型。

方法

.updateMesh()

根据跟踪的 XR 关节数据更新网格。
