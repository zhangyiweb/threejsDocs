CSS2DObject

CSS2DRenderer 支持的唯一一种 3D 对象类型。

导入

CSS2DObject 是一个插件，必须显式导入，参见 Installation#Addons。

import { CSS2DObject } from 'three/addons/renderers/CSS2DRenderer.js';

构造函数

new CSS2DObject( element : HTMLElement )

构造一个新的 CSS2D 对象。

element

DOM 元素。

属性

.center : Vector2

该 3D 对象的中心点。 ( 0, 0 ) 为左下角，( 1, 1 ) 为右上角。

默认值为 (0.5,0.5)。

.element : HTMLElement (readonly)

定义此 3D 对象外观的 DOM 元素。

默认值为 true。

.isCSS2DObject : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
