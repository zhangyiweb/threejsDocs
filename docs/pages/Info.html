Info

此渲染器模块提供一系列关于 GPU 内存与渲染过程的 统计信息。对调试与监控很有用。

构造函数

new Info()

构造一个新的信息组件。

属性

.autoReset : boolean

与帧相关的指标是否应自动重置。 自行管理动画循环的应用应将此属性设为 false， 然后必须在每帧手动调用 renderer.info.reset()。

默认值为 true。

.calls : number (readonly)

自应用启动以来的渲染调用次数。

默认值为 0。

.compute : Object (readonly)

与计算相关的指标。

calls
number

自应用启动以来的计算调用次数。
frameCalls
number

当前帧的计算调用次数。
timestamp
number

使用 renderer.computeAsync() 时该帧的时间戳。

.frame : number (readonly)

当前帧 ID。此 ID 由 NodeFrame 管理。

默认值为 0。

.memory : Object (readonly)

与内存相关的指标。

attributes
number

活动属性的数量。
attributesSize
number

活动属性的内存大小（字节）。
geometries
number

活动几何体的数量。
indexAttributes
number

活动索引属性的数量。
indexAttributesSize
number

活动索引属性的内存大小（字节）。
indirectStorageAttributes
number

活动间接存储属性的数量。
indirectStorageAttributesSize
number

活动间接存储属性的内存大小（字节）。
programs
number

活动程序的数量。
programsSize
number

活动程序的内存大小（字节）。
readbackBuffers
number

活动回读缓冲区的数量。
readbackBuffersSize
number

活动回读缓冲区的内存大小（字节）。
renderTargets
number

活动渲染目标的数量。
storageAttributes
number

活动存储属性的数量。
storageAttributesSize
number

活动存储属性的内存大小（字节）。
textures
number

活动纹理的数量。
texturesSize
number

活动纹理的内存大小（字节）。
uniformBuffers
number

活动 uniform 缓冲区的数量。
uniformBuffersSize
number

活动 uniform 缓冲区的内存大小（字节）。
total
number

总内存大小（字节）。

.render : Object (readonly)

与渲染相关的指标。

calls
number

自应用启动以来的渲染调用次数。
frameCalls
number

当前帧的渲染调用次数。
drawCalls
number

当前帧的绘制调用次数。
triangles
number

当前帧已渲染的三角形图元数量。
points
number

当前帧已渲染的点图元数量。
lines
number

当前帧已渲染的线图元数量。
timestamp
number

该帧的时间戳。

.createAttribute( attribute : BufferAttribute )

显式跟踪常规属性的内存。

attribute

要跟踪的属性。

显式跟踪索引属性的内存。

attribute

要跟踪的索引属性。

.createIndirectStorageAttribute( attribute : BufferAttribute )

显式跟踪间接存储属性的内存。

attribute

要跟踪的间接存储属性。

.createProgram( program : ProgrammableStage )

program

显式跟踪 uniform 缓冲区的内存。

uniformBuffer

要跟踪的 uniform 缓冲区。

.dispose()

对该对象执行完整重置。

.reset()

重置与帧相关的指标。

.update( object : Object3D, count : number, instanceCount : number )

此方法应在每次绘制调用时执行，并更新相应的指标。

object

即将被渲染的 3D 对象。
count

顶点或索引数量。
instanceCount

实例数量。
