InstancedBufferAttribute

缓冲属性的实例化版本。

构造函数

new InstancedBufferAttribute( array : TypedArray, itemSize : number, normalized : boolean, meshPerAttribute : number )

构造一个新的实例化缓冲属性。

array

保存属性数据的数组。
itemSize

每个顶点的分量数量。
normalized

数据是否已归一化。

默认值为 false。
meshPerAttribute

该缓冲属性的值应重复使用的次数。

默认值为 1。

属性

.isInstancedBufferAttribute : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.meshPerAttribute : number

定义该缓冲属性的值应重复使用的次数。值为 1 表示实例化属性的每个值用于单个实例；值为 2 表示每个值用于连续的两个实例（以此类推）。

默认值为 1。
