Object3DNode

此节点可用于访问 3D 对象与变换相关的度量。 根据所选作用域，着色器中以 uniform 形式表示不同的度量。支持以下作用域：

  • POSITION：对象在世界空间中的位置。
  • VIEW_POSITION：对象在视图/相机空间中的位置。
  • DIRECTION：对象在世界空间中的方向。
  • SCALE：对象在世界空间中的缩放。
  • WORLD_MATRIX：对象在世界空间中的矩阵。

构造函数

new Object3DNode( scope : 'position' | 'viewPosition' | 'direction' | 'scale' | 'worldMatrix', object3d : Object3D )

构造一个新的 3D 对象节点。

scope

根据作用域的不同，该节点表示不同类型的变换。
object3d

3D 对象。

默认值为 null。

属性

.object3d : Object3D

3D 对象。

默认值为 null。

.scope : 'position' | 'viewPosition' | 'direction' | 'scale' | 'worldMatrix'

根据作用域的不同，该节点报告不同类型的变换。

.uniformNode : UniformNode

以 uniform 形式保存节点的值。

.updateType : string

已重写，因为此类节点按对象更新。

默认值为 'object'。

重写： Node#updateType

方法

.generate( builder : NodeBuilder ) : string

builder

当前的节点构建器。
重写： Node#generate
返回值： 生成的代码片段。

.generateNodeType() : 'mat4' | 'vec3' | 'float'

已重写，因为节点类型从作用域推断。

重写： Node#generateNodeType
返回值： 节点类型。

.update( frame : NodeFrame )

根据作用域更新 uniform 值。

frame

当前的节点帧。
重写： Node#update
