MeshPhongMaterial

用于带有镜面高光的光泽表面的材质。

该材质使用基于非物理的 Blinn-Phong 模型计算反射。与 MeshLambertMaterial 中使用的 Lambertian 模型不同，它可以模拟带有高光的光泽表面（例如上漆的木材）。MeshPhongMaterial 使用逐片段着色。

使用此材质时性能通常优于 MeshStandardMaterial 或 MeshPhysicalMaterial，但会在一定程度上牺牲图形精度。

构造函数

new MeshPhongMaterial( parameters : Object )

构造一个新的 mesh phong 材质。

parameters

包含一个或多个定义材质外观属性的对象。材质的任何属性（包括继承材质的任何属性）都可在此传入。颜色值可以是 Color#set 接受的任意类型。

属性

.alphaMap : Texture

alpha 贴图是一张灰度纹理，用于控制表面各处的不透明度（黑色：完全透明；白色：完全不透明）。

仅使用纹理的颜色，若存在 alpha 通道则会被忽略。对于 RGB 和 RGBA 纹理，渲染器在采样时会使用绿色通道，因为在 DXT 压缩格式以及未压缩的 RGB 565 格式中，绿色通道具有额外的精度位。仅亮度以及亮度/alpha 纹理仍可按预期工作。

alphaMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.aoMap : Texture

该纹理的红色通道用作环境光遮蔽贴图。 需要第二套 UV。

aoMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.aoMapIntensity : number

环境光遮蔽效果的强度。范围为 [0,1]，其中 0 禁用环境光遮蔽。当强度为 1 且 AO 贴图的 红色通道也为 1 时，表面上的环境光会被完全遮蔽。

默认值为 1。

.bumpMap : Texture

用于创建凹凸贴图的纹理。黑白值映射为相对于光照的感知深度。凹凸实际上不会改变物体的几何形状，只影响光照。若已定义法线贴图，则此贴图会被忽略。

bumpMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.bumpScale : number

凹凸贴图对材质的影响程度。典型范围为 [0,1]。

默认值为 1。

.color : Color

材质的颜色。

默认值为 (1,1,1)。

.combine : MultiplyOperation | MixOperation | AddOperation

如何将表面颜色的结果与环境贴图（若有）进行合成。

当设置为 MixOperation 时，会使用 MeshBasicMaterial#reflectivity 在两种颜色之间进行混合。

默认值为 MultiplyOperation。

.displacementBias : number

位移贴图像素值在网格顶点上的偏移量。 该偏置会加到经缩放的位移贴图采样值上。 未设置位移贴图时，此值不会生效。

默认值为 0。

.displacementMap : Texture

位移贴图会影响网格顶点的位置。与仅影响材质明暗的其他贴图不同，被位移的顶点可以投射阴影、遮挡其他物体，并在其他方面表现得像真实几何体。位移纹理是一幅图像，其中每个像素的值（白色最高）会映射并重新定位网格顶点。为获得最佳效果，请将位移贴图与匹配的法线贴图配对使用，因为渲染器无法根据位移后的顶点重新计算表面法线。

displacementMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.displacementScale : number

位移贴图对网格的影响程度（黑色表示无位移，白色表示最大位移）。未设置位移贴图时，此值不会生效。

默认值为 0。

.emissive : Color

材质的自发光（发光）颜色，本质上是一种不受其他光照影响的纯色。

默认值为 (0,0,0)。

.emissiveIntensity : number

自发光的强度。用于调制自发光颜色。

默认值为 1。

.emissiveMap : Texture

设置自发光（辉光）贴图。自发光贴图颜色会受自发光颜色与自发光强度调制。若使用了自发光贴图，请务必将自发光颜色设置为非黑色。

emissiveMap 表示颜色数据，纹理必须指定 Texture#colorSpace。大多数 emissiveMap 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。

.envMap : Texture

环境贴图。

envMap 表示亮度数据，纹理必须指定 Texture#colorSpace。大多数 envMap 纹理会设置 texture.colorSpace = LinearSRGBColorSpace，并使用浮点类型格式， 例如 .exr 或 .hdr。

默认值为 null。

.envMapIntensity : number

通过乘以其颜色来缩放环境贴图的效果。

默认值为 1。

.envMapRotation : Euler

环境贴图的旋转角度，单位为弧度。

默认值为 (0,0,0)。

.flatShading : boolean

材质是否以平面着色方式渲染。

默认值为 false。

.fog : boolean

材质是否受雾影响。

默认值为 true。

.isMeshPhongMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.lightMap : Texture

光照贴图。需要第二套 UV。

lightMap 表示预烘焙的照度数据，纹理必须指定 Texture#colorSpace。大多数 lightMap 纹理会设置 texture.colorSpace = LinearSRGBColorSpace，并使用浮点类型格式， 例如 .exr 或 .hdr。

默认值为 null。

.lightMapIntensity : number

烘焙光照的强度。

默认值为 1。

.map : Texture

颜色贴图。可选地包含 alpha 通道，通常与 Material#transparent 或 Material#alphaTest 配合使用。纹理贴图颜色会受漫反射 color 调制。

map 表示颜色数据，纹理必须指定 Texture#colorSpace。大多数 map 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。

.normalMap : Texture

用于创建法线贴图的纹理。RGB 值会影响每个像素片段的表面法线，并改变颜色的受光方式。法线贴图不会改变表面的实际形状，只影响光照。若材质的法线贴图按左手坐标系编写，则应将 normalScale 的 y 分量取反，以补偿手性差异。

normalMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.normalMapType : TangentSpaceNormalMap | ObjectSpaceNormalMap

法线贴图的类型。

默认值为 TangentSpaceNormalMap。

.normalScale : Vector2

法线贴图对材质的影响程度。典型取值范围为 [0,1]。

默认值为 (1,1)。

.reflectivity : number

环境贴图对表面的影响程度。 有效范围在 0（无反射）到 1（完全反射）之间。

默认值为 1。

.refractionRatio : number

空气的折射率（IOR，约等于 1）除以材质的折射率。用于环境贴图模式 CubeRefractionMapping 和 EquirectangularRefractionMapping。 折射比不应超过 1。

默认值为 0.98。

.shininess : number

镜面高光的光泽程度；值越高，高光越锐利。

默认值为 30。

.specular : Color

材质的高光颜色。默认颜色为 0x111111（非常深的灰色）。

这定义了材质的光泽程度及其高光颜色。

.specularMap : Texture

高光贴图的值既影响表面镜面高光的贡献程度，也影响环境贴图对表面的影响程度。

specularMap 表示颜色数据，纹理必须指定 Texture#colorSpace。大多数 specularMap 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。

.wireframe : boolean

将几何体渲染为线框。

默认值为 false。

.wireframeLinecap : 'round' | 'bevel' | 'miter'

定义线框端点的外观。

仅可与 SVGRenderer 一起使用。

默认值为 'round'。

.wireframeLinejoin : 'round' | 'bevel' | 'miter'

定义线框连接处的外观。

仅可与 SVGRenderer 一起使用。

默认值为 'round'。

.wireframeLinewidth : number

控制线框的粗细。

仅可与 SVGRenderer 一起使用。

默认值为 1。
