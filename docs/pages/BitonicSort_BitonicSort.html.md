# BitonicSort

### new BitonicSort( renderer : Renderer, dataBuffer : StorageBufferNode, options : Object )

构造一个新的双调排序器。

**renderer**

当前场景的渲染器。

**dataBuffer**

待排序的数据缓冲区。

**options**

用于调整双调排序的选项。

默认值为 `{}`。
