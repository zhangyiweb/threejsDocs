TubePainter

此模块可用于沿一系列点绘制管状网格。 该模块用于 XR 绘画演示。

代码示例

const painter = new TubePainter();
scene.add( painter.mesh );

导入

TubePainter 是一个插件，必须显式导入，参见 Installation#Addons。

import { TubePainter } from 'three/addons/misc/TubePainter.js';

构造函数

new TubePainter()

属性

.mesh : Mesh

已「绘制」的管状网格。必须添加到场景中。

方法

.lineTo( position : Vector3 )

从当前位置绘制到给定位置的笔画。 使用 XR 控制器绘制时会延伸管状几何体。

position

目标位置。

.moveTo( position : Vector3 )

将当前绘制位置移动到给定值。

position

新的绘制位置。

.setColor( color : Color )

设置新渲染管段的颜色。

color

颜色。

.setSize( size : number )

设置新渲染管段的尺寸。

size

尺寸。

.update()

更新内部几何体缓冲区，以渲染新绘制的管段。
