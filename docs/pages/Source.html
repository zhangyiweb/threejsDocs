Source

表示纹理的数据源。

此类的主要目的是将数据定义与纹理定义解耦， 以便相同的数据可被多个纹理实例使用。

构造函数

new Source( data : any )

构造一个新的数据源。

data

纹理的数据定义。

默认值为 null。

属性

.data : any

纹理的数据定义。

.dataReady : boolean

此属性仅在 Source#needsUpdate 设为 true 时相关， 可更精细地控制纹理数据的处理方式。当 dataReady 设为 false 时，引擎会执行内存分配（如有必要），但不会将 数据传输到 GPU 内存。

默认值为 true。

.id : number (readonly)

数据源的 ID。

.isSource : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.needsUpdate : boolean

当该属性设为 true 时，引擎会为纹理分配内存 （如有必要），并在下次使用该数据源时触发实际的纹理上传到 GPU。

默认值为 false。

.uuid : string (readonly)

数据源的 UUID。

.version : number (readonly)

从 0 开始，统计 Source#needsUpdate 被设为 true 的次数。

默认值为 0。

方法

.getSize( target : Vector2 | Vector3 ) : Vector2 | Vector3

将数据源的尺寸写入给定的目标向量并返回。

target

用于写入结果的目标对象。
返回值： 数据源的尺寸。

.toJSON( meta : Object | string ) : Object

将数据源序列化为 JSON。

meta

可选值，保存有关序列化的元信息。
参见：
  • ObjectLoader#parse
返回值： 表示序列化数据源的 JSON 对象。
