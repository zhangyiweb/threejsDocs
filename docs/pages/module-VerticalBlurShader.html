VerticalBlurShader

导入

VerticalBlurShader 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { VerticalBlurShader } from 'three/addons/shaders/VerticalBlurShader.js';

属性

.VerticalBlurShader : ShaderMaterial~Shader (inner, constant)

双通道高斯模糊滤镜（水平与垂直模糊着色器）

  • 参见 http://www.cake23.de/traveling-wavefronts-lit-up.html

  • 每通道 9 个采样

  • 标准差 2.7

  • "h" 和 "v" 参数应分别设为 "1 / width" 和 "1 / height"
