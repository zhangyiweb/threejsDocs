SkyscraperGenerator

根据少量参数生成精致的三段式「美术学院派 / 新哥特式」赤陶摩天楼。

体量读作一个占地多边形（带一个倒角的矩形），拆成垂直立面， 每个立面再分为三层——高大的拱廊基座、重复的中段与华丽的冠部—— 然后进一步拆成楼层与开间。少量预制构件（墩柱、窗户、檐口剖面、 哥特拱）在整座塔楼上实例化，再与定制的基座拱廊一起烘焙进 单个非索引 BufferGeometry，并用逐顶点 partId（PartId）标记， 以便一种材质就能为每个区域着色。

该生成器与材质无关——只产出几何体。传入单一材质 （例如按 partId 分支的 TSL 节点材质）即可装扮。

代码示例

const generator = new SkyscraperGenerator( { seed: 35, totalHeight: 140 }, material );
scene.add( generator.build() ); // a single Mesh

构造函数

new SkyscraperGenerator()
