DirectionalLight

一种沿特定方向发出的光源。该光源的行为如同位于无限远处， 所产生的光线全部平行。常见用途是模拟日光；太阳足够远， 其位置可视为无穷远，且来自太阳的所有光线都是平行的。

关于平行光，一个常见的误解是设置旋转不会产生任何效果。 这是因为 three.js 中的 DirectionalLight 等效于其他应用中常说的 “目标平行光”（Target Direct Light）。

这意味着其方向是从光源的 Object3D#position 指向 DirectionalLight#target 的位置计算得出的 （而不是仅具有旋转分量的“自由平行光”）。

该光源可以投射阴影——详情参见 DirectionalLightShadow。

代码示例

// White directional light at half intensity shining from the top.
const directionalLight = new THREE.DirectionalLight( 0xffffff, 0.5 );
scene.add( directionalLight );

构造函数

new DirectionalLight( color : number | Color | string, intensity : number )

构造一个新的平行光。

color

光源颜色。

默认值为 0xffffff。
intensity

光源强度。

默认值为 1。

属性

.isDirectionalLight : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.shadow : DirectionalLightShadow

该属性保存光源的阴影配置。

.target : Object3D

平行光从其自身位置指向目标的位置。

若要将目标位置改为默认值以外的任何值， 必须将其添加到场景中。

也可以将目标设置为场景中的另一个 3D 对象。 此时光源会跟踪该目标对象。
