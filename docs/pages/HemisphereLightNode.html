HemisphereLightNode

用于将半球光表示为节点的模块。

构造函数

new HemisphereLightNode( light : HemisphereLight )

构造一个新的半球光节点。

light

半球光源。

默认值为 null。

属性

.groundColorNode : UniformNode.<vec3>

表示灯光地面颜色的 Uniform 节点。

.lightDirectionNode : Node.<vec3>

表示灯光方向的节点。

.lightPositionNode : UniformNode.<vec3>

表示灯光位置的 Uniform 节点。

方法

.update( frame : NodeFrame )

重写以更新半球光特有的 Uniform。

frame

当前节点帧的引用。
重写： AnalyticLightNode#update
