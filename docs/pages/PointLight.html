PointLight

从单个点向所有方向发射的光源。常见用途是 模拟裸灯泡发出的光。

此光源可投射阴影——详情参见 PointLightShadow。

代码示例

const light = new THREE.PointLight( 0xff0000, 1, 100 );
light.position.set( 50, 50, 50 );
scene.add( light );

构造函数

new PointLight( color : number | Color | string, intensity : number, distance : number, decay : number )

构造一个新的点光源。

color

光源颜色。

默认值为 0xffffff。
intensity

光源强度，单位为坎德拉 (cd)。

默认值为 1。
distance

灯光的最大照射范围。0 表示无限制。

默认值为 0。
decay

灯光沿距离衰减的程度。

默认值为 2。

属性

.decay : number

灯光沿距离衰减的程度。在物理正确渲染的语境下， 不应更改默认值。

默认值为 2。

.distance : number

当距离为零时，灯光将按平方反比定律衰减至无限远。 当距离非零时，灯光将按平方反比定律衰减，直至接近距离截止点， 然后快速平滑地衰减至 0。本质上，截止不是 物理正确的。

默认值为 0。

.isPointLight : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.power : number

灯光功率。功率是以流明 (lm) 为单位的光通量。 更改功率也会更改灯光强度。

.shadow : PointLightShadow

此属性保存灯光的阴影配置。
