GLTFLoader

用于加载 glTF 2.0 格式的加载器。

glTF（GL Transmission Format）是一种开放格式规范，在可能的情况下使用。请注意，ImageBitmap 在不再被引用时不会自动被垃圾回收，在销毁过程中需要特殊处理。

GLTFLoader 支持以下 glTF 2.0 扩展：

  • KHR_draco_mesh_compression
  • KHR_lights_punctual
  • KHR_materials_anisotropy
  • KHR_materials_clearcoat
  • KHR_materials_dispersion
  • KHR_materials_emissive_strength
  • KHR_materials_ior
  • KHR_materials_specular
  • KHR_materials_transmission
  • KHR_materials_iridescence
  • KHR_materials_unlit
  • KHR_materials_volume
  • KHR_mesh_quantization
  • KHR_meshopt_compression
  • KHR_texture_basisu
  • KHR_texture_transform
  • EXT_materials_bump
  • EXT_meshopt_compression
  • EXT_mesh_gpu_instancing
  • EXT_texture_avif
  • EXT_texture_webp

以下 glTF 2.0 扩展由外部用户插件支持：

  • KHR_materials_variants
  • MSFT_texture_dds
  • KHR_animation_pointer
  • NEEDLE_progressive

代码示例

const loader = new GLTFLoader();
// Optional: Provide a DRACOLoader instance to decode compressed mesh data
const dracoLoader = new DRACOLoader();
dracoLoader.setDecoderPath( '/examples/jsm/libs/draco/' );
loader.setDRACOLoader( dracoLoader );
const gltf = await loader.loadAsync( 'models/gltf/duck/duck.gltf' );
scene.add( gltf.scene );

导入

GLTFLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { GLTFLoader } from 'three/addons/loaders/GLTFLoader.js';

构造函数

new GLTFLoader( manager : LoadingManager )

构造一个新的 glTF 加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 glTF 资源传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( data : string | ArrayBuffer, path : string, onLoad : function, onError : onErrorCallback )

解析给定的 glTF 数据并返回生成的组。

data

原始 glTF 数据。
path

URL 基础路径。
onLoad

加载完成时执行。
onError

发生错误时执行。
重写： Loader#parse

.parseAsync( data : string | ArrayBuffer, path : string ) : Promise.<GLTFLoader~LoadObject> (async)

GLTFLoader#parse 的异步版本。

data

原始 glTF 数据。
path

URL 基础路径。
返回值： 在解析完成后解析并返回已加载 glTF 的 Promise。

.register( callback : function ) : GLTFLoader

注册插件回调。此 API 在内部用于实现各种 glTF 扩展，也可供第三方代码向加载器添加额外逻辑。

callback

要注册的回调函数。
返回值： 此加载器的引用。

.setDRACOLoader( dracoLoader : DRACOLoader ) : GLTFLoader

将指定的 Draco 加载器设置到此加载器。解码使用 KHR_draco_mesh_compression 扩展压缩的资源时需要。

dracoLoader

要设置的 Draco 加载器。
返回值： 此加载器的引用。

.setKTX2Loader( ktx2Loader : KTX2Loader ) : GLTFLoader

将指定的 KTX2 加载器设置到此加载器。加载 KTX2 压缩纹理时需要。

ktx2Loader

要设置的 KTX2 加载器。
返回值： 此加载器的引用。

.setMeshoptDecoder( meshoptDecoder : Object ) : GLTFLoader

设置指定的 meshopt 解码器。解码使用 EXT_meshopt_compression 扩展压缩的资源时需要。

meshoptDecoder

要设置的 meshopt 解码器。
返回值： 此加载器的引用。

.unregister( callback : function ) : GLTFLoader

注销插件回调。

callback

要注销的回调函数。
返回值： 此加载器的引用。

类型定义

.LoadObject

GLTFLoader 的加载器结果。

animations
Array.<AnimationClip>

动画剪辑数组。
asset
Object

已加载资源的元数据。
cameras
Array.<Camera>

相机数组。
parser
GLTFParser

内部解析器的引用。
scene
Group

默认场景。
scenes
Array.<Group>

glTF 资源可能定义多个场景。
userData
Object

附加数据。
