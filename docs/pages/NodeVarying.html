NodeVarying

NodeBuilder 在构建节点的过程中会创建此类的实例。 它们表示构建器最终将生成的着色器 varying。为此，节点 varying 数组维护在 NodeBuilder#varyings 中。

构造函数

new NodeVarying( name : string, type : string, interpolationType : string, interpolationSampling : string )

构造一个新的节点 varying。

name

varying 的名称。
type

varying 的类型。
interpolationType

varying 的插值类型。

默认值为 null。
interpolationSampling

varying 的插值采样类型。

默认值为 null。

属性

.interpolationSampling : string

varying 数据的插值采样类型。

默认值为 null。

.interpolationType : string

varying 数据的插值类型。

默认值为 null。

.isNodeVarying : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.needsInterpolation : boolean

此 varying 是否需要插值。此属性可用于检查该 varying 是否可优化为 variable。

默认值为 false。
