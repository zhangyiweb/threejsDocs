SubsurfaceScatteringShader

导入

SubsurfaceScatteringShader 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { SubsurfaceScatteringShader } from 'three/addons/shaders/SubsurfaceScatteringShader.js';

属性

.SubsurfaceScatteringShader : ShaderMaterial~Shader (inner, constant)

次表面散射着色器。

基于 GDC 2011 – Approximating Translucency for a Fast, Cheap and Convincing Subsurface Scattering Look
