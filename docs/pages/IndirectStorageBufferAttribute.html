IndirectStorageBufferAttribute

此特殊类型的缓冲属性用于计算着色器。 可用于编码间接绘制调用的绘制参数。

注意：此类型的缓冲属性只能与 WebGPURenderer 及 WebGPU 后端一起使用。

构造函数

new IndirectStorageBufferAttribute( count : number | Uint32Array, itemSize : number )

构造一个新的存储缓冲属性。

count

元素数量。也可以传入 Uint32Array 作为参数。 此时后续参数将失效。
itemSize

每个元素的大小。

属性

.isIndirectStorageBufferAttribute : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
