ReferenceNode

此类节点建立对其他对象属性的引用。 这样，节点的值会自动与引用对象的值关联。引用节点在内部将关联的值 表示为 uniform。

构造函数

new ReferenceNode( property : string, uniformType : string, object : Object, count : number )

构造一个新的引用节点。

property

节点所引用属性的名称。
uniformType

用于表示属性值的 uniform 类型。
object

属性所属的对象。

默认值为 null。
count

当关联属性为类数组类型时，此参数定义其长度。

默认值为 null。

属性

.count : number

当关联属性为数组时，此参数定义其长度。

默认值为 null。

.group : UniformGroupNode

内部 uniform 的 uniform 组。

默认值为 null。

.name : string

内部 uniform 节点的可选标签。

默认值为 null。

重写： Node#name

.node : UniformNode

保存引用节点值的 uniform 节点。

默认值为 null。

.object : Object

属性所属的对象。

默认值为 null。

.properties : Array.<string>

属性名称可能包含点号以便引用嵌套属性。 名称层级存储在此数组中。

.property : string

节点所引用属性的名称。

.reference : Object

指向当前引用的对象。此属性与 ReferenceNode#object 并存， 因为最终引用可能会由调用代码更新。

默认值为 null。

.uniformType : string

用于表示属性值的 uniform 类型。

.updateType : string

已重写，因为引用节点按对象更新。

默认值为 'object'。

重写： Node#updateType

方法

.element( indexNode : IndexNode ) : ReferenceElementNode

当被引用属性为类数组类型时，可使用此方法 通过索引节点访问元素。

indexNode

索引节点。
返回值： 对元素的引用。

.generateNodeType( builder : NodeBuilder ) : string

此方法已重写，因为节点类型从 引用节点的类型推断。

builder

当前节点构建器。
重写： Node#generateNodeType
返回值： 节点类型。

.getValueFromReference( object : Object ) : any

从给定引用对象返回属性值。

object

要从中检索属性值的对象。

默认值为 this.reference。
返回值： 值。

.label( name : string ) : ReferenceNode

设置内部 uniform 的标签。

name

要设置的标签。
已弃用： 是
返回值： 对此节点的引用。

.setGroup( group : UniformGroupNode ) : ReferenceNode

设置此引用节点的 uniform 组。

group

要设置的 uniform 组。
返回值： 对此节点的引用。

.setName( name : string ) : ReferenceNode

设置内部 uniform 的名称。

name

要设置的标签。
返回值： 对此节点的引用。

.setNodeType( uniformType : string )

设置节点类型，自动定义内部 uniform 类型。

uniformType

要设置的类型。

.setup( builder : NodeBuilder ) : UniformNode

引用节点的输出为内部 uniform 节点。

builder

当前节点构建器。
重写： Node#setup
返回值： 输出节点。

.update( frame : NodeFrame )

已重写以更新内部 uniform 值。

frame

对当前节点帧的引用。
重写： Node#update

.updateReference( state : NodeFrame | NodeBuilder ) : Object

允许根据给定状态更新引用。仅当未设置 ReferenceNode#object 时才会评估状态。

state

当前状态。
重写： Node#updateReference
返回值： 更新后的引用。

.updateValue()

从引用对象属性检索值并用其 更新内部 uniform。
