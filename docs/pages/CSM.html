CSM

级联阴影贴图（Cascade Shadow Maps，CSM）的一种实现。

此模块只能与 WebGLRenderer 一起使用。使用 WebGPURenderer 时， 请改用 CSMShadowNode。

导入

CSM 是一个插件，必须显式导入，参见 Installation#Addons。

import { CSM } from 'three/addons/csm/CSM.js';

构造函数

new CSM( data : CSM~Data )

构造一个新的 CSM 实例。

data

CSM 数据。

类

CSM

属性

.breaks : Array.<number>

取值范围为 [0,1] 的数字数组，用于定义如何分割 主 CSM 视锥体。

.camera : Camera

场景的相机。

.cascades : number

级联数量。

默认值为 3。

.customSplitsCallback : function

使用 mode='custom' 时的自定义分割回调。

.fade : boolean

是否在级联之间进行淡入淡出过渡。

默认值为 false。

.frustums : Array.<CSMFrustum>

表示各级联的视锥体数组。

.lightDirection : Vector3

光照方向。

.lightFar : number

灯光的远裁剪面值。

默认值为 2000。

.lightIntensity : number

灯光强度。

默认值为 3。

.lightMargin : number

灯光边距。

默认值为 200。

.lightNear : number

灯光的近裁剪面值。

默认值为 1。

.lights : Array.<DirectionalLight>

为不同级联投射阴影的平行光数组。 每个级联对应一盏平行光。

.mainFrustum : CSMFrustum

主视锥体。

.maxFar : number

最大远裁剪面值。

默认值为 100000。

.mode : 'practical' | 'uniform' | 'logarithmic' | 'custom'

视锥体分割模式。

默认值为 'practical'。

.parent : Object3D

父对象，通常为场景。

.shaders : Map.<Material, Object>

保存增强后材质着色器的 Map。

.shadowBias : number

阴影偏移。

默认值为 0.000001。

.shadowMapSize : number

阴影贴图尺寸。

默认值为 2048。

方法

.dispose()

释放此实例分配的 GPU 相关资源。当应用中 不再使用此实例时，请调用此方法。

.remove()

当应用从场景中移除 CSM 时，必须调用此方法。

.setupMaterial( material : Material )

对于所有应受 CSM 影响的材质，应用都必须调用此方法。

material

要配置为支持 CSM 的材质。

.update()

更新 CSM。必须在动画循环中、调用 renderer.render() 之前调用此方法。

.updateFrustums()

每当更改相机或 CSM 设置时，应用都必须调用此方法。
