OrbitControls

轨道控制器允许相机围绕目标点旋转。

OrbitControls 执行环绕、推拉（缩放）与平移。与 TrackballControls 不同， 它会保持「上」方向 object.up（默认为 +Y）。

  • 环绕：鼠标左键 / 触摸：单指移动。
  • 缩放：鼠标中键或滚轮 / 触摸：双指张开或捏合。
  • 平移：鼠标右键，或鼠标左键 + ctrl/meta/shiftKey，或方向键 / 触摸：双指移动。

代码示例

const controls = new OrbitControls( camera, renderer.domElement );
// controls.update() must be called after any manual changes to the camera's transform
camera.position.set( 0, 20, 100 );
controls.update();
function animate() {
	// required if controls.enableDamping or controls.autoRotate are set to true
	controls.update();
	renderer.render( scene, camera );
}

导入

OrbitControls 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { OrbitControls } from 'three/addons/controls/OrbitControls.js';

构造函数

new OrbitControls( object : Object3D, domElement : HTMLElement )

构造一个新的控制器实例。

object

由控制器管理的对象。
domElement

用于事件监听的 HTML 元素。

默认值为 null。

属性

.autoRotate : boolean

设为 true 可自动围绕目标旋转。

注意，若启用此功能，必须在动画循环中调用 update()。 若希望自动旋转速度与帧率（显示器刷新率）无关，必须将时间 deltaTime（单位为秒）传给 update()。

默认值为 false。

.autoRotateSpeed : number

当 autoRotate 为 true 时，围绕目标旋转的速度。默认值相当于在 60fps 下每圈 30 秒。

注意，若启用 autoRotate，必须在动画循环中调用 update()。

默认值为 2。

.cursor : Vector3

minTargetRadius 与 maxTargetRadius 限制的焦点。 可随时手动更新，以改变 target 的兴趣中心。

.cursorStyle : 'auto' | 'grab'

定义光标的视觉表现。

默认值为 'auto'。

.dampingFactor : number

当 enableDamping 设为 true 时使用的阻尼惯性。

注意，要使阻尼生效，必须在动画循环中调用 update()。

默认值为 0.05。

.enableDamping : boolean

设为 true 以启用阻尼（惯性），可用于为控制器增加重量感。 注意，若启用此功能，必须在动画循环中调用 update()。

默认值为 false。

.enablePan : boolean

启用或禁用相机平移。

默认值为 true。

.enableRotate : boolean

启用或禁用相机的水平与垂直旋转。

注意，可通过将 minPolarAngle 或 minAzimuthAngle 的最小值与最大值设为相同值来禁用单个轴， 从而使垂直或水平旋转固定在该值。

默认值为 true。

.enableZoom : boolean

启用或禁用相机缩放（推拉）。

默认值为 true。

.keyPanSpeed : number

使用键盘时相机平移的速度，单位为每次按键的像素数。

默认值为 7。

.keyRotateSpeed : number

使用键盘时相机旋转的速度。

默认值为 1。

.keys : Object

此对象包含用于控制相机平移的键码引用。

controls.keys = {
	LEFT: 'ArrowLeft', //left arrow
	UP: 'ArrowUp', // up arrow
	RIGHT: 'ArrowRight', // right arrow
	BOTTOM: 'ArrowDown' // down arrow
}
重写： Controls#keys

.maxAzimuthAngle : number

水平环绕的上限。若设置，区间 [ min, max ] 必须是 [ - 2 PI, 2 PI ] 的子区间，且满足 ( max - min < 2 PI )。

默认值为 -Infinity。

.maxDistance : number

可向外推拉的最远距离（仅透视相机）。

默认值为 Infinity。

.maxPolarAngle : number

垂直环绕的上限。范围为 [0, Math.PI] 弧度。

默认值为 Math.PI。

.maxTargetRadius : number

可将目标移离 3D cursor 的最远距离。

默认值为 Infinity。

.maxZoom : number

可缩小的最远程度（仅正交相机）。

默认值为 Infinity。

.minAzimuthAngle : number

水平环绕的下限。若设置，区间 [ min, max ] 必须是 [ - 2 PI, 2 PI ] 的子区间，且满足 ( max - min < 2 PI )。

默认值为 -Infinity。

.minDistance : number

可向内推拉的最近距离（仅透视相机）。

默认值为 0。

.minPolarAngle : number

垂直环绕的下限。范围为 [0, Math.PI] 弧度。

默认值为 0。

.minTargetRadius : number

目标可接近 3D cursor 的最近距离。

默认值为 0。

.minZoom : number

可放大的最近程度（仅正交相机）。

默认值为 0。

.mouseButtons : Object

此对象包含控制器所使用的鼠标操作引用。

controls.mouseButtons = {
	LEFT: THREE.MOUSE.ROTATE,
	MIDDLE: THREE.MOUSE.DOLLY,
	RIGHT: THREE.MOUSE.PAN
}
重写： Controls#mouseButtons

.panSpeed : number

平移速度。

默认值为 1。

.position0 : Vector3

由 saveState() 和 reset() 内部使用。

.rotateSpeed : number

旋转速度。

默认值为 1。

.screenSpacePanning : boolean

定义平移时相机位置的平移方式。若为 true，相机在屏幕空间中平移。 否则，相机在垂直于相机上方向的平面内平移。

默认值为 true。

.target : Vector3

控制器的焦点，object 围绕此点环绕。 可随时手动更新以改变控制器的焦点。

.target0 : Vector3

由 saveState() 和 reset() 内部使用。

.touches : Object

此对象包含控制器所使用的触摸操作引用。

controls.mouseButtons = {
	ONE: THREE.TOUCH.ROTATE,
	TWO: THREE.TOUCH.DOLLY_PAN
}
重写： Controls#touches

.zoom0 : number

由 saveState() 和 reset() 内部使用。

.zoomSpeed : number

缩放 / 推拉速度。

默认值为 1。

.zoomToCursor : boolean

将此属性设为 true 可缩放至光标位置。

默认值为 false。

方法

.dollyIn( dollyScale : number )

以编程方式向内推拉（透视相机为放大）。

dollyScale

推拉缩放因子。

.dollyOut( dollyScale : number )

以编程方式向外推拉（透视相机为缩小）。

dollyScale

推拉缩放因子。

.getAzimuthalAngle() : number

获取当前水平旋转角度（弧度）。

返回值： 当前水平旋转角度（弧度）。

.getDistance() : number

返回相机到目标的距离。

返回值： 相机到目标的距离。

.getPolarAngle() : number

获取当前垂直旋转角度（弧度）。

返回值： 当前垂直旋转角度（弧度）。

.listenToKeyEvents( domElement : HTMLElement )

向给定的 DOM 元素添加键盘事件监听器。 使用此方法时，推荐将 window 作为参数。

domElement

DOM 元素。

.pan( deltaX : number, deltaY : number )

以编程方式平移相机。

deltaX

水平平移量（像素）。
deltaY

垂直平移量（像素）。

.reset()

将控制器重置为上次调用 saveState() 时的状态，或初始状态。

.rotateLeft( angle : number )

以编程方式使相机向左旋转（绕垂直轴）。

angle

旋转角度（弧度）。

.rotateUp( angle : number )

以编程方式使相机向上旋转（绕水平轴）。

angle

旋转角度（弧度）。

.saveState()

保存控制器的当前状态。之后可通过 reset() 恢复。

.stopListenToKeyEvents()

移除先前通过 listenToKeyEvents() 定义的键盘事件监听器。

事件

.change

当相机已被控制器变换时触发。

类型：
  • Object

.end

当交互结束时触发。

类型：
  • Object

.start

当交互开始时触发。

类型：
  • Object
