Rhino3dmLoader

用于加载 Rhinoceros 3D 文件和对象的加载器。

Rhinoceros 是一款 3D 建模软件，用于创建、编辑、分析、记录、渲染、 动画化以及转换 NURBS 曲线、曲面、Brep、拉伸体、点云， 以及多边形网格和 SubD 对象。rhino3dm.js 由开源几何库 openNURBS 编译为 WebAssembly。 此加载器当前使用 rhino3dm.js 8.4.0。

代码示例

const loader = new Rhino3dmLoader();
loader.setLibraryPath( 'https://cdn.jsdelivr.net/npm/rhino3dm@8.17.0/' );
const object = await loader.loadAsync( 'models/3dm/Rhino_Logo.3dm' );
scene.add( object );

导入

Rhino3dmLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { Rhino3dmLoader } from 'three/addons/loaders/3DMLoader.js';

构造函数

new Rhino3dmLoader( manager : LoadingManager )

构造一个新的 Rhino 3DM 加载器。

manager

加载管理器。

方法

.debug()

向浏览器控制台打印调试信息。

.decodeObjects( buffer : ArrayBuffer, url : string ) : Promise.<Object3D>

使用 Web Worker 解码 3DM 资源数据。

buffer

原始 3DM 资源数据 ArrayBuffer。
url

资源 URL。
返回值： 解析后以解码后的 3D 对象兑现的 Promise。

.dispose()

释放内部资源。不再需要此加载器时应调用本方法。

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 3DM 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( data : ArrayBuffer, onLoad : function, onError : onErrorCallback )

解析给定的 3DM 数据，并将加载完成的 3DM 资源 传递给 onLoad() 回调。

data

原始 3DM 资源数据 ArrayBuffer。
onLoad

加载完成时执行。
onError

发生错误时执行。
重写： Loader#parse

.setLibraryPath( path : string ) : Rhino3dmLoader

包含 JS 和 WASM 库的文件夹路径。

path

要设置的库路径。
返回值： 对此加载器的引用。

.setWorkerLimit( workerLimit : number ) : Rhino3dmLoader

设置解码期间使用的 Web Worker 最大数量。 若 Worker 在应用中还需承担其他任务， 可适当降低此限制。

workerLimit

Worker 数量上限。
返回值： 对此加载器的引用。
