ArcCurve

表示圆弧的曲线。

构造函数

new ArcCurve( aX : number, aY : number, aRadius : number, aStartAngle : number, aEndAngle : number, aClockwise : boolean )

构造一条新的圆弧曲线。

aX

椭圆的 X 中心。

默认值为 0。
aY

椭圆的 Y 中心。

默认值为 0。
aRadius

椭圆在 x 方向上的半径。

默认值为 1。
aStartAngle

曲线的起始角度（弧度），从正 X 轴开始计量。

默认值为 0。
aEndAngle

曲线的结束角度（弧度），从正 X 轴开始计量。

默认值为 Math.PI*2。
aClockwise

椭圆是否按顺时针方向绘制。

默认值为 false。

属性

.isArcCurve : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
