SpriteNodeMaterial

SpriteMaterial 的节点材质版本。

构造函数

new SpriteNodeMaterial( parameters : Object )

构造一个新的精灵节点材质。

parameters

配置参数。

属性

.isSpriteNodeMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.positionNode : Node.<vec2>

此属性可用节点定义精灵的位置。 当材质用于实例化渲染，且节点数据通过实例化 attribute 节点定义时， 会很有用：

const positionAttribute = new InstancedBufferAttribute( new Float32Array( positions ), 3 );
material.positionNode = instancedBufferAttribute( positionAttribute );

另一种方式是用计算着色器计算实例化数据：

const positionBuffer = instancedArray( particleCount, 'vec3' );
particleMaterial.positionNode = positionBuffer.toAttribute();

默认值为 null。

重写： NodeMaterial#positionNode

.rotationNode : Node.<float>

精灵材质的旋转默认由 rotation 属性推断。此节点属性可覆盖默认行为，改用 节点定义旋转。

如果不想覆盖旋转，而只是修改现有 值，请使用 materialRotation。

默认值为 null。

.scaleNode : Node.<vec2>

此节点属性提供了在 Object3D.scale 之外 缩放精灵的额外方式。基于 Object3D.scale 的缩放变换 会在顶点着色器中与此节点的缩放值相乘。

默认值为 null。

.sizeAttenuation : boolean

是否使用尺寸衰减。

默认值为 true。

.transparent : boolean

在精灵中，transparent 属性默认启用。

默认值为 true。

重写： NodeMaterial#transparent

方法

.setupPositionView( builder : NodeBuilder ) : Node.<vec3>

设置视图空间中的位置节点。此方法实现了 精灵专用的顶点着色器。

builder

当前节点构建器。
重写： NodeMaterial#setupPositionView
返回值： 视图空间中的位置。
