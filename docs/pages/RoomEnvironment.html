RoomEnvironment

此类表示一个带有基础房间布置的场景，可用作 PMREMGenerator#fromScene 的输入。生成的 PMREM 表示该房间的光照，可通过赋值给 Scene#environment，或直接作为环境贴图赋给 PBR 材质，用于基于图像的光照（IBL）。

实现基于 model-viewer 项目中的 EnvironmentScene 组件。

代码示例

const environment = new RoomEnvironment();
const pmremGenerator = new THREE.PMREMGenerator( renderer );
const envMap = pmremGenerator.fromScene( environment ).texture;
scene.environment = envMap;

导入

RoomEnvironment 是一个插件，必须显式导入，参见 Installation#Addons。

import { RoomEnvironment } from 'three/addons/environments/RoomEnvironment.js';

构造函数

new RoomEnvironment()

方法

.dispose()

释放内部资源。当不再需要该环境时应调用此方法。
