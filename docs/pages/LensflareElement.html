LensflareElement

表示可添加到 Lensflare 容器中的单个光晕。

导入

LensflareElement 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { LensflareElement } from 'three/addons/objects/Lensflare.js';

构造函数

new LensflareElement( texture : Texture, size : number, distance : number, color : Color )

构造一个新的镜头光晕元素。

texture

光晕的纹理。
size

以像素为单位的尺寸。

默认值为 1。
distance

距光源的归一化距离（[0,1]）。 值为 0 表示光晕位于光源处。

默认值为 0。
color

光晕的颜色

属性

.color : Color

光晕的颜色

默认值为 (1,1,1)。

.distance : number

距光源的归一化距离（[0,1]）。 值为 0 表示光晕位于光源处。

默认值为 0。

.size : number

以像素为单位的尺寸。

默认值为 1。

.texture : Texture

光晕的纹理。
