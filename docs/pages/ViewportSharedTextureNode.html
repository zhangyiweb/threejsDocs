ViewportSharedTextureNode

ViewportTextureNode 会为每个节点实例创建内部纹理。此模块 则在所有 ViewportSharedTextureNode 实例之间共享同一纹理。出于性能考虑， 在使用默认/屏幕帧缓冲数据时应优先选择本模块。

构造函数

new ViewportSharedTextureNode( uvNode : Node, levelNode : Node )

构造一个新的视口共享纹理节点。

uvNode

uv 节点。

默认值为 screenUV。
levelNode

level 节点。

默认值为 null。

方法

.getTextureForReference() : FramebufferTexture

已重写，使该方法始终返回唯一的共享 帧缓冲纹理。

重写： ViewportTextureNode#getTextureForReference
返回值： 共享的帧缓冲纹理。
