ProjectorLight

SpotLight 的投影仪灯光版本。仅可与 WebGPURenderer 一起使用。

构造函数

new ProjectorLight( color : number | Color | string, intensity : number, distance : number, angle : number, penumbra : number, decay : number )

构造一个新的投影仪灯光。

color

光源颜色。

默认值为 0xffffff。
intensity

光源强度，单位为坎德拉 (cd)。

默认值为 1。
distance

灯光的最大照射范围。0 表示无限制。

默认值为 0。
angle

光线相对其方向的最大散射角度，上限为 Math.PI/2。

默认值为 Math.PI/3。
penumbra

聚光灯锥体中因半影而衰减的百分比。取值范围为 [0,1]。

默认值为 0。
decay

灯光沿距离衰减的程度。

默认值为 2。

属性

.aspect : number

灯光的宽高比。设为 null 以使用纹理宽高比。

默认值为 null。
