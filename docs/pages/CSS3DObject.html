CSS3DObject

CSS3DRenderer 支持的基础 3D 对象。

导入

CSS3DObject 是一个插件，必须显式导入，参见 Installation#Addons。

import { CSS3DObject } from 'three/addons/renderers/CSS3DRenderer.js';

构造函数

new CSS3DObject( element : HTMLElement )

构造一个新的 CSS3D 对象。

element

DOM 元素。

属性

.element : HTMLElement (readonly)

定义此 3D 对象外观的 DOM 元素。

默认值为 true。

.isCSS3DObject : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
