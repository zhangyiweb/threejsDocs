TrefoilPolynomialKnot

三叶多项式纽结。

导入

TrefoilPolynomialKnot 是一个插件，必须显式导入，参见 Installation#Addons。

import { TrefoilPolynomialKnot } from 'three/addons/curves/CurveExtras.js';

构造函数

new TrefoilPolynomialKnot( scale : number )

构造一个新的三叶多项式纽结。

scale

曲线的缩放。

默认值为 10。

属性

.scale : number

曲线的缩放。

默认值为 10。

方法

.getPoint( t : number, optionalTarget : Vector3 ) : Vector3

此方法根据给定的插值因子返回三维空间中的向量。

t

表示曲线上位置的插值因子，必须在 [0,1] 范围内。
optionalTarget

可选的目标向量，结果将写入其中。
重写： Curve#getPoint
返回值： 曲线上的位置。
