BitcastNode

该节点表示一种操作：将某一类型值的位表示 重新解释为另一类型的值。

构造函数

new BitcastNode( valueNode : Node, conversionType : string, inputType : string )

构造一个新的位转换节点。

valueNode

要转换的值。
conversionType

要转换到的类型。
inputType

位转换操作所期望的输入数据类型。

默认值为 null。

属性

.conversionType : string

值将被转换到的类型。

.inputType : string

位转换操作所期望的输入数据类型。

默认值为 null。

.isBitcastNode : boolean (readonly)

该标志可用于类型检测。

默认值为 true。

.valueNode : Node

要位转换为新类型的数据。
