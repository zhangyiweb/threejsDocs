OutlineEffect

用于卡通着色器的描边效果。

注意：此类仅可与 WebGLRenderer 配合使用。 使用 WebGPURenderer 时，请使用 ToonOutlinePassNode。

代码示例

const effect = new OutlineEffect( renderer );
function render() {
	effect.render( scene, camera );
}

导入

OutlineEffect 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { OutlineEffect } from 'three/addons/effects/OutlineEffect.js';

构造函数

new OutlineEffect( renderer : WebGLRenderer, parameters : OutlineEffect~Options )

构造一个新的描边效果。

renderer

渲染器。
parameters

配置参数。

方法

.render( scene : Object3D, camera : Camera )

使用此效果时，应调用此方法而非默认的 WebGLRenderer#render。

scene

要渲染的场景。
camera

相机。

.renderOutline( scene : Object3D, camera : Camera )

此方法可用于在 VR 中渲染描边。

const effect = new OutlineEffect( renderer );
let renderingOutline = false;
scene.onAfterRender = function () {
	if ( renderingOutline ) return;
	renderingOutline = true;
	effect.renderOutline( scene, camera );
	renderingOutline = false;
};
function render() {
	renderer.render( scene, camera );
}
scene

要渲染的场景。
camera

相机。

.setSize( width : number, height : number )

调整效果尺寸。

width

效果的宽度（逻辑像素）。
height

效果的高度（逻辑像素）。

类型定义

.Options

此类型表示 OutlineEffect 的配置设置。

defaultThickness
number

描边粗细。

默认值为 0.003。
defaultColor
Array.<number>

描边颜色。

默认值为 [0,0,0]。
defaultAlpha
number

描边的透明度。

默认值为 1。
defaultKeepAlive
boolean

是否保留缓存的内部材质。

默认值为 false。
