ShapeUtils

包含形状相关实用函数的类。

静态方法

.area( contour : Array.<Vector2> ) : number

计算（二维）轮廓多边形的面积。

contour

二维点数组。
返回值： 面积。

.isClockWise( pts : Array.<Vector2> ) : boolean

若给定轮廓使用顺时针绕序，则返回 true。

pts

定义多边形的二维点数组。
返回值： 给定轮廓是否使用顺时针绕序。

.triangulateShape( contour : Array.<Vector2>, holes : Array.<Array.<Vector2>> ) : Array.<Array.<number>>

对给定的形状定义进行三角剖分。

contour

定义轮廓的二维点数组。
holes

保存二维点数组的数组，用于定义孔洞。
返回值： 数组中每个面定义对应一个包含三个索引的数组。
