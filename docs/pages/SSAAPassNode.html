SSAAPassNode

使用 SSAA（超采样抗锯齿）渲染场景的特殊渲染通道节点。 这种手动 SSAA 方法会对每个采样点在相机抖动下重新渲染一次场景，并累积结果。

此节点可产生高质量的抗锯齿输出，但由于需要多次重新渲染整个场景， 采用暴力方式，开销也极大。

参考：https://en.wikipedia.org/wiki/Supersampling

导入

SSAAPassNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { ssaaPass } from 'three/addons/tsl/display/SSAAPassNode.js';

构造函数

new SSAAPassNode( scene : Scene, camera : Camera )

构造一个新的 SSAA 通道节点。

scene

要渲染的场景。
camera

用于渲染场景的相机。

属性

.isSSAAPassNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.sampleLevel : number

采样级别。以 n 表示，采样数为 2^n， 因此 sampleLevel = 4 表示 2^4 = 16 个采样。

默认值为 4。

.sampleWeight : UniformNode.<float>

表示采样权重的 uniform 节点。

默认值为 1。

.unbiased : boolean

是否应缓解舍入误差。

默认值为 true。

方法

.dispose()

释放内部资源。当通道不再使用时应调用此方法。

重写： PassNode#dispose

.setup( builder : NodeBuilder ) : PassTextureNode

此方法用于设置效果的 MRT 配置和四边形网格。

builder

当前的节点构建器。
重写： PassNode#setup

.updateBefore( frame : NodeFrame )

此方法用于每帧渲染一次 SSAA 效果。

frame

当前的节点帧。
重写： PassNode#updateBefore
