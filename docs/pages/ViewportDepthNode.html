ViewportDepthNode

此节点提供片元着色器中深度逻辑相关的一组功能。 根据 ViewportDepthNode#scope，可用于为当前片元定义深度值， 或用于深度评估。

构造函数

new ViewportDepthNode( scope : 'depth' | 'depthBase' | 'linearDepth', valueNode : Node )

构造一个新的视口深度节点。

scope

节点的作用域。
valueNode

值节点。

默认值为 null。

属性

.isViewportDepthNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.scope : 'depth' | 'depthBase' | 'linearDepth'

节点的行为会根据所选作用域而有所不同。

  • ViewportDepthNode.DEPTH_BASE：允许为当前片元的深度定义一个值。
  • ViewportDepthNode.DEPTH：表示当前片元的深度值（忽略 valueNode）。
  • ViewportDepthNode.LINEAR_DEPTH：表示当前片元的线性（正交）深度值。 若设置了 valueNode，该作用域可用于将透视深度数据转换为线性数据。

.valueNode : Node

可用于定义自定义深度值。 在 ViewportDepthNode.DEPTH 作用域中会忽略此属性。

默认值为 null。
