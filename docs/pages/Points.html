Points

用于显示点或点云的类。

构造函数

new Points( geometry : BufferGeometry, material : Material | Array.<Material> )

构造一个新的点云。

geometry

点几何体。
material

点材质。

属性

.geometry : BufferGeometry

点几何体。

.isPoints : boolean (readonly)

该标志可用于类型检测。

默认值为 true。

.material : Material | Array.<Material>

点材质。

默认值为 PointsMaterial。

.morphTargetDictionary : Object.<string, number> | undefined

表示几何体中变形目标的字典。键为变形目标名称， 值为其属性索引。该成员默认值为 undefined， 仅在几何体中检测到变形目标时才会被设置。

默认值为 undefined。

.morphTargetInfluences : Array.<number> | undefined

权重数组，通常取值范围为 [0,1]，用于指定变形的应用程度。 该成员默认值为 undefined，仅在几何体中检测到变形目标时才会被设置。

默认值为 undefined。

方法

.raycast( raycaster : Raycaster, intersects : Array.<Object> )

计算投射光线与该点云之间的交点。

raycaster

光线投射器。
intersects

用于保存交点的目标数组。
重写： Object3D#raycast

.updateMorphTargets()

设置 Points#morphTargetDictionary 和 Points#morphTargetInfluences 的值，以确保现有变形目标能够影响该 3D 对象。
