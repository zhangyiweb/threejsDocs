VolumeSlice

此类用于保存体数据的一个切片。

导入

VolumeSlice 是一个插件，必须显式导入，参见 Installation#Addons。

import { VolumeSlice } from 'three/addons/misc/VolumeSlice.js';

构造函数

new VolumeSlice( volume : Volume, index : number, axis : 'x' | 'y' | 'z' )

构造一个新的体数据切片。

volume

关联的体数据。
index

切片的索引。

默认值为 0。
axis

目前仅支持 'x'、'y' 或 'z'，后续可能改为法向量。

默认值为 'z'。
参见：
  • Volume

属性

.axis : 'x' | 'y' | 'z'

法线轴。

.canvas : HTMLCanvasElement

用于纹理的最终 canvas。

.canvasBuffer : HTMLCanvasElement

用于绘制数据的中间 canvas。

.ctx : CanvasRenderingContext2D

canvas 的渲染上下文。

.ctxBuffer : CanvasRenderingContext2D

canvas 缓冲区的渲染上下文。

.geometryNeedsUpdate : boolean

若设为 true，下次 repaint 时将触发 updateGeometry()。

默认值为 true。

.iLength : number

切片在原始坐标系中的宽度，对应缓冲区 canvas 的宽度。

默认值为 0。

.index : number

切片的索引，若更改，将在下次 repaint 时自动调用 updateGeometry。

默认值为 0。

.jLength : number

切片在原始坐标系中的高度，对应缓冲区 canvas 的高度。

默认值为 0。

.mesh : Mesh

可直接用于场景的网格。

.sliceAccess : function

允许切片访问正确数据的函数。

参见：
  • Volume#extractPerpendicularPlane

.volume : Volume

关联的体数据。

方法

.repaint()

刷新纹理；若 geometryNeedsUpdate 设为 true，则同时刷新几何体。

.updateGeometry()

根据 axis 和 index 刷新几何体。

参见：
  • Volume#extractPerpendicularPlane
