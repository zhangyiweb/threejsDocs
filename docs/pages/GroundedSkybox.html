GroundedSkybox

一种地面投影天空盒。

默认情况下该对象以相机为中心，因此通常建议设置 skybox.position.y = height，以使地面位于原点。

代码示例

const height = 15, radius = 100;
const skybox = new GroundedSkybox( envMap, height, radius );
skybox.position.y = height;
scene.add( skybox );

导入

GroundedSkybox 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { GroundedSkybox } from 'three/addons/objects/GroundedSkybox.js';

构造函数

new GroundedSkybox( map : Texture, height : number, radius : number, resolution : number )

构造一个新的地面投影天空盒。

map

要使用的环境贴图。
height

高度表示拍摄照片时相机距地面的距离。 较大的值会放大图像的下方部分。
radius

天空盒的半径。必须足够大，以确保场景中的相机始终位于其内部。
resolution

天空盒的几何分辨率。

默认值为 128。
