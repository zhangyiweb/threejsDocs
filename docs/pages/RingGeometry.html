RingGeometry

用于生成二维环形几何体的类。

代码示例

const geometry = new THREE.RingGeometry( 1, 5, 32 );
const material = new THREE.MeshBasicMaterial( { color: 0xffff00, side: THREE.DoubleSide } );
const mesh = new THREE.Mesh( geometry, material );
scene.add( mesh );

构造函数

new RingGeometry( innerRadius : number, outerRadius : number, thetaSegments : number, phiSegments : number, thetaStart : number, thetaLength : number )

构造一个新的环形几何体。

innerRadius

环的内半径。

默认值为 0.5.
outerRadius

环的外半径。

默认值为 1.
thetaSegments

分段数。数值越大，环越接近圆形。最小值为 3。

默认值为 32.
phiSegments

每个环向分段的分段数。最小值为 1。

默认值为 1.
thetaStart

起始角度（弧度）。

默认值为 0.
thetaLength

圆心角（弧度）。

默认值为 Math.PI*2.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

静态方法

.fromJSON( data : Object ) : RingGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
