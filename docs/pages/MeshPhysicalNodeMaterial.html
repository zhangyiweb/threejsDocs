MeshPhysicalNodeMaterial

MeshPhysicalMaterial 的节点材质版本。

构造函数

new MeshPhysicalNodeMaterial( parameters : Object )

构造一个新的 mesh physical 节点材质。

parameters

配置参数。

属性

.anisotropyNode : Node.<float>

物理材质的各向异性默认从 anisotropy 属性推断。此节点属性允许覆盖默认值， 改用节点定义各向异性。

若不想覆盖各向异性，而只想修改现有值， 请使用 materialAnisotropy。

默认值为 null。

.attenuationColorNode : Node.<vec3>

物理材质的衰减颜色默认从 attenuationColor 属性推断。此节点属性允许覆盖默认值， 改用节点定义衰减颜色。

若不想覆盖衰减颜色，而只想修改现有值， 请使用 materialAttenuationColor。

默认值为 null。

.attenuationDistanceNode : Node.<float>

默认值为 null。
