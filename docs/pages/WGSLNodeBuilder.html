WGSLNodeBuilder

面向 WGSL 的节点构建器。

此模块从节点材质生成 WGSL 着色器代码，并生成相应的绑定与顶点缓冲区定义。这些数据随后由渲染器用于为渲染对象创建渲染与计算管线。

构造函数

new WGSLNodeBuilder( object : Object3D, renderer : Renderer )

构造一个新的 WGSL 节点构建器。

object

3D 对象。
renderer

渲染器。

属性

.allowEarlyReturns : boolean

指示是否允许提前返回的标志。

默认值为 true。

.allowGlobalVariables : boolean

指示是否允许全局变量的标志。

默认值为 true。

.builtins : Object.<string, Map.<string, Object>>

为每个着色器阶段保存 builtins Map 的字典。

.directives : Object.<string, Set.<string>>

为每个着色器阶段保存指令 Set 的字典。

.scopedArrays : Map.<string, Object>

用于管理作用域数组的映射。仅在计算着色器上下文中使用 WorkgroupInfoNode 时相关。

.uniformGroups : Object.<string, Object.<string, NodeUniformsGroup>>

为每个着色器阶段（'vertex'、'fragment'、'compute'）保存另一个字典的字典，该字典按组（'render'、'frame'、'object'）管理 UBO。

.uniformGroupsBindings : Object.<string, {index: number, id: number}>

为每个 uniform 组保存已分配绑定索引的字典。这确保所有着色器阶段使用相同的绑定索引。

name

builtin 名称。
shaderStage

为此代码片段生成的着色器阶段。

默认值为 this.shaderStage。
返回值： 给定 builtin 是否在给定着色器阶段中已定义。

.isAvailable( name : string ) : boolean

所请求的特性是否可用。

name

所请求的特性。
重写： NodeBuilder#isAvailable
返回值： 所请求的特性是否受支持。

.isFlipY() : boolean

是否沿垂直轴翻转纹理数据。

重写： NodeBuilder#isFlipY
返回值： 在 WGSL 上下文中始终返回 false。
