BoxLineGeometry

专为 LineSegments 设计的特殊长方体几何体。

代码示例

const geometry = new THREE.BoxLineGeometry();
const material = new THREE.LineBasicMaterial( { color: 0x00ff00 } );
const lines = new THREE.LineSegments( geometry, material );
scene.add( lines );

导入

BoxLineGeometry 是一个插件，必须显式导入，参见 Installation#Addons。

import { BoxLineGeometry } from 'three/addons/geometries/BoxLineGeometry.js';

构造函数

new BoxLineGeometry( width : number, height : number, depth : number, widthSegments : number, heightSegments : number, depthSegments : number )

构造一个新的长方体线框几何体。

width

宽度。即平行于 X 轴的边的长度。

默认值为 1。
height

高度。即平行于 Y 轴的边的长度。

默认值为 1。
depth

深度。即平行于 Z 轴的边的长度。

默认值为 1。
widthSegments

沿侧面宽度方向分段的矩形面数量。

默认值为 1。
heightSegments

沿侧面高度方向分段的矩形面数量。

默认值为 1。
depthSegments

沿侧面深度方向分段的矩形面数量。

默认值为 1。
