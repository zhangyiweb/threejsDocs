RectAreaLightTexturesLib

RectAreaLight 的纹理库。该类将 LTC BRDF 数据保存在数据纹理中，供渲染器后续使用。

参考：Eric Heitz、Jonathan Dupuy、Stephen Hill 和 David Neubelt 的 Real-Time Polygonal-Light Shading with Linearly Transformed Cosines。代码。

注意：这是基于 Eric Heitz 工作（见引用）的 BRDF 近似纹理数据的临时存放位置。RectAreaLight 的 BRDF 数据当前使用约 80kb 大小的预计算纹理进行近似。希望在将完整的 RectAreaLight 实现纳入主构建文件之前，能找到更好的方式来包含这些大型纹理数据。

导入

RectAreaLightTexturesLib 是一个插件，必须显式导入，参见 Installation#Addons。

import { RectAreaLightTexturesLib } from 'three/addons/lights/RectAreaLightTexturesLib.js';

属性

.LTC_FLOAT_1 : DataTexture

第一个 LTC FP32 数据纹理。

默认值为 null。

.LTC_FLOAT_2 : DataTexture

第二个 LTC FP32 数据纹理。

默认值为 null。

.LTC_HALF_1 : DataTexture

第一个 LTC FP16 数据纹理。

默认值为 null。

.LTC_HALF_2 : DataTexture

第二个 LTC FP16 数据纹理。

默认值为 null。

静态方法

.init() : RectAreaLightTexturesLib

初始化纹理库。
