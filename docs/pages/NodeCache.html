NodeCache

此工具类在 NodeBuilder 中用作节点数据的内部 缓存数据结构。

构造函数

new NodeCache( parent : NodeCache )

构造一个新的节点缓存。

parent

对父缓存的引用。

默认值为 null。

属性

.id : number (readonly)

缓存的 id。

.nodesData : WeakMap.<Node, Object>

用于管理节点数据的弱映射。

.parent : NodeCache

对父节点缓存的引用。

默认值为 null。

方法

.getData( node : Node ) : Object

返回给定节点的数据。

node

节点。
返回值： 该节点的数据。

.setData( node : Node, data : Object )

为给定节点设置数据。

node

节点。
data

应被缓存的数据。
