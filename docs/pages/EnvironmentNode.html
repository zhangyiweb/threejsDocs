EnvironmentNode

表示基于图像的光照（IBL）的物理模型。环境 通过等距柱状投影、立方体贴图或 cubeUV（PMREM）格式的环境贴图定义。 EnvironmentNode 面向如 MeshStandardNodeMaterial 这类 PBR 材质。

构造函数

new EnvironmentNode( envNode : Node )

构造一个新的环境节点。

envNode

表示环境的节点。

默认值为 null。

属性

.envNode : Node

表示环境的节点。

默认值为 null。
