UnpackFloatNode

此节点表示从 32 位无符号整数解包数值，并将结果重新解释为浮点向量的操作。

构造函数

new UnpackFloatNode( encoding : 'snorm' | 'unorm' | 'float16', uintNode : Node )

encoding

描述整数值如何映射到浮点范围的数值编码。
uintNode

待解包的 uint 节点。

属性

.encoding : string

数值编码。

.isUnpackFloatNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.uintNode : Node

待解包的无符号整数。
