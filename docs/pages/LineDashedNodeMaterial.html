LineDashedNodeMaterial

LineDashedMaterial 的节点材质版本。

构造函数

new LineDashedNodeMaterial( parameters : Object )

构造一个新的虚线节点材质。

parameters

配置参数。

属性

.dashOffset : number

虚线偏移量。

默认值为 0。

.dashScaleNode : Node.<float>

虚线材质的缩放默认从 scale 属性推断。此节点属性允许覆盖默认值， 改用节点定义缩放。

若不想覆盖缩放，而只想修改已有值， 请使用 materialLineScale。

默认值为 null。

.dashSizeNode : Node.<float>

虚线材质的虚线段长度默认从 dashSize 属性推断。此节点属性允许覆盖默认值， 改用节点定义虚线段长度。

若不想覆盖虚线段长度，而只想修改已有值， 请使用 materialLineDashSize。

默认值为 null。

.gapSizeNode : Node.<float>

虚线材质的间隙大小默认从 gapSize 属性推断。此节点属性允许覆盖默认值， 改用节点定义间隙大小。

若不想覆盖间隙大小，而只想修改已有值， 请使用 materialLineGapSize。

默认值为 null。

.isLineDashedNodeMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.offsetNode : Node.<float>

虚线材质的偏移默认从 dashOffset 属性推断。此节点属性允许覆盖默认值， 改用节点定义偏移。

若不想覆盖偏移，而只想修改已有值， 请使用 materialLineDashOffset。

默认值为 null。

方法

.setupVariants( builder : NodeBuilder )

设置虚线专用的节点变量。

builder

当前节点构建器。
