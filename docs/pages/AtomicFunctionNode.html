AtomicFunctionNode

AtomicFunctionNode 表示可在着色器中对原子变量类型进行操作的任意函数。 在原子函数中，对原子变量的任何修改都会作为不可分割的步骤发生， 并且相对于其他修改具有确定的顺序。因此，即使多个原子函数同时修改同一个原子变量， 原子操作也不会相互干扰。

此节点只能与 WebGPU 后端一起使用。

构造函数

new AtomicFunctionNode( method : string, pointerNode : Node, valueNode : Node )

构造一个新的原子函数节点。

method

要构造的原子函数的签名。
pointerNode

原子变量或原子缓冲区的元素。
valueNode

用于修改原子变量的值。

属性

.method : string

要构造的原子函数的签名。

.parents : boolean

创建此节点的父节点列表，用于检测该节点是否需要返回值。

默认值为 true。

重写： Node#parents

.pointerNode : Node

原子变量或原子缓冲区的元素。

.valueNode : Node

用于修改原子变量的值。

方法

.generateNodeType( builder : NodeBuilder ) : string

由于节点类型从输入类型推断得出，因此覆盖了此方法。

builder

当前的节点构建器。
重写： Node#generateNodeType
返回值： 节点类型。

.getInputType( builder : NodeBuilder ) : string

覆盖默认实现，返回指针节点的类型。

builder

当前的节点构建器。
返回值： 输入类型。
