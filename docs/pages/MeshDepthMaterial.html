MeshDepthMaterial

按深度绘制几何体的材质。深度基于相机的 近平面与远平面。白色最近，黑色最远。

构造函数

new MeshDepthMaterial( parameters : Object )

构造一个新的深度网格材质。

parameters

包含一个或多个属性的对象，用于定义材质外观。 材质的任意属性（包括继承材质中的任意属性）都可在此传入。 颜色值可以是 Color#set 所接受的任意类型。

属性

.alphaMap : Texture

alpha 贴图是控制表面不透明度的灰度纹理 （黑色：完全透明；白色：完全不透明）。

仅使用纹理的颜色，若存在 alpha 通道则忽略。 对于 RGB 和 RGBA 纹理，渲染器在采样此纹理时会使用绿色通道， 因为 DXT 压缩格式以及未压缩的 RGB 565 格式为绿色提供了额外精度。 仅亮度以及亮度/alpha 纹理仍可按预期工作。

alphaMap 表示非颜色数据。所指定的任意纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.depthPacking : BasicDepthPacking | RGBADepthPacking | RGBDepthPacking | RGDepthPacking

深度打包类型。

默认值为 BasicDepthPacking。

.displacementBias : number

位移贴图值在网格顶点上的偏移量。 偏置会加到位移贴图缩放后的采样值上。 未设置位移贴图时，此值不会应用。

默认值为 0。

.displacementMap : Texture

位移贴图会影响网格顶点的位置。与 仅影响材质光影的其他贴图不同， 被位移的顶点可以投射阴影、阻挡其他对象，并以其他方式 充当真实几何体。位移纹理是一幅图像，其中每个像素的值 （白色最高）会映射并重新定位网格的顶点。

displacementMap 表示非颜色数据。所指定的任意纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.displacementScale : number

位移贴图对网格的影响程度（黑色表示无位移， 白色表示最大位移）。未设置位移贴图时， 此值不会应用。

默认值为 0。

.isMeshDepthMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.map : Texture

颜色贴图。可选择包含 alpha 通道，通常与 Material#transparent 或 Material#alphaTest 结合使用。

map 表示颜色数据，纹理必须指定 Texture#colorSpace。大多数 map 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。

.wireframe : boolean

将几何体渲染为线框。

默认值为 false。

.wireframeLinewidth : number

控制线框粗细。

WebGL 和 WebGPU 会忽略此属性，并始终渲染 1 像素宽的线条。

默认值为 1。
