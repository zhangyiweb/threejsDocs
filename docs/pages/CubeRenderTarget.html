CubeRenderTarget

此类表示立方体渲染目标。它是 WebGLCubeRenderTarget 的特殊版本，可与 WebGPURenderer 兼容。

构造函数

new CubeRenderTarget( size : number, options : RenderTarget~Options )

构造一个新的立方体渲染目标。

size

渲染目标的尺寸。

默认值为 1。
options

配置对象。

属性

.isCubeRenderTarget : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.texture : DataArrayTexture

被重写为不同的纹理类型。

重写： RenderTarget#texture

方法

.clear( renderer : Renderer, color : boolean, depth : boolean, stencil : boolean )

清除此立方体渲染目标。

renderer

渲染器。
color

是否应清除颜色缓冲。

默认值为 true。
depth

是否应清除深度缓冲。

默认值为 true。
stencil

是否应清除模板缓冲。

默认值为 true。

.fromEquirectangularTexture( renderer : Renderer, texture : Texture ) : CubeRenderTarget

将给定的等距柱状投影纹理转换为立方体贴图。

renderer

渲染器。
texture

等距柱状投影纹理。
返回值： 对此立方体渲染目标的引用。
