AnimationMixer

AnimationMixer 是场景中特定对象的动画播放器。当场景中多个对象需要独立播放动画时， 可为每个对象分别使用一个 AnimationMixer。

构造函数

new AnimationMixer( root : Object3D )

构造一个新的动画混合器。

root

由该混合器播放动画的对象。

属性

.time : number

混合器的全局时间（秒；混合器创建时从 0 开始）。

默认值为 0。

.timeScale : number

全局时间的缩放因子。

注意：将此成员设为 0 后再设回 1，可用于暂停/恢复该混合器控制的所有动作。

默认值为 1。

方法

.clipAction( clip : AnimationClip | string, optionalRoot : Object3D, blendMode : NormalAnimationBlendMode | AdditiveAnimationBlendMode ) : AnimationAction

返回传入片段对应的 AnimationAction 实例。

若尚不存在匹配片段和根对象参数的动作，此方法会创建它。使用相同的片段和根对象参数多次调用此方法，始终返回同一个动作。

clip

动画片段，或动画片段的名称。
optionalRoot

可选的替代根对象。
blendMode

混合模式。
返回值： 动画动作。

.existingAction( clip : AnimationClip | string, optionalRoot : Object3D ) : AnimationAction

返回传入片段对应的已有动画动作。

clip

动画片段，或动画片段的名称。
optionalRoot

可选的替代根对象。
返回值： 动画动作。若未找到则返回 null。

.getRoot() : Object3D

返回该混合器的根对象。

返回值： 混合器的根对象。

.setTime( time : number ) : AnimationMixer

将混合器的全局时间设置到指定时刻，并相应更新动画。

当需要跳转到动画的精确时刻时很有用。输入参数会乘以 AnimationMixer#timeScale。

time

要设置的时间（秒）。
返回值： 对该动画混合器的引用。

.stopAllAction() : AnimationMixer

停用该混合器上所有已调度的动作。

返回值： 对该动画混合器的引用。

.uncacheAction( clip : AnimationClip | string, optionalRoot : Object3D )

释放某个动作占用的所有内存资源。动作由给定片段和可选根对象标识。使用此方法前， 请确保调用 AnimationAction#stop 停用该动作。

clip

动画片段，或动画片段的名称。
optionalRoot

可选的替代根对象。

.uncacheClip( clip : AnimationClip )

释放某个片段占用的所有内存资源。使用此方法前， 请确保对所有相关动作调用 AnimationAction#stop。

clip

要清除缓存的片段。

.uncacheRoot( root : Object3D )

释放某个根对象占用的所有内存资源。使用此方法前， 请确保对所有相关动作调用 AnimationAction#stop， 或在混合器仅操作单个根对象时调用 AnimationMixer#stopAllAction。

root

要清除缓存的根对象。

.update( deltaTime : number ) : AnimationMixer

推进混合器的全局时间并更新动画。

通常在渲染循环中完成，传入来自 Clock 或 Timer 的时间增量。

deltaTime

时间增量（秒）。
返回值： 对该动画混合器的引用。
