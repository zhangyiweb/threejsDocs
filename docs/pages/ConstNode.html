ConstNode

用于在着色器中表示常量值的类。

构造函数

new ConstNode( value : any, nodeType : string )

构造一个新的输入节点。

value

此节点的值。通常为 JS 原始类型或 three.js 对象（向量、矩阵、颜色）。
nodeType

节点类型。若未显式定义类型，节点会尝试从其值推导类型。

默认值为 null。

属性

.isConstNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.generateConst( builder : NodeBuilder ) : string

使用当前节点构建器生成该值的着色器字符串。

builder

当前的节点构建器。
返回值： 生成的值作为着色器字符串。
