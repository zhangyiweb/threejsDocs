NURBSVolume

此类表示一个 NURBS 体积。

实现基于带权重 w=weight 的控制点 (x, y [, z=0 [, w=1]])。

导入

NURBSVolume 是一个插件，必须显式导入，参见 Installation#Addons。

import { NURBSVolume } from 'three/addons/curves/NURBSVolume.js';

构造函数

new NURBSVolume( degree1 : number, degree2 : number, degree3 : number, knots1 : Array.<number>, knots2 : Array.<number>, knots3 : Array.<number>, controlPoints : Array.<Array.<Array.<(Vector2|Vector3|Vector4)>>> )

构造一个新的 NURBS 体积。

degree1

第一维 NURBS 阶数。
degree2

第二维 NURBS 阶数。
degree3

第三维 NURBS 阶数。
knots1

第一维节点，以数字平铺数组表示。
knots2

第二维节点，以数字平铺数组表示。
knots3

第三维节点，以数字平铺数组表示。
controlPoints

保存控制点的三维数组。

方法

.getPoint( t1 : number, t2 : number, t3 : number, target : Vector3 )

此方法根据给定的插值因子返回三维空间中的向量。该向量位于 NURBS 体积内。

t1

表示体积内 u 位置的第一插值因子，必须在 [0,1] 范围内。
t2

表示体积内 v 位置的第二插值因子，必须在 [0,1] 范围内。
t3

表示体积内 w 位置的第三插值因子，必须在 [0,1] 范围内。
target

结果写入的目标向量。
