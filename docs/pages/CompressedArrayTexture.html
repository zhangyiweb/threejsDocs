CompressedArrayTexture

基于压缩形式的数据创建二维纹理数组。

此类纹理通常通过 CompressedTextureLoader 加载。

构造函数

new CompressedArrayTexture( mipmaps : Array.<Object>, width : number, height : number, depth : number, format : number, type : number )

构造一个新的压缩数组纹理。

mipmaps

该数组保存所有 mipmap（包括基础 mip） 的数据与尺寸。
width

纹理的宽度。
height

纹理的高度。
depth

纹理的深度。
format

纹理格式。

默认值为 RGBAFormat。
type

纹理类型。

默认值为 UnsignedByteType。

属性

.image : Object

压缩纹理的 image 属性仅定义其尺寸。

重写： CompressedTexture#image

.isCompressedArrayTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.layerUpdates : Set.<number>

需要在纹理中更新的所有层的集合。

.wrapR : RepeatWrapping | ClampToEdgeWrapping | MirroredRepeatWrapping

定义纹理在深度方向上的包裹方式，对应 UVW 映射中的 W。

默认值为 ClampToEdgeWrapping。

方法

.addLayerUpdate( layerIndex : number )

标记纹理的特定层需要更新。 通常当 Texture#needsUpdate 设为 true 时， 整个压缩纹理数组都会发送到 GPU。标记特定层 只会传输与数组中特定深度相关的 mipmap 子集， 这通常性能更好。

layerIndex

应当更新的层索引。

.clearLayerUpdates()

重置层更新注册表。
