VRMLLoader

用于加载 VRML 格式的加载器。

代码示例

const loader = new VRMLLoader();
const object = await loader.loadAsync( 'models/vrml/house.wrl' );
scene.add( object );

导入

VRMLLoader 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { VRMLLoader } from 'three/addons/loaders/VRMLLoader.js';

构造函数

new VRMLLoader( manager : LoadingManager )

构造一个新的 VRML 加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 VRML 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( data : string, path : string ) : Scene

解析给定的 VRML 数据并返回结果场景。

data

原始 VRML 数据，以字符串形式提供。
path

URL 基路径。
重写： Loader#parse
返回值： 解析后的场景。
