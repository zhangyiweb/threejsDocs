TeapotGeometry

将 Martin Newell 著名的犹他茶壶数据细分（tessellate）为三角形。

茶壶通常应作为双面对象渲染，因为某些曲面片的两侧都可见，例如盖子周围的缝隙以及壶嘴内部。

分段数 n 决定输出的三角形数量。总三角形数 = 322nn - 8n（顶部和底部尖点处的退化三角形已删除）。

代码基于 SPD software，为 Udacity 课程 Interactive Rendering 创建。

代码示例

const geometry = new TeapotGeometry( 50, 18 );
const material = new THREE.MeshBasicMaterial( { color: 0x00ff00 } );
const teapot = new THREE.Mesh( geometry, material );
scene.add( teapot );

导入

TeapotGeometry 是一个插件，必须显式导入，参见 Installation#Addons。

import { TeapotGeometry } from 'three/addons/geometries/TeapotGeometry.js';

构造函数

new TeapotGeometry( size : number, segments : number, bottom : boolean, lid : boolean, body : boolean, fitLid : boolean, blinn : boolean )

构造一个新的茶壶几何体。

size

茶壶的相对缩放。

默认值为 50。
segments

细分每个曲面片边的线段数量。

默认值为 10。
bottom

是否生成茶壶底部。

默认值为 true。
lid

是否生成盖子。

默认值为 true。
body

是否生成壶身。

默认值为 true。
fitLid

是否略微拉伸盖子以防止壶身与盖子之间出现缝隙。

默认值为 true。
blinn

是否对茶壶进行垂直缩放以获得更好的视觉效果。

默认值为 true。
