EventDispatcher → Node → TempNode →

PMREMNode

此节点表示 PMREM，这是一种专为 PBR 材质设计的特殊预处理环境贴图。

代码示例

const material = new MeshStandardNodeMaterial();
material.envNode = pmremTexture( envMap );

构造函数

new PMREMNode( value : Texture, uvNode : Node.<vec2>, levelNode : Node.<float> )

构造一个新的函数重载节点。

value

输入纹理。
uvNode

UV 节点。

默认值为 null。
levelNode

层级节点。

默认值为 null。

属性

.levelNode : Node.<float>

层级节点。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.RENDER。

默认值为 'render'。

重写： TempNode#updateBeforeType

.uvNode : Node.<vec2>

UV 节点。

.value : Texture

节点的纹理值。

方法

.updateFromTexture( texture : Texture )

使用给定的 PMREM 纹理更新内部值。

texture

PMREM 纹理。
