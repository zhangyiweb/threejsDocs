WebXRManager

此类是对 WebXR Device API 的抽象，由 WebGLRenderer 内部使用。WebXRManager 还提供了公共 接口，允许用户启用/禁用 XR，并执行与 XR 相关的 任务，例如获取控制器。

属性

.cameraAutoUpdate : boolean

管理器的 XR 相机是否应自动更新。

默认值为 true。

.enabled : boolean

此标志通知渲染器已准备好进行 XR 渲染。若你的应用 将使用 XR，请将其设为 true。

默认值为 false。

.isPresenting : boolean (readonly)

XR 呈现是否处于活动状态。

默认值为 false。

方法

.getBaseLayer() : XRWebGLLayer | XRProjectionLayer

返回当前的基础层。

当目标 XR 设备支持 WebXR Layers API 时，这是一个 XRProjectionLayer； 否则为 XRWebGLLayer。

返回值： XR 基础层。

.getBinding() : XRWebGLBinding

返回当前的 XR 绑定。

若需要且浏览器支持，则会创建新的绑定。

返回值： XR 绑定。若无法创建则返回 null。

.getCamera() : ArrayCamera

返回一个 ArrayCamera 实例，表示活动 XR 会话的 XR 相机。 它对每个视图持有一个单独的相机对象。

该相机的 fov 当前未被使用，也不反映 XR 相机的视场角。 若需要在应用层使用视场角，必须从 XR 相机的投影矩阵 手动计算。

返回值： XR 相机。

.getCameraTexture( xrCamera : XRCamera ) : Texture

从与视图对齐的 XRCamera 获取不透明纹理。 仅在当前动画循环期间可用。

xrCamera

要查询的相机。
返回值： 表示当前原始相机帧的不透明纹理。

.getController( index : number ) : Group

返回表示 XR 控制器 target ray 空间的组。 使用此空间可视化可帮助用户完成指向任务 （如 UI 交互）的 3D 对象。

index

控制器的索引。
返回值： 表示 target ray 空间的组。

.getControllerGrip( index : number ) : Group

返回表示 XR 控制器 grip 空间的组。 使用此空间可视化可帮助用户完成指向任务 （如 UI 交互）的 3D 对象。

注意：若既要在用户手中显示物体，又要同时提供 指向射线，应将手持物体附加到 getControllerGrip() 返回的组，并将射线附加到 getController() 返回的组。其思路是为同一个 WebXR 控制器在两个不同的坐标空间中使用两个不同的组。

index

控制器的索引。
返回值： 表示 grip 空间的组。

.getDepthSensingMesh() : Mesh

返回深度感知网格。

参见 WebXRDepthSensing#getMesh。

返回值： 深度感知网格。

.getDepthTexture() : Texture

返回通过深度感知计算得到的当前深度纹理。

参见 WebXRDepthSensing#getDepthTexture。

返回值： 深度纹理。

.getEnvironmentBlendMode() : 'opaque' | 'additive' | 'alpha-blend' | undefined

返回当前 XR 会话的环境混合模式。

返回值： 环境混合模式。在 XR 会话之外使用时返回 undefined。

.getFoveation() : number | undefined

返回 XR 合成器对投影层使用的注视点渲染（foveation）量。

返回值： 注视点渲染量。

.getFrame() : XRFrame

返回当前的 XR 帧。

返回值： XR 帧。在 XR 会话之外使用时返回 null。

.getHand( index : number ) : Group

返回表示 XR 控制器 hand 空间的组。 使用此空间可视化可帮助用户完成指向任务 （如 UI 交互）的 3D 对象。

index

控制器的索引。
返回值： 表示 hand 空间的组。

.getReferenceSpace() : XRReferenceSpace

返回 XR 参考空间。

返回值： XR 参考空间。

.getSession() : XRSession

返回当前的 XR 会话。

返回值： XR 会话。在 XR 会话之外使用时返回 null。

.hasDepthSensing() : boolean

若支持深度感知则返回 true。

返回值： 是否支持深度感知。

.setFoveation( value : number )

设置注视点渲染（foveation）值。

value

范围为 [0,1] 的数值，其中 0 表示无注视点渲染（全分辨率）， 1 表示最大注视点渲染（边缘以较低分辨率渲染）。

.setFramebufferScaleFactor( value : number )

设置帧缓冲缩放因子。

此方法不能在 XR 会话期间使用。

value

帧缓冲缩放因子。

.setReferenceSpace( space : XRReferenceSpace )

设置自定义 XR 参考空间。

space

XR 参考空间。

.setReferenceSpaceType( value : string )

设置参考空间类型。可用于配置与用户物理环境的空间关系。 根据用户在 3D 空间中的移动方式，设置合适的参考空间可改善追踪。 默认值为 local-floor。有效值可参见 https://developer.mozilla.org/en-US/docs/Web/API/XRReferenceSpace#reference_space_types。

此方法不能在 XR 会话期间使用。

value

参考空间类型。

.setSession( value : XRSession ) : Promise (async)

在通常通过某个 *Button 模块请求 XR 会话后， 使用此方法将其注入渲染器。此方法会触发实际 XR 渲染的开始。

value

要设置的 XR 会话。
返回值： 在会话设置完成后 resolve 的 Promise。

.updateCamera( camera : Camera )

更新 XR 相机的状态。若在应用层将 cameraAutoUpdate 设为 false，请使用此方法。该方法需要将场景的 非 XR 相机作为参数。调用此方法时，传入相机的变换 会自动调整为 XR 相机的位置。

camera

相机。
