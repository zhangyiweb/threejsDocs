Int32BufferAttribute

便捷类，用于使用普通 Array 实例创建 Int32 缓冲属性。

构造函数

new Int32BufferAttribute( array : Array.<number> | Int32Array, itemSize : number, normalized : boolean )

构造一个新的缓冲属性。

array

保存属性数据的数组。
itemSize

每个顶点的分量数量。
normalized

数据是否已归一化。

默认值为 false。
