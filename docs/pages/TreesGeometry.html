TreesGeometry

程序化树木几何体。

导入

TreesGeometry 是一个插件，必须显式导入，参见 Installation#Addons。

import { TreesGeometry } from 'three/addons/misc/RollerCoaster.js';

构造函数

new TreesGeometry( landscape : Mesh )

构造一个新的几何体。

landscape

表示地形的 mesh。树木将随机分布在地形表面上。
