BitcountNode

new BitcountNode( method : 'countTrailingZeros' | 'countLeadingZeros' | 'countOneBits', aNode : Node )

构造一个新的数学节点。

method

方法名称。
aNode

第一个输入。

属性

.isBitcountNode : boolean (readonly)

该标志可用于类型检测。

默认值为 true。
