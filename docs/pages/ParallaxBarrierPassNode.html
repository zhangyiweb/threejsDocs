ParallaxBarrierPassNode

创建视差屏障效果的渲染通道节点。

导入

ParallaxBarrierPassNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { parallaxBarrierPass } from 'three/addons/tsl/display/ParallaxBarrierPassNode.js';

构造函数

new ParallaxBarrierPassNode( scene : Scene, camera : Camera )

构造一个新的视差屏障通道节点。

scene

要渲染的场景。
camera

用于渲染场景的相机。

属性

.isParallaxBarrierPassNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.setup( builder : NodeBuilder ) : PassTextureNode

此方法用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： StereoCompositePassNode#setup
