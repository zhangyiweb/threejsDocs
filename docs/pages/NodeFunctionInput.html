NodeFunctionInput

描述 NodeFunction 的输入。

构造函数

new NodeFunctionInput( type : string, name : string, count : number, qualifier : 'in' | 'out' | 'inout', isConst : boolean )

构造一个新的节点函数输入。

type

输入类型。
name

输入名称。
count

若输入为 Array，count 表示其长度。

默认值为 null。
qualifier

参数限定符（仅与 GLSL 相关）。

默认值为 ''。
isConst

输入是否使用 const 限定符（仅与 GLSL 相关）。

默认值为 false。

属性

.count : number

若输入为 Array，count 表示其长度。

默认值为 null。

.isConst : boolean

输入是否使用 const 限定符（仅与 GLSL 相关）。

默认值为 false。

.name : string

输入名称。

.qualifier : 'in' | 'out' | 'inout'

参数限定符（仅与 GLSL 相关）。

默认值为 ''。

.type : string

输入类型。
