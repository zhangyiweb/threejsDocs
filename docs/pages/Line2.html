Line2

在顶点之间绘制的折线。

相较于 Line，它提供了更多功能，例如任意线宽，以及将线宽改为世界单位。 它扩展自 LineSegments2，简化了由一连串点构造线段的过程。

此模块只能与 WebGLRenderer 一起使用。使用 WebGPURenderer 时， 请从 lines/webgpu/Line2.js 导入该类。

代码示例

const geometry = new LineGeometry();
geometry.setPositions( positions );
geometry.setColors( colors );
const material = new LineMaterial( { linewidth: 5, vertexColors: true } );
const line = new Line2( geometry, material );
scene.add( line );

导入

Line2 是一个插件，必须显式导入，参见 Installation#Addons。

import { Line2 } from 'three/addons/lines/Line2.js';

构造函数

new Line2( geometry : LineGeometry, material : LineMaterial )

构造一条新的宽线。

geometry

线几何体。
material

线材质。

属性

.isLine2 : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
