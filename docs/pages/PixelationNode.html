PixelationNode

实现实际像素化 TSL 代码的内部节点定义。

构造函数

new PixelationNode( textureNode : TextureNode, depthNode : TextureNode, normalNode : TextureNode, pixelSize : Node.<float>, normalEdgeStrength : Node.<float>, depthEdgeStrength : Node.<float> )

构造一个新的像素化节点。

textureNode

表示 beauty pass 的纹理节点。
depthNode

表示 beauty pass 深度的纹理。
normalNode

表示 beauty pass 法线的纹理。
pixelSize

像素大小。
normalEdgeStrength

法线边缘强度。
depthEdgeStrength

深度边缘强度。

属性

.depthEdgeStrength : Node.<float>

深度边缘强度。

.depthNode : TextureNode

表示 beauty pass 深度的纹理。

.normalEdgeStrength : Node.<float>

像素大小。

.normalNode : TextureNode

表示 beauty pass 法线的纹理。

.pixelSize : Node.<float>

像素大小。

.textureNode : TextureNode

表示 beauty pass 的纹理节点。

.updateType : string

updateType 设置为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧更新一次内部 uniform。

默认值为 'frame'。

重写： TempNode#updateType

方法

.setup( builder : NodeBuilder ) : ShaderCallNodeInternal

此方法用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.update( frame : NodeFrame )

此方法用于每帧更新 uniform。

frame

当前的节点帧。
重写： TempNode#update
