ProgressiveLightMap

渐进式光照贴图累加器，作者：zalo。

使用方法：构造 ProgressiveLightMap 对象， 一次性调用 plmap.addObjectsToLightMap(object) 将半静态对象和灯光数组添加到该类， 然后每帧调用 plmap.update(camera) 开始累积光照采样。

这将开始累积应用于对象的 lightmap， 你可以开始抖动光照以实现所需的纹理空间效果。

此类只能与 WebGLRenderer 一起使用。 使用 WebGPURenderer 时，请从 ProgressiveLightMapGPU.js 导入。

导入

ProgressiveLightMap 是一个插件，必须显式导入，参见 Installation#Addons。

import { ProgressiveLightMap } from 'three/addons/misc/ProgressiveLightMap.js';

构造函数

new ProgressiveLightMap( renderer : WebGLRenderer, res : number )

构造一个新的渐进式光照贴图。

renderer

渲染器。
res

光照贴图总尺寸的边长。

默认值为 1024。

属性

.renderer : WebGLRenderer

渲染器。

.res : number

光照贴图总尺寸的边长。

默认值为 1024。

方法

.addObjectsToLightMap( objects : Array.<Object3D> )

设置这些对象的 material 的 lightmap，并修改其 uv1。

objects

用于设置光照贴图的对象和灯光数组。

.dispose()

释放所有内部资源。

.showDebugLightmap( visible : boolean, position : Vector3 )

在主场景中绘制光照贴图。添加对象后调用此方法。

visible

调试平面是否可见。
position

调试平面的绘制位置。

.update( camera : Camera, blendWindow : number, blurEdges : boolean )

此函数将每个网格逐一渲染到各自的表面贴图中。

camera

用于渲染场景的相机。
blendWindow

当 >1 时，采样将随时间累积。

默认值为 100。
blurEdges

是否通过模糊修复 UV 边缘。

默认值为 true。
