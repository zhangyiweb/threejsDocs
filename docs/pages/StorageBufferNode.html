StorageBufferNode

此节点用于计算着色器场景，用于定义数据的 存储缓冲区。典型工作流程是通过便捷函数 attributeArray() 或 instancedArray() 创建该节点的实例，设置向缓冲区写入数据的计算着色器， 然后将存储缓冲区转换为属性节点以进行渲染。

代码示例

const positionBuffer = instancedArray( particleCount, 'vec3' ); // the storage buffer node
const computeInit = Fn( () => { // the compute shader
	const position = positionBuffer.element( instanceIndex );
	// compute position data
	position.x = 1;
	position.y = 1;
	position.z = 1;
} )().compute( particleCount );
const particleMaterial = new THREE.SpriteNodeMaterial();
particleMaterial.positionNode = positionBuffer.toAttribute();
renderer.computeAsync( computeInit );

构造函数

new StorageBufferNode( value : StorageBufferAttribute | StorageInstancedBufferAttribute | BufferAttribute, bufferType : string | Struct, bufferCount : number )

构造一个新的存储缓冲区节点。

value

缓冲区数据。
bufferType

缓冲区类型（例如 'vec3'）。

默认值为 null。
bufferCount

缓冲区计数。

默认值为 0。

属性

.access : string

纹理节点的访问类型。

默认值为 'readWrite'。

.global : boolean

StorageBufferNode 默认将此属性设置为 true。

默认值为 true。

重写： BufferNode#global

.isAtomic : boolean

该节点是否为原子节点。

默认值为 false。

.isPBO : boolean

该节点是否表示 PBO。 仅与 WebGL 相关。

默认值为 false。

.isStorageBufferNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.structTypeNode : StructTypeNode

缓冲区结构体类型。

默认值为 null。
