UltraHDRLoader

用于加载 Ultra HDR 图像格式的加载器。

现有的 HDR 或 EXR 纹理可通过此工具转换为 Ultra HDR。

当前功能集：

  • JPEG 头部（必需）
  • XMP 元数据（旧格式，已支持）
  • ISO 21496-1 元数据（当前标准，已支持）
  • XMP 验证（未实现）
  • EXIF 配置文件（未实现）
  • ICC 配置文件（未实现）
  • SDR 与 HDR 图像的二进制存储（必需）
  • Gainmap 元数据（必需）
  • 非 JPEG 图像格式（未实现）
  • 主图像为 HDR 图像（未实现）

代码示例

const loader = new UltraHDRLoader();
const texture = await loader.loadAsync( 'textures/equirectangular/ice_planet_close.jpg' );
texture.mapping = THREE.EquirectangularReflectionMapping;
scene.background = texture;
scene.environment = texture;

导入

UltraHDRLoader 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { UltraHDRLoader } from 'three/addons/loaders/UltraHDRLoader.js';

构造函数

new UltraHDRLoader( manager : LoadingManager )

构造一个新的 Ultra HDR 加载器。

manager

加载管理器。

属性

.type : HalfFloatType | FloatType

纹理类型。

默认值为 HalfFloatType。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback ) : DataTexture

从指定 URL 开始加载，并将加载完成的 Ultra HDR 纹理 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load
返回值： Ultra HDR 纹理。

.parse( buffer : ArrayBuffer, onLoad : function )

解析给定的 Ultra HDR 纹理数据。

buffer

原始纹理数据。
onLoad

onLoad 回调。
重写： Loader#parse

.setDataType( value : HalfFloatType | FloatType ) : UltraHDRLoader

设置纹理类型。

value

要设置的纹理类型。
返回值： 此加载器的引用。
