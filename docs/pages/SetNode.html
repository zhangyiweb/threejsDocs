SetNode

此模块属于 TSL 核心，通常不会在应用层代码中直接使用。 SetNode 表示一次 set 操作，用于实现节点对象上的任意 setXYZW()、setRGBA() 和 setSTPQ() 方法调用。 例如：

代码示例

materialLine.colorNode = color( 0, 0, 0 ).setR( float( 1 ) );

构造函数

new SetNode( sourceNode : Node, components : string, targetNode : Node )

构造一个新的 set 节点。

sourceNode

应被更新的节点。
components

应被更新的分量。
targetNode

值节点。

属性

.components : string

应被更新的分量。

.sourceNode : Node

应被更新的节点。

.targetNode : Node

值节点。

方法

.generateNodeType( builder : NodeBuilder ) : string

该方法被重写，因为节点类型是从 SetNode#sourceNode 推断的。

builder

当前的节点构建器。
重写： TempNode#generateNodeType
返回值： 节点类型。
