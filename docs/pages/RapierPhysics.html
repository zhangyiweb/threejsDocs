RapierPhysics

可将 Rapier 作为物理引擎集成到 three.js 应用中。可通过以下方式初始化 API：

该组件会自动从 CDN 导入 Rapier，因此使用时需要保持网络连接。

代码示例

const physics = await RapierPhysics();

导入

RapierPhysics 是一个插件，必须显式导入，参见 Installation#Addons。

import { RapierPhysics } from 'three/addons/physics/RapierPhysics.js';

方法

.addHeightfield( mesh : Mesh, width : number, depth : number, heights : Float32Array, scale : Object ) : RigidBody

向物理模拟添加高度场地形。

mesh

表示地形的 Three.js 网格。
width

高度场沿宽度（x 轴）的顶点数。
depth

高度场沿深度（z 轴）的顶点数。
heights

高度场中每个顶点的高度值数组。
scale

高度场各维度的缩放因子。

x

宽度缩放因子。
y

高度缩放因子。
z

深度缩放因子。
返回值： 为该高度场创建的 Rapier 刚体。

.addMesh( mesh : Mesh, mass : number, restitution : number )

将给定网格添加到物理模拟中。

mesh

要添加的网格。
mass

网格的质量（千克）。

默认值为 0。
restitution

网格的恢复系数，通常取值范围为 0 到 1。表示物体碰撞时的弹性程度。

默认值为 0。

.addScene( scene : Object3D )

将给定场景添加到物理模拟中。仅处理在 Object3D#userData 字段中包含 physics 对象的网格。 该对象可用于存储网格的质量和恢复系数，例如：

box.userData.physics = { mass: 1, restitution: 0 };
scene

要添加的场景或任意 3D 对象。

.applyImpulse( mesh : Mesh, impulse : Vector3, index : number )

对参与物理模拟的给定网格施加冲量。

mesh

要施加冲量的网格。
impulse

要施加的冲量。
index

若网格为实例化网格，索引表示实例 ID。

默认值为 0。

.removeMesh( mesh : Mesh )

从物理模拟中移除给定网格。

mesh

要移除的网格。

.setMeshPosition( mesh : Mesh, position : Vector3, index : number )

设置参与物理模拟的给定网格的位置。调用此方法会重置该网格当前模拟的速度。

mesh

要更新位置的网格。
position

新位置。
index

若网格为实例化网格，索引表示实例 ID。

默认值为 0。

.setMeshVelocity( mesh : Mesh, velocity : Vector3, index : number )

设置参与物理模拟的给定网格的速度。

mesh

要更新速度的网格。
velocity

新速度。
index

若网格为实例化网格，索引表示实例 ID。

默认值为 0。
