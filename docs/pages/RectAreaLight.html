RectAreaLight

该类在矩形平面表面上均匀发光。 此光源类型可用于模拟明亮的窗户或灯带等光源。

重要说明：

  • 不支持阴影。
  • 仅支持 PBR 材质。
  • 必须在应用中引入 RectAreaLightUniformsLib（WebGLRenderer）或 RectAreaLightTexturesLib（WebGPURenderer） 并初始化相应的 uniform / 纹理。

代码示例

RectAreaLightUniformsLib.init(); // only relevant for WebGLRenderer
THREE.RectAreaLightNode.setLTC( RectAreaLightTexturesLib.init() ); //  only relevant for WebGPURenderer
const intensity = 1; const width = 10; const height = 10;
const rectLight = new THREE.RectAreaLight( 0xffffff, intensity, width, height );
rectLight.position.set( 5, 5, 0 );
rectLight.lookAt( 0, 0, 0 );
scene.add( rectLight )

构造函数

new RectAreaLight( color : number | Color | string, intensity : number, width : number, height : number )

构造一个新的矩形区域光。

color

光源颜色。

默认值为 0xffffff。
intensity

光源强度。

默认值为 1。
width

灯光宽度。

默认值为 10。
height

灯光高度。

默认值为 10。

属性

.height : number

灯光高度。

默认值为 10。

.isRectAreaLight : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.power : number

灯光功率。功率是以流明 (lm) 为单位的光通量。 更改功率也会更改灯光强度。

.width : number

灯光宽度。

默认值为 10。
