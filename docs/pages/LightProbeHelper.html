LightProbeHelper

渲染一个球体以可视化场景中的光探针。

此辅助对象只能与 WebGLRenderer 一起使用。 使用 WebGPURenderer 时，请从 LightProbeHelperGPU.js 导入。

代码示例

const helper = new LightProbeHelper( lightProbe );
scene.add( helper );

导入

LightProbeHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { LightProbeHelper } from 'three/addons/helpers/LightProbeHelper.js';

构造函数

new LightProbeHelper( lightProbe : LightProbe, size : number )

构造一个新的光探针辅助对象。

lightProbe

要可视化的光探针。
size

辅助对象的大小。

默认值为 1。

属性

.lightProbe : LightProbe

要可视化的光探针。

.size : number

辅助对象的大小。

默认值为 1。

方法

.dispose()

释放此实例分配的 GPU 相关资源。 当应用中不再使用此实例时，请调用此方法。
