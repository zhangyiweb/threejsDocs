AnimationPathHelper

根据 AnimationClip 中的位置关键帧，可视化动画对象的运动路径。

代码示例

const clip = model.animations[ 0 ];
const helper = new AnimationPathHelper( model, clip, object );
scene.add( helper );

导入

AnimationPathHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { AnimationPathHelper } from 'three/addons/helpers/AnimationPathHelper.js';

构造函数

new AnimationPathHelper( root : Object3D, clip : AnimationClip, object : Object3D, options : Object )

构造一个新的动画路径辅助对象。

root

包含动画剪辑的根对象。
clip

包含位置关键帧的动画剪辑。
object

要显示路径的特定对象。
options

配置选项。

默认值为 {}。

color

路径线条颜色。

默认值为 0x00ff00。
markerColor

关键帧标记颜色。

默认值为 0xff0000。
divisions

用于平滑路径插值的采样数量。

默认值为 100。
showMarkers

是否在关键帧位置显示标记。

默认值为 true。
markerSize

关键帧标记的大小（像素）。

默认值为 5。

属性

.clip : AnimationClip

包含位置关键帧的动画剪辑。

.divisions : number

用于平滑路径插值的采样数量。

默认值为 100。

.isAnimationPathHelper : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.line : Line

表示动画路径的线条。

.object : Object3D

正在可视化其路径的对象。

.points : Points | null

标记关键帧位置的点。

.root : Object3D

包含动画剪辑的根对象。

方法

.dispose()

释放此实例分配的 GPU 相关资源。

.setColor( color : number | Color | string )

设置路径线条颜色。

color

新的颜色。

.setMarkerColor( color : number | Color | string )

设置关键帧标记颜色。

color

新的颜色。

.updateMatrixWorld( force : boolean )

更新辅助对象的变换，使其与对象的父级匹配。

force

强制更新矩阵。
重写： Object3D#updateMatrixWorld
