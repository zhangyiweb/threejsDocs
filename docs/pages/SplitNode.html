SplitNode

此模块属于 TSL 核心，通常不会在应用层代码中直接使用。 SplitNode 表示一次属性访问操作，用于实现节点对象上的任意 .xyzw、.rgba 和 stpq 用法。 例如：

代码示例

const redValue = color.r;

构造函数

new SplitNode( node : Node, components : string )

构造一个新的 split 节点。

node

应被访问的节点。
components

应被访问的分量。

默认值为 'x'。

属性

.components : string

应被访问的分量。

.isSplitNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.node : Node

应被访问的节点。

方法

.generateNodeType( builder : NodeBuilder ) : string

该方法被重写，因为节点类型是从所请求的分量推断的。

builder

当前的节点构建器。
重写： Node#generateNodeType
返回值： 节点类型。

.getComponentType( builder : NodeBuilder ) : string

返回该节点类型的分量类型。

builder

当前的节点构建器。
返回值： 分量类型。

.getScope() : Node

返回该节点的作用域。

重写： Node#getScope
返回值： 该节点的作用域。

.getVectorLength() : number

返回向量长度，该长度根据所请求的分量计算得出。

返回值： 向量长度。
