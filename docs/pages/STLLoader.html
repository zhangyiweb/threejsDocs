STLLoader

用于加载 STL 格式的加载器，该格式由 Solidworks 等 CAD 程序创建。

支持二进制和 ASCII 编码文件。加载器返回非索引 buffer geometry。

限制：

  • 二进制解码支持 "Magics" 颜色格式（http://en.wikipedia.org/wiki/STL_(file_format)#Color_in_binary_STL）。
  • 始终假设小端字节序是否合理，或许存在疑问。
  • ASCII 解码假定文件为 UTF-8 编码。
const loader = new STLLoader();
const geometry = await loader.loadAsync( './models/stl/slotted_disk.stl' )
scene.add( new THREE.Mesh( geometry ) );

对于二进制 STL，几何体可能包含顶点颜色。使用方法：

// use the same code to load STL as above
if ( geometry.hasColors ) {
	material = new THREE.MeshPhongMaterial( { opacity: geometry.alpha, vertexColors: true } );
}
const mesh = new THREE.Mesh( geometry, material );

对于包含多个实体的 ASCII STL，每个实体被分配到不同的组。 可通过定义与 geometry.groups 长度相同的材质数组， 并将其传递给 Mesh 构造函数，为各组指定不同颜色：

const materials = [];
const nGeometryGroups = geometry.groups.length;
for ( let i = 0; i < nGeometryGroups; i ++ ) {
	const material = new THREE.MeshPhongMaterial( { color: colorMap[ i ], wireframe: false } );
	materials.push( material );
}
const mesh = new THREE.Mesh(geometry, materials);

导入

STLLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { STLLoader } from 'three/addons/loaders/STLLoader.js';

构造函数

new STLLoader( manager : LoadingManager )

构造一个新的 STL 加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 STL 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( data : ArrayBuffer ) : BufferGeometry

解析给定的 STL 数据并返回生成的几何体。

data

原始 STL 数据 ArrayBuffer。
重写： Loader#parse
返回值： 解析后的几何体。
