Quaternion

用于表示四元数的类。three.js 使用四元数表示旋转。

遍历向量实例将按相应顺序得到其分量 (x, y, z, w)。

注意：three.js 期望四元数已归一化。

代码示例

const quaternion = new THREE.Quaternion();
quaternion.setFromAxisAngle( new THREE.Vector3( 0, 1, 0 ), Math.PI / 2 );
const vector = new THREE.Vector3( 1, 0, 0 );
vector.applyQuaternion( quaternion );

构造函数

new Quaternion( x : number, y : number, z : number, w : number )

构造一个新四元数。

x

此四元数的 x 值。

默认值为 0。
y

此四元数的 y 值。

默认值为 0。
z

此四元数的 z 值。

默认值为 0。
w

此四元数的 w 值。

默认值为 1。

属性

.isQuaternion : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.w : number

此四元数的 w 值。

默认值为 1。

.x : number

此四元数的 x 值。

默认值为 0。

.y : number

此四元数的 y 值。

默认值为 0。

.z : number

此四元数的 z 值。

默认值为 0。

方法

.angleTo( q : Quaternion ) : number

返回此四元数与给定四元数之间的夹角（弧度）。

q

用于计算夹角的四元数。
返回值： 以弧度为单位的角度。

.clone() : Quaternion

返回一个从此实例复制值的新四元数。

返回值： 此实例的克隆。

.conjugate() : Quaternion

返回此四元数的旋转共轭。四元数的共轭表示绕同一旋转轴、方向相反的相同旋转。

返回值： 此四元数的引用。

.copy( quaternion : Quaternion ) : Quaternion

将给定四元数的值复制到此实例。

quaternion

要复制的四元数。
返回值： 此四元数的引用。

.dot( v : Quaternion ) : number

计算此四元数与给定四元数的点积。

v

用于计算点积的四元数。
返回值： 点积结果。

.equals( quaternion : Quaternion ) : boolean

若此四元数与给定四元数相等，则返回 true。

quaternion

用于相等性测试的四元数。
返回值： 此四元数是否与给定四元数相等。

.fromArray( array : Array.<number>, offset : number ) : Quaternion

根据给定数组设置此四元数的分量。

array

保存四元数分量值的数组。
offset

数组中的偏移量。

默认值为 0。
返回值： 此四元数的引用。

.fromBufferAttribute( attribute : BufferAttribute, index : number ) : Quaternion

根据给定缓冲属性设置此四元数的分量。

attribute

保存四元数数据的缓冲属性。
index

属性中的索引。
返回值： 此四元数的引用。

.identity() : Quaternion

将此四元数设为单位四元数，即表示「无旋转」的四元数。

返回值： 此四元数的引用。

.invert() : Quaternion

通过 Quaternion#conjugate 求此四元数的逆。假定该四元数具有单位长度。

返回值： 此四元数的引用。

.length() : number

计算此四元数（视为四维向量）的欧几里得长度（直线长度）。

返回值： 欧几里得长度。

.lengthSq() : number

计算此四元数（视为四维向量）的欧几里得长度平方。在比较两个四元数长度时很有用，因为比 Quaternion#length 的计算略高效。

返回值： 欧几里得长度的平方。

.multiply( q : Quaternion ) : Quaternion

将此四元数与给定四元数相乘。

q

四元数。
返回值： 此四元数的引用。

.multiplyQuaternions( a : Quaternion, b : Quaternion ) : Quaternion

将给定四元数相乘，并将结果存储在此实例中。

a

第一个四元数。
b

第二个四元数。
返回值： 此四元数的引用。

.normalize() : Quaternion

归一化此四元数——即计算执行相同旋转、但长度为 1 的四元数。

返回值： 此四元数的引用。

.premultiply( q : Quaternion ) : Quaternion

用给定四元数左乘此四元数。

q

四元数。
返回值： 此四元数的引用。

.random() : Quaternion

将此四元数设为均匀随机的归一化四元数。

返回值： 此四元数的引用。

.rotateTowards( q : Quaternion, step : number ) : Quaternion

将此四元数按给定角步长朝目标四元数旋转。 该方法确保最终四元数不会越过 q。

q

目标四元数。
step

以弧度为单位的角步长。
返回值： 此四元数的引用。

.set( x : number, y : number, z : number, w : number ) : Quaternion

设置四元数分量。

x

此四元数的 x 值。
y

此四元数的 y 值。
z

此四元数的 z 值。
w

此四元数的 w 值。
返回值： 此四元数的引用。

.setFromAxisAngle( axis : Vector3, angle : number ) : Quaternion

根据给定轴和角度设置此四元数。

axis

归一化的轴。
angle

以弧度为单位的角度。
返回值： 此四元数的引用。

.setFromEuler( euler : Euler, update : boolean ) : Quaternion

根据给定欧拉角指定的旋转设置此四元数。

euler

欧拉角。
update

是否应执行内部 onChange 回调。

默认值为 true。
返回值： 此四元数的引用。

.setFromRotationMatrix( m : Matrix4 ) : Quaternion

根据给定旋转矩阵设置此四元数。

m

一个 4x4 矩阵，其上三角 3x3 部分为纯旋转矩阵（即未缩放）。
返回值： 此四元数的引用。

.setFromUnitVectors( vFrom : Vector3, vTo : Vector3 ) : Quaternion

将此四元数设为把方向向量 vFrom 旋转到方向向量 vTo 所需的旋转。

vFrom

第一个（归一化）方向向量。
vTo

第二个（归一化）方向向量。
返回值： 此四元数的引用。

.slerp( qb : Quaternion, t : number ) : Quaternion

在此四元数与目标四元数之间执行球面线性插值。

qb

目标四元数。
t

插值因子。范围 [0,1] 内的值进行插值；范围外的值进行外推。
返回值： 此四元数的引用。

.slerpQuaternions( qa : Quaternion, qb : Quaternion, t : number ) : Quaternion

在给定四元数之间执行球面线性插值，并将结果存储在此四元数中。

qa

源四元数。
qb

目标四元数。
t

闭区间 [0, 1] 内的插值因子。
返回值： 此四元数的引用。

.toArray( array : Array.<number>, offset : number ) : Array.<number>

将此四元数的分量写入给定数组。若未提供数组，则方法返回新实例。

array

保存四元数分量的目标数组。

默认值为 []。
offset

数组中第一个元素的索引。

默认值为 0。
返回值： 四元数分量。

.toJSON() : Array.<number>

定义此类的序列化结果。以 [x, y, z, w] 格式的数组返回此四元数的数值元素。

返回值： 序列化后的四元数。
