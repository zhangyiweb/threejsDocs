RapierHelper

此类以轮廓线形式显示所有 Rapier 碰撞体。

导入

RapierHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { RapierHelper } from 'three/addons/helpers/RapierHelper.js';

构造函数

new RapierHelper( world : RAPIER.world )

构造一个新的 Rapier 调试辅助对象。

world

要可视化的 Rapier 世界。

属性

.world : RAPIER.world

要可视化的 Rapier 世界。

方法

.dispose()

释放此实例分配的 GPU 相关资源。 当应用中不再使用此实例时，请调用此方法。

.update()

在渲染循环中调用此方法以更新轮廓线。
