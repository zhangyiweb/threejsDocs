EventDispatcher → Material →

ShadowMaterial

此材质可以接收阴影，除此之外完全透明。

代码示例

const geometry = new THREE.PlaneGeometry( 2000, 2000 );
geometry.rotateX( - Math.PI / 2 );
const material = new THREE.ShadowMaterial();
material.opacity = 0.2;
const plane = new THREE.Mesh( geometry, material );
plane.position.y = -200;
plane.receiveShadow = true;
scene.add( plane );

构造函数

new ShadowMaterial( parameters : Object )

构造一个新的阴影材质。

parameters

包含一个或多个属性的对象， 用于定义材质的外观。材质的任何属性 （包括从继承材质中获得的任何属性）都可以传入 此处。颜色值可以传入 Color#set 所接受的任意类型的值。

属性

.color : Color

材质的颜色。

默认值为 (0,0,0)。

.fog : boolean

材质是否受雾效影响。

默认值为 true。

.isShadowMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.transparent : boolean

已重写，因为阴影材质默认 为透明。

默认值为 true。

重写： Material#transparent
