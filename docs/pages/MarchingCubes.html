MarchingCubes

一种移动立方体（marching cubes）实现。

移植自：http://webglsamples.org/blob/blob.html

导入

MarchingCubes 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { MarchingCubes } from 'three/addons/objects/MarchingCubes.js';

构造函数

new MarchingCubes( resolution : number, material : Material, enableUvs : boolean, enableColors : boolean, maxPolyCount : number )

构造一个新的移动立方体实例。

resolution

效果的分辨率。
material

立方体的材质。
enableUvs

是否应动画化纹理坐标。

默认值为 false。
enableColors

是否应动画化颜色。

默认值为 false。
maxPolyCount

几何缓冲区的最大大小。

默认值为 10000。

属性

.enableColors : boolean

是否应动画化颜色。

默认值为 false。

.enableUvs : boolean

是否应动画化纹理坐标。

默认值为 false。

.isMarchingCubes : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.addBall( ballx : number, bally : number, ballz : number, strength : number, subtract : number, colors : Color )

添加一个倒数球（外观圆润、呈团块状）。为提高速度，该球在由 strength 与 subtract 决定的固定距离后衰减为零。

ballx

球的 x 坐标。
bally

球的 y 坐标。
ballz

球的 z 坐标。
strength

强度因子。
subtract

减去因子。
colors

颜色。

.addPlaneX( strength : number, subtract : number )

沿 x 轴添加一个平面。

strength

强度因子。
subtract

减去因子。

.addPlaneY( strength : number, subtract : number )

沿 y 轴添加一个平面。

strength

强度因子。
subtract

减去因子。

.addPlaneZ( strength : number, subtract : number )

沿 z 轴添加一个平面。

strength

强度因子。
subtract

减去因子。

.blur( intensity : number )

以给定强度应用模糊。

intensity

模糊强度。

默认值为 1。

.getCell( x : number, y : number, z : number ) : number

返回给定坐标处的单元值。

x

x 值。
y

y 值。
z

z 值。
返回值： 该值。

.reset()

重置效果。

.setCell( x : number, y : number, z : number, value : number )

设置给定坐标处的单元值。

x

x 值。
y

y 值。
z

z 值。
value

要设置的值。

.update()

更新效果。
