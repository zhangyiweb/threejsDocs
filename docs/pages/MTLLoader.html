MTLLoader

用于加载 MTL 格式的加载器。

Material Template Library 格式（MTL）或 .MTL 文件格式是与 OBJ 配套的文件格式， 用于描述一个或多个 OBJ 文件中对象的表面着色（材质）属性。

代码示例

const loader = new MTLLoader();
const materials = await loader.loadAsync( 'models/obj/male02/male02.mtl' );
const objLoader = new OBJLoader();
objLoader.setMaterials( materials );

导入

MTLLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { MTLLoader } from 'three/addons/loaders/MTLLoader.js';

构造函数

new MTLLoader()

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 MTL 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( text : string, path : string ) : MaterialCreator

解析给定的 MTL 数据并返回生成的材质创建器。

text

原始 MTL 数据字符串。
path

URL 基础路径。
重写： Loader#parse
返回值： 材质创建器。

.setMaterialOptions( value : MTLLoader~MaterialOptions ) : MTLLoader

设置材质选项。

value

材质选项。
返回值： 对此加载器的引用。

类型定义

.MaterialOptions

MTLLoader 的材质选项。

side
FrontSide | BackSide | DoubleSide

材质应用到哪一面。

默认值为 FrontSide。
wrap
RepeatWrapping | ClampToEdgeWrapping | MirroredRepeatWrapping

纹理使用的包裹类型。

默认值为 RepeatWrapping。
normalizeRGB
boolean

是否将 RGB 颜色从 0-255 归一化到 0-1。

默认值为 false。
ignoreZeroRGBs
boolean

忽略 RGB（Ka、Kd、Ks）全为 0 的值。

默认值为 false。
