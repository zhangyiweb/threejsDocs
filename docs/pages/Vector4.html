Vector4

表示四维向量的类。四维向量是四个有序数字 （标记为 x、y、z 和 w），可用于表示多种事物，例如：

  • 四维空间中的点。
  • 四维空间中的方向与长度。在 three.js 中，长度始终是从 (0, 0, 0, 0) 到 (x, y, z, w) 的欧几里得距离（直线距离）， 方向也是从 (0, 0, 0, 0) 指向 (x, y, z, w)。
  • 任意有序的四元组数字。

四维向量还可用于表示其他事物，但以上是 three.js 中最常见的用途。

遍历向量实例将按相应顺序产生其分量 (x, y, z, w)。

代码示例

const a = new THREE.Vector4( 0, 1, 0, 0 );
//no arguments; will be initialised to (0, 0, 0, 1)
const b = new THREE.Vector4( );
const d = a.dot( b );

构造函数

new Vector4( x : number, y : number, z : number, w : number )

构造一个新的四维向量。

x

此向量的 x 值。

默认值为 0。
y

此向量的 y 值。

默认值为 0。
z

此向量的 z 值。

默认值为 0。
w

此向量的 w 值。

默认值为 1。

.clamp( min : Vector4, max : Vector4 ) : Vector4

如果此向量的 x、y、z 或 w 值大于最大向量的对应值，则替换为对应值。 如果此向量的 x、y、z 或 w 值小于最小向量的对应值，则替换为对应值。

min

max

期望范围内的最大 x、y 和 z 值。
返回值： 对此向量的引用。

.clampLength( min : number, max : number ) : Vector4

如果此向量的长度大于最大值，则替换为最大值。 如果此向量的长度小于最小值，则替换为最小值。

min

max

返回值： 对此向量的引用。

返回值： 此向量长度的平方。

.lerp( v : Vector4, alpha : number ) : Vector4

alpha

返回值： 对此向量的引用。

v1

第一个向量。
v2

alpha
