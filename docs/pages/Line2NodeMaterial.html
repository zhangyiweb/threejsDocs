Line2NodeMaterial

此节点材质可用于渲染线宽大于一像素的线条， 通过将线条表示为实例化网格来实现。

构造函数

new Line2NodeMaterial( parameters : Object )

构造一个用于宽线渲染的新节点材质。

parameters

配置参数。

默认值为 {}。

属性

.alphaToCoverage : boolean

是否应使用 alpha to coverage。

默认值为 true。

重写： NodeMaterial#alphaToCoverage

.blending : number

混合模式设为 NoBlending，因为目前尚不支持透明度。

默认值为 0。

重写： NodeMaterial#blending

.dashOffset : number

虚线偏移量。

默认值为 0。

.dashScaleNode : Node.<float>

定义虚线缩放。

默认值为 null。

.dashSizeNode : Node.<float>

定义虚线段长度。

默认值为 null。

.dashed : boolean

线条是否应为虚线。

默认值为 false。

.gapSizeNode : Node.<float>

定义间隙大小。

默认值为 null。

.isLine2NodeMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.lineColorNode : Node.<vec3>

定义线条颜色。

已弃用： 自 r185 起。请改用 NodeMaterial#colorNode。

.offsetNode : Node.<float>

定义偏移量。

默认值为 null。

.vertexColors : boolean

是否应使用顶点颜色。

默认值为 false。

重写： NodeMaterial#vertexColors

.worldUnits : boolean

线条尺寸是否应使用世界单位。 设为 false 时单位为像素。

默认值为 false。

方法

.setupDiffuseColor( builder : NodeBuilder )

在片元阶段设置线材质的漫反射颜色。 重写基类设置，以纳入线条/虚线渲染与混合。

builder

当前节点构建器。
重写： NodeMaterial#setupDiffuseColor

.setupModelViewProjection( builder : NodeBuilder ) : Node.<vec4>

为粗线的顶点阶段设置裁剪空间位置。 重写默认的模型-视图-投影，以返回扩展后的粗线顶点坐标。

builder

当前节点构建器。
重写： NodeMaterial#setupModelViewProjection
返回值： 粗线顶点在裁剪空间中的位置。
