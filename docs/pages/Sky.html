Sky

表示用于场景背景的天空穹顶。基于 A Practical Analytic Model for Daylight （即 Preetham 模型），该模型是解析天空穹顶的事实标准。

注意：该类只能与 WebGLRenderer 一起使用。 使用 WebGPURenderer 时，请改用 SkyMesh。

更多参考：

  • http://simonwallner.at/project/atmospheric-scattering/
  • http://blenderartists.org/forum/showthread.php?245954-preethams-sky-impementation-HDR
const sky = new Sky();
sky.scale.setScalar( 10000 );
scene.add( sky );

在生成环境贴图时隐藏太阳圆盘有助于避免伪影

// disable before rendering environment map
sky.material.uniforms.showSunDisc.value = false;
// ...
// re-enable before scene sky box rendering
sky.material.uniforms.showSunDisc.value = true;

导入

Sky 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { Sky } from 'three/addons/objects/Sky.js';

构造函数

new Sky()

构造一个新的天空穹顶。

属性

.isSky : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
