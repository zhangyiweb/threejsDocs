ReadbackBuffer

回读缓冲用于将数据从 GPU 传输到 CPU。 主要用于回读计算着色器的结果。

构造函数

new ReadbackBuffer( maxByteLength : number )

构造一个新的回读缓冲。

maxByteLength

要回读的缓冲的最大大小。

属性

.buffer : ArrayBuffer | null

已映射的回读数组缓冲。

.isReadbackBuffer : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.maxByteLength : number

要回读的缓冲的最大大小。

.name : string

用于调试的名称。

方法

.dispose()

释放内部资源。

.release()

释放已映射的缓冲数据，以便 GPU 缓冲 可再次被 GPU 使用。

注意：调用此方法后，与此回读缓冲关联的任何 ArrayBuffer 数据都将被移除且不再可访问。
