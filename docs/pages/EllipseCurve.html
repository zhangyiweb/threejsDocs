EllipseCurve

表示椭圆的曲线。

代码示例

const curve = new THREE.EllipseCurve(
	0, 0,
	10, 10,
	0, 2 * Math.PI,
	false,
	0
);
const points = curve.getPoints( 50 );
const geometry = new THREE.BufferGeometry().setFromPoints( points );
const material = new THREE.LineBasicMaterial( { color: 0xff0000 } );
// Create the final object to add to the scene
const ellipse = new THREE.Line( geometry, material );

构造函数

new EllipseCurve( aX : number, aY : number, xRadius : number, yRadius : number, aStartAngle : number, aEndAngle : number, aClockwise : boolean, aRotation : number )

构造一条新的椭圆曲线。

aX

椭圆的 X 中心。

默认值为 0。
aY

椭圆的 Y 中心。

默认值为 0。
xRadius

椭圆在 x 方向上的半径。

默认值为 1。
yRadius

椭圆在 y 方向上的半径。

默认值为 1。
aStartAngle

曲线的起始角度（弧度），从正 X 轴开始计量。

默认值为 0。
aEndAngle

曲线的结束角度（弧度），从正 X 轴开始计量。

默认值为 Math.PI*2。
aClockwise

椭圆是否按顺时针方向绘制。

默认值为 false。
aRotation

椭圆的旋转角度（弧度），从正 X 轴逆时针计量。

默认值为 0。

属性

.aClockwise : boolean

椭圆是否按顺时针方向绘制。

默认值为 false。

.aEndAngle : number

曲线的结束角度（弧度），从正 X 轴开始计量。

默认值为 Math.PI*2。

.aRotation : number

椭圆的旋转角度（弧度），从正 X 轴逆时针计量。

默认值为 0。

.aStartAngle : number

曲线的起始角度（弧度），从正 X 轴开始计量。

默认值为 0。

.aX : number

椭圆的 X 中心。

默认值为 0。

.aY : number

椭圆的 Y 中心。

默认值为 0。

.isEllipseCurve : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.xRadius : number

椭圆在 x 方向上的半径。若将该值设为与 EllipseCurve#yRadius 相等，则结果为圆。

默认值为 1。

.yRadius : number

椭圆在 y 方向上的半径。若将该值设为与 EllipseCurve#xRadius 相等，则结果为圆。

默认值为 1。

方法

.getPoint( t : number, optionalTarget : Vector2 ) : Vector2

返回曲线上的一个点。

t

表示曲线上位置的插值因子，必须在 [0,1] 范围内。
optionalTarget

可选的目标向量，结果将写入其中。
重写： Curve#getPoint
返回值： 曲线上的位置。
