CapsuleGeometry

用于表示胶囊体的几何体类。

代码示例

const geometry = new THREE.CapsuleGeometry( 1, 1, 4, 8, 1 );
const material = new THREE.MeshBasicMaterial( { color: 0x00ff00 } );
const capsule = new THREE.Mesh( geometry, material );
scene.add( capsule );

构造函数

new CapsuleGeometry( radius : number, height : number, capSegments : number, radialSegments : number, heightSegments : number )

构造一个新的胶囊体几何体。

radius

胶囊体的半径。

默认值为 1.
height

中间部分的高度。

默认值为 1.
capSegments

用于构建每个端盖的曲线分段数。

默认值为 4.
radialSegments

沿胶囊体圆周方向的分段面数量。必须为大于等于 3 的整数。

默认值为 8.
heightSegments

沿中间部分高度方向的面行数。必须为大于等于 1 的整数。

默认值为 1.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

静态方法

.fromJSON( data : Object ) : CapsuleGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
