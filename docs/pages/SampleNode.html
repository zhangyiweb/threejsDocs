EventDispatcher → Node →

SampleNode

表示使用提供的回调函数采样值的节点类。

构造函数

new SampleNode( callback : function, uvNode : Node.<vec2> )

创建 SampleNode 实例。

callback

采样时调用的函数。应接受 UV 节点并返回值。
uvNode

用于纹理采样的 UV 节点。

默认值为 null。

属性

.isSampleNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.uvNode : Node.<(vec2|vec3)>

表示纹理坐标。

默认值为 null。

.type : string (readonly)

返回节点的类型。

方法

.sample( uv : Node.<vec2> ) : Node

使用提供的 UV 节点调用回调函数。

uv

传递给回调的 UV 节点或值。
返回值： 回调函数的返回值。

.setup() : Node

使用默认 UV 访问器进行采样以设置节点。

重写： Node#setup
返回值： 使用 UV 节点调用回调函数时的返回值。
