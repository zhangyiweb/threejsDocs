Water

基础的平面反射水面效果。

注意：该类只能与 WebGLRenderer 一起使用。 使用 WebGPURenderer 时，请改用 WaterMesh。

参考：

  • Flat mirror for three.js
  • An implementation of water shader based on the flat mirror
  • Water shader explanations in WebGL

导入

Water 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { Water } from 'three/addons/objects/Water.js';

构造函数

new Water( geometry : BufferGeometry, options : Water~Options )

构造一个新的水面实例。

geometry

水面的几何体。
options

配置选项。

属性

.isWater : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

类型定义

.Options

Water 的构造函数选项。

textureWidth
number

纹理宽度。值越高反射越清晰，但开销也越大。

默认值为 512。
textureHeight
number

纹理高度。值越高反射越清晰，但开销也越大。

默认值为 512。
clipBias
number

裁剪偏置。

默认值为 0。
alpha
number

透明度值。

默认值为 1。
time
number

时间值。

默认值为 0。
waterNormals
Texture

水面的法线贴图。

默认值为 null。
sunDirection
Vector3

太阳方向。

默认值为 (0.70707,0.70707,0.0)。
sunColor
number | Color | string

太阳颜色。

默认值为 0xffffff。
waterColor
number | Color | string

水面颜色。

默认值为 0x7F7F7F。
eye
Vector3

视点向量。
distortionScale
number

扭曲缩放。

默认值为 20。
side
FrontSide | BackSide | DoubleSide

水面材质的 side 属性。

默认值为 FrontSide。
fog
boolean

水面是否受雾影响。

默认值为 false。
