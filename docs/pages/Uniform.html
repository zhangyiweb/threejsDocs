Uniform

表示一个 uniform，即全局着色器变量，会传递给着色器程序。

在 ShaderMaterial 中声明 uniform 时，可按值或按对象声明。

由于该类仅可在 ShaderMaterial 上下文中使用，因此仅支持 WebGLRenderer。

代码示例

uniforms: {
	time: { value: 1.0 },
	resolution: new Uniform( new Vector2() )
};

构造函数

new Uniform( value : any )

构造一个新的 uniform。

value

uniform 的值。

属性

.boundary : number

用于按 STD140 布局构建 uniform 缓冲区。派生的 uniform 会将该属性设置为特定数据类型对应的值。

.index : number

由 UniformsGroup 设置，标记在 uniform 数组中的索引位置。

.itemSize : number

分量数量。派生的 uniform 会将该属性设置为特定数据类型对应的值。

.name : string

uniform 的名称。

.offset : number

由 UniformsGroup 设置，标记在 uniform 缓冲区中的起始位置。

.value : any

uniform 的值。

.value : any

uniform 的值。

方法

.clone() : Uniform

返回一个复制了当前实例值的新 uniform。若该值具有 clone() 方法，则该值也会被克隆。

返回值： 当前实例的克隆。

.getValue() : any

返回 uniform 的值。

返回值： 该值。

.setValue( value : any )

设置 uniform 的值。

value

要设置的值。
