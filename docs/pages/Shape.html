Shape

使用路径（可含可选孔洞）定义任意二维平面形状。可与 ExtrudeGeometry、ShapeGeometry 配合使用，用于获取点，或获取三角化面。

代码示例

const heartShape = new THREE.Shape();
heartShape.moveTo( 25, 25 );
heartShape.bezierCurveTo( 25, 25, 20, 0, 0, 0 );
heartShape.bezierCurveTo( - 30, 0, - 30, 35, - 30, 35 );
heartShape.bezierCurveTo( - 30, 55, - 10, 77, 25, 95 );
heartShape.bezierCurveTo( 60, 77, 80, 55, 80, 35 );
heartShape.bezierCurveTo( 80, 35, 80, 0, 50, 0 );
heartShape.bezierCurveTo( 35, 0, 25, 25, 25, 25 );
const extrudeSettings = {
	depth: 8,
	bevelEnabled: true,
	bevelSegments: 2,
	steps: 2,
	bevelSize: 1,
	bevelThickness: 1
};
const geometry = new THREE.ExtrudeGeometry( heartShape, extrudeSettings );
const mesh = new THREE.Mesh( geometry, new THREE.MeshBasicMaterial() );

构造函数

new Shape( points : Array.<Vector2> )

构造一个新形状。

points

定义该形状的二维点数组。

属性

.holes : Array.<Path> (readonly)

定义形状中的孔洞。孔洞定义必须使用与外形相反的绕序（CW/CCW）。

.uuid : string (readonly)

该形状的 UUID。

方法

.extractPoints( divisions : number ) : Object

返回一个对象，其中包含该形状及其孔洞的轮廓数据（以二维点数组形式）。

divisions

结果的精细度。
返回值： 包含轮廓数据的对象。

.getPointsHoles( divisions : number ) : Array.<Array.<Vector2>>

返回一个数组，其中每个元素表示一个孔洞轮廓的二维点列表。

divisions

结果的精细度。
返回值： 以一系列二维点表示的孔洞。
