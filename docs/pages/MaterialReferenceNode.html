MaterialReferenceNode

此节点是一种特殊类型的引用节点，用于将材质属性与节点值关联。

当更改 material.opacity 时，opacityNode 的节点值将自动更新。

代码示例

const opacityNode = materialReference( 'opacity', 'float', material );

构造函数

new MaterialReferenceNode( property : string, inputType : string, material : Material )

构造一个新的材质引用节点。

property

节点所引用的属性名称。
inputType

用于表示属性值的 uniform 类型。
material

属性所属的材质。未设置材质时， 节点引用当前渲染对象的材质。

默认值为 null。

属性

.isMaterialReferenceNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.material : Material

属性所属的材质。未设置材质时， 节点引用当前渲染对象的材质。

默认值为 null。

方法

.updateReference( state : NodeFrame | NodeBuilder ) : Object

根据给定状态更新引用。仅当未设置 MaterialReferenceNode#material 时才会评估该状态。

state

当前状态。
重写： ReferenceNode#updateReference
返回值： 更新后的引用。
