ShadowBaseNode

所有阴影节点的基类。

阴影节点封装了与阴影相关的逻辑，并且始终与光照节点配对。 光照节点可能共享相同的阴影节点类型，也可能根据自身需求使用特定的阴影节点。

构造函数

new ShadowBaseNode( light : Light )

构造一个新的阴影基类节点。

light

投射阴影的灯光。

属性

.isShadowBaseNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.light : Light

投射阴影的灯光。

.updateBeforeType : string

被重写，因为阴影默认按每次渲染进行更新。

默认值为 'render'。

重写： Node#updateBeforeType

方法

.setupShadowPosition( object : NodeBuilder )

设置阴影位置节点，默认使用预定义的 TSL 节点对象 shadowPositionWorld。

object

配置对象，至少须包含材质引用。
