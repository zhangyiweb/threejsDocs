ImageLoader

用于加载图像的加载器。该类使用 HTML Image API 加载图像。

请注意，自 r84 起，ImageLoader 已不再支持进度事件。 若需要支持进度事件的 ImageLoader，请参见 此讨论。

代码示例

const loader = new THREE.ImageLoader();
const image = await loader.loadAsync( 'image.png' );

构造函数

new ImageLoader( manager : LoadingManager )

创建一个新的图像加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback ) : Image

从给定 URL 开始加载，并将加载的图像传递给 onLoad() 回调。 该方法还会返回一个新的 Image 对象，可直接用于创建纹理。 若以此方式使用，纹理可能会在相应加载过程完成后出现在场景中。

url

要加载文件的路径/URL。也可以是 data URI。
onLoad

加载过程完成时执行。
onProgress

此加载器不支持。
onError

发生错误时执行。
重写： Loader#load
返回值： 图像。
