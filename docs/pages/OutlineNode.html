OutlineNode

用于在选中对象周围渲染轮廓的后处理节点。该节点可根据需求灵活组合最终轮廓外观。

代码示例

const renderPipeline = new THREE.RenderPipeline( renderer );
const scenePass = pass( scene, camera );
// 轮廓参数
const edgeStrength = uniform( 3.0 );
const edgeGlow = uniform( 0.0 );
const edgeThickness = uniform( 1.0 );
const visibleEdgeColor = uniform( new THREE.Color( 0xffffff ) );
const hiddenEdgeColor = uniform( new THREE.Color( 0x4e3636 ) );
outlinePass = outline( scene, camera, {
	selectedObjects,
	edgeGlow,
	edgeThickness
} );
// 合成自定义轮廓
const { visibleEdge, hiddenEdge } = outlinePass;
const outlineColor = visibleEdge.mul( visibleEdgeColor ).add( hiddenEdge.mul( hiddenEdgeColor ) ).mul( edgeStrength );
renderPipeline.outputNode = outlineColor.add( scenePass );

导入

OutlineNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { outline } from 'three/addons/tsl/display/OutlineNode.js';

构造函数

new OutlineNode( scene : Scene, camera : Camera, params : Object )

构造一个新的轮廓节点。

scene

场景的引用。
camera

用于渲染场景的相机。
params

配置参数。

selectedObjects

选中对象的数组。
edgeThickness

边缘的粗细。

默认值为 float(1)。
edgeGlow

可用于实现动画发光/脉冲效果。

默认值为 float(0)。
downSampleRatio

降采样比例。

默认值为 2。

属性

.camera : Camera

用于渲染场景的相机。

.downSampleRatio : number

降采样比例。

默认值为 2。

.edgeGlowNode : Node.<float>

可用于实现动画发光/脉冲效果。

.edgeThicknessNode : Node.<float>

边缘的粗细。

.hiddenEdge

表示隐藏边缘的遮罩值。

.scene : Scene

场景的引用。

.selectedObjects : Array.<Object3D>

选中对象的数组。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

.visibleEdge

表示可见边缘的遮罩值。

方法

.dispose()

释放内部资源。当不再需要该效果时应调用此方法。

重写： TempNode#dispose

.getTextureNode() : PassTextureNode

以纹理节点的形式返回效果结果。

返回值： 表示效果结果的纹理节点。

.setSize( width : number, height : number )

设置效果的尺寸。

width

效果的宽度。
height

效果的高度。

.setup( builder : NodeBuilder ) : PassTextureNode

此方法用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

此方法用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
