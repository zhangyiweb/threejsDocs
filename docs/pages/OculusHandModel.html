OculusHandModel

表示 Oculus 手部模型。

导入

OculusHandModel 是一个插件，必须显式导入，参见 Installation#Addons。

import { OculusHandModel } from 'three/addons/webxr/OculusHandModel.js';

构造函数

new OculusHandModel( controller : Group, loader : Loader, onLoad : function )

构造一个新的 Oculus 手部模型。

controller

手部控制器。
loader

用于加载手部模型的加载器。

默认值为 null。
onLoad

手部模型加载完成后执行的回调。

默认值为 null。

属性

.controller : Group

手部控制器。

.envMap : Texture

模型的环境贴图。

默认值为 null。

.loader : Loader

用于加载手部模型的加载器。

默认值为 null。

.mesh : Mesh

模型网格。

默认值为 null。

.motionController : MotionController

运动控制器。

默认值为 null。

.onLoad : function

手部模型加载完成后执行的回调。

默认值为 null。

.path : string

模型资源库的路径。

默认值为 null。

方法

.checkButton( button : Object )

根据与给定按钮的交互状态执行相应操作。

button

按钮。

.getPointerPosition() : Vector3

返回指针位置，即食指指尖的位置。

返回值： 指针位置。若未找到食指指尖关节，则返回 null。

.intersectBoxObject( boxObject : Mesh ) : boolean

若当前指针位置（食指指尖）与给定盒体对象相交，则返回 true。

boxObject

盒体对象。
返回值： 是否检测到相交。

.updateMatrixWorld( force : boolean )

已用自定义实现重写。确保运动控制器更新网格。

force

设为 true 时，即使 Object3D#matrixWorldAutoUpdate 设为 false，也会强制重新计算世界矩阵。

默认值为 false。
重写： Object3D#updateMatrixWorld
