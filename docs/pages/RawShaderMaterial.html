RawShaderMaterial

该类的工作方式与 ShaderMaterial 相同，区别在于内置 uniform 和 attribute 的定义不会自动前置到 GLSL 着色器代码中。

RawShaderMaterial 只能与 WebGLRenderer 一起使用。

构造函数

new RawShaderMaterial( parameters : Object )

构造一个新的原始着色器材质。

parameters

包含一个或多个属性的对象， 用于定义材质的外观。材质的任何属性 （包括从继承材质中获得的任何属性）都可以传入 此处。颜色值可以传入 Color#set 所接受的任意类型的值。

属性

.isRawShaderMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
