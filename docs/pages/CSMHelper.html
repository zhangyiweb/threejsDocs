CSMHelper

用于可视化 CSM 实例级联的辅助对象。

导入

CSMHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { CSMHelper } from 'three/addons/csm/CSMHelper.js';

构造函数

new CSMHelper( csm : CSM | CSMShadowNode )

构造一个新的 CSM 辅助对象。

csm

要可视化的 CSM 实例。

属性

.csm : CSM | CSMShadowNode

要可视化的 CSM 实例。

.displayFrustum : boolean

是否显示 CSM 视锥体。

默认值为 true。

.displayPlanes : boolean

是否显示级联平面。

默认值为 true。

.displayShadowBounds : boolean

是否显示阴影边界。

默认值为 true。

方法

.dispose()

释放该实例分配的 GPU 相关资源。当应用中不再使用此实例时，请调用此方法。

.update()

更新辅助对象。应在应用的动画循环中调用此方法。

.updateVisibility()

若在运行时更改了某个 display* 属性，则必须调用此方法。
