Audio

表示一个非空间定位（全局）的音频对象。

本类及相关音频模块使用 Web Audio API。

代码示例

// create an AudioListener and add it to the camera
const listener = new THREE.AudioListener();
camera.add( listener );
// create a global audio source
const sound = new THREE.Audio( listener );
// load a sound and set it as the Audio object's buffer
const audioLoader = new THREE.AudioLoader();
audioLoader.load( 'sounds/ambient.ogg', function( buffer ) {
	sound.setBuffer( buffer );
	sound.setLoop( true );
	sound.setVolume( 0.5 );
	sound.play();
});

构造函数

new Audio( listener : AudioListener )

构造一个新的音频对象。

listener

全局音频监听器。

属性

.autoplay : boolean

是否自动开始播放。

默认值为 false。

.buffer : AudioBuffer (readonly)

指向音频缓冲区的引用。

通过 Audio#setBuffer 定义。

默认值为 null。

.context : AudioContext (readonly)

音频上下文。

.detune : number (readonly)

修改音高，以音分为单位。+/- 100 为一个半音， +/- 1200 为一个八度。

通过 Audio#setDetune 定义。

默认值为 0。

.duration : undefined | number

重写音频的默认时长。

默认值为 undefined。

.filters : Array.<AudioNode> (readonly)

可用于应用多种低阶滤波器以创建 更复杂的声音效果，例如通过 BiquadFilterNode。

该属性由 Audio#setFilters 自动设置。

.gain : GainNode (readonly)

用于音量控制的增益节点。

.hasPlaybackControl : boolean (readonly)

指示是否可以使用 Audio#play 或 Audio#pause 等方法 控制音频播放。

定义音频源时会自动设置此标志。

默认值为 true。

.isPlaying : boolean (readonly)

指示音频是否正在播放。

使用 Audio#play、 Audio#pause、Audio#stop 时会自动设置此标志。

默认值为 false。

.listener : AudioListener (readonly)

全局音频监听器。

.loop : boolean (readonly)

音频是否循环播放。

通过 Audio#setLoop 定义。

默认值为 false。

.loopEnd : number

定义音频缓冲区中重播停止的位置， 单位为秒。

默认值为 0。

.loopStart : number

定义音频缓冲区中重播开始的位置， 单位为秒。

默认值为 0。

.offset : number

播放开始时在音频缓冲区内的起始时间偏移量， 单位为秒。

默认值为 0。

.playbackRate : number (readonly)

播放速度。

通过 Audio#setPlaybackRate 定义。

默认值为 1。

.source : AudioNode (readonly)

保存对当前音频源的引用。

该属性由某个 set*() 方法自动设置。

默认值为 null。

.sourceType : 'empty' | 'audioNode' | 'mediaNode' | 'mediaStreamNode' | 'buffer' (readonly)

定义源类型。

该属性由某个 set*() 方法自动设置。

默认值为 'empty'。

方法

.connect() : Audio

连接到音频源。在初始化以及设置/移除滤波器时内部使用。

返回值： 此实例的引用。

.disconnect() : Audio | undefined

断开与音频源的连接。在初始化以及设置/移除滤波器时内部使用。

返回值： 此实例的引用。

.getDetune() : number

返回振荡的音分偏离值。

返回值： 振荡的音分偏离值。

.getFilter() : AudioNode | undefined

返回滤波器列表中的第一个滤波器。

返回值： 滤波器列表中的第一个滤波器。

.getFilters() : Array.<AudioNode>

返回当前设置的滤波器。

返回值： 滤波器列表。

.getLoop() : boolean

返回循环标志。

仅可用于支持播放控制的兼容音频源。

返回值： 音频是否循环播放。

.getOutput() : GainNode

返回输出音频节点。

返回值： 输出节点。

.getPlaybackRate() : number

返回当前播放速率。

返回值： 播放速率。

.getVolume() : number

返回音量。

返回值： 音量。

.onEnded()

播放完成时自动调用。

.pause() : Audio | undefined

暂停音频播放。

仅可用于支持播放控制的兼容音频源。

返回值： 此实例的引用。

.play( delay : number ) : Audio | undefined

开始播放音频。

仅可用于支持播放控制的兼容音频源。

delay

音频开始播放的延迟时间，单位为秒。

默认值为 0。
返回值： 此实例的引用。

.setBuffer( audioBuffer : AudioBuffer ) : Audio

将给定的音频缓冲区设置为该实例的源。

Audio#sourceType 设置为 buffer，Audio#hasPlaybackControl 设置为 true。

audioBuffer

音频缓冲区。
返回值： 此实例的引用。

.setDetune( value : number ) : Audio

定义振荡的音分偏离值。

value

振荡的音分偏离值。
返回值： 此实例的引用。

.setFilter( filter : AudioNode ) : Audio

对音频应用单个滤波器节点。

filter

要设置的滤波器。
返回值： 此实例的引用。

.setFilters( value : Array.<AudioNode> ) : Audio

设置滤波器数组并将其与音频源连接。

value

滤波器列表。
返回值： 此实例的引用。

.setLoop( value : boolean ) : Audio | undefined

设置循环标志。

仅可用于支持播放控制的兼容音频源。

value

音频是否循环播放。
返回值： 此实例的引用。

.setLoopEnd( value : number ) : Audio

设置循环结束值，定义音频缓冲区中重播停止的位置， 单位为秒。

value

循环结束值。
返回值： 此实例的引用。

.setLoopStart( value : number ) : Audio

设置循环开始值，定义音频缓冲区中重播开始的位置， 单位为秒。

value

循环开始值。
返回值： 此实例的引用。

.setMediaElementSource( mediaElement : HTMLMediaElement ) : Audio

将给定的媒体元素设置为该实例的源。

Audio#sourceType 设置为 mediaNode，Audio#hasPlaybackControl 设置为 false。

mediaElement

媒体元素。
返回值： 此实例的引用。

.setMediaStreamSource( mediaStream : MediaStream ) : Audio

将给定的媒体流设置为该实例的源。

Audio#sourceType 设置为 mediaStreamNode，Audio#hasPlaybackControl 设置为 false。

mediaStream

媒体流。
返回值： 此实例的引用。

.setNodeSource( audioNode : AudioNode ) : Audio

将给定的音频节点设置为该实例的源。

Audio#sourceType 设置为 audioNode，Audio#hasPlaybackControl 设置为 false。

audioNode

音频节点，例如 OscillatorNode 的实例。
返回值： 此实例的引用。

.setPlaybackRate( value : number ) : Audio | undefined

设置播放速率。

仅可用于支持播放控制的兼容音频源。

value

要设置的播放速率。
返回值： 此实例的引用。

.setVolume( value : number ) : Audio

设置音量。

value

要设置的音量。
返回值： 此实例的引用。

.stop( delay : number ) : Audio | undefined

停止音频播放。

仅可用于支持播放控制的兼容音频源。

delay

音频停止播放的延迟时间，单位为秒。

默认值为 0。
返回值： 此实例的引用。
