Interpolations

Interpolations 包含样条与贝塞尔函数，供具体曲线类内部使用。

贝塞尔曲线公式来源：https://en.wikipedia.org/wiki/B%C3%A9zier_curve

方法

.CatmullRom( t : number, p0 : number, p1 : number, p2 : number, p3 : number ) : number (inner)

计算 Catmull-Rom 样条上的一个点。

t

插值因子。
p0

第一个控制点。
p1

第二个控制点。
p2

第三个控制点。
p3

第四个控制点。
返回值： Catmull-Rom 样条上计算出的点。

.CubicBezier( t : number, p0 : number, p1 : number, p2 : number, p3 : number ) : number (inner)

计算三次贝塞尔曲线上的一个点。

t

插值因子。
p0

第一个控制点。
p1

第二个控制点。
p2

第三个控制点。
p3

第四个控制点。
返回值： 三次贝塞尔曲线上计算出的点。

.QuadraticBezier( t : number, p0 : number, p1 : number, p2 : number ) : number (inner)

t

p0

第一个控制点。
p1
