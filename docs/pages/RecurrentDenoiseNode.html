RecurrentDenoiseNode

用于对时序累积的屏幕空间效果（如 SSGI（环境光遮蔽 / 间接漫反射）和 SSR（镜面反射））进行降噪的后处理节点。

降噪核在构造时通过 mode 选择：'diffuse'（SSGI）或 'specular'（SSR）。该核使用固定的 8 采样 Vogel 圆盘。

导入

RecurrentDenoiseNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { recurrentDenoise } from 'three/addons/tsl/display/RecurrentDenoiseNode.js';

构造函数

new RecurrentDenoiseNode( inputTexture : TextureNode, camera : Camera, options : RecurrentDenoiseNodeOptions )

inputTexture

待降噪的时序滤波输入（例如 TRAA 输出）。
camera
options

默认值为 {}。

属性

.accumulate : boolean

当为 true 时，在空间降噪后应用时序混合。当为 false 时，仅输出空间滤波后的颜色（alpha 从输入的时序通道透传）。

.alphaSource : DenoiseAlphaSource

原始纹理的哪个通道驱动基于 alpha 的边缘停止。 'raylength' — alpha 编码 SSR 射线长度；'ao' — alpha 编码 AO 因子； 'none' — 跳过基于 alpha 的边缘停止。

默认值为 'raylength'。

.mode : DenoiseMode

降噪核类型。

方法

.getRenderTarget() : RenderTarget

返回内部输出渲染目标（例如用于时序重投影 / SSGI 时序反馈循环）。
