AnimationClip

一组可复用的关键帧轨道，用于表示一个动画。

构造函数

new AnimationClip( name : string, duration : number, tracks : Array.<KeyframeTrack>, blendMode : NormalAnimationBlendMode | AdditiveAnimationBlendMode )

构造一个新的动画片段。

注意：除了直接使用构造函数实例化 AnimationClip 外，也可以使用该类的静态接口创建片段。不过在大多数情况下，导入带动画的 3D 资源时，加载器会自动创建动画片段。

name

片段名称。

默认值为 ''。
duration

片段持续时间（秒）。若传入负值， 将根据传入的关键帧自动计算持续时间。

默认值为 -1。
tracks

关键帧轨道数组。
blendMode

定义当两个或多个动画同时播放时， 动画如何混合/组合。

默认值为 NormalAnimationBlendMode。

属性

.blendMode : NormalAnimationBlendMode | AdditiveAnimationBlendMode

定义当两个或多个动画同时播放时，动画如何混合/组合。

.duration : number

片段持续时间（秒）。

.name : string

片段名称。

.tracks : Array.<KeyframeTrack>

关键帧轨道数组。

.userData : Object

可用于存储动画片段自定义数据的对象。 不应保存函数引用，因为函数不会被克隆。

.uuid : string (readonly)

动画片段的 UUID。

方法

.clone() : AnimationClip

返回一个复制了当前实例值的新动画片段。

返回值： 当前实例的克隆。

.optimize() : AnimationClip

通过移除等效的连续关键帧（在变形目标序列中很常见）来优化每条轨道。

返回值： 对该动画片段的引用。

.resetDuration() : AnimationClip

将该片段的持续时间设置为其最长关键帧轨道的持续时间。

返回值： 对该动画片段的引用。

.toJSON() : Object

将该动画片段序列化为 JSON。

返回值： JSON 对象。

.trim() : AnimationClip

将所有轨道裁剪到片段的持续时间范围内。

返回值： 对该动画片段的引用。

.validate() : boolean

对片段中的每条轨道执行最小验证。若所有轨道均有效，则返回 true。

返回值： 片段的关键帧是否有效。

静态方法

.CreateClipsFromMorphTargetSequences( morphTargets : Array.<Object>, fps : number, noLoop : boolean ) : Array.<AnimationClip>

从几何体的变形目标序列创建新的 AnimationClip 数组，尝试将变形目标名称按动画组模式排序，例如 "Walk_001, Walk_002, Run_001, Run_002..."。

请参阅 MD2Loader#parse 了解该方法的使用示例。

morphTargets

变形目标序列。
fps

每秒帧数（FPS）。
noLoop

片段是否不循环播放。
返回值： 新动画片段数组。

.CreateFromMorphTargetSequence( name : string, morphTargetSequence : Array.<Object>, fps : number, noLoop : boolean ) : AnimationClip

从几何体的变形目标数组创建新的动画片段，需指定名称和每秒帧数。

注意：fps 参数是必需的，但可通过 AnimationAction#setDuration 覆盖动画播放速度。

name

动画片段名称。
morphTargetSequence

变形目标序列。
fps

每秒帧数（FPS）。
noLoop

片段是否不循环播放。
返回值： 新的动画片段。

.findByName( objectOrClipArray : Array.<AnimationClip> | Object3D, name : string ) : AnimationClip

按名称搜索动画片段。第一个参数可以是片段数组，或包含名为 "animations" 属性的网格/几何体。

objectOrClipArray

要搜索的数组或对象。
name

要搜索的名称。
返回值： 找到的动画片段。若未找到则返回 null。

.parse( json : Object ) : AnimationClip

从给定 JSON 创建动画片段的工厂方法。

json

序列化后的动画片段。
返回值： 新的动画片段。

.toJSON( clip : AnimationClip ) : Object

将给定动画片段序列化为 JSON。

clip

要序列化的动画片段。
返回值： JSON 对象。
