BoxGeometry

用于创建具有给定宽度、高度和深度的长方体的几何体类。创建时，长方体以原点为中心，每条边平行于坐标轴之一。

代码示例

const geometry = new THREE.BoxGeometry( 1, 1, 1 );
const material = new THREE.MeshBasicMaterial( { color: 0x00ff00 } );
const cube = new THREE.Mesh( geometry, material );
scene.add( cube );

构造函数

new BoxGeometry( width : number, height : number, depth : number, widthSegments : number, heightSegments : number, depthSegments : number )

构造一个新的长方体几何体。

width

宽度。即平行于 X 轴的边的长度。

默认值为 1.
height

高度。即平行于 Y 轴的边的长度。

默认值为 1.
depth

深度。即平行于 Z 轴的边的长度。

默认值为 1.
widthSegments

沿侧面宽度方向分段的矩形面数量。

默认值为 1.
heightSegments

沿侧面高度方向分段的矩形面数量。

默认值为 1.
depthSegments

沿侧面深度方向分段的矩形面数量。

默认值为 1.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

静态方法

.fromJSON( data : Object ) : BoxGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
