MaterialLoader

用于加载材质的类。文件在内部通过 FileLoader 加载。

此加载器不支持节点材质。请改用 NodeMaterialLoader。

代码示例

const loader = new THREE.MaterialLoader();
const material = await loader.loadAsync( 'material.json' );

构造函数

new MaterialLoader( manager : LoadingManager )

构造一个新的材质加载器。

manager

加载管理器。

属性

.textures : Object.<string, Texture>

保存材质所用纹理的字典。

方法

.createMaterialFromType( type : string ) : Material

根据给定类型创建材质。

type

材质类型。
返回值： 新的材质。

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的材质传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( json : Object ) : Material

解析给定的 JSON 对象并返回材质。

json

序列化后的材质。
重写： Loader#parse
返回值： 解析后的材质。

.setTextures( value : Object ) : MaterialLoader

纹理不会嵌入在材质 JSON 中，因此必须在 加载过程开始前注入。

value

保存材质属性所用纹理的字典。
返回值： 此材质加载器的引用。

静态方法

.createMaterialFromType( type : string ) : Material

根据给定类型创建材质。

type

材质类型。
返回值： 新的材质。

.registerMaterial( type : string, materialClass : Material.constructor )

将给定材质注册到内部 材质库中。

type

材质类型。
materialClass

材质类。
