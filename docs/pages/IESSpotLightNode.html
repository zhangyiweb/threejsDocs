IESSpotLightNode

默认聚光灯节点的 IES 版本。

构造函数

new IESSpotLightNode( light : SpotLight )

构造一个新的 IES 聚光灯节点。

light

聚光灯光源。

默认值为 null。

属性

._iesTextureNode : TextureNode

表示 IES 纹理的纹理节点。

默认值为 null。

方法

.getSpotAttenuation( builder : NodeBuilder, angleCosine : Node.<float> ) : Node.<float>

重写默认实现，以计算符合 IES 规范的聚光灯衰减。

builder

节点构建器。
angleCosine

用于计算聚光灯衰减的角度。
重写： SpotLightNode#getSpotAttenuation
返回值： 聚光灯衰减。

.update( frame : NodeFrame )

重写以更新 IES 聚光灯纹理。

frame

当前节点帧的引用。
重写： SpotLightNode#update
