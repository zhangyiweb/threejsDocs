RenderOutputNode

通常，色调映射和颜色转换会在输出像素到默认（屏幕）帧缓冲区之前自动进行。在某些 后期处理设置中这太晚了，因为某些效果（如 FXAA） 需要 sRGB 输入。对于此类场景，可使用 RenderOutputNode 在效果链的任意位置应用色调映射和颜色空间转换。

使用此节点手动应用色调映射和颜色空间转换时， 必须将 RenderPipeline#outputColorTransform 设为 false。

代码示例

const postProcessing = new RenderPipeline( renderer );
postProcessing.outputColorTransform = false;
const scenePass = pass( scene, camera );
const outputPass = renderOutput( scenePass );
postProcessing.outputNode = outputPass;

构造函数

new RenderOutputNode( colorNode : Node, toneMapping : number, outputColorSpace : string )

构造一个新的渲染输出节点。

colorNode

要处理的颜色节点。
toneMapping

色调映射类型。
outputColorSpace

输出颜色空间。

属性

.colorNode : Node

要处理的颜色节点。

.isRenderOutputNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.outputColorSpace : string

输出颜色空间。

方法

.getToneMapping() : number

获取色调映射类型。

返回值： 色调映射类型。

.setToneMapping( value : number ) : ToneMappingNode

设置色调映射类型。

value

色调映射类型。
返回值： 对此节点的引用。
