ModelNode

此类节点是 Object3DNode 的特化版本， 提供了更多与模型相关的度量。与 Object3DNode 不同， ModelNode 从当前节点帧状态中提取 3D 对象的引用。

构造函数

new ModelNode( scope : 'position' | 'viewPosition' | 'direction' | 'scale' | 'worldMatrix' )

构造一个新的对象模型节点。

scope

根据作用域的不同，该节点表示不同类型的变换。

方法

.update( frame : NodeFrame )

从帧状态中提取模型引用，然后 根据作用域更新 uniform 值。

frame

当前节点帧。
重写： Object3DNode#update
