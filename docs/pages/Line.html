Line

连续折线。通过用直线连接连续顶点来渲染线条。

代码示例

const material = new THREE.LineBasicMaterial( { color: 0x0000ff } );
const points = [];
points.push( new THREE.Vector3( - 10, 0, 0 ) );
points.push( new THREE.Vector3( 0, 10, 0 ) );
points.push( new THREE.Vector3( 10, 0, 0 ) );
const geometry = new THREE.BufferGeometry().setFromPoints( points );
const line = new THREE.Line( geometry, material );
scene.add( line );

构造函数

new Line( geometry : BufferGeometry, material : Material | Array.<Material> )

构造一个新的线条。

geometry

线条几何体。
material

线条材质。

属性

.geometry : BufferGeometry

线条几何体。

.isLine : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.material : Material | Array.<Material>

线条材质。

默认值为 LineBasicMaterial。

.morphTargetDictionary : Object.<string, number> | undefined

表示几何体中变形目标的字典。键为变形目标名称， 值为其属性索引。该成员默认为 undefined， 仅在几何体中检测到变形目标时才会设置。

默认值为 undefined。

.morphTargetInfluences : Array.<number> | undefined

权重数组，通常取值范围为 [0,1]，用于指定变形的应用程度。 该成员默认为 undefined，仅在几何体中检测到变形目标时才会设置。

默认值为 undefined。

方法

.computeLineDistances() : Line

计算渲染虚线所需的距离值数组。 对于几何体中的每个顶点，该方法会计算从当前点到线条起点的累计长度。

返回值： 此线条的引用。

.raycast( raycaster : Raycaster, intersects : Array.<Object> )

计算投射光线与此线条之间的交点。

raycaster

光线投射器。
intersects

保存交点的目标数组。
重写： Object3D#raycast

.updateMorphTargets()

设置 Line#morphTargetDictionary 和 Line#morphTargetInfluences 的值， 以确保现有变形目标能够影响此 3D 对象。
