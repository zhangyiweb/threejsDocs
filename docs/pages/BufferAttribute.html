BufferAttribute

该类用于存储与几何体关联的某个属性（例如顶点位置、面索引、法线、颜色、UV 以及任意自定义属性）的数据，从而更高效地将数据传递给 GPU。

在处理类似向量的数据时，向量类与颜色类上的 fromBufferAttribute( attribute, index ) 辅助方法可能会很有用，例如：Vector3#fromBufferAttribute。

构造函数

new BufferAttribute( array : TypedArray, itemSize : number, normalized : boolean )

构造一个新的缓冲属性。

array

保存属性数据的数组。
itemSize

单个条目的大小。
normalized

数据是否进行归一化。

默认值为 false。

属性

.array : TypedArray

保存属性数据的数组。它应包含 itemSize * numVertices 个元素，其中 numVertices 为关联几何体中的顶点数量。

.count : number (readonly)

表示此缓冲属性存储的条目数量。内部通过将 array 的长度除以 itemSize 计算得出。

.gpuType : FloatType | IntType

配置绑定的 GPU 类型，供着色器使用。

注意：此属性仅对整数数组有效，浮点数组不可配置。如需更低精度的浮点类型，请使用 Float16BufferAttribute。

默认值为 FloatType。

.id : number (readonly)

缓冲属性的 ID。

.isBufferAttribute : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.itemSize : number

数组中与单个顶点关联的数值个数。例如，若此属性存储 3 分量向量（如位置、法线或颜色），则该值应为 3。

.name : string

缓冲属性的名称。

.needsUpdate : number

标记此属性已更改、需要重新传输到 GPU。修改数组中的值时，请将其设为 true。

默认值为 false。

.normalized : boolean

仅适用于整数数据。表示缓冲区中的底层数据如何映射到 GLSL 代码中的值。例如，若 array 为 UInt16Array 的实例，且 normalized 为 true，则数组中 0 - +65535 的值将映射为 GLSL 属性中的 0.0f - +1.0f。若 normalized 为 false，数值将直接转换为浮点数而不做映射，即 65535 变为 65535.0f。

.updateRanges : Array.<Object>

可用于仅更新所存储向量的部分分量（例如仅更新与颜色相关的分量）。使用 addUpdateRange() 函数向此数组添加更新范围。

.usage : StaticDrawUsage | DynamicDrawUsage | StreamDrawUsage | StaticReadUsage | DynamicReadUsage | StreamReadUsage | StaticCopyUsage | DynamicCopyUsage | StreamCopyUsage

定义数据存储的预期用法模式，以便优化。

注意：缓冲区在首次使用后，其用法不可再更改。如需更改，请创建新实例并在下次渲染前设置所需用法。

默认值为 StaticDrawUsage。

.version : number

版本号，每当 needsUpdate 设为 true 时递增。

方法

.addUpdateRange( start : number, count : number )

添加数据数组中需要在 GPU 上更新的数据范围。

start

开始更新的位置。
count

要更新的分量数量。

.clearUpdateRanges()

.clone() : BufferAttribute

返回值： 本实例的克隆。

返回值： 对该实例的引用。

.copyArray( array : TypedArray | Array ) : BufferAttribute

将给定的数组数据复制到此缓冲属性。

要复制的数组。
返回值： 对该实例的引用。

.copyAt( index1 : number, attribute : BufferAttribute, index2 : number ) : BufferAttribute

从给定缓冲属性复制一个向量到此缓冲属性。属性缓冲区中的源位置与目标位置由给定索引表示。

index1

此缓冲属性中的目标索引。
attribute

要复制的源缓冲属性。
index2

给定缓冲属性中的源索引。
返回值： 对该实例的引用。

.dispose()

销毁该缓冲属性。仅在 WebGPURenderer 中可用。

.onUpload( callback : function ) : BufferAttribute

设置给定回调函数，在 Renderer 将属性数组数据传输到 GPU 后执行。可用于在上传完成后、CPU 侧不再需要属性数据时执行清理操作。

callback

onUpload() 回调。
返回值： 对该实例的引用。

.onUploadCallback()

在 renderer 将属性数组数据传输到 GPU 后执行的回调函数。
