InstancedInterleavedBuffer

交错缓冲区的实例化版本。

构造函数

new InstancedInterleavedBuffer( array : TypedArray, stride : number, meshPerAttribute : number )

构造一个新的实例化交错缓冲区。

array

共享缓冲区中存储属性数据的类型化数组。
stride

每个顶点对应的类型化数组元素数量。
meshPerAttribute

定义该交错缓冲区的值应重复使用的次数。

默认值为 1。

属性

.isInstancedInterleavedBuffer : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.meshPerAttribute : number

定义该缓冲属性的值应重复使用的次数，参见 InstancedBufferAttribute#meshPerAttribute。

默认值为 1。
