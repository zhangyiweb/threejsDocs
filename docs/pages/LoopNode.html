LoopNode

此模块提供了多种在 TSL 中实现循环的方式。其基本形式为：

Loop( count, ( { i } ) => {
} );

此外，也可以定义起始和结束范围、数据类型以及循环条件：

Loop( { start: int( 0 ), end: int( 10 ), type: 'int', condition: '<' }, ( { i } ) => {
} );

嵌套循环可以用紧凑形式定义：

Loop( 10, 5, ( { i, j } ) => {
} );

需要反向运行的循环可以这样定义：

Loop( { start: 10 }, () => {} );

也可以使用布尔值执行，类似于 while 语法。

const value = float( 0 ).toVar();
Loop( value.lessThan( 10 ), () => {
	value.addAssign( 1 );
} );

该模块还提供了用于循环控制的 Break() 和 Continue() TSL 表达式。

构造函数

new LoopNode( params : Array.<any> )

构造一个新的循环节点。

params

根据循环类型，数组中保存循环的不同参数化值。

方法

.getProperties( builder : NodeBuilder ) : Object

返回关于此节点的属性。

builder

当前的节点构建器。
返回值： 节点属性。

.getVarName( index : number ) : string

根据索引返回循环变量名。模式为 0 = i，1= j，2= k，以此类推。

index

索引。
返回值： 循环变量名。
