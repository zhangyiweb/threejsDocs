MorphAnimMesh

一种特殊的动画网格，提供简洁的动画播放接口。 它只允许播放单个动画，切换动画时无过渡或淡入淡出效果。

导入

MorphAnimMesh 是一个插件，必须显式导入，参见 Installation#Addons。

import { MorphAnimMesh } from 'three/addons/misc/MorphAnimMesh.js';

构造函数

new MorphAnimMesh( geometry : BufferGeometry, material : Material | Array.<Material> )

构造一个新的 morph 动画网格。

geometry

网格几何体。
material

网格材质。

属性

.activeAction : AnimationAction

当前活动的动画动作。

默认值为 null。

.mixer : AnimationMixer

内部动画混合器。

方法

.playAnimation( label : string, fps : number )

播放指定的动画片段。实现假设动画片段存储在 Object3D#animations 或几何体中。

label

动画片段的名称。
fps

动画片段的 FPS。

.setDirectionBackward()

将动画播放方向设置为「向后」。

.setDirectionForward()

将动画播放方向设置为「向前」。

.updateAnimation( delta : number )

更新网格的动画。必须在动画循环中调用。

delta

增量时间（秒）。
