HDRLoader

用于加载 RGBE HDR 纹理格式的加载器。

代码示例

const loader = new HDRLoader();
const envMap = await loader.loadAsync( 'textures/equirectangular/blouberg_sunrise_2_1k.hdr' );
envMap.mapping = THREE.EquirectangularReflectionMapping;
scene.environment = envMap;

导入

HDRLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { HDRLoader } from 'three/addons/loaders/HDRLoader.js';

构造函数

new HDRLoader( manager : LoadingManager )

构造一个新的 RGBE/HDR 加载器。

manager

加载管理器。

属性

.type : HalfFloatType | FloatType

纹理类型。

默认值为 HalfFloatType。

方法

.parse( buffer : ArrayBuffer ) : DataTextureLoader~TexData

解析给定的 RGBE 纹理数据。

buffer

原始纹理数据。
重写： DataTextureLoader#parse
返回值： 表示已解析纹理数据的对象。

.setDataType( value : HalfFloatType | FloatType ) : HDRLoader

设置纹理类型。

value

要设置的纹理类型。
返回值： 此加载器的引用。
