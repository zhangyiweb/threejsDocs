Matrix2

表示一个 2x2 矩阵。

关于行主序与列主序的说明：

构造函数和 Matrix2#set 方法按 行主序 接收参数，而内部则按列主序存储在 Matrix2#elements 数组中。 这意味着调用：

const m = new THREE.Matrix2();
m.set( 11, 12,
       21, 22 );

会使 elements 数组包含：

m.elements = [ 11, 21,
               12, 22 ];

并且内部所有计算都使用列主序进行。 不过，由于实际排序在数学上并无差异，且大多数人习惯按行主序思考矩阵， three.js 文档以行主序展示矩阵。请记住：若阅读源码，需要将此处列出的矩阵取转置，才能对应理解计算过程。

构造函数

new Matrix2( n11 : number, n12 : number, n21 : number, n22 : number )

构造一个新的 2x2 矩阵。参数应按行主序提供。若未提供参数，构造函数将矩阵初始化为单位矩阵。

n11

1-1 矩阵元素。
n12

1-2 矩阵元素。
n21

2-1 矩阵元素。
n22

2-2 矩阵元素。

类

Matrix2

属性

.elements : Array.<number>

按列主序排列的矩阵值列表。

方法

.fromArray( array : Array.<number>, offset : number ) : Matrix2

根据给定数组设置矩阵元素。

array

按列主序排列的矩阵元素。
offset

数组中第一个元素的索引。

默认值为 0。
返回值： 此矩阵的引用。

.identity() : Matrix2

将此矩阵设为 2x2 单位矩阵。

返回值： 此矩阵的引用。

.set( n11 : number, n12 : number, n21 : number, n22 : number ) : Matrix2

设置矩阵元素。参数应按行主序提供。

n11

1-1 矩阵元素。
n12

1-2 矩阵元素。
n21

2-1 矩阵元素。
n22

2-2 矩阵元素。
返回值： 此矩阵的引用。
