TemporalReprojectNode

用于对屏幕空间效果（SSGI、SSR 等）进行降噪的时域重投影通道。

两种模式共享几何加权 4 点双线性历史采样与 YCoCg 方差裁剪。表面速度重投影始终优先采样。镜面（Specular）模式随后在该表面结果之上混合命中点视差历史。漫反射（Diffuse）模式则应用速度场散度以检测表面拉伸。

与基于抖动的 TAA/TAAU 不同，此节点不应用相机亚像素抖动——仅使用运动向量重投影并累积历史帧。

导入

TemporalReprojectNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { temporalReproject } from 'three/addons/tsl/display/TemporalReprojectNode.js';

属性

.accumulate : boolean

当为 true 时，每帧将解析输出复制到内部历史缓冲区。 当为 false 时，历史帧通过 TemporalReprojectNode#setHistoryTexture 从外部提供。

.mode : TemporalReprojectMode

方法

.setHistoryTexture( source : Object | Texture )

提供外部历史源（例如 RecurrentDenoiseNode 或其纹理）。仅在 TemporalReprojectNode#accumulate 为 false 时使用。
