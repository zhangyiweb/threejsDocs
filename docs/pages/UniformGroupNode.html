UniformGroupNode

此节点可用于将 UniformNode 的单个实例分组， 并将它们作为 uniform 缓冲区进行管理。

在大多数情况下，定义 UniformNode#groupNode 属性时 会使用预定义节点 objectGroup、renderGroup 和 frameGroup。

  • objectGroup：每个对象一个 uniform 缓冲区。
  • renderGroup：共享的 uniform 缓冲区，每次渲染调用更新一次。
  • frameGroup：共享的 uniform 缓冲区，每帧更新一次。

构造函数

new UniformGroupNode( name : string, shared : boolean, order : number, updateType : string | null )

构造一个新的 uniform 组节点。

name

uniform 组节点的名称。
shared

此 uniform 组节点是否共享。

默认值为 false。
order

影响内部排序。

默认值为 1。
updateType

uniform 组节点的更新类型。

默认值为 null。
