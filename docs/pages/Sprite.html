Sprite

精灵是一个始终面向相机的平面，通常会应用 部分透明的纹理。

精灵不会投射阴影，将 Object3D#castShadow 设为 true 不会产生任何效果。

代码示例

const map = new THREE.TextureLoader().load( 'sprite.png' );
const material = new THREE.SpriteMaterial( { map: map } );
const sprite = new THREE.Sprite( material );
scene.add( sprite );

构造函数

new Sprite( material : SpriteMaterial | SpriteNodeMaterial )

构造一个新的精灵。

material

精灵材质。

属性

.center : Vector2

精灵的锚点，也是精灵旋转所围绕的点。 值为 (0.5, 0.5) 对应精灵的中点。值为 (0, 0) 对应精灵的左下角。

默认值为 (0.5,0.5)。

.count : number

该精灵的实例数量。 仅可与 WebGPURenderer 一起使用。

默认值为 1。

.geometry : BufferGeometry

精灵几何体。

.isSprite : boolean (readonly)

该标志可用于类型检测。

默认值为 true。

.material : SpriteMaterial | SpriteNodeMaterial

精灵材质。

方法

.raycast( raycaster : Raycaster, intersects : Array.<Object> )

计算投射光线与该精灵之间的交点。

raycaster

光线投射器。
intersects

用于保存交点的目标数组。
重写： Object3D#raycast
