Projector

此类可将给定的 3D 空间场景投影为用于 2D API 渲染的 2D 表示。 Projector 目前由 SVGRenderer 使用， 此前也曾被旧版 CanvasRenderer 使用。

导入

Projector 是一个插件，必须显式导入，参见 Installation#Addons。

import { Projector } from 'three/addons/renderers/Projector.js';

构造函数

new Projector()

构造一个新的投影器。

方法

.projectScene( scene : Object3D, camera : Camera, sortObjects : boolean, sortElements : boolean ) : Object

将给定的 3D 空间场景投影为 2D 表示。结果 是一个包含可渲染项的对象。

scene

场景或其他类型的 3D 对象。
camera

相机。
sortObjects

是否对对象进行排序。
sortElements

是否对元素（面、线和精灵）进行排序。
返回值： 投影后的场景，表示为可渲染对象。
