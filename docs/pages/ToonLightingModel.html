ToonLightingModel

表示卡通材质的光照模型。用于 MeshToonNodeMaterial。

构造函数

new ToonLightingModel()

方法

.direct( lightData : Object, builder : NodeBuilder )

实现直接光照。不同于常规的平滑辐照，辐照会被 离散为少量色阶，以营造类似漫画的平面风格。

lightData

光照数据。
builder

当前节点构建器。
重写： LightingModel#direct

.indirect( builder : NodeBuilder )

实现间接光照。

builder

当前节点构建器。
重写： LightingModel#indirect
