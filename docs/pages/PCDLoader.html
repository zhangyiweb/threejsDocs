PCDLoader

用于加载 Point Cloud Data（PCD）格式的加载器。

PCDLoader 支持 ASCII 和（压缩）二进制文件，以及以下 PCD 字段：

  • x y z
  • rgb
  • normal_x normal_y normal_z
  • intensity
  • label

代码示例

const loader = new PCDLoader();
const points = await loader.loadAsync( './models/pcd/binary/Zaghetto.pcd' );
points.geometry.center(); // optional
points.geometry.rotateX( Math.PI ); // optional
scene.add( points );

导入

PCDLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { PCDLoader } from 'three/addons/loaders/PCDLoader.js';

构造函数

new PCDLoader( manager : LoadingManager )

构造一个新的 PCD 加载器。

manager

加载管理器。

属性

.littleEndian : boolean

是否使用小端字节序。

默认值为 true。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 PCD 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( data : ArrayBuffer ) : Points

解析给定的 PCD 数据并返回点云。

data

原始 PCD 数据 ArrayBuffer。
重写： Loader#parse
返回值： 解析后的点云。
