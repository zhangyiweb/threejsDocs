LineBasicMaterial

用于渲染线条图元的材质。

材质定义可渲染 3D 对象的外观。

代码示例

const material = new THREE.LineBasicMaterial( { color: 0xffffff } );

构造函数

new LineBasicMaterial( parameters : Object )

构造一个新的基础线材质。

parameters

包含一个或多个属性的对象，用于定义材质外观。 材质的任意属性（包括继承材质中的任意属性）都可在此传入。 颜色值可以是 Color#set 所接受的任意类型。

属性

.color : Color

材质颜色。

默认值为 (1,1,1)。

.fog : boolean

材质是否受雾影响。

默认值为 true。

.isLineBasicMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.linecap : 'butt' | 'round' | 'square'

定义线端外观。

仅可与 SVGRenderer 一起使用。

默认值为 'round'。

.linejoin : 'round' | 'bevel' | 'miter'

定义线连接处外观。

仅可与 SVGRenderer 一起使用。

默认值为 'round'。

.linewidth : number

控制线条粗细。

仅可与 SVGRenderer 一起使用。WebGL 和 WebGPU 会忽略此设置，并始终以一像素宽度渲染线条图元。

默认值为 1。

.map : Texture

使用纹理数据设置线条颜色。纹理贴图颜色 会与漫反射 color 相乘调制。

map 表示颜色数据，纹理必须指定 Texture#colorSpace。大多数 map 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。
