XRManager

XR 管理器基于 WebXR Device API 构建， 用于通过渲染器后端管理 XR 会话。

构造函数

new XRManager( renderer : Renderer, multiview : boolean )

构造一个新的 XR 管理器。

renderer

渲染器。
multiview

若设备支持，则启用多视图（multiview）。

默认值为 false。

属性

.cameraAutoUpdate : boolean

XR 相机是否应自动更新。

默认值为 true。

.enabled : boolean

此标志全局启用 XR 渲染。

默认值为 false。

.isPresenting : boolean (readonly)

XR 设备当前是否正在呈现。

默认值为 false。

方法

.createCylinderLayer( radius : number, centralAngle : number, aspectratio : number, translation : Vector3, quaternion : Quaternion, pixelwidth : number, pixelheight : number, rendercall : function, attributes : Object ) : Mesh

此方法可在 XR 应用中用于创建柱面层，以呈现单独的 渲染场景。

radius

圆柱半径（世界单位）。
centralAngle

圆柱中心角（弧度）。
aspectratio

宽高比。
translation

层平面的位置/平移（世界单位）。
quaternion

层平面的朝向，以四元数表示。
pixelwidth

层渲染目标的宽度（像素）。
pixelheight

层渲染目标的高度（像素）。
rendercall

渲染该层的回调函数。与默认动画循环中的代码类似， 此方法可用于更新/变换层场景中的 3D 对象。
attributes

用于配置层的渲染目标。

默认值为 {}。
返回值： 表示柱面 XR 层的网格。此网格应添加到 XR 场景中。

.createQuadLayer( width : number, height : number, translation : Vector3, quaternion : Quaternion, pixelwidth : number, pixelheight : number, rendercall : function, attributes : Object ) : Mesh

此方法可在 XR 应用中用于创建四边形层，以呈现单独的 渲染场景。

width

层平面的宽度（世界单位）。
height

层平面的高度（世界单位）。
translation

层平面的位置/平移（世界单位）。
quaternion

层平面的朝向，以四元数表示。
pixelwidth

层渲染目标的宽度（像素）。
pixelheight

层渲染目标的高度（像素）。
rendercall

渲染该层的回调函数。与默认动画循环中的代码类似， 此方法可用于更新/变换层场景中的 3D 对象。
attributes

用于配置层的渲染目标。

默认值为 {}。
返回值： 表示四边形 XR 层的网格。此网格应添加到 XR 场景中。

.foveateBoundTexture( renderTarget : RenderTarget )

对第一个 XR 渲染通道在合成到投影层之前所使用的 内部后处理渲染目标应用 WebXR 固定注视点渲染（fixed foveation）。

浏览器端 XRWebGLBinding.foveateBoundTexture() 的失败视为 非致命错误，以免中断渲染。

renderTarget

内部渲染目标。

.getBaseLayer() : XRWebGLLayer | XRProjectionLayer

返回当前的基础层。

当目标 XR 设备支持 WebXR Layers API 时，这是一个 XRProjectionLayer， 否则为 XRWebGLLayer。

返回值： XR 基础层。

.getBinding() : XRWebGLBinding

返回当前的 XR 绑定。

如有需要且浏览器支持，会创建新的绑定。

返回值： XR 绑定。若无法创建则返回 null。

.getCamera() : ArrayCamera

返回 XR 相机。

返回值： XR 相机。

.getController( index : number ) : Group

返回一个表示 XR 控制器在目标射线空间（target ray space）中 变换的 THREE.Group 实例。所请求的控制器由给定索引定义。

index

XR 控制器的索引。
返回值： 表示控制器变换的组。

.getControllerGrip( index : number ) : Group

返回一个表示 XR 控制器在握持空间（grip space）中 变换的 THREE.Group 实例。所请求的控制器由给定索引定义。

index

XR 控制器的索引。
返回值： 表示控制器变换的组。

.getEnvironmentBlendMode() : 'opaque' | 'additive' | 'alpha-blend' | undefined

从当前 XR 会话返回环境混合模式。

返回值： 环境混合模式。在 XR 会话外使用时返回 undefined。

.getFoveation() : number | undefined

返回注视点渲染（foveation）值。

返回值： 注视点渲染值。

.getFrame() : XRFrame

返回当前的 XR 帧。

返回值： XR 帧。在 XR 会话外使用时返回 null。

.getFramebufferScaleFactor() : number

返回帧缓冲缩放因子。

返回值： 帧缓冲缩放因子。

.getHand( index : number ) : Group

返回一个表示 XR 控制器在手部空间（hand space）中 变换的 THREE.Group 实例。所请求的控制器由给定索引定义。

index

XR 控制器的索引。
返回值： 表示控制器变换的组。

.getReferenceSpace() : XRReferenceSpace

返回 XR 参考空间。

返回值： XR 参考空间。

.getReferenceSpaceType() : XRReferenceSpaceType

返回参考空间类型。

返回值： 参考空间类型。

.getSession() : XRSession

返回当前的 XR 会话。

返回值： XR 会话。在 XR 会话外使用时返回 null。

.getWebGPUBinding() : XRGPUBinding

返回当前的 XR WebGPU 绑定。

如有需要且浏览器支持，会创建新的绑定。

返回值： XR WebGPU 绑定。若无法创建则返回 null。

.renderLayers()

渲染先前已添加到场景中的 XR 层。

此方法通常在动画循环中、通过 renderer.render( scene, camera ); 渲染实际场景之前调用。

.setFoveation( foveation : number )

设置注视点渲染（foveation）值。

foveation

[0,1] 范围内的数值，其中 0 表示无注视点渲染（全分辨率）， 1 表示最大注视点渲染（边缘以较低分辨率渲染）。

.setFramebufferScaleFactor( factor : number )

设置帧缓冲缩放因子。

此方法不能在 XR 会话期间使用。

factor

帧缓冲缩放因子。

.setReferenceSpace( space : XRReferenceSpace )

设置自定义 XR 参考空间。

space

XR 参考空间。

.setReferenceSpaceType( type : XRReferenceSpaceType )

设置参考空间类型。

此方法不能在 XR 会话期间使用。

type

参考空间类型。

.setSession( session : XRSession ) : Promise (async)

在通常通过某个 *Button 模块请求 XR 会话之后， 使用此方法将其注入渲染器。此方法会触发实际 XR 渲染的开始。

session

要设置的 XR 会话。
返回值： 在会话设置完成时解析的 Promise。

.updateCamera( camera : PerspectiveCamera )

此方法由渲染器每帧调用，并根据给定相机更新 XR 相机 及其子相机。给定的相机是在应用层创建、用于非 XR 渲染的 “用户”相机。

camera

相机。

.useMultiview() : boolean

若引擎渲染到多视图目标，则返回 true。

返回值： 引擎是否渲染到多视图渲染目标。
