Skeleton

用于表示 three.js 中骨骼结构的类。骨架 由骨骼层级定义。

代码示例

const bones = [];
const shoulder = new THREE.Bone();
const elbow = new THREE.Bone();
const hand = new THREE.Bone();
shoulder.add( elbow );
elbow.add( hand );
bones.push( shoulder , elbow, hand);
shoulder.position.y = -5;
elbow.position.y = 0;
hand.position.y = 5;
const armSkeleton = new THREE.Skeleton( bones );

构造函数

new Skeleton( bones : Array.<Bone>, boneInverses : Array.<Matrix4> )

构造一个新的骨架。

bones

骨骼数组。
boneInverses

骨骼逆矩阵数组。 若未提供，这些矩阵将通过 Skeleton#calculateInverses 自动计算。

属性

.boneInverses : Array.<Matrix4>

骨骼逆矩阵数组。

.boneMatrices : Float32Array

保存骨骼数据的数组缓冲区。 作为 Skeleton#boneTexture 的输入数据。

默认值为 null。

.boneTexture : DataTexture

保存骨骼数据的纹理，供 顶点着色器使用。

默认值为 null。

.bones : Array.<Bone>

定义骨架的骨骼数组。

方法

.calculateInverses()

计算骨骼逆矩阵。此方法会重置 Skeleton#boneInverses 并用新的矩阵填充它。

.clone() : Skeleton

返回一个从此实例复制值的新骨架。

返回值： 此实例的克隆。

.computeBoneTexture() : Skeleton

计算用于将骨骼数据传递给顶点着色器的数据纹理。

返回值： 此实例的引用。

.dispose()

释放此实例分配的与 GPU 相关的资源。当应用中 不再使用此实例时，应调用此方法。

.fromJSON( json : Object, bones : Object.<string, Bone> ) : Skeleton

根据给定的 JSON 和骨骼设置骨架。

json

序列化为 JSON 的骨架数据。
bones

骨骼数组。
返回值： 此实例的引用。

.getBoneByName( name : string ) : Bone | undefined

在骨架的骨骼数组中搜索，并返回第一个名称匹配的骨骼。

name

骨骼名称。
返回值： 找到的骨骼。若未找到则返回 undefined。

.init()

初始化骨架。此方法会由构造函数自动调用， 但根据骨架的创建方式，有时可能需要手动调用此方法。

.pose()

将骨架重置为基础姿势。

.toJSON() : Object

将骨架序列化为 JSON。

参见：
  • ObjectLoader#parse
返回值： 表示序列化骨架的 JSON 对象。

.update()

将骨架重置为基础姿势。
