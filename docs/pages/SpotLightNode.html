SpotLightNode

用于将聚光灯表示为节点的模块。

构造函数

new SpotLightNode( light : SpotLight )

构造一个新的聚光灯节点。

light

聚光灯光源。

默认值为 null。

属性

.colorNode : UniformNode.<Color>

表示灯光颜色的 uniform 节点。

重写： AnalyticLightNode#colorNode

.coneCosNode : UniformNode.<float>

表示锥体余弦值的 uniform 节点。

.cutoffDistanceNode : UniformNode.<float>

表示截止距离的 uniform 节点。

.decayExponentNode : UniformNode.<float>

表示衰减指数的 uniform 节点。

.penumbraCosNode : UniformNode.<float>

表示半影余弦值的 uniform 节点。

方法

.getSpotAttenuation( builder : NodeBuilder, angleCosine : Node.<float> ) : Node.<float>

计算给定角度的聚光灯衰减。

builder

节点构建器。
angleCosine

用于计算聚光灯衰减的角度。
返回值： 聚光灯衰减。

.update( frame : NodeFrame )

被重写以更新聚光灯特有的 uniform。

frame

当前节点帧的引用。
重写： AnalyticLightNode#update
