NodeMaterial

所有节点材质的基类。

构造函数

new NodeMaterial()

构造一个新的节点材质。

属性

.alphaTestNode : Node.<float>

节点材质的 alpha 测试默认由 alphaTest 属性推断。此节点属性允许覆盖默认值，改用 节点定义 alpha 测试。

若不想覆盖 alpha 测试，而只想修改现有 值，请使用 materialAlphaTest。

默认值为 null。

.aoNode : Node.<float>

节点材质的光照可能受环境光遮蔽影响。 默认 AO 由赋给 aoMap 的环境光遮蔽贴图 以及相应的 aoMapIntensity 推断。此节点属性允许覆盖 默认值，改用自定义节点定义环境光遮蔽。

若不想覆盖漫反射颜色，而只想修改现有 值，请使用 materialAO。

默认值为 null。

.backdropAlphaNode : Node.<float>

此节点可用于调节 backdropNode 对出射光的影响。

默认值为 null。

.backdropNode : Node.<vec3>

此节点可用于实现多种类似滤镜的效果。思路是 将当前渲染结果存入纹理（例如通过 viewportSharedTexture()），用它 创建任意效果，然后将节点组合赋给此属性。 使用此材质的对象后方的一切现在都会受滤镜影响。

const material = new NodeMaterial()
material.transparent = true;
// everything behind the object will be monochromatic
material.backdropNode = saturation( viewportSharedTexture().rgb, 0 );

背景计算是光照的一部分，因此只有受光材质才能使用此属性。

默认值为 null。

.castShadowNode : Node.<vec4>

此节点可用于影响使用此节点材质的对象 如何投射阴影。要为阴影应用颜色，可以简单地这样做：

material.castShadowNode = vec4( 1, 0, 0, 1 );

这对于伪造半透明对象的彩色阴影很有用。也 常见的做法是将该属性与 Fn 函数一起使用，以便按片段执行检查。

materialCustomShadow.castShadowNode = Fn( () => {
	hash( vertexIndex ).greaterThan( 0.5 ).discard();
	return materialColor;
} )();

默认值为 null。

.castShadowPositionNode : Node.<float>

允许覆盖用于阴影贴图投影的几何体位置，默认 为 positionLocal（局部空间中的顶点位置）。

默认值为 null。

.colorNode : Node.<vec3>

节点材质的漫反射颜色默认由 color 和 map 属性推断。此节点属性允许覆盖默认值， 改用节点定义漫反射颜色。

material.colorNode = color( 0xff0000 ); // define red color

若不想覆盖漫反射颜色，而只想修改现有 值，请使用 materialColor。

material.colorNode = materialColor.mul( color( 0xff0000 ) ); // give diffuse colors a red tint

默认值为 null。

.contextNode : ContextNode

此节点可用作该材质的全局上下文管理组件。

默认值为 null。

.depthNode : Node.<float>

允许在片段着色器中覆盖深度值。

默认值为 null。

.envNode : Node.<vec3>

节点材质的环境可由赋给 envMap 属性的环境 贴图定义；若节点材质是 PBR 材质，也可由 Scene.environment 定义。此节点属性允许覆盖 默认行为，改用自定义节点定义环境。

material.envNode = pmremTexture( renderTarget.texture );

默认值为 null。

.fog : boolean

此材质是否受雾影响。

默认值为 true。

.fragmentNode : Node.<vec4>

若需要在实现片段着色器时获得完全自由，可使用此节点属性。 赋给节点将替换片段阶段中使用的内置材质逻辑。

默认值为 null。

.geometryNode : function

此节点属性面向一次性或每动画步修改几何体数据的逻辑。 应用通常会把此类逻辑随意放在初始化例程或动画循环中。 geometryNode 旨在作为专用 API，以便有明确的位置来实现 几何体修改。

思路是赋一个持有几何体修改逻辑的 Fn 定义。典型示例 是基于 GPU 的粒子系统，它在应用层提供节点材质。粒子 模拟会实现为计算着色器，并在 Fn 函数内管理。该函数最终 会赋给 geometryNode。

默认值为 null。

.isNodeMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.lights : boolean

此材质是否受灯光影响。

默认值为 false。

.lightsNode : LightsNode

将 lights 属性设为 true 的节点材质 会受场景中所有灯光影响。有时需要选择性 光照，即场景中只有部分灯光 影响材质。可以通过创建带有选择性 灯光列表的 LightsNode 实例，并将该节点赋给此属性来实现。

const customLightsNode = lights( [ light1, light2 ] );
material.lightsNode = customLightsNode;

默认值为 null。

.maskNode : Node.<bool>

若遮罩值为 false，则丢弃该片段。

默认值为 null。

.maskShadowNode : Node.<bool>

此节点可用于为材质实现阴影遮罩。

默认值为 null。

.mrtNode : MRTNode

MRT 配置在渲染器或通道级别完成。此节点允许在材质级别 覆盖写入 MRT 目标的值。这 对于实现只应影响特定对象的选择性 FX 功能很有用。

默认值为 null。

.normalNode : Node.<vec3>

节点材质的法线默认由 normalMap/normalScale 或 bumpMap/bumpScale 属性推断。此节点属性允许覆盖默认值， 改用节点定义法线。

若不想覆盖法线，而只想修改现有值， 请使用 materialNormal。

默认值为 null。

.opacityNode : Node.<float>

节点材质的不透明度默认由 opacity 和 alphaMap 属性推断。此节点属性允许覆盖默认值， 改用节点定义不透明度。

若不想覆盖不透明度，而只想修改现有 值，请使用 materialOpacity。

默认值为 null。

.outputNode : Node.<vec4>

此节点可用于定义材质的最终输出。

TODO: Explain the differences to fragmentNode.

默认值为 null。

.positionNode : Node.<vec3>

局部顶点位置基于属性数据、变形或蒙皮等多个因素计算。 此节点属性允许覆盖 默认值，改用节点定义局部顶点位置。

若不想覆盖顶点位置，而只想修改现有 值，请使用 positionLocal。

material.positionNode = positionLocal.add( displace );

默认值为 null。

.receivedShadowNode : function | FunctionNode.<vec4>

此节点可用于影响使用此节点材质的对象 如何接收阴影。

const totalShadows = float( 1 ).toVar();
material.receivedShadowNode = Fn( ( [ shadow ] ) => {
	totalShadows.mulAssign( shadow );
	//return float( 1 ); // bypass received shadows
	return shadow.mix( color( 0xff0000 ), 1 ); // modify shadow color
} );

默认值为 null。

.receivedShadowPositionNode : Node.<float>

允许覆盖用于阴影贴图渲染的位置，默认 为 positionWorld（世界空间中的顶点位置）。

默认值为 null。

.type : string

表示节点材质的类型。

默认值为 null。

.setupOutput( builder : NodeBuilder, outputNode : Node.<vec4> ) : Node.<vec4>

设置输出节点。

派生材质可以实现此方法以扩展材质输出的功能， 或完全替换它。

class ColoredShadowMaterial extends MeshPhongNodeMaterial {
  constructor( parameters ) {
    super( parameters );
    this._shadeColor = uniform( new Color( parameters.shadeColor ?? 0xff0000 ) );
  }
  setupOutput( builder, outputNode ) {
	   // Modify the native output of the MeshPhongNodeMaterial fragment shader
    const brightness = min( outputNode.r, 1.0 );
    const mixedColor = mix( this._shadeColor, diffuseColor.rgb, brightness );
	   // Return new output back into NodeMaterial flow
    return super.setupOutput( builder, vec4( mixedColor, outputNode.a ) );
  }
}
builder

当前节点构建器。
outputNode

现有的输出节点。
返回值： 输出节点。
