ForestGenerator

在 TerrainGenerator（或任何暴露 sampleHeight、 sampleSlope、minY、maxY 和 parameters.size 的对象）上铺满一片森林， 用单次绘制调用渲染数十万棵树。

每棵树都是仍能辨认为树的最简形态：约 20 面的二十面体球被压成锥状泪滴， 再叠一点噪声，并带有预烘焙的底部暗、顶部亮渐变。每棵仅数十个三角形， 因此五十万棵树组成的单个 THREE.InstancedMesh 只需一次绘制调用。树木 通过拒绝采样按生态规则放置——海拔上下限（雾层之上、雪线以下）、坡度限制（悬崖上不种） 以及打开空地的低频密度遮罩——再在偏航、倾斜与（平方偏置的）缩放上抖动， 使林分不会读起来像复制粘贴。

代码示例

const forest = new ForestGenerator( { count: 500000 } );
scene.add( forest.build( terrain ) );

构造函数

new ForestGenerator()
