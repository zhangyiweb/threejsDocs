SVGLoader

用于加载 SVG 格式的加载器。

可缩放矢量图形（Scalable Vector Graphics）是一种基于 XML 的二维矢量图像格式， 支持交互与动画。

代码示例

const loader = new SVGLoader();
const data = await loader.loadAsync( 'data/svgSample.svg' );
const paths = data.paths;
const group = new THREE.Group();
for ( let i = 0; i < paths.length; i ++ ) {
	const path = paths[ i ];
	const material = new THREE.MeshBasicMaterial( {
		color: path.color,
		side: THREE.DoubleSide,
		depthWrite: false
	} );
	const shapes = SVGLoader.createShapes( path );
	for ( let j = 0; j < shapes.length; j ++ ) {
		const shape = shapes[ j ];
		const geometry = new THREE.ShapeGeometry( shape );
		const mesh = new THREE.Mesh( geometry, material );
		group.add( mesh );
	}
}
scene.add( group );

导入

SVGLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { SVGLoader } from 'three/addons/loaders/SVGLoader.js';

构造函数

new SVGLoader( manager : LoadingManager )

构造一个新的 SVG 加载器。

manager

加载管理器。

属性

.defaultDPI : number

默认每英寸点数。

默认值为 90。

.defaultUnit : 'mm' | 'cm' | 'in' | 'pt' | 'pc' | 'px'

默认单位。

默认值为 'px'。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 SVG 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( text : string ) : Object

解析给定的 SVG 数据并返回生成的数据。

text

原始 SVG 数据字符串。
重写： Loader#parse
返回值： 包含形状路径数组和 SVG XML 文档的对象。

静态方法

.createFillMaterial( shapePath : ShapePath ) : MeshBasicMaterial

为给定路径的填充渲染创建材质。

shapePath

形状路径。
返回值： 填充材质。若路径无填充则为 null。

.createShapes( shapePath : ShapePath ) : Array.<Shape>

根据给定形状路径创建形状数组。

shapePath

形状路径。
已弃用： 自 185 版本起。
返回值： 形状数组。

.createStrokeMaterial( shapePath : ShapePath ) : MeshBasicMaterial

为给定路径的描边渲染创建材质。

shapePath

形状路径。
返回值： 描边材质。若路径无描边则为 null。

.getStrokeStyle( width : number, color : string, lineJoin : 'round' | 'bevel' | 'miter' | 'miter-limit', lineCap : 'round' | 'square' | 'butt', miterLimit : number ) : Object

根据给定参数返回描边样式对象。

width

描边宽度。

默认值为 1。
color

描边颜色，由 Color#getStyle 返回的格式。

默认值为 '#000'。
lineJoin

线段连接样式。

默认值为 'miter'。
lineCap

线段端点样式。

默认值为 'butt'。
miterLimit

最大连接长度，以 width 参数的倍数表示（超出该距离时连接将被截断）。

默认值为 4。
返回值： 样式对象。

.pointsToStroke( points : Array.<Vector2>, style : Object, arcDivisions : number, minDistance : number ) : BufferGeometry

从点数组创建描边。

points

二维空间中的点。至少需要 2 个点。路径可以是开放的或闭合的（最后一个点等于第一个点）。
style

包含 SVG 属性的对象，由 SVGLoader.getStrokeStyle() 返回，或由 SVGLoader.parse() 写入 path.userData.style 对象。
arcDivisions

圆角连接和端帽的圆弧分段数。

默认值为 12。
minDistance

距离小于此值的点将被合并。

默认值为 0.001。
返回值： 描边几何体。会生成 UV 坐标（'u' 沿路径方向，'v' 横跨路径，从左到右）。 若未生成几何体则返回 null。

.pointsToStrokeWithBuffers( points : Array.<Vector2>, style : Object, arcDivisions : number, minDistance : number, vertices : Array.<number>, normals : Array.<number>, uvs : Array.<number>, vertexOffset : number ) : number

从点数组创建描边。

points

二维空间中的点。至少需要 2 个点。
style

包含 SVG 属性的对象，由 SVGLoader.getStrokeStyle() 返回，或由 SVGLoader.parse() 写入 path.userData.style 对象。
arcDivisions

圆角连接和端帽的圆弧分段数。

默认值为 12。
minDistance

距离小于此值的点将被合并。

默认值为 0.001。
vertices

存储顶点的数组。
normals

存储法线的数组。
uvs

存储 UV 的数组。
vertexOffset

顶点偏移量。

默认值为 0。
返回值： 顶点数量。
