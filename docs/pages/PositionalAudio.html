EventDispatcher → Object3D → Audio →

PositionalAudio

表示空间定位音频对象。

代码示例

// create an AudioListener and add it to the camera
const listener = new THREE.AudioListener();
camera.add( listener );
// create the PositionalAudio object (passing in the listener)
const sound = new THREE.PositionalAudio( listener );
// load a sound and set it as the PositionalAudio object's buffer
const audioLoader = new THREE.AudioLoader();
audioLoader.load( 'sounds/song.ogg', function( buffer ) {
	sound.setBuffer( buffer );
	sound.setRefDistance( 20 );
	sound.play();
});
// create an object for the sound to play from
const sphere = new THREE.SphereGeometry( 20, 32, 16 );
const material = new THREE.MeshPhongMaterial( { color: 0xff2200 } );
const mesh = new THREE.Mesh( sphere, material );
scene.add( mesh );
// finally add the sound to the mesh
mesh.add( sound );

构造函数

new PositionalAudio( listener : AudioListener )

构造一个空间音频对象。

listener

全局音频监听器。

属性

.panner : PannerNode (readonly)

声像定位节点，表示 3D 空间中音频源的位置、方向和行为。

方法

.getDistanceModel() : 'linear' | 'inverse' | 'exponential'

返回当前距离模型。

返回值： 距离模型。

.getMaxDistance() : number

返回当前最大距离。

返回值： 最大距离。

.getRefDistance() : number

返回当前参考距离。

返回值： 参考距离。

.getRolloffFactor() : number

返回当前衰减因子。

返回值： 衰减因子。

.setDirectionalCone( coneInnerAngle : number, coneOuterAngle : number, coneOuterGain : number ) : PositionalAudio

设置可听到音频的定向锥体范围。

coneInnerAngle

内锥角度（度），在此锥体内音量不衰减。
coneOuterAngle

外锥角度（度），在此锥体外音量将按 coneOuterGain 参数定义的恒定值衰减。
coneOuterGain

在 coneOuterAngle 定义的锥体之外的音量衰减量。设为 0 时完全听不到声音。
返回值： 对该实例的引用。

.setDistanceModel( value : 'linear' | 'inverse' | 'exponential' ) : PositionalAudio

定义音频源远离监听器时降低音量所使用的算法。

更多细节请参阅 规范。

value

要设置的距离模型。
返回值： 对该实例的引用。

.setMaxDistance( value : number ) : PositionalAudio

定义音频源与监听器之间的最大距离，超过此距离后音量不再进一步降低。

此值仅由 linear 距离模型使用。

value

最大距离。
返回值： 对该实例的引用。

.setRefDistance( value : number ) : PositionalAudio

定义参考距离：音频源远离监听器时开始降低音量的距离， 即音量衰减开始生效的距离。

value

要设置的参考距离。
返回值： 对该实例的引用。

.setRolloffFactor( value : number ) : PositionalAudio

定义音频源远离监听器时音量降低的速度。

value

衰减因子。
返回值： 对该实例的引用。
