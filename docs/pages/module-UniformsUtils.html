UniformsUtils

提供用于管理 uniform 的工具函数。

静态方法

.cloneUniforms( src : Object ) : Object

通过深拷贝克隆给定的 uniform 定义。这意味着 如果某个 uniform 的值引用了像 Vector3 或 Texture 这样的对象， 克隆后的 uniform 将引用新的对象。

src

表示 uniform 定义的对象。
返回值： 克隆后的 uniform。

.mergeUniforms( uniforms : Array ) : Object

将给定的 uniform 定义合并为单个对象。由于该方法 内部使用 cloneUniforms()，在生成合并后的 uniform 定义时 会执行深拷贝。

uniforms

包含 uniform 定义的对象数组。
返回值： 合并后的 uniform。
