Flow

一个用于使网格沿曲线弯曲的修改器。

此模块只能与 WebGLRenderer 一起使用。使用 WebGPURenderer 时， 请从 CurveModifierGPU.js 导入该类。

导入

Flow 是一个插件，必须显式导入，参见 Installation#Addons。

import { Flow } from 'three/addons/modifiers/CurveModifier.js';

构造函数

new Flow( mesh : Mesh, numberOfCurves : number )

构造一个新的 Flow 实例。

mesh

要克隆并修改以使其沿曲线弯曲的网格。
numberOfCurves

应为额外曲线预分配的空间数量。

默认值为 1。

类

Flow

方法

.moveAlongCurve( amount : number )

沿曲线移动网格。

amount

偏移量。

.updateCurve( index : number, curve : Curve )

更新给定曲线索引对应的曲线。

index

曲线索引。
curve

应用于弯曲网格的曲线。
