PropertyBinding

持有场景图中真实属性的引用，供内部使用。

构造函数

new PropertyBinding( rootNode : Object, path : string, parsedPath : Object )

构造一个新的属性绑定。

rootNode

根节点。
path

路径。
parsedPath

已解析的路径。

属性

.node : Object

拥有该动画属性的对象。

.parsedPath : Object

包含路径信息的对象。

.path : string

指向动画属性的对象路径。

.rootNode : Object3D | Skeleton

根节点。

方法

.bind()

为该绑定跟踪的属性创建 getter / setter 对。

.unbind()

解除属性绑定。

静态方法

.create( root : Object, path : string, parsedPath : Object ) : PropertyBinding | Composite

根据给定参数创建属性绑定的工厂方法。

root

根节点。
path

路径。
parsedPath

已解析的路径。
返回值： 创建的属性绑定或组合绑定。

.findNode( root : Object, nodeName : string | number ) : Object

在给定根对象的层级结构中，按节点名称搜索节点。

root

根对象。
nodeName

节点名称。
返回值： 找到的节点。若未找到对象则返回 null。

.parseTrackName( trackName : string ) : Object

解析给定的轨道名称（指向动画属性的对象路径）， 并返回包含路径信息的对象。可匹配以下格式的字符串：

  • nodeName.property
  • nodeName.property[accessor]
  • nodeName.material.property[accessor]
  • uuid.property[accessor]
  • uuid.objectName[objectIndex].propertyName[propertyIndex]
  • parentName/nodeName.property
  • parentName/parentName/nodeName.property[index]
  • .bone[Armature.DEF_cog].position
  • scene:helium_balloon_model:helium_balloon_model.position
trackName

要解析的轨道名称。
返回值： 解析后的轨道名称对象。

.sanitizeNodeName( name : string ) : string

将空格替换为下划线，并移除节点名称中不支持的字符， 以确保与 parseTrackName() 兼容。

name

要清理的节点名称。
返回值： 清理后的节点名称。
