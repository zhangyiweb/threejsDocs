Curve

用于创建解析曲线对象的抽象基类，包含插值相关方法。

构造函数

new Curve() (abstract)

构造一条新曲线。

属性

.arcLengthDivisions : number

该值决定通过 Curve#getLengths 计算曲线累计段长度时的划分数量。若曲线非常大，为确保使用 Curve#getSpacedPoints 等方法时的精度，建议增大此属性的值。

默认值为 200。

.needsUpdate : boolean

若曲线参数已更改，必须将其设为 true。

默认值为 false。

.type : string (readonly)

type 属性用于在序列化/反序列化上下文中检测对象类型。

方法

.clone() : Curve

返回一个复制了当前实例值的新曲线。

返回值： 当前实例的克隆。

.computeFrenetFrames( segments : number, closed : boolean ) : Object

生成 Frenet 标架。要求曲线定义在三维空间中。用于 TubeGeometry 或 ExtrudeGeometry 等几何体。

segments

分段数量。
closed

曲线是否闭合。

默认值为 false。
返回值： Frenet 标架。

.copy( source : Curve ) : Curve

将给定曲线的值复制到当前实例。

source

要复制的曲线。
返回值： 对该曲线的引用。

.fromJSON( json : Object ) : Curve

从给定的 JSON 反序列化曲线。

json

保存已序列化曲线的 JSON。
返回值： 对该曲线的引用。

.getLength() : number

返回曲线的总弧长。

返回值： 曲线长度。

.getLengths( divisions : number ) : Array.<number>

返回曲线的累计段长度数组。

divisions

划分数量。

默认值为 this.arcLengthDivisions。
返回值： 保存累计段长度的数组。

.getPoint( t : number, optionalTarget : Vector2 | Vector3 ) : Vector2 | Vector3 (abstract)

根据给定的插值因子，返回二维或三维空间中的向量（取决于曲线定义）。

t

表示曲线上位置的插值因子，必须在 [0,1] 范围内。
optionalTarget

可选的目标向量，结果将写入其中。
返回值： 曲线上的位置。根据曲线定义，可以是二维或三维向量。

.getPointAt( u : number, optionalTarget : Vector2 | Vector3 ) : Vector2 | Vector3

根据给定的插值因子，返回二维或三维空间中的向量（取决于曲线定义）。与 Curve#getPoint 不同，此方法会考虑曲线长度，从而得到等距采样点。

u

表示曲线上位置的插值因子，必须在 [0,1] 范围内。
optionalTarget

可选的目标向量，结果将写入其中。
返回值： 曲线上的位置。根据曲线定义，可以是二维或三维向量。

.getPoints( divisions : number ) : Array.<(Vector2|Vector3)>

此方法通过 Curve#getPoint 对曲线进行采样，并返回表示曲线形状的点数组。

divisions

划分数量。

默认值为 5。
返回值： 保存采样曲线值的数组。点数为 divisions + 1。

.getSpacedPoints( divisions : number ) : Array.<(Vector2|Vector3)>

此方法通过 Curve#getPointAt 对曲线进行采样，并返回表示曲线形状的点数组。与 Curve#getPoints 不同，此方法返回沿整条曲线等距分布的点。

divisions

划分数量。

默认值为 5。
返回值： 保存采样曲线值的数组。点数为 divisions + 1。

.getTangent( t : number, optionalTarget : Vector2 | Vector3 ) : Vector2 | Vector3

返回给定插值因子处的单位切向量。若派生曲线未实现其切线推导，则会使用相距一个小增量的两个点来求梯度，这通常能给出合理的近似。

t

插值因子。
optionalTarget

可选的目标向量，结果将写入其中。
返回值： 切向量。

.getTangentAt( u : number, optionalTarget : Vector2 | Vector3 ) : Vector2 | Vector3

与 Curve#getTangent 相同，但使用等距采样。

u

插值因子。
optionalTarget

可选的目标向量，结果将写入其中。
参见：
  • Curve#getPointAt
返回值： 切向量。

.getUtoTmapping( u : number, distance : number ) : number

给定范围在 [0,1] 内的插值因子，此方法返回同一范围内更新后的插值因子，可用于从曲线上等距采样点。

u

插值因子。
distance

可选的曲线上距离。

默认值为 null。
返回值： 更新后的插值因子。

.toJSON() : Object

将曲线序列化为 JSON。

参见：
  • ObjectLoader#parse
返回值： 表示已序列化曲线的 JSON 对象。

.updateArcLengths()

更新累计段距离缓存。每当曲线参数更改时必须调用此方法。若更新后的曲线属于 CurvePath 这类组合曲线，也必须在该组合曲线上调用此方法。
