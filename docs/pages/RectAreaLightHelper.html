RectAreaLightHelper

为矩形区域光创建可视化辅助对象。

RectAreaLightHelper 必须作为灯光的子对象添加。

代码示例

const light = new THREE.RectAreaLight( 0xffffbb, 1.0, 5, 5 );
const helper = new RectAreaLightHelper( light );
light.add( helper );

导入

RectAreaLightHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { RectAreaLightHelper } from 'three/addons/helpers/RectAreaLightHelper.js';

构造函数

new RectAreaLightHelper( light : RectAreaLight, color : number | Color | string )

构造一个新的矩形区域光辅助对象。

light

要可视化的灯光。
color

辅助对象的颜色。 若未设置，辅助对象将采用灯光的颜色。

属性

.color : number | Color | string | undefined

辅助对象的颜色。若为 undefined，辅助对象将采用灯光的颜色。

.light : RectAreaLight

要可视化的灯光。

方法

.dispose()

释放此实例分配的 GPU 相关资源。 当应用中不再使用此实例时，请调用此方法。
