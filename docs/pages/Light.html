Light

所有光源的抽象基类——其他光源类型均继承此处描述的属性和方法。

构造函数

new Light( color : number | Color | string, intensity : number ) (abstract)

构造一个新的光源。

color

光源颜色。

默认值为 0xffffff。
intensity

光源强度。

默认值为 1。

属性

.color : Color

光源颜色。

.intensity : number

光源强度。

默认值为 1。

.isLight : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.dispose()

释放该实例分配的 GPU 相关资源。当应用中不再使用该实例时， 请调用此方法。
