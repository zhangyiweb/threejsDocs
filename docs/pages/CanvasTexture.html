CanvasTexture

从 canvas 元素创建纹理。

这与基础纹理类几乎相同，区别在于它会立即将 Texture#needsUpdate 设为 true，因为 canvas 可以直接用于渲染。

构造函数

new CanvasTexture( canvas : HTMLCanvasElement, mapping : number, wrapS : number, wrapT : number, magFilter : number, minFilter : number, format : number, type : number, anisotropy : number )

构造一个新的纹理。

canvas

HTML canvas 元素。
mapping

纹理映射方式。

默认值为 Texture.DEFAULT_MAPPING。
wrapS

wrapS 值。

默认值为 ClampToEdgeWrapping。
wrapT

wrapT 值。

默认值为 ClampToEdgeWrapping。
magFilter

放大过滤器值。

默认值为 LinearFilter。
minFilter

缩小过滤器值。

默认值为 LinearMipmapLinearFilter。
format

纹理格式。

默认值为 RGBAFormat。
type

纹理类型。

默认值为 UnsignedByteType。
anisotropy

各向异性值。

默认值为 Texture.DEFAULT_ANISOTROPY。

属性

.isCanvasTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
