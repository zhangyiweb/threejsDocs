WebGPUTimestampQueryPool

管理用于性能测量的 WebGPU 时间戳查询池。 扩展基础的 TimestampQueryPool，提供 WebGPU 特定的实现。

构造函数

new WebGPUTimestampQueryPool( device : GPUDevice, type : string, maxQueries : number )

创建一个新的 WebGPU 时间戳查询池。

device

用于创建查询的 WebGPU 设备。
type

此查询池的类型标识符。
maxQueries

此池可容纳的最大查询数量。

默认值为 2048。

方法

.allocateQueriesForContext( uid : string ) : number

为给定的渲染上下文分配一对查询。

uid

渲染上下文的唯一标识符。
重写： TimestampQueryPool#allocateQueriesForContext
返回值： 已分配查询的基偏移量；若分配失败则返回 null。

.dispose() : Promise (async)

销毁该查询池。

重写： TimestampQueryPool#dispose
返回值： 在销毁执行完成后 resolve 的 Promise。

.resolveQueriesAsync() : Promise.<number> (async)

异步解析所有待处理的查询，并返回总时长。 若已有待处理的解析操作，则返回该 Promise。

重写： TimestampQueryPool#resolveQueriesAsync
返回值： 总时长（毫秒）；若解析失败则返回上一个有效值。
