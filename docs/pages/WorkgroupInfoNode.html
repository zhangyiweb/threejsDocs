WorkgroupInfoNode

允许用户在计算着色器上下文中创建 workgroup 作用域缓冲区。 通常，workgroup 作用域缓冲区用于保存从全局存储作用域传输到 本地 workgroup 作用域的数据。对于 workgroup 内的调用，访问 workgroup 作用域缓冲区的速度可能显著快于访问全局可访问的存储缓冲区。

此节点仅可与 WebGPU 后端一起使用。

构造函数

new WorkgroupInfoNode( scope : string, bufferType : string, bufferCount : number )

构造一个作用域限定的新缓冲区。

scope

待补充。
bufferType

workgroup 作用域缓冲区元素的数据类型。
bufferCount

缓冲区中的元素数量。

默认值为 0。

属性

.bufferCount : number

缓冲区元素数量。

默认值为 0。

.bufferType : string

缓冲区类型。

.elementType : string

数组缓冲区的数据类型。

.isWorkgroupInfoNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.name : string

workgroup 作用域缓冲区的名称。

默认值为 ''。

重写： Node#name

.scope : string

待补充。
