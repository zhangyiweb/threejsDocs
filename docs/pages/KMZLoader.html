KMZLoader

用于加载 KMZ 格式的加载器。

代码示例

const loader = new KMZLoader();
const kmz = await loader.loadAsync( './models/kmz/Box.kmz' );
scene.add( kmz.scene );

导入

KMZLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { KMZLoader } from 'three/addons/loaders/KMZLoader.js';

构造函数

new KMZLoader( manager : LoadingManager )

构造一个新的 KMZ 加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 KMZ 资源传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( data : ArrayBuffer ) : Object

解析给定的 KMZ 数据并返回包含场景的对象。

data

原始 KMZ 数据（ArrayBuffer 格式）。
重写： Loader#parse
返回值： 已解析的 KMZ 资源。
