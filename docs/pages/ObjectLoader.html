ObjectLoader

用于加载 JSON Object/Scene format 格式 JSON 资源的加载器。 文件在内部通过 FileLoader 加载。

代码示例

const loader = new THREE.ObjectLoader();
const obj = await loader.loadAsync( 'models/json/example.json' );
scene.add( obj );
// Alternatively, to parse a previously loaded JSON structure
const object = await loader.parseAsync( a_json_object );
scene.add( object );

构造函数

new ObjectLoader( manager : LoadingManager )

构造一个新的对象加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 3D 对象传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.loadAsync( url : string, onProgress : onProgressCallback ) : Promise.<Object3D> (async)

ObjectLoader#load 的异步版本。

url

要加载的文件路径/URL，也可以是 data URI。
onProgress

加载进行中时执行。
重写： Loader#loadAsync
返回值： 加载完成后以加载的 3D 对象兑现的 Promise。

.parse( json : Object, onLoad : onLoad ) : Object3D

解析给定的 JSON。此方法由 ObjectLoader#load 在内部使用， 也可直接用于解析先前已加载的 JSON 结构。

json

序列化后的 3D 对象。
onLoad

当所有资源（例如纹理）已完全加载时执行。
重写： Loader#parse
返回值： 解析后的 3D 对象。

.parseAsync( json : Object ) : Promise.<Object3D> (async)

ObjectLoader#parse 的异步版本。

json

序列化后的 3D 对象。
返回值： 解析完成后以解析后的 3D 对象兑现的 Promise。

静态方法

.registerGeometry( type : string, geometryClass : BufferGeometry.constructor )

将给定几何体注册到内部 几何体库中。

type

几何体类型。
geometryClass

几何体类。
