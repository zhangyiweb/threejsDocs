Text2D

导入

Text2D 是一个插件，必须显式导入，参见 Installation#Addons。

import * as Text2D from 'three/addons/webxr/Text2D.js';

方法

.createText( message : string, height : number ) : Mesh (inner)

用于创建简单平面网格的辅助函数，该网格可用作文本标签。网格的材质包含显示给定消息的 canvas 纹理。

message

要显示的消息。
height

标签的高度。
返回值： 表示文本标签的平面网格。
