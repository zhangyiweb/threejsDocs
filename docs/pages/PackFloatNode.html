PackFloatNode

此节点表示将向量的浮点值打包到无符号 32 位整数中的操作。

构造函数

new PackFloatNode( encoding : 'snorm' | 'unorm' | 'float16', vectorNode : Node )

encoding

描述浮点值如何映射到整数范围的数值编码。
vectorNode

待打包的向量节点。

属性

.encoding : string

数值编码。

.isPackFloatNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.vectorNode : Node

待打包的向量。
