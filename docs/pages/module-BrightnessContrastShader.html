BrightnessContrastShader

导入

BrightnessContrastShader 是一个插件，必须显式导入，参见 Installation#Addons。

import { BrightnessContrastShader } from 'three/addons/shaders/BrightnessContrastShader.js';

属性

.BrightnessContrastShader : ShaderMaterial~Shader (inner, constant)

亮度和对比度调整 https://github.com/evanw/glfx.js。 亮度：-1 到 1（-1 为纯黑，0 为无变化，1 为纯白） 对比度：-1 到 1（-1 为纯灰，0 为无变化，1 为最大对比度）
