ArcballControls

Arcball 控制器允许通过虚拟轨迹球控制相机，具备完整的触摸支持与高级导航功能。 光标/手指的位置与移动会映射到由辅助控件（gizmo）表示的虚拟轨迹球表面，并转化为直观、一致的相机运动。 拖动光标/手指会使相机以保守方式绕轨迹球中心旋转（返回起点时，相机会回到起始朝向）。

除支持平移、缩放与捏合手势外，双击/双触还可聚焦到某一点，直观地将物体的兴趣点移到虚拟轨迹球中心。 聚焦功能便于在复杂环境中更好地检视与导航。 此外，Arcball 控制器还支持以晕眩效果（vertigo）方式操控 FOV，以及绕 z 轴旋转。也支持相机状态的保存与恢复，并可通过剪贴板完成（使用 ctrl+c 与 ctrl+v 快捷键复制和粘贴状态）。

与 OrbitControls 和 TrackballControls 不同，当动画开启时，ArcballControls 无需在动画循环中从外部调用 update()。

导入

ArcballControls 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { ArcballControls } from 'three/addons/controls/ArcballControls.js';

构造函数

new ArcballControls( camera : Camera, domElement : HTMLElement, scene : Scene )

构造一个新的控制器实例。

camera

要控制的相机。该相机不得是其他对象的子对象，除非该对象就是场景本身。
domElement

用于事件监听的 HTML 元素。

默认值为 null。
scene

由相机渲染的场景。若未提供，则无法显示辅助控件（gizmo）。

默认值为 null。

属性

.adjustNearFar : boolean

若设为 true，每次缩放时都会调整相机的 near 与 far 值， 以尽量保持与初始 near、far 值所对应的相同可见范围。 仅适用于透视相机。

该功能仅在创建控制器之前正确配置了相机的初始状态（位置、near 与 far 值）时才能按预期工作。 否则应用程序必须调用 ArcballControls#setCamera。

默认值为 false。

.cursorZoom : boolean

设为 true 可使缩放以光标为中心。

默认值为 false。

.dampingFactor : number

当 enableAnimations 设为 true 时使用的阻尼惯性。

默认值为 25。

.enableAnimations : boolean

设为 true 以启用旋转（阻尼）与聚焦操作的动画。

默认值为 true。

.enableFocus : boolean

启用或禁用双触（或双击）时的相机聚焦操作。

默认值为 true。

.enableGizmos : boolean

启用或禁用辅助控件（gizmo）。

默认值为 true。

.enableGrid : boolean

若设为 true，在执行平移操作时会显示网格 （仅桌面端交互）。

默认值为 false。

.enablePan : boolean

启用或禁用相机平移。

默认值为 true。

.enableRotate : boolean

启用或禁用相机旋转。

默认值为 true。

.enableZoom : boolean

启用或禁用相机缩放。

默认值为 true。

.focusAnimationTime : number

聚焦动画的持续时间（毫秒）。

默认值为 500。

.maxDistance : number

可向外推拉的最大距离。仅适用于透视相机。

默认值为 Infinity。

.maxFov : number

最大 FOV（以度为单位）。

默认值为 90。

.maxZoom : number

可向外缩放的最大程度。仅适用于正交相机。

默认值为 Infinity。

.minDistance : number

可向内推拉的最小距离。仅适用于透视相机。

默认值为 0。

.minFov : number

最小 FOV（以度为单位）。

默认值为 5。

.minZoom : number

可向内缩放的最大程度。仅适用于正交相机。

默认值为 0。

.mouseActions : Array.<Object>

保存该控制器的鼠标操作。此属性由方法 setMouseAction() 与 unsetMouseAction() 维护。

.radiusFactor : number

辅助控件（gizmo）相对于屏幕宽度与高度的尺寸。

默认值为 0.67。

.rotateSpeed : number

旋转速度。

默认值为 1。

.scaleFactor : number

执行缩放操作时使用的缩放因子。

默认值为 1.1。

.scene : Scene

由相机渲染的场景。若未提供，则无法显示辅助控件（gizmo）。

默认值为 null。

.target : Vector3

控制器的焦点。

.wMax : number

旋转动画开始时允许的最大角速度。

默认值为 20。

方法

.activateGizmos( isActive : boolean )

使旋转辅助控件（gizmo）更明显或更不明显。

isActive

若设为 true，辅助控件会更明显。

.copyState()

将当前状态复制到剪贴板（作为可读的 JSON 文本）。

.disposeGrid()

从场景中移除网格。

.getRaycaster() : Raycaster

返回用于用户交互的射线投射器。该对象在所有 ArcballControls 实例之间共享。

返回值： 内部的射线投射器。

.pasteState()

从剪贴板设置控制器状态，假定剪贴板中存储的是由 copyState() 保存的 JSON 文本。

.reset()

重置控制器。

.saveState()

保存控制器的当前状态。之后可通过 reset() 恢复。

.setCamera( camera : Camera )

设置要控制的相机。若要更换被控制的相机，必须调用此方法。

camera

要控制的相机。

.setGizmosVisible( value : boolean )

设置辅助控件（gizmo）的可见性。

value

辅助控件的可见性值。

.setMouseAction( operation : 'PAN' | 'ROTATE' | 'ZOOM' | 'FOV', mouse : 0 | 1 | 2 | 'WHEEL', key : 'CTRL' | 'SHIFT' ) : boolean

通过指定要执行的操作以及鼠标/按键组合来设置新的鼠标操作。若存在冲突，则替换已有操作。

operation

要执行的操作（'PAN'、'ROTATE'、'ZOOM'、'FOV'）。
mouse

鼠标按键（0、1、2）或表示滚轮刻度的 'WHEEL'。
key

键盘修饰键（'CTRL'、'SHIFT'）；若不需要按键则为 null。

默认值为 null。
返回值： 若鼠标操作已成功添加则为 true，否则为 false。

.setTbRadius( value : number )

设置辅助控件半径因子并重新绘制辅助控件。

value

半径因子的值。

.unsetMouseAction( mouse : 0 | 1 | 2 | 'WHEEL', key : 'CTRL' | 'SHIFT' ) : boolean

通过指定鼠标/按键组合移除鼠标操作。

mouse

鼠标按键（0、1、2）或表示滚轮刻度的 'WHEEL'。
key

键盘修饰键（'CTRL'、'SHIFT'）；若不需要按键则为 null。

默认值为 null。
返回值： 若操作已成功移除则为 true，否则为 false。

事件

.change

当相机被控制器变换时触发。

类型：
  • Object

.end

当一次交互结束时触发。

类型：
  • Object

.start

当一次交互开始时触发。

类型：
  • Object
