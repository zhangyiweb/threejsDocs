PhysicalLightingModel

表示 PBR 材质的光照模型。

构造函数

new PhysicalLightingModel( clearcoat : boolean, sheen : boolean, iridescence : boolean, anisotropy : boolean, transmission : boolean, dispersion : boolean )

构造一个新的物理光照模型。

clearcoat

是否支持清漆。

默认值为 false。
sheen

是否支持光泽。

默认值为 false。
iridescence

是否支持虹彩。

默认值为 false。
anisotropy

是否支持各向异性。

默认值为 false。
transmission

是否支持透射。

默认值为 false。
dispersion

是否支持色散。

默认值为 false。

属性

.anisotropy : boolean

是否支持各向异性。

默认值为 false。

.clearcoat : boolean

是否支持清漆。

默认值为 false。

.clearcoatRadiance : Node

清漆辐射度。

默认值为 null。

.clearcoatSpecularDirect : Node

清漆直接镜面反射。

默认值为 null。

.clearcoatSpecularIndirect : Node

清漆间接镜面反射。

默认值为 null。

.dispersion : boolean

是否支持色散。

默认值为 false。

.iridescence : boolean

是否支持虹彩。

默认值为 false。

.iridescenceF0 : Node

虹彩 F0。

默认值为 null。

.iridescenceF0Dielectric : Node

虹彩 F0 电介质。

默认值为 null。

.iridescenceF0Metallic : Node

虹彩 F0 金属。

默认值为 null。

.iridescenceFresnel : Node

虹彩 Fresnel。

默认值为 null。

.sheen : boolean

是否支持光泽。

默认值为 false。

.sheenSpecularDirect : Node

光泽直接镜面反射。

默认值为 null。

.sheenSpecularIndirect : Node

光泽间接镜面反射。

默认值为 null。

.transmission : boolean

是否支持透射。

默认值为 false。
