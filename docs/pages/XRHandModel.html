XRHandModel

表示一个 XR 手部模型。

构造函数

new XRHandModel( controller : Group )

构造一个新的 XR 手部模型。

controller

手部控制器。

属性

.controller : Group

手部控制器。

.envMap : Texture

控制器的环境贴图。

默认值为 null。

.mesh : Mesh

模型网格。

默认值为 null。

.motionController : MotionController

运动控制器。

默认值为 null。

方法

.updateMatrixWorld( force : boolean )

使用自定义实现重写。确保运动控制器更新网格。

force

设为 true 时，即使 Object3D#matrixWorldAutoUpdate 为 false，也会强制重新计算世界矩阵。

默认值为 false。
重写： Object3D#updateMatrixWorld
