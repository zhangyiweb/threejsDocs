LineGeometry

由一连串顶点组成的折线。

在 Line2 中用于描述形状。

代码示例

const points = [
	new THREE.Vector3( - 10, 0, 0 ),
	new THREE.Vector3( 0, 5, 0 ),
	new THREE.Vector3( 10, 0, 0 ),
];
const geometry = new LineGeometry();
geometry.setFromPoints( points );

导入

LineGeometry 是一个插件，必须显式导入，参见 Installation#Addons。

import { LineLineGeometry2 } from 'three/addons/lines/LineGeometry.js';

构造函数

new LineGeometry()

构造一个新的线几何体。

属性

.isLineGeometry : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.fromLine( line : Line ) : LineGeometry

根据给定的线设置此线段几何体。

line

应作为此几何体数据源的线。
返回值： 对此几何体的引用。

.setColors( array : Float32Array | Array.<number> ) : LineGeometry

为此几何体设置给定的线颜色。

array

要设置的位置数据。
重写： LineSegmentsGeometry#setColors
返回值： 对此几何体的引用。

.setFromPoints( points : Array.<(Vector3|Vector2)> ) : LineGeometry

根据给定的点序列设置此线段几何体。

points

二维或三维空间中的点数组。
返回值： 对此几何体的引用。

.setPositions( array : Float32Array | Array.<number> ) : LineGeometry

为此几何体设置给定的线位置。

array

要设置的位置数据。
重写： LineSegmentsGeometry#setPositions
返回值： 对此几何体的引用。
