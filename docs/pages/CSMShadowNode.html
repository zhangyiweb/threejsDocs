CSMShadowNode

级联阴影贴图（CSM）的一种实现。

此模块只能与 WebGPURenderer 一起使用。使用 WebGLRenderer 时，请改用 CSM。

导入

CSMShadowNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { CSMShadowNode } from 'three/addons/csm/CSMShadowNode.js';

构造函数

new CSMShadowNode( light : DirectionalLight, data : CSMShadowNode~Data )

构造一个新的 CSM 阴影节点。

light

CSM 光源。
data

CSM 数据。

默认值为 {}。

属性

.breaks : Array.<number>

取值范围为 [0,1] 的数字数组，用于定义如何分割主 CSM 视锥体。

.camera : Camera

场景的相机。

默认值为 null。

.cascades : number

级联数量。

默认值为 3。

.customSplitsCallback : function

当使用 mode='custom' 时的自定义分割回调。

.fade : boolean

是否在级联之间淡入淡出。

默认值为 false。

.frustums : Array.<CSMFrustum>

表示各级联的视锥体数组。

.lightMargin : number

光源边距。

默认值为 200。

.lights : Array.<DirectionalLight>

为不同级联投射阴影的平行光数组。每个级联对应一盏平行光。

.mainFrustum : CSMFrustum

主视锥体。

默认值为 null。

.maxFar : number

最大远裁剪面值。

默认值为 100000。

.mode : 'practical' | 'uniform' | 'logarithmic' | 'custom'

视锥体分割模式。

默认值为 'practical'。

方法

.dispose()

释放该实例分配的 GPU 相关资源。当应用中不再使用此实例时，请调用此方法。

重写： ShadowBaseNode#dispose

.updateFrustums()

每当更改相机或 CSM 设置时，应用必须调用此方法。
