AMFLoader

用于加载 AMF 格式的加载器。

该加载器支持材质、颜色以及 ZIP 压缩文件。 暂不支持 constellation（星座/组合体）功能。

代码示例

const loader = new AMFLoader();
const object = await loader.loadAsync( './models/amf/rook.amf' );
scene.add( object );

导入

AMFLoader 属于附加组件，需要显式导入，请参阅 Installation#Addons。

import { AMFLoader } from 'three/addons/loaders/AMFLoader.js';

构造函数

new AMFLoader( manager : LoadingManager )

构造一个新的 AMF 加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 AMF 资源传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( data : ArrayBuffer ) : Group

解析给定的 AMF 数据并返回生成的组（Group）。

data

原始 AMF 资源数据（ArrayBuffer 格式）。
重写： Loader#parse
返回值： 表示已解析资源的组（Group）。
