RangeNode

RangeNode 在指定范围内生成随机的实例化属性数据。 此工具节点的一个典型用例是生成每实例的 随机颜色：

代码示例

const material = new MeshBasicNodeMaterial();
material.colorNode = range( new Color( 0x000000 ), new Color( 0xFFFFFF ) );
const mesh = new InstancedMesh( geometry, material, count );

构造函数

new RangeNode( minNode : Node.<any>, maxNode : Node.<any> )

构造一个新的范围节点。

minNode

定义范围下限的节点。

默认值为 float()。
maxNode

定义范围上限的节点。

默认值为 float()。

属性

.maxNode : Node.<any>

定义范围上限的节点。

默认值为 float()。

.minNode : Node.<any>

定义范围下限的节点。

默认值为 float()。

方法

.generateNodeType( builder : NodeBuilder ) : string

此方法被重写，因为节点类型是从范围定义推断的。

builder

当前节点构建器。
重写： Node#generateNodeType
返回值： 节点类型。

.getConstNode( node : Node ) : Node

通过遍历给定节点返回常量节点。

node

要遍历的节点。
返回值： 常量节点（若找到）。

.getVectorLength( builder : NodeBuilder ) : number

根据范围定义计算并返回向量长度。

builder

当前节点构建器。
返回值： 向量长度。
