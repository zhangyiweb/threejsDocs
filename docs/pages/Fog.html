Fog

此类可用于定义线性雾，其密度随距离线性增加。

代码示例

const scene = new THREE.Scene();
scene.fog = new THREE.Fog( 0xcccccc, 10, 15 );

构造函数

new Fog( color : number | Color, near : number, far : number )

构造一个新的雾效果。

color

雾的颜色。
near

开始应用雾效果的最小距离。

默认值为 1。
far

停止计算并应用雾效果的最大距离。

默认值为 1000。

属性

.color : Color

雾的颜色。

.far : number

停止计算并应用雾效果的最大距离。 距离活动相机超过 far 单位的物体将不受雾影响。

默认值为 1000。

.isFog : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.name : string

雾的名称。

.near : number

开始应用雾效果的最小距离。距离活动相机少于 near 单位的物体将不受雾影响。

默认值为 1。

方法

.clone() : Fog

返回一个从此实例复制数值的新雾对象。

返回值： 此实例的克隆。

.toJSON( meta : Object | string ) : Object

将雾序列化为 JSON。

meta

可选值，保存有关序列化的元信息。
返回值： 表示序列化雾的 JSON 对象
