EXRExporter

EXR 导出器。

EXR（Extended Dynamic Range，扩展动态范围）是电影行业专业级图像存储的开放格式规范。其目的是准确、高效地表示高动态范围的场景线性图像数据及相关元数据。该库广泛应用于对精度要求极高的宿主应用软件，如真实感渲染、纹理访问、图像合成、深度合成以及 DI（数字中间片）。

代码示例

const exporter = new EXRExporter();
const result = await exporter.parse( renderer, options );

导入

EXRExporter 是一个插件，必须显式导入，参见 Installation#Addons。

import { EXRExporter } from 'three/addons/exporters/EXRExporter.js';

构造函数

new EXRExporter()

方法

.parse( arg1 : DataTexture | WebGPURenderer | WebGLRenderer, arg2 : EXRExporter~Options | RenderTarget, arg3 : EXRExporter~Options ) : Promise.<Uint8Array> (async)

此方法有两种调用形式。

  • 导出数据纹理时，接收两个参数：纹理和导出选项。
  • 导出渲染目标（例如 PMREM）时，接收三个参数：渲染器、渲染目标和导出选项。
arg1

要导出的数据纹理，或渲染器。
arg2

导出选项，或渲染目标。
arg3

导出选项。
返回值： 解析为导出的 EXR 的 Promise。

类型定义

.Options

EXRExporter 的导出选项。

type
HalfFloatType | FloatType

输出数据类型。

默认值为 HalfFloatType。
type
NO_COMPRESSION | ZIP_COMPRESSION | ZIPS_COMPRESSION

压缩算法。

默认值为 ZIP_COMPRESSION。
