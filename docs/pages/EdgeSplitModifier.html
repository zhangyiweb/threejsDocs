EdgeSplitModifier

该修改器可用于在锐边上拆分面。这样可以在计算法线时不对边缘进行平滑处理，从而改善视觉效果。

代码示例

const modifier = new EdgeSplitModifier();
geometry = modifier.modify( geometry, Math.PI * 0.4 );

导入

EdgeSplitModifier 是一个插件，必须显式导入，参见 Installation#Addons。

import { EdgeSplitModifier } from 'three/addons/modifiers/EdgeSplitModifier.js';

构造函数

new EdgeSplitModifier()

方法

.modify( geometry : BufferGeometry, cutOffAngle : number, tryKeepNormals : boolean ) : BufferGeometry

通过对给定几何体应用边缘拆分操作，返回一个新的、已修改的几何体版本。 请注意，结果几何体始终是索引几何体。

geometry

要修改的几何体。
cutOffAngle

截断角度，以弧度为单位。
tryKeepNormals

是否尝试保留法线。

默认值为 true。
返回值： 一个新的、已修改的几何体。
