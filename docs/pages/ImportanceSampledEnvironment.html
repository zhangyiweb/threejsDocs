ImportanceSampledEnvironment

管理预处理后的 HDR 环境贴图（CDF 纹理、uniform），并提供用于 BRDF 方向查找与 MIS 重要性采样的 TSL 辅助函数。

构造函数

new ImportanceSampledEnvironment( importanceSampling : boolean )

importanceSampling

为 true 时，构建亮度 CDF 表并启用 MIS 环境采样。

默认值为 false。
参见：
  • https://github.com/gkjohnson/three-gpu-pathtracer

方法

.sampleEnvironmentBRDF( params : Object ) : Node.<vec3>

针对屏幕空间未命中，仅使用 BRDF / 反射射线方向进行环境反射。

params
cameraWorldMatrix
viewReflectDir

视图空间 GGX 采样的反射射线。
N

视图空间着色法线。
V

视图空间指向相机的方向。
alpha

GGX 粗糙度（alpha）。
f0

.sampleEnvironmentMIS( params : Object ) : Node.<vec3>

针对屏幕空间未命中，使用多重重要性采样（MIS）在 BRDF / 反射射线方向与环境亮度 CDF 方向之间估计环境反射。两种技术使用一致的立体角 PDF（D·G1(N·V)/(4·N·V)），因此幂启发式是无偏的。改编自 three-gpu-pathtracer。

params
cameraWorldMatrix
viewReflectDir

视图空间 GGX 采样的反射射线。
N

视图空间着色法线。
V

视图空间指向相机的方向。
alpha

GGX 粗糙度（alpha）。
f0
Xi2

第二个蓝噪声采样（zw 用于 CDF）。
参见：
  • https://github.com/gkjohnson/three-gpu-pathtracer

.sampleReflect( params : Object ) : Node.<vec3>

沿反射方向的简单环境查找（无 MIS）。

params
cameraWorldMatrix
viewReflectDir
sampleWeight

可选的辐射度缩放（默认为 1）。

.updateFrom( hdr : Texture )

hdr

等距柱状投影 HDR 环境贴图。
