AnimationUtils

提供多种辅助动画处理方法的工具类。

静态方法

.convertArray( array : TypedArray | Array, type : TypedArray.constructor ) : TypedArray

将数组转换为指定类型。

array

要转换的数组。
type

类型化数组的构造函数。
返回值： 转换后的数组。

.flattenJSON( jsonKeys : Array.<number>, times : Array.<number>, values : Array.<number>, valuePropertyName : string )

用于解析 AOS 关键帧格式。

jsonKeys

JSON 关键帧列表。
times

此方法将用关键帧时间填充该数组。
values

此方法将用关键帧值填充该数组。
valuePropertyName

要使用的属性名称。

.getKeyframeOrder( times : Array.<number> ) : Array.<number>

返回用于对时间和值进行排序的数组。

times

关键帧时间值。
返回值： 排序数组。

.isTypedArray( object : any ) : boolean

若给定对象是类型化数组，则返回 true。

object

要检查的对象。
返回值： 给定对象是否为类型化数组。

.makeClipAdditive( targetClip : AnimationClip, referenceFrame : number, referenceClip : AnimationClip, fps : number ) : AnimationClip

将给定动画片段的关键帧转换为叠加（additive）格式。

targetClip

要转换为叠加格式的片段。
referenceFrame

参考帧。

默认值为 0。
referenceClip

参考片段。

默认值为 targetClip。
fps

帧率（FPS）。

默认值为 30。
返回值： 已转换为叠加格式的更新后片段。

.sortedArray( values : Array.<number>, stride : number, order : Array.<number> ) : Array.<number>

根据 getKeyframeOrder() 先前计算的顺序对给定数组进行排序。

values

要排序的值。
stride

步长。
order

排序顺序。
返回值： 排序后的值。

.subclip( sourceClip : AnimationClip, name : string, startFrame : number, endFrame : number, fps : number ) : AnimationClip

创建新片段，仅包含原始片段在给定帧范围内的部分。

sourceClip

源动画片段。
name

片段名称。
startFrame

起始帧。
endFrame

结束帧。
fps

帧率（FPS）。

默认值为 30。
返回值： 新的子片段。
