StereoCamera

一种特殊相机，使用两个带立体投影的透视相机。可用于渲染立体效果，例如 3D 红蓝立体（Anaglyph） 或 视差屏障（Parallax Barrier）。

构造函数

new StereoCamera()

构造一个新的立体相机。

属性

.aspect : number

宽高比。

默认值为 1。

.cameraL : PerspectiveCamera

表示左眼的相机。该相机被添加到图层 1，因此需要由左相机渲染的物体也必须添加到该图层。

.cameraR : PerspectiveCamera

表示右眼的相机。该相机被添加到图层 2，因此需要由右相机渲染的物体也必须添加到该图层。

.eyeSep : number

眼间距，表示左右相机之间的距离。

默认值为 0.064。

.type : string (readonly)

type 属性用于在序列化/反序列化上下文中检测对象类型。

方法

.update( camera : PerspectiveCamera )

根据给定的透视相机更新立体相机。

camera

透视相机。
