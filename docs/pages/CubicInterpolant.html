CubicInterpolant

快速且简洁的三次样条插值器。

它由埃尔米特构造导出，将每个采样位置处的一阶导数 设为相邻位置之间在其参数区间上的线性斜率。

构造函数

new CubicInterpolant( parameterPositions : TypedArray, sampleValues : TypedArray, sampleSize : number, resultBuffer : TypedArray )

构造一个新的三次插值器。

parameterPositions

参数位置，保存插值因子。
sampleValues

采样值。
sampleSize

采样大小
resultBuffer

结果缓冲区。
