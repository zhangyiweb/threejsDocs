TriangleBlurShader

导入

TriangleBlurShader 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { TriangleBlurShader } from 'three/addons/shaders/TriangleBlurShader.js';

属性

.TechnicolorShader : ShaderMaterial~Shader (inner, constant)

模拟 20 世纪初电影中流行的双色条 technicolor 工艺外观。 更多历史信息见：http://www.widescreenmuseum.com/oldcolor/technicolor1.htm 演示见：http://charliehoey.com/technicolor_shader/shader_test.html

.ToonShader1 : ShaderMaterial~Shader (inner, constant)

Toon1 着色器。

.ToonShader2 : Object (inner, constant)

Toon2 着色器。

.ToonShaderDotted : Object (inner, constant)

Toon 点状着色器。

.ToonShaderHatching : Object (inner, constant)

Toon 阴影线着色器。

.TriangleBlurShader : ShaderMaterial~Shader (inner, constant)

基于 glfx.js triangle blur shader 的三角模糊着色器。

一种基本的模糊滤镜，将图像与金字塔滤镜进行卷积。 金字塔滤镜是可分离的，并作为两个互相垂直的三角滤镜应用。
