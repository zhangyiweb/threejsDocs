CubeMapNode

此节点可用于将等距柱状投影格式的环境贴图 自动转换为立方体贴图格式。

构造函数

new CubeMapNode( envNode : Node )

构造一个新的立方体贴图节点。

envNode

表示环境贴图的节点。

属性

.envNode : Node

表示环境贴图的节点。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.RENDER，因为该节点会在 CubeMapNode#updateBefore 方法中每次渲染更新一次纹理。

默认值为 'render'。

重写： TempNode#updateBeforeType
