RenderTarget3D

表示 3D 渲染目标。

构造函数

new RenderTarget3D( width : number, height : number, depth : number, options : RenderTarget~Options )

构造一个新的 3D 渲染目标。

width

渲染目标的宽度。

默认值为 1。
height

渲染目标的高度。

默认值为 1。
depth

渲染目标的深度。

默认值为 1。
options

配置对象。

属性

.isRenderTarget3D : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.texture : Data3DTexture

使用不同纹理类型覆盖。

重写： RenderTarget#texture
