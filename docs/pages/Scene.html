Scene

场景用于设置 three.js 要渲染的内容及其位置。 网格、线条或灯光等 3D 对象都放置在场景中。

构造函数

new Scene()

构造一个新的场景。

属性

.background : Color | Texture

定义场景的背景。有效输入为：

  • 颜色：定义均匀着色的背景。
  • 纹理：定义（平面）纹理背景。
  • 立方体贴图或等距柱状投影纹理：定义天空盒。

默认值为 null。

.backgroundBlurriness : number

设置背景的模糊程度。仅影响赋给 Scene#background 的环境贴图。有效输入为 0 到 1 之间的浮点数。

默认值为 0。

.backgroundIntensity : number

衰减背景颜色。仅适用于背景纹理。

默认值为 1。

.backgroundRotation : Euler

背景的旋转（以弧度为单位）。仅影响赋给 Scene#background 的环境贴图。

默认值为 (0,0,0)。

.environment : Texture

为场景中所有物理材质设置环境贴图。不过， 无法覆盖已赋给材质 envMap 属性的现有纹理。

默认值为 null。

.environmentIntensity : number

衰减环境颜色。仅影响赋给 Scene#environment 的环境贴图。

默认值为 1。

.environmentRotation : Euler

环境贴图的旋转（以弧度为单位）。仅在使用 Scene#environment 时影响场景中的物理材质。

默认值为 (0,0,0)。

.fog : Fog | FogExp2

雾实例，定义影响场景中所有渲染内容的雾类型。

默认值为 null。

.isScene : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.overrideMaterial : Material

强制场景中的所有内容使用指定材质渲染。可通过将 Material#allowOverride 设为 false 来排除某些材质不被覆盖。

默认值为 null。
