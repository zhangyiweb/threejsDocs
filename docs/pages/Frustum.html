Frustum

视锥体用于判断物体是否位于相机视野之内。 它们有助于加快渲染过程——位于相机视锥体之外的对象 可以安全地从渲染中排除。

此类主要供渲染器内部使用。

构造函数

new Frustum( p0 : Plane, p1 : Plane, p2 : Plane, p3 : Plane, p4 : Plane, p5 : Plane )

构造一个新的视锥体。

p0

围成视锥体的第一个平面。
p1

围成视锥体的第二个平面。
p2

围成视锥体的第三个平面。
p3

围成视锥体的第四个平面。
p4

围成视锥体的第五个平面。
p5

围成视锥体的第六个平面。

属性

.planes : Array.<Plane>

该数组保存围成视锥体的各平面。

方法

.clone() : Frustum

返回一个新的视锥体，其值从本实例复制而来。

返回值： 本实例的克隆。

.containsPoint( point : Vector3 ) : boolean

若给定点位于视锥体内，则返回 true。

point

要测试的点。
返回值： 该点是否位于本视锥体内。

.copy( frustum : Frustum ) : Frustum

将给定视锥体的值复制到本实例。

frustum

要复制的视锥体。
返回值： 对本视锥体的引用。

.intersectsBox( box : Box3 ) : boolean

若给定包围盒与本视锥体相交，则返回 true。

box

要测试的包围盒。
返回值： 包围盒是否与本视锥体相交。

.intersectsObject( object : Object3D ) : boolean

若三维对象的包围球与本视锥体相交，则返回 true。

注意：该三维对象必须具有几何体，以便能够计算包围球。

object

要测试的三维对象。
返回值： 三维对象的包围球是否与本视锥体相交。

.intersectsSphere( sphere : Sphere ) : boolean

若给定包围球与本视锥体相交，则返回 true。

sphere

要测试的包围球。
返回值： 包围球是否与本视锥体相交。

.intersectsSprite( sprite : Sprite ) : boolean

若给定精灵与本视锥体相交，则返回 true。

sprite

要测试的精灵。
返回值： 精灵是否与本视锥体相交。

.set( p0 : Plane, p1 : Plane, p2 : Plane, p3 : Plane, p4 : Plane, p5 : Plane ) : Frustum

通过复制给定平面来设置视锥体各平面。

p0

围成视锥体的第一个平面。
p1

围成视锥体的第二个平面。
p2

围成视锥体的第三个平面。
p3

围成视锥体的第四个平面。
p4

围成视锥体的第五个平面。
p5

围成视锥体的第六个平面。
返回值： 对本视锥体的引用。

.setFromProjectionMatrix( m : Matrix4, coordinateSystem : WebGLCoordinateSystem | WebGPUCoordinateSystem, reversedDepth : boolean ) : Frustum

根据给定投影矩阵设置视锥体各平面。

m

投影矩阵。
coordinateSystem

坐标系。
reversedDepth

是否使用反向深度。

默认值为 false。
返回值： 对本视锥体的引用。
