PhongLightingModel

表示 Phong 材质的照明模型。用于 MeshPhongNodeMaterial。

构造函数

new PhongLightingModel( specular : boolean )

构造一个新的 Phong 照明模型。

specular

是否支持镜面反射。

默认值为 true。

属性

.specular : boolean

是否支持镜面反射。若设为 false，则得到类似 Lambert 的材质，适用于非光泽表面，无镜面高光。

默认值为 true。

方法

.direct( lightData : Object )

实现直接光照。镜面反射部分为可选，可通过 PhongLightingModel#specular 标志控制。

lightData

光照数据。
重写： BasicLightingModel#direct

.indirect( builder : NodeBuilder )

实现间接光照。

builder

当前节点构建器。
重写： BasicLightingModel#indirect
