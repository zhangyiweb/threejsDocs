DRACOLoader

Draco 格式的加载器。

Draco 是一个用于压缩和 解压缩 3D 网格与点云的开源库。压缩后的几何体体积可显著减小， 但会在客户端设备上增加额外的解码时间。

独立的 Draco 文件扩展名为 .drc，包含顶点位置、法线、颜色 及其他属性。Draco 文件不包含材质、纹理、动画或节点层级—— 若要使用这些功能，请将 Draco 几何体嵌入 glTF 文件中。可使用 glTF-Pipeline 将普通 glTF 文件 转换为 Draco 压缩的 glTF 文件。 在 glTF 中使用 Draco 时，GLTFLoader 会在内部使用 DRACOLoader 实例。

建议创建一个 DRACOLoader 实例并重复使用，以避免加载和创建 多个解码器实例。

DRACOLoader 会根据浏览器能力自动使用 JS 或 WASM 解码库。

代码示例

const loader = new DRACOLoader();
const geometry = await loader.loadAsync( 'models/draco/bunny.drc' );
geometry.computeVertexNormals(); // optional
loader.dispose();

导入

DRACOLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { DRACOLoader } from 'three/addons/loaders/DRACOLoader.js';

构造函数

new DRACOLoader( manager : LoadingManager )

构造一个新的 Draco 加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 Draco 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( buffer : ArrayBuffer, onLoad : function, onError : onErrorCallback )

解析给定的 Draco 数据。

buffer

作为 ArrayBuffer 的原始 Draco 数据。
onLoad

加载/解析完成时执行。
onError

发生错误时执行。
重写： Loader#parse

.setDecoderConfig( config : Object ) : DRACOLoader

为解码库提供配置。解码开始后无法更改配置。

config

解码器配置。
已弃用： 是
返回值： 对此加载器的引用。

.setDecoderPath( path : string | Object ) : DRACOLoader

为解码库提供配置。解码开始后无法更改配置。

path

解码器路径，或包含各解码器文件显式 URL 的配置对象。
返回值： 对此加载器的引用。

.setWorkerLimit( workerLimit : number ) : DRACOLoader

设置解码期间使用的 Web Worker 最大数量。 若 Worker 在应用中还需承担其他任务，较低的限制可能更合适。

workerLimit

Worker 数量限制。
返回值： 对此加载器的引用。
