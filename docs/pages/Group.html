EventDispatcher → Object3D →

Group

该类与 Object3D 几乎完全相同。其目的是 使对一组对象的操作在语法上更清晰。

代码示例

// Create a group and add the two cubes.
// These cubes can now be rotated / scaled etc as a group.
const group = new THREE.Group();
group.add( meshA );
group.add( meshB );
scene.add( group );

构造函数

new Group()

属性

.isGroup : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
