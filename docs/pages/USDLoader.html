USDLoader

用于加载 USD 格式（USD、USDA、USDC、USDZ）的加载器。

同时支持 ASCII（USDA）与二进制（USDC）USD 文件，以及包含上述任一格式的 USDZ 压缩包。

代码示例

const loader = new USDLoader();
const model = await loader.loadAsync( 'model.usdz' );
scene.add( model );

导入

USDLoader 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { USDLoader } from 'three/addons/loaders/USDLoader.js';

构造函数

new USDLoader( manager : LoadingManager )

构造一个新的 USDZ 加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 USDZ 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( buffer : ArrayBuffer | string, path : string, onLoad : function, onError : onErrorCallback ) : Group

解析给定的 USDZ 数据并返回结果组。

返回的组是同步创建的，但任何引用的纹理 会异步加载。提供 onLoad 以便在所有 纹理加载完成后收到通知。

buffer

原始 USDZ 数据，以数组缓冲区形式提供。
path

URL 基路径。

默认值为 ''。
onLoad

当组及其所有纹理就绪时执行。
onError

发生错误时执行。
重写： Loader#parse
返回值： 解析后的资源，作为组返回。
