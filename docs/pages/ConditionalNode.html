ConditionalNode

表示逻辑上的 if/else 语句。可用作 If()/Else() 语法的替代方案。

select() 方法以链式方式在条件上调用。select() 的参数节点 决定整个语句的结果。

代码示例

velocity = position.greaterThanEqual( limit ).select( velocity.negate(), velocity );

构造函数

new ConditionalNode( condNode : Node, ifNode : Node, elseNode : Node )

构造一个新的条件节点。

condNode

定义条件的节点。
ifNode

条件为 true 时求值的节点。
elseNode

条件为 false 时求值的节点。

默认值为 null。

属性

.condNode : Node

定义条件的节点。

.elseNode : Node

条件为 false 时求值的节点。

默认值为 null。

.ifNode : Node

条件为 true 时求值的节点。

方法

.generateNodeType( builder : NodeBuilder ) : string

此方法被重写，因为节点类型由 if/else 节点推断。

builder

当前的节点构建器。
重写： Node#generateNodeType
返回值： 节点类型。
