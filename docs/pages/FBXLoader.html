FBXLoader

FBX 格式的加载器。

要求 FBX 文件为 ASCII 格式且版本 >= 7.0，或二进制格式且版本 >= 6400。 低于此版本的文件可能可以加载，但很可能出现错误。

需要支持：

  • 形变法线 / 混合形状法线

FBX 格式参考：

  • C++ SDK 参考

二进制格式规范：

  • FBX 二进制文件格式规范

代码示例

const loader = new FBXLoader();
const object = await loader.loadAsync( 'models/fbx/stanford-bunny.fbx' );
scene.add( object );

导入

FBXLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { FBXLoader } from 'three/addons/loaders/FBXLoader.js';

构造函数

new FBXLoader( manager : LoadingManager )

构造一个新的 FBX 加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 FBX 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( FBXBuffer : ArrayBuffer, path : string ) : Group

解析给定的 FBX 数据并返回结果组。

FBXBuffer

作为 ArrayBuffer 的原始 FBX 数据。
path

URL 基础路径。
重写： Loader#parse
返回值： 表示已解析资源的对象。
