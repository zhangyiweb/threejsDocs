LoftGeometry

此类可通过放样（蒙皮）一系列横截面来生成几何体。每个截面是三维空间中的点数组，且所有截面必须具有相同数量的点。

LoftGeometry 是诸如 LatheGeometry（将固定轮廓绕轴旋转）或 TubeGeometry（将圆形截面沿路径扫掠）这类几何体的更一般情形：截面可以是任意形状，并且可以从一个截面到下一个截面改变形状、大小、位置和朝向。

截面围绕放样路径缠绕，使得当从放样末端回望起点时，若每个截面按逆时针顺序排列，则生成的面法线指向外侧。如果表面看起来内外颠倒，请反转每个截面中点的顺序。

代码示例

const sections = [];
for ( let i = 0; i <= 10; i ++ ) {
	const points = [];
	const radius = 2 + Math.sin( i * 0.8 );
	for ( let j = 0; j < 32; j ++ ) {
		const angle = j / 32 * Math.PI * 2;
		points.push( new THREE.Vector3( Math.sin( angle ) * radius, i, Math.cos( angle ) * radius ) );
	}
	sections.push( points );
}
const geometry = new LoftGeometry( sections, { capStart: true, capEnd: true } );
const material = new THREE.MeshStandardMaterial( { color: 0x00ff00 } );
const mesh = new THREE.Mesh( geometry, material );
scene.add( mesh );

导入

LoftGeometry 是一个插件，必须显式导入，参见 Installation#Addons。

import { LoftGeometry } from 'three/addons/geometries/LoftGeometry.js';

构造函数

new LoftGeometry( sections : Array.<Array.<Vector3>>, options : Object )

构造一个新的放样几何体。

sections

用于蒙皮的横截面。至少需要两个截面，且所有截面必须具有相同数量的点。
options

放样选项。

默认值为 {}。

closed

每个截面是视为闭合环（例如机身）还是开放条带（例如丝带）。

默认值为 true。
capStart

是否用封盖封闭第一个截面。

默认值为 false。
capEnd

是否用封盖封闭最后一个截面。

默认值为 false。

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。
