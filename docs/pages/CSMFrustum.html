CSMFrustum

表示 CSM 实例的视锥体。

导入

CSMFrustum 是一个插件，必须显式导入，参见 Installation#Addons。

import { CSMFrustum } from 'three/addons/csm/CSMFrustum.js';

构造函数

new CSMFrustum( data : CSMFrustum~Data )

构造一个新的 CSM 视锥体。

data

CSM 数据。

属性

.vertices : Object

表示近平面与远平面在视图空间中 顶点的对象。

.zFar : number

zFar 值。

.zNear : number

zNear 值。该值取决于 CSM 是与 WebGL 还是 WebGPU 一起使用。两种 API 的投影矩阵约定不同。

方法

.setFromProjectionMatrix( projectionMatrix : Matrix4, maxFar : number ) : Object

根据给定的投影矩阵和最大远裁剪面值设置此 CSM 视锥体。

projectionMatrix

投影矩阵，通常为场景相机的投影矩阵。
maxFar

最大远裁剪面值。
返回值： 表示近平面与远平面在视图空间中顶点的对象。

.split( breaks : Array.<number>, target : Array.<CSMFrustum> )

按给定数组分割 CSM 视锥体。新的 CSM 视锥体将被推入给定的 目标数组。

breaks

取值范围为 [0,1] 的数字数组，用于定义如何分割 CSM 视锥体。
target

用于存放新 CSM 视锥体的目标数组。

.toSpace( cameraMatrix : Matrix4, target : CSMFrustum )

将给定的目标 CSM 视锥体变换到由给定相机矩阵所定义的 另一坐标系中。

cameraMatrix

定义新坐标系的矩阵。
target

要转换的 CSM。

类型定义

.Data

CSMFrustum 的构造函数数据。

webGL
boolean

此 CSM 视锥体是与 WebGL 还是 WebGPU 一起使用。
reversedDepth
boolean

是否启用反向深度缓冲。
projectionMatrix
Matrix4

投影矩阵，通常为场景相机的投影矩阵。
maxFar
number

最大远裁剪面值。
