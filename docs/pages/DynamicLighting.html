DynamicLighting

自定义光照实现，将支持的解析光源批量放入 uniform 数组，从而使光源数量变化不会导致材质重新编译。

代码示例

const lighting = new DynamicLighting( { maxPointLights: 64 } );
renderer.lighting = lighting;

导入

DynamicLighting 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { DynamicLighting } from 'three/addons/lighting/DynamicLighting.js';

构造函数

new DynamicLighting( options : Object )

构造一个新的动态光照系统。

options

动态光照配置。

默认值为 {}。

maxDirectionalLights

批量平行光的最大数量。

默认值为 8。
maxPointLights

批量点光源的最大数量。

默认值为 16。
maxSpotLights

批量聚光灯的最大数量。

默认值为 16。
maxHemisphereLights

批量半球光的最大数量。

默认值为 4。

类

DynamicLighting

方法

.createNode( lights : Array.<Light> ) : DynamicLightsNode

为给定的光源数组创建一个新的动态光源节点。

lights

要绑定到节点的光源列表。
重写： Lighting#createNode
返回值： 动态光源节点。

.getNode( scene : Scene ) : LightsNode

返回给定场景的光源节点。

scene

场景。
重写： Lighting#getNode
返回值： 光源节点。
