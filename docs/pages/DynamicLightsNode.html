DynamicLightsNode

LightsNode 的自定义版本，将受支持的解析光源批量写入 uniform 数组并通过循环处理。

不受支持的光源、节点光源、投射阴影的光源以及投影聚光灯仍走默认的逐光源路径。

导入

DynamicLightsNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { DynamicLightsNode } from 'three/addons/tsl/lighting/DynamicLightsNode.js';

构造函数

new DynamicLightsNode( options : Object )

构造一个新的动态光源节点。

options

动态光照配置。

默认值为 {}。

maxDirectionalLights

批量方向光的最大数量。

默认值为 8。
maxPointLights

批量点光源的最大数量。

默认值为 16。
maxSpotLights

批量聚光灯的最大数量。

默认值为 16。
maxHemisphereLights

批量半球光的最大数量。

默认值为 4。
