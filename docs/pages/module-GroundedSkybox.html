GroundedSkybox

导入

GroundedSkybox 是一个插件，必须显式导入，参见 Installation#Addons。

import { getGroundProjectedNormal } from 'three/addons/tsl/utils/GroundedSkybox.js';

静态方法

.getGroundProjectedNormal( radiusNode : Node.<float>, heightNode : Node.<float> ) : Node.<vec3>

将世界坐标投影到球面上，球体底部由地面圆盘裁剪， 然后返回可用于采样环境立方体贴图的方向向量。

radiusNode

投影球体的半径。必须足够大，以确保场景中的相机始终位于其内部。
heightNode

高度表示拍摄照片时相机距地面的距离。较大的值会放大图像的下方部分。
返回值： 用于采样环境立方体贴图的方向向量。
