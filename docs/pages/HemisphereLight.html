HemisphereLight

一种位于场景正上方的光源，颜色从天空色渐变到地面色。

该光源无法用于投射阴影。

代码示例

const light = new THREE.HemisphereLight( 0xffffbb, 0x080820, 1 );
scene.add( light );

构造函数

new HemisphereLight( skyColor : number | Color | string, groundColor : number | Color | string, intensity : number )

构造一个新的半球光。

skyColor

光源的天空色。

默认值为 0xffffff。
groundColor

光源的地面色。

默认值为 0xffffff。
intensity

光源强度。

默认值为 1。

属性

.groundColor : Color

光源的地面色。

.isHemisphereLight : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
