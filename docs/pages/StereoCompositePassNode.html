StereoCompositePassNode

一种特殊的（抽象）渲染通道节点，将场景渲染为立体图像。与 StereoPassNode 不同，此节点将左右眼的图像合并为单张图像。红蓝立体（Anaglyph）或视差屏障（parallax barrier）等效果需要此节点。

导入

StereoCompositePassNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { StereoCompositePassNode } from 'three/addons/tsl/display/StereoCompositePassNode.js';

构造函数

new StereoCompositePassNode( scene : Scene, camera : Camera ) (abstract)

构造一个新的立体合成通道节点。

scene

要渲染的场景。
camera

用于渲染场景的相机。

属性

.isStereoCompositePassNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.stereo : StereoCamera

用于渲染场景的内部立体相机。

方法

.dispose()

释放内部资源。当通道不再使用时，应调用此方法。

重写： PassNode#dispose

.setSize( width : number, height : number )

设置通道的尺寸。

width

通道的宽度。
height

通道的高度。
重写： PassNode#setSize

.updateBefore( frame : NodeFrame )

此方法用于每帧渲染一次效果。

frame

当前的节点帧。
重写： PassNode#updateBefore

.updateStereoCamera( coordinateSystem : number )

更新内部立体相机。

coordinateSystem

当前的坐标系。
