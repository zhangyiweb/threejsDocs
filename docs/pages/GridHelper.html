GridHelper

该辅助对象用于定义网格。网格是由线段组成的二维阵列。

代码示例

const size = 10;
const divisions = 10;
const gridHelper = new THREE.GridHelper( size, divisions );
scene.add( gridHelper );

构造函数

new GridHelper( size : number, divisions : number, color1 : number | Color | string, color2 : number | Color | string )

构造一个新的网格辅助对象。

size

网格的尺寸。

默认值为 10.
divisions

网格的划分数量。

默认值为 10.
color1

中心线的颜色。

默认值为 0x444444.
color2

网格线的颜色。

默认值为 0x888888.

方法

.dispose()

释放该实例分配的 GPU 相关资源。当应用中不再使用该实例时，应调用此方法。
