FirstPersonControls

此类是 FlyControls 的另一种实现。

导入

FirstPersonControls 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { FirstPersonControls } from 'three/addons/controls/FirstPersonControls.js';

构造函数

new FirstPersonControls( object : Object3D, domElement : HTMLElement )

构造一个新的控制器实例。

object

由控制器管理的对象。
domElement

用于事件监听的 HTML 元素。

默认值为 null。

属性

.autoForward : boolean

相机是否自动向前移动。

默认值为 false。

.constrainVertical : boolean

环顾四周时是否在垂直方向上受 verticalMin 和 verticalMax 约束。

默认值为 false。

.dampingFactor : number

移动与环顾速度追上输入的快慢。值越低手感越重（惯性越大），1 表示禁用阻尼。

默认值为 0.1。

.heightCoef : number

决定当相机的 y 分量接近 heightMax 时，移动速度加快多少。

默认值为 1。

.heightMax : number

用于调整移动速度的相机高度上限。

默认值为 1。

.heightMin : number

用于调整移动速度的相机高度下限。

默认值为 0。

.heightSpeed : boolean

相机高度是否影响向前移动速度。可使用属性 heightCoef、heightMin 和 heightMax 进行配置。

默认值为 false。

.lookSpeed : number

环顾四周的速度。

默认值为 0.005。

.lookVertical : boolean

是否可以在垂直方向上环顾四周。

默认值为 true。

.mouseDragOn : boolean (readonly)

鼠标是否处于按下状态。

默认值为 false。

.movementSpeed : number

移动速度。

默认值为 1。

.verticalMax : number

垂直方向环顾的上限。范围以弧度为单位，从 0 到 Math.PI。

默认值为 0。

.verticalMin : number

垂直方向环顾的下限。范围以弧度为单位，从 0 到 Math.PI。

默认值为 0。

方法

.handleResize()

.lookAt( x : number | Vector3, y : number, z : number ) : FirstPersonControls

将相机旋转朝向指定的目标位置。

x

目标位置的 x 坐标，或者表示目标位置的向量。
y

目标位置的 y 坐标。
z

目标位置的 z 坐标。
返回值： 对该控制器的引用。
