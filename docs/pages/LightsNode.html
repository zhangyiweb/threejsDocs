LightsNode

此节点表示场景的光照，并管理当前构建的 3D 对象的光照模型生命周期。 它负责在给定的光照上下文中计算总出射光。

构造函数

new LightsNode()

构造一个新的光照节点。

属性

.global : boolean

LightsNode 默认将此属性设为 true。

默认值为 true。

重写： Node#global

.hasLights : boolean

场景中是否有灯光。

.outgoingLightNode : Node.<vec3>

表示出射光的节点。

.totalDiffuseNode : Node.<vec3>

表示总漫反射光的节点。

.totalSpecularNode : Node.<vec3>

表示总镜面反射光的节点。

方法

.analyze( builder : NodeBuilder )

通过构建所有嵌套的光照节点和输出节点来分析节点的依赖关系。

builder

对当前节点构建器的引用。
重写： Node#analyze

.customCacheKey() : number

通过将光照数据纳入缓存键，覆盖默认的 Node#customCacheKey 实现。

重写： Node#customCacheKey
返回值： 自定义缓存键。

.getHash( builder : NodeBuilder ) : string

计算用于标识当前光照节点配置的哈希值。

builder

对当前节点构建器的引用。
重写： Node#getHash
返回值： 计算得到的哈希值。

.getLights() : Array.<Light>

返回场景灯光的数组。

返回值： 场景中的灯光。

.setLights( lights : Array.<Light> ) : LightsNode

使用灯光数组配置此节点。

lights

灯光数组。
返回值： 对此节点的引用。

.setup( builder : NodeBuilder ) : Node.<vec3>

该实现确保场景中的每个灯光都有对应的光照节点。 通过构建光照节点并评估光照模型来计算出射光。

builder

对当前节点构建器的引用。
重写： Node#setup
返回值： 表示出射光的节点。

.setupDirectLight( builder : Object, lightNode : Object, lightData : Object )

在光照模型中设置直接光。

builder

包含上下文和栈的构建器对象。
lightNode

光照节点。
lightData

包含颜色和方向属性的灯光对象。

.setupDirectRectAreaLight( builder : Object, lightNode : Object, lightData : Object )

在光照模型中设置直接矩形区域光。

builder

包含上下文和栈的构建器对象。
lightNode

光照节点。
lightData

包含颜色和区域光属性的灯光对象。

.setupLights( builder : NodeBuilder, lightNodes : Array.<LightingNode> )

通过构建所有相应的光照节点来设置内部灯光。

builder

对当前节点构建器的引用。
lightNodes

光照节点数组。

.setupLightsNode( builder : NodeBuilder ) : Array.<LightingNode>

为场景中的每个灯光创建光照节点。这使得可以在节点系统中进一步处理灯光。

builder

对当前节点构建器的引用。
返回值： 光照节点数组。
