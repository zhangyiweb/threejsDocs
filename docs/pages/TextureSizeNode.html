TextureSizeNode

表示纹理尺寸的节点。纹理尺寸通过内置着色器函数 如 textureDimensions() 或 textureSize() 在着色器中获取。

构造函数

new TextureSizeNode( textureNode : TextureNode, levelNode : Node.<int> )

构造一个新的纹理尺寸节点。

textureNode

需要获取尺寸的纹理节点。
levelNode

定义所请求 mip 级别的节点。

默认值为 null。

属性

.isTextureSizeNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.levelNode : Node.<int>

定义所请求 mip 级别的节点。

默认值为 null。

.textureNode : TextureNode

需要获取尺寸的纹理节点。
