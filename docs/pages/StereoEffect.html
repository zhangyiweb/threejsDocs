StereoEffect

用于创建立体效果的类。

注意：此类仅可与 WebGLRenderer 配合使用。 使用 WebGPURenderer 时，请使用 StereoPassNode。

导入

StereoEffect 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { StereoEffect } from 'three/addons/effects/StereoEffect.js';

构造函数

new StereoEffect( renderer : WebGLRenderer )

构造一个新的立体效果。

renderer

渲染器。

方法

.render( scene : Object3D, camera : Camera )

使用此效果时，应调用此方法而非默认的 WebGLRenderer#render。

scene

要渲染的场景。
camera

相机。

.setEyeSeparation( eyeSep : number )

设置给定的瞳距。

eyeSep

要设置的瞳距。

.setSize( width : number, height : number )

调整效果尺寸。

width

效果的宽度（逻辑像素）。
height

效果的高度（逻辑像素）。
