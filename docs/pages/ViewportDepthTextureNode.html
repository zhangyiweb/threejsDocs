ViewportDepthTextureNode

将当前视口的深度表示为纹理。此模块 可与视口纹理配合使用，以实现需要深度评估的效果。

构造函数

new ViewportDepthTextureNode( uvNode : Node, levelNode : Node, depthTexture : DepthTexture )

构造一个新的视口深度纹理节点。

uvNode

uv 节点。

默认值为 screenUV。
levelNode

level 节点。

默认值为 null。
depthTexture

深度纹理。若未提供，则使用共享深度纹理。

默认值为 null。
