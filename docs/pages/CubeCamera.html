CubeCamera

一种特殊相机，放置在 3D 空间中以将其周围环境渲染到立方体渲染目标。随后可将该渲染目标用作环境贴图，为场景中的实时反射提供支持。

代码示例

// Create cube render target
const cubeRenderTarget = new THREE.WebGLCubeRenderTarget( 256, { generateMipmaps: true, minFilter: THREE.LinearMipmapLinearFilter } );
// Create cube camera
const cubeCamera = new THREE.CubeCamera( 1, 100000, cubeRenderTarget );
scene.add( cubeCamera );
// Create car
const chromeMaterial = new THREE.MeshLambertMaterial( { color: 0xffffff, envMap: cubeRenderTarget.texture } );
const car = new THREE.Mesh( carGeometry, chromeMaterial );
scene.add( car );
// Update the render target cube
car.visible = false;
cubeCamera.position.copy( car.position );
cubeCamera.update( renderer, scene );
// Render the scene
car.visible = true;
renderer.render( scene, camera );

构造函数

new CubeCamera( near : number, far : number, renderTarget : WebGLCubeRenderTarget )

构造一个新的立方体相机。

near

相机的近裁剪面。
far

相机的远裁剪面。
renderTarget

立方体渲染目标。

属性

.activeMipmapLevel : number

当前活动的 mipmap 级别。

默认值为 0。

.coordinateSystem : WebGLCoordinateSystem | WebGPUCoordinateSystem

当前活动的坐标系。

默认值为 null。

.renderTarget : WebGLCubeRenderTarget

对立方体渲染目标的引用。

方法

.update( renderer : Renderer | WebGLRenderer, scene : Scene )

调用此方法将使用给定渲染器将给定场景渲染到相机的立方体渲染目标中。

renderer

渲染器。
scene

要渲染的场景。

.updateCoordinateSystem()

当立方体相机的坐标系发生更改时必须调用。
