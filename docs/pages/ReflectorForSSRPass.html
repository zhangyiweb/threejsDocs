ReflectorForSSRPass

专用于与 SSRPass 配合使用的特殊版 Reflector。

导入

ReflectorForSSRPass 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { ReflectorForSSRPass } from 'three/addons/objects/ReflectorForSSRPass.js';

构造函数

new ReflectorForSSRPass( geometry : BufferGeometry, options : ReflectorForSSRPass~Options )

构造一个新的反射器。

geometry

反射器的几何体。
options

配置选项。

方法

.dispose()

释放此实例分配的 GPU 相关资源。 当应用中不再使用此实例时，请调用此方法。

.getRenderTarget() : WebGLRenderTarget

返回反射器的内部渲染目标。

返回值： 内部渲染目标

类型定义

.Options

ReflectorForSSRPass 的构造函数选项。

color
number | Color | string

反射器的颜色。

默认值为 0x7F7F7F。
textureWidth
number

纹理宽度。值越高，反射越清晰，但开销也更大。

默认值为 512。
textureHeight
number

纹理高度。值越高，反射越清晰，但开销也更大。

默认值为 512。
clipBias
number

裁剪偏置。

默认值为 0。
shader
Object

可用于传入自定义着色器，以定义如何将反射视图投影到反射器的几何体上。
useDepthTexture
boolean

是否将深度值存储到纹理中。

默认值为 true。
resolution
Vector2

Reflector Pass 的分辨率。
