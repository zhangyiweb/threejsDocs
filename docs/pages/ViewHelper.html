ViewHelper

一种特殊的辅助对象，在视口的小区域中以坐标轴辅助的形式 可视化相机的变换。此类辅助对象在 3D 建模工具或场景编辑器中 （例如 three.js editor）经常需要。

该辅助对象允许点击 X、Y 和 Z 轴，从而动画化相机， 使其沿所选轴方向观察。

导入

ViewHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { ViewHelper } from 'three/addons/helpers/ViewHelper.js';

构造函数

new ViewHelper( camera : Camera, domElement : HTMLElement )

构造一个新的视图辅助对象。

camera

要可视化其变换的相机。
domElement

用于渲染视图的 DOM 元素。

属性

.animating : boolean (readonly)

辅助对象当前是否正在执行动画。

默认值为 false。

.camera : Camera

正在可视化其变换的相机。可随时重新赋值， 以将辅助对象绑定到另一台相机。

.center : Vector3

辅助对象的中心点。

.isViewHelper : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.location : Object

控制辅助对象在视口中的位置。 使用 top/bottom 进行垂直定位，使用 left/right 进行水平定位。 若 left 为 null，则使用 right。若 top 为 null，则使用 bottom。

方法

.dispose()

释放此实例分配的 GPU 相关资源。 当应用中不再使用此实例时，请调用此方法。

.handleClick( event : PointerEvent ) : boolean

当应用中发生点击或指针事件时，应调用此方法。

event

要处理的事件。
返回值： 是否检测到与辅助对象的相交。

.render( renderer : WebGLRenderer | WebGPURenderer )

在视口中的独立视图内渲染辅助对象。 位置由 location 属性控制。

renderer

渲染器。

.setLabelStyle( font : string, color : string, radius : number )

设置标签样式。当坐标轴未标注时无效。

font

标签字体。

默认值为 '24px Arial'。
color

标签颜色。

默认值为 '#000000'。
radius

标签半径。

默认值为 14。

.setLabels( labelX : string | undefined, labelY : string | undefined, labelZ : string | undefined )

为各坐标轴设置标签。默认情况下无标签。

labelX

X 轴的标签。
labelY

Y 轴的标签。
labelZ

Z 轴的标签。

.update( delta : number )

更新辅助对象。应在应用的动画循环中调用此方法。

delta

以秒为单位的时间增量。
