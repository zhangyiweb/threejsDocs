NodeUniform

NodeBuilder 在构建节点的过程中会创建此类的实例。 它们表示构建器最终将生成的着色器 uniform。为此，节点 uniform 字典维护在 NodeBuilder#uniforms 中。

构造函数

new NodeUniform( name : string, type : string, node : UniformNode )

构造一个新的节点 uniform。

name

uniform 的名称。
type

uniform 的类型。
node

对节点的引用。

属性

.groupNode : UniformGroupNode

uniform 节点的组。

.id : number

uniform 节点的 ID。

.isNodeUniform : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.name : string

uniform 的名称。

.node : UniformNode

对节点的引用。

.type : string

uniform 的类型。

.value : any

uniform 节点的值。
