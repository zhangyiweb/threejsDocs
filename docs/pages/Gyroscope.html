Gyroscope

一种特殊的 3D 对象，其位置来自场景图层级， 但使用局部旋转作为世界旋转。它的工作方式类似真实世界的陀螺仪—— 你可以通过层级移动它，而其朝向相对于世界保持固定。

导入

Gyroscope 是一个插件，必须显式导入，参见 Installation#Addons。

import { Gyroscope } from 'three/addons/misc/Gyroscope.js';

构造函数

new Gyroscope()

构造一个新的陀螺仪。
