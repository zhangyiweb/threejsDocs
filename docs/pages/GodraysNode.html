GodraysNode

用于对场景应用屏幕空间光线步进体积光（godrays）的后处理节点。

体积光计算完成后，建议对结果应用双边模糊（Bilateral Blur），以减轻光线步进与噪声伪影。

与场景通道的合成理想情况下应使用 depthAwareBlend()，可减轻锯齿与漏光。

限制：

  • 目前仅支持点光源与方向光。
  • 该效果需要完整的阴影配置：渲染器须启用阴影，3D 对象须投射并接收阴影，且主光源须投射阴影。

参考：该节点源自 three-good-godrays。

代码示例

const godraysPass = godrays( scenePassDepth, camera, light );
const blurPass = bilateralBlur( godraysPassColor ); // 可选模糊
const outputBlurred = depthAwareBlend( scenePassColor, blurPassColor, scenePassDepth, camera, { blendColor, edgeRadius, edgeStrength } ); // 合成

导入

GodraysNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { godrays } from 'three/addons/tsl/display/GodraysNode.js';

构造函数

new GodraysNode( depthNode : TextureNode, camera : Camera, light : DirectionalLight | PointLight )

构造一个新的 Godrays 节点。

depthNode

表示场景深度的纹理节点。
camera

用于渲染场景的相机。
light

用于渲染体积光的光源。

属性

.density : UniformNode.<float>

体积光的累积速率。数值越高，大致相当于空气更潮湿/雾更浓。

默认值为 0.7。

.depthNode : TextureNode

表示 beauty pass 深度的节点。

.distanceAttenuation : UniformNode.<float>

数值越高，离光源越远时体积光累积越少。

默认值为 2。

.maxDensity : UniformNode.<float>

体积光的最大密度，用于限制体积光的最大亮度。

默认值为 0.5。

.raymarchSteps : UniformNode.<uint>

光线步进的步数。

默认值为 60。

.resolutionScale : number

分辨率缩放。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.FRAME，因为该节点会在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

方法

.dispose()

释放内部资源。当不再需要该效果时应调用此方法。

重写： TempNode#dispose

.getTextureNode() : PassTextureNode

将效果结果作为纹理节点返回。

返回值： 表示效果结果的纹理节点。

.setSize( width : number, height : number )

设置效果的尺寸。

width

效果的宽度。
height

效果的高度。

.setup( builder : NodeBuilder ) : PassTextureNode

用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
