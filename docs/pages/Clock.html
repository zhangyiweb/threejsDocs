Clock

用于跟踪时间的类。

构造函数

new Clock( autoStart : boolean )

创建一个新的时钟。

autoStart

当第一次调用 getDelta() 时，是否自动启动时钟。

默认值为 true。
已废弃：自 r183 起。

属性

.autoStart : boolean

如果设为 true，时钟会在第一次调用 getDelta() 时自动启动。

默认值为 true。

.elapsedTime : number

记录时钟运行的总时长。

默认值为 0。

.oldTime : number

保存时钟上次调用 start()、getElapsedTime() 或 getDelta() 的时间。

默认值为 0。

.running : boolean

时钟是否正在运行。

默认值为 true。

.startTime : number

保存时钟上次调用 start() 方法的时间。

默认值为 0。

方法

.getDelta() : number

返回以秒为单位的时间增量。

返回值：时间增量。

.getElapsedTime() : number

返回以秒为单位的已过时间。

返回值：已过时间。

.start()

启动时钟。当 autoStart 设为 true 时，该方法会由类自动调用。

.stop()

停止时钟。
