FlakesTexture

用于生成片状纹理图像的工具类。该图像可用作 法线贴图，以产生类似汽车漆面的效果。

导入

FlakesTexture 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { FlakesTexture } from 'three/addons/textures/FlakesTexture.js';

构造函数

new FlakesTexture( width : number, height : number ) : HTMLCanvasElement

生成一张新的片状纹理图像。返回的 canvas 可作为 CanvasTexture 的输入。

width

图像宽度。

默认值为 512。
height

图像高度。

默认值为 512。
返回值： 生成的图像。
