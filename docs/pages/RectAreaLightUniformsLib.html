RectAreaLightUniformsLib

此类仅在将 RectAreaLight 与 WebGLRenderer 一起使用时相关。

在使用矩形面光源之前，必须用以下代码增强渲染器的内部 uniform 库。

代码示例

RectAreaLightUniformsLib.init();

导入

RectAreaLightUniformsLib 是一个插件，必须显式导入，参见 Installation#Addons。

import { RectAreaLightUniformsLib } from 'three/addons/lights/RectAreaLightUniformsLib.js';

静态方法

.init()

初始化使用矩形面光源时所需的 uniform 库。
