Object3D

这是 three.js 中大多数对象的基类，提供一组用于在 3D 空间中操作对象的属性和方法。

构造函数

new Object3D()

构造一个新的 3D 对象。

属性

.animations : Array.<AnimationClip>

存储该 3D 对象动画片段的数组。

.castShadow : boolean

当设置为 true 时，该 3D 对象会被渲染到阴影贴图中。

默认值为 false.

.children : Array.<Object3D>

存储该实例子 3D 对象的数组。

.customDepthMaterial : Material | undefined

渲染到深度贴图时使用的自定义深度材质。仅可在网格上下文中使用。使用 DirectionalLight 或 SpotLight 投射阴影时，若在顶点着色器中修改了顶点位置，则必须指定自定义深度材质才能获得正确的阴影效果。

仅在 WebGLRenderer 上下文中相关。

默认值为 undefined.

.customDistanceMaterial : Material | undefined

与 Object3D#customDepthMaterial 相同，但用于 PointLight。

仅在 WebGLRenderer 上下文中相关。

默认值为 undefined.

.frustumCulled : boolean

当设置为 true 时，该 3D 对象会参与视锥体剔除。

默认值为 true.

.id : number (readonly)

该 3D 对象的 ID。

.isObject3D : boolean (readonly)

此标志可用于类型检测。

默认值为 true.

.layers : Layers

该 3D 对象的图层成员关系。仅当 3D 对象与当前使用的相机至少共有一个图层时才可见。使用 Raycaster 进行射线相交检测时，此属性也可用于过滤不需要的对象。

.matrix : Matrix4

表示对象在本地空间中的变换矩阵。

.matrixAutoUpdate : boolean

当设置为 true 时，引擎会在每一帧根据 position、rotation 和 scale 自动计算本地矩阵。若设置为 false，则由应用程序负责通过调用 updateMatrix() 重新计算本地矩阵。

所有 3D 对象的默认值由 Object3D.DEFAULT_MATRIX_AUTO_UPDATE 定义。

默认值为 true.

.matrixWorld : Matrix4

表示对象在世界空间中的变换矩阵。若 3D 对象没有父对象，则与世界变换矩阵相同。

.matrixWorldAutoUpdate : boolean

当设置为 true 时，引擎会根据当前本地矩阵和对象的变换层级自动计算世界矩阵。若设置为 false，则由应用程序负责通过直接更新 matrixWorld 属性来重新计算世界矩阵。

所有 3D 对象的默认值由 Object3D.DEFAULT_MATRIX_WORLD_AUTO_UPDATE 定义。

默认值为 true.

.matrixWorldNeedsUpdate : boolean

当设置为 true 时，会在该帧计算世界矩阵，并将此属性重置为 false。

默认值为 false.

.modelViewMatrix : Matrix4

表示对象的模型视图矩阵。

.name : string

该 3D 对象的名称。

.normalMatrix : Matrix3

表示对象的法线矩阵。

.parent : Object3D

对父对象的引用。

默认值为 null.

.pivot : Vector3

旋转和缩放变换的枢轴点。设置后，旋转和缩放将围绕此点而非对象原点进行。

默认值为 null.

.position : Vector3

表示对象的本地位置。

默认值为 (0,0,0).

.quaternion : Quaternion

以四元数表示对象的本地旋转。

.receiveShadow : boolean

当设置为 true 时，该 3D 对象会受场景中阴影的影响。

默认值为 false.

.renderOrder : number

此值允许覆盖场景图对象的默认渲染顺序，但不透明对象和透明对象仍分别独立排序。当为 Group 实例设置此属性时，所有后代对象将一起排序并渲染。排序按 renderOrder 从低到高进行。

默认值为 0.

.rotation : Euler

以欧拉角（弧度）表示对象的本地旋转。

默认值为 (0,0,0).

.scale : Vector3

表示对象的本地缩放。

默认值为 (1,1,1).

.static : boolean

该 3D 对象是否应为静态。若设置为 true，表示 3D 对象在初始渲染后不会再发生变化，包括几何体和材质设置。静态 3D 对象可被渲染器略快处理，因为可以跳过某些状态检查。

仅在 WebGPURenderer 上下文中相关。

默认值为 false.

.type : string (readonly)

type 属性用于在序列化/反序列化上下文中检测对象类型。

.up : Vector3

定义 3D 对象的 up 方向，会影响 Object3D#lookAt 等方法的方向计算。

所有 3D 对象的默认值由 Object3D.DEFAULT_UP 定义。

.userData : Object

可用于存储与该 3D 对象相关的自定义数据的对象。不应保存对函数的引用，因为这些引用不会被克隆。

.uuid : string (readonly)

该 3D 对象的 UUID。

.visible : boolean

当设置为 true 时，该 3D 对象会被渲染。

默认值为 true.

.DEFAULT_MATRIX_AUTO_UPDATE : boolean

新创建 3D 对象的 Object3D#matrixAutoUpdate 默认设置。

默认值为 true.

.DEFAULT_MATRIX_WORLD_AUTO_UPDATE : boolean

新创建 3D 对象的 Object3D#matrixWorldAutoUpdate 默认设置。

默认值为 true.

.DEFAULT_UP : Vector3

对象的默认 up 方向，也用作 DirectionalLight 和 HemisphereLight 的默认位置。

默认值为 (0,1,0).

方法

.add( object : Object3D ) : Object3D

将给定 3D 对象作为子对象添加到该 3D 对象。可添加任意数量的对象。传入对象的当前父对象会被移除，因为一个对象最多只能有一个父对象。

object

要添加的 3D 对象。
触发：
  • Object3D#event:added
  • Object3D#event:childadded
返回值： 对该实例的引用。

.applyMatrix4( matrix : Matrix4 )

将给定变换矩阵应用于对象，并更新对象的位置、旋转和缩放。

matrix

变换矩阵。

.applyQuaternion( q : Quaternion ) : Object3D

将给定四元数所表示的旋转应用于 3D 对象。

q

四元数。
返回值： 对该实例的引用。

.attach( object : Object3D ) : Object3D

将给定 3D 对象作为该 3D 对象的子对象添加，同时保持对象的世界变换。此方法不支持包含非均匀缩放节点的场景图。

object

要附加的 3D 对象。
触发：
  • Object3D#event:added
  • Object3D#event:childadded
返回值： 对该实例的引用。

.clear() : Object3D

移除所有子对象。

触发：
  • Object3D#event:removed
  • Object3D#event:childremoved
返回值： 对该实例的引用。

.clone( recursive : boolean ) : Object3D

返回一个从该实例复制值的新 3D 对象。

recursive

当设置为 true 时，3D 对象的后代也会被克隆。

默认值为 true.
返回值： 当前实例的克隆。

.copy( source : Object3D, recursive : boolean ) : Object3D

将给定 3D 对象的值复制到该实例。

source

要复制的 3D 对象。
recursive

当设置为 true 时，3D 对象的后代会被克隆。

默认值为 true.
返回值： 对该实例的引用。

.getObjectById( id : number ) : Object3D | undefined

从该 3D 对象自身开始，遍历该 3D 对象及其子对象，返回第一个 ID 匹配的对象。

id

ID。
返回值： 找到的 3D 对象。若未找到 3D 对象则返回 undefined。

.getObjectByName( name : string ) : Object3D | undefined

从该 3D 对象自身开始，遍历该 3D 对象及其子对象，返回第一个名称匹配的对象。

name

名称。
返回值： 找到的 3D 对象。若未找到 3D 对象则返回 undefined。

.getObjectByProperty( name : string, value : any ) : Object3D | undefined

从该 3D 对象自身开始，遍历该 3D 对象及其子对象，返回第一个属性值匹配的对象。

name

属性名称。
value

值。
返回值： 找到的 3D 对象。若未找到 3D 对象则返回 undefined。

.getObjectsByProperty( name : string, value : any, result : Array.<Object3D> ) : Array.<Object3D>

从该 3D 对象自身开始，遍历该 3D 对象及其子对象，返回所有属性值匹配的 3D 对象。

name

属性名称。
value

值。
result

此方法将结果存储在该数组中。
返回值： 找到的 3D 对象。

.traverse( callback : function )

在该 3D 对象及其所有后代上执行回调。

注意：不建议在回调中修改场景图。

callback

用于处理当前 3D 对象的回调函数。

.traverseAncestors( callback : function )

与 Object3D#traverse 类似，但回调仅对所有祖先执行。

注意：不建议在回调中修改场景图。

callback

用于处理当前 3D 对象的回调函数。

.traverseVisible( callback : function )

与 Object3D#traverse 类似，但回调仅对可见的 3D 对象执行。不可见 3D 对象的后代不会被遍历。

注意：不建议在回调中修改场景图。

callback

用于处理当前 3D 对象的回调函数。

.updateMatrix()

根据当前的 position、rotation 和 scale 值计算并更新本地空间的变换矩阵。

.updateMatrixWorld( force : boolean )

更新该 3D 对象及其后代在世界空间中的变换矩阵。

为确保结果正确，此方法还会重新计算 3D 对象在本地空间的变换矩阵。本地矩阵和世界矩阵的计算可通过 Object3D#matrixAutoUpdate 和 Object3D#matrixWorldAutoUpdate 标志控制，两者默认均为 true。若需要更多控制，可将这些标志设为 false。

force

当设置为 true 时，即使 Object3D#matrixWorldNeedsUpdate 为 false，也会强制重新计算世界矩阵。

默认值为 false.

.updateWorldMatrix( updateParents : boolean, updateChildren : boolean, force : boolean )

Object3D#updateMatrixWorld 的替代版本，可对祖先节点和后代节点的更新进行更精细的控制。

updateParents

是否更新祖先节点。

默认值为 false.
updateChildren

是否更新后代节点。

默认值为 false.
force

当设置为 true 时，即使 Object3D#matrixWorldNeedsUpdate 为 false，也会强制重新计算世界矩阵。

默认值为 false.
