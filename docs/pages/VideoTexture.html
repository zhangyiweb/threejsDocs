VideoTexture

用于视频的纹理。

注意：在 WebGPURenderer 中使用视频纹理时，必须将 Texture#colorSpace 设为 THREE.SRGBColorSpace。

注意：纹理首次使用后，其尺寸、格式和类型 不能更改。应改为调用 Texture#dispose 并实例化一个新纹理。

代码示例

// assuming you have created a HTML video element with id="video"
const video = document.getElementById( 'video' );
const texture = new THREE.VideoTexture( video );

构造函数

new VideoTexture( video : HTMLVideoElement, mapping : number, wrapS : number, wrapT : number, magFilter : number, minFilter : number, format : number, type : number, anisotropy : number )

构造一个新的视频纹理。

video

用作纹理数据源的视频元素。
mapping

纹理映射方式。

默认值为 Texture.DEFAULT_MAPPING。
wrapS

wrapS 值。

默认值为 ClampToEdgeWrapping。
wrapT

wrapT 值。

默认值为 ClampToEdgeWrapping。
magFilter

放大过滤器值。

默认值为 LinearFilter。
minFilter

缩小过滤器值。

默认值为 LinearFilter。
format

纹理格式。

默认值为 RGBAFormat。
type

纹理类型。

默认值为 UnsignedByteType。
anisotropy

各向异性值。

默认值为 Texture.DEFAULT_ANISOTROPY。

属性

.generateMipmaps : boolean

是否为纹理生成 mipmap（若可能）。

已覆盖，默认设置为 false。

默认值为 false。

重写： Texture#generateMipmaps

.isVideoTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.update()

此方法由渲染器自动调用，并在每次有新帧可用时将 Texture#needsUpdate 设为 true。

仅在浏览器不支持 requestVideoFrameCallback 时相关。
