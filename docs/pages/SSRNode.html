SSRNode

用于计算屏幕空间反射（SSR）的后处理节点。

参考：https://lettier.github.io/3d-game-shaders-for-beginners/screen-space-reflection.html

导入

SSRNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { ssr } from 'three/addons/tsl/display/SSRNode.js';

构造函数

new SSRNode( colorNode : Node.<vec4>, depthNode : Node.<float>, normalNode : Node.<vec3>, options : SSRNodeOptions )

构造一个新的 SSR 节点。

colorNode

表示 beauty 通道的节点。
depthNode

表示 beauty 通道深度的节点。
normalNode

表示 beauty 通道法线的节点。
options

材质和环境数据的可选输入。

属性

._binaryRefine : boolean

启用检测命中后的子步长二分搜索精化。开启后，对粗略穿越点进行二分以逼近精确交点（更锐利的命中，减少步进锯齿）， 但需额外深度采样。作为编译时常量烘焙进着色器；赋新值会重建 SSR 材质。

默认值为 false。

._blurQuality : number

模糊质量。必须是 [1,3] 范围内的整数。

作为编译时常量烘焙进模糊着色器，使 (size*2+1)² 采样循环展开；赋新值会重新编译模糊材质。

默认值为 2。

._reflectNonMetals : boolean

仅在 SSRNode#stochastic 为 false 时使用。当为 false 时，丢弃非金属表面以显著节省性能；设为 true 可同时反射电介质。作为编译时常量烘焙进着色器；赋新值会重新编译 SSR 材质。

默认值为 false。

._screenEdgeFadeBlack : boolean

当为 true 时，SSR 在屏幕边缘附近淡出至零，而非混合向环境贴图。命中由反射采样 UV 淡出；未命中由表面像素 UV 淡出。

作为编译时常量烘焙进着色器以消除未使用的淡出分支；赋新值会重新编译 SSR 材质。

默认值为 false。

._stepExponent : number

非线性步进分布指数。1 = 均匀步进；> 1 将采样集中在射线原点附近——大多数短程反射在此被遗漏—— 并随 maxDistance 增大间距，公式为 s = (i / steps) ^ stepExponent。

作为编译时常量烘焙进着色器，使 pow() 折叠为少量乘法；赋新值会重新编译 SSR 材质。仅用于随机反射路径。

默认值为 2。

.binaryRefine : boolean

是否启用子步长二分搜索命中精化（编译时常量）。 赋新值会重建 SSR 材质。

.blurQuality : number

模糊核大小（编译时常量）。赋新值会重新编译模糊材质。

.camera : Camera

用于渲染场景的相机。

.colorNode : Node.<vec4>

表示 beauty 通道的节点。

.depthNode : Node.<float>

表示 beauty 通道深度的节点。

.diffuseNode : Node.<vec4>

表示场景漫反射颜色的节点（通常为 MRT diffuseColor 附件）。 当为 null 时，着色器使用 vec3(1)。

.envImportanceSampling : boolean

当为 true 时，构建环境亮度 CDF 表，并对环境未命中使用 MIS。 在构造时固定。

.envMapIntensity : UniformNode.<float>

重要性采样环境贡献的强度乘数。 仅在调用 setEnvMap 后可用。

.environmentIntensity : UniformNode.<float>

应用于屏幕空间未命中和屏幕边缘处环境贴图反射的强度乘数。默认为 π，以匹配原先硬编码的乘数。

默认值为 Math.PI。

.environmentNode : Texture

用于屏幕空间未命中的 HDR 环境贴图。

.historyTexture : Texture

表示多次反弹反射历史纹理的节点。

.intensity : UniformNode.<float>

整体反射强度的乘数。1 保持反射不变，较低值使其变暗，较高值使其增强。

默认值为 1。

.maxDistance : UniformNode.<float>

控制片段可反射的距离。增大此值会增加计算开销，但也会增大反射距离。

.maxLuminance : UniformNode.<float>

绝对环境亮度上限。超过此值的 HDR 环境采样会被缩小（保留色相）。

默认值为 10。

.metalnessNode : Node.<float>

逐像素金属度，用于驱动 GGX 反射采样和非金属提前退出。当为 null 时，着色器将表面视为非金属。

.mirrorBias : UniformNode.<float>

随机 GGX 采样的镜面偏置。将反射射线集中在波瓣的窄（近镜面）核心，以少量偏置换取更少噪点。 0 采样完整 VNDF 波瓣；趋近 1 的值收紧锥体。范围 [0,1]。

默认值为 0.5。

.normalNode : Node.<vec3>

表示 beauty 通道法线的节点。

.quality : UniformNode.<float>

此参数控制光线步进的精细程度。 取值范围为 [0,1]，其中 1 表示最佳质量（最大光线步进迭代/采样数），0 表示完全不采样。

对于大多数用例，0.5 的质量通常已足够。请尽量保持此参数尽可能低。较大的值会导致明显更多的开销。

.reflectNonMetals : boolean

非随机路径中是否反射电介质（编译时常量）。 赋新值会重建 SSR 材质。

.resolutionScale : number

分辨率缩放。有效值在 [0,1] 范围内。1 表示最佳质量，但也会导致 更多计算开销。设为 0.5 表示以半分辨率计算效果。

默认值为 1。

.roughnessNode : Node.<float>

逐像素粗糙度，用于驱动 GGX 反射采样和模糊 mip 选择。当为 null 时，着色器将表面视为完全光滑。

.screenEdgeFade : UniformNode.<float>

屏幕边缘淡出宽度（UV 单位）。当屏幕空间命中接近屏幕边界时，反射在此距离内淡出—— 要么向环境反射淡出（SSRNode#screenEdgeFadeBlack 为 false）， 要么淡出至零强度（true）。0 表示禁用。

默认值为 0.2。

.screenEdgeFadeBlack : boolean

SSR 是否在屏幕边缘附近淡出至黑色（编译时常量）。赋新值会重新编译 SSR 材质。

.stepExponent : number

非线性步进分布指数（编译时常量）。详见后备字段。赋新值会重新编译 SSR 材质。

.stochastic : boolean

当为 true 时，每像素以随机 GGX 射线变化反射方向（第二代 SSR）。当为 false 时，追踪单一镜面反射， 并通过模糊通道柔化粗糙度（第一代 SSR）。

.thickness : UniformNode.<float>

控制可能反射命中与未命中之间的截止阈值。

.updateBeforeType : string

updateBeforeType 被设置为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

.velocityTexture : Node.<vec2>

表示用于重投影的速度纹理的节点。

方法

.dispose()

释放内部资源。当效果不再使用时应调用此方法。

重写： TempNode#dispose

.getTextureNode() : PassTextureNode

以纹理节点形式返回效果结果。

返回值： 表示效果结果的纹理节点。

.setEnvMap( hdr : Texture | null )

设置屏幕空间射线未命中时用于重要性采样环境光照的环境贴图。 当场景环境贴图变化时应调用此方法。

使用 ImportanceSampledEnvironment（CDF + MIS，改编自 three-gpu-pathtracer）。

hdr

等距柱状投影 HDR 环境贴图，或 null 以禁用。
参见：
  • https://github.com/gkjohnson/three-gpu-pathtracer

.setHistory( history : Texture, velocity : Node.<vec2> )

连接多次反弹反射的反馈输入：上一帧的降噪结果（history）和用于重投影的速度缓冲区（velocity）。 history 可接受产生该结果的节点（例如 RecurrentDenoiseNode）——使用其输出渲染目标——或原始纹理。 两者均传入 null 可禁用多次反弹。

history
velocity

.setSize( width : number, height : number )

设置效果的尺寸。

width

效果的宽度。
height

效果的高度。

.setup( builder : NodeBuilder ) : PassTextureNode

此方法用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

此方法用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
