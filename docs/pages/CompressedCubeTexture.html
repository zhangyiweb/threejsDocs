CompressedCubeTexture

基于压缩形式的数据创建立方体纹理。

此类纹理通常通过 CompressedTextureLoader 加载。

构造函数

new CompressedCubeTexture( images : Array.<CompressedTexture>, format : number, type : number )

构造一个新的压缩纹理。

images

压缩纹理数组。
format

纹理格式。

默认值为 RGBAFormat。
type

纹理类型。

默认值为 UnsignedByteType。

属性

.isCompressedCubeTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.isCubeTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
