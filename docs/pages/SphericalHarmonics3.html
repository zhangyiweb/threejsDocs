SphericalHarmonics3

表示三阶球谐函数（SH）。光照探针使用此类 来编码光照信息。

  • 主要参考：https://graphics.stanford.edu/papers/envmap/envmap.pdf
  • 次要参考：https://www.ppsloan.org/publications/StupidSH36.pdf

构造函数

new SphericalHarmonics3()

构造一个新的球谐函数。

属性

.coefficients : Array.<Vector3>

保存（9 个）SH 系数的数组。

.isSphericalHarmonics3 : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.add( sh : SphericalHarmonics3 ) : SphericalHarmonics3

将给定的 SH 加到本实例上。

sh

要相加的 SH。
返回值： 对此球谐函数的引用。

.addScaledSH( sh : SphericalHarmonics3, s : number ) : SphericalHarmonics3

便捷方法，可一次执行 SphericalHarmonics3#add 和 SphericalHarmonics3#scale。

sh

要相加的 SH。
s

缩放因子。
返回值： 对此球谐函数的引用。

.clone() : SphericalHarmonics3

返回一个新球谐函数，其值从本实例复制而来。

返回值： 本实例的克隆。

.copy( sh : SphericalHarmonics3 ) : SphericalHarmonics3

将给定球谐函数的值复制到本实例。

sh

要复制的球谐函数。
返回值： 对此球谐函数的引用。

.equals( sh : SphericalHarmonics3 ) : boolean

如果此球谐函数与给定球谐函数相等，则返回 true。

sh

要测试相等性的球谐函数。
返回值： 此球谐函数是否与给定球谐函数相等。

.fromArray( array : Array.<number>, offset : number ) : SphericalHarmonics3

从给定数组设置本实例的 SH 系数。

array

保存 SH 系数的数组。
offset

开始复制的数组偏移量。

默认值为 0。
返回值： 本实例的克隆。

.getAt( normal : Vector3, target : Vector3 ) : Vector3

返回给定法线方向上的辐射度。

normal

法线向量（假定为单位长度）
target

用于存储方法结果的目标向量。
返回值： 辐射度。

.getIrradianceAt( normal : Vector3, target : Vector3 ) : Vector3

返回给定法线方向上的辐照度（辐射度与余弦瓣卷积的结果）。

normal

法线向量（假定为单位长度）
target

用于存储方法结果的目标向量。
返回值： 辐照度。

.lerp( sh : SphericalHarmonics3, alpha : number ) : SphericalHarmonics3

按给定的 alpha 因子，在给定 SH 与本实例之间进行线性插值。

sh

用于插值的 SH。
alpha

alpha 因子。
返回值： 对此球谐函数的引用。

.scale( s : number ) : SphericalHarmonics3

按给定缩放因子缩放此 SH。

s

缩放因子。
返回值： 对此球谐函数的引用。

.set( coefficients : Array.<Vector3> ) : SphericalHarmonics3

通过复制给定值，将给定的 SH 系数设置到本实例。

coefficients

SH 系数。
返回值： 对此球谐函数的引用。

.toArray( array : Array.<number>, offset : number ) : Array.<number>

返回包含 SH 系数的数组，或将其复制到提供的数组中。 系数以数字形式表示。

array

目标数组。

默认值为 []。
offset

开始复制的数组偏移量。

默认值为 0。
返回值： 包含扁平化 SH 系数的数组。

.zero() : SphericalHarmonics3

将所有 SH 系数设为 0。

返回值： 对此球谐函数的引用。

静态方法

.getBasisAt( normal : Vector3, shBasis : Array.<number> )

normal

法线。
shBasis

保存 SH 基的目标数组。
