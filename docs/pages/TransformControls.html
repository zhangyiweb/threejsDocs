TransformControls

该类可用于在 3D 空间中变换对象，其交互模型类似于 Blender 等 DCC 工具。 与其他控制器不同，它并非用于变换场景相机。

TransformControls 要求其所附加的 3D 对象属于场景图的一部分。

导入

TransformControls 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { TransformControls } from 'three/addons/controls/TransformControls.js';

构造函数

new TransformControls( camera : Camera, domElement : HTMLElement )

构造一个新的控制器实例。

camera

所渲染场景的相机。
domElement

用于事件监听的 HTML 元素。

默认值为 null。

属性

.axis : string

当前的变换轴。

.camera : Camera

所渲染场景的相机。

.dragging : boolean (readonly)

当前是否正在执行拖拽。

默认值为 false。

.maxX : number

平移期间允许的最大 X 位置。

默认值为 Infinity。

.maxY : number

平移期间允许的最大 Y 位置。

默认值为 Infinity。

.maxZ : number

平移期间允许的最大 Z 位置。

默认值为 Infinity。

.minX : number

平移期间允许的最小 X 位置。

默认值为 -Infinity。

.minY : number

平移期间允许的最小 Y 位置。

默认值为 -Infinity。

.minZ : number

平移期间允许的最小 Z 位置。

默认值为 -Infinity。

.mode : 'translate' | 'rotate' | 'scale'

当前的变换模式。

默认值为 'translate'。

.rotationSnap : number

默认情况下，3D 对象会连续旋转。若将此属性设为数值（弧度）， 则可定义 3D 对象旋转的步进。

默认值为 null。

.scaleSnap : number

默认情况下，3D 对象会连续缩放。若将此属性设为数值， 则可定义 3D 对象缩放的步进。

默认值为 null。

.showE : boolean

e 旋转辅助对象是否可见。

默认值为 true。

.showX : boolean

x 轴辅助对象是否可见。

默认值为 true。

.showXY : boolean

xy 平面辅助对象是否可见。

默认值为 true。

.showXYZE : boolean

xyze 旋转辅助对象是否可见。

默认值为 true。

.showXZ : boolean

xz 轴辅助对象是否可见。

默认值为 true。

.showY : boolean

y 轴辅助对象是否可见。

默认值为 true。

.showYZ : boolean

yz 平面辅助对象是否可见。

默认值为 true。

.showZ : boolean

z 轴辅助对象是否可见。

默认值为 true。

.size : number

辅助 UI（轴/平面）的尺寸。

默认值为 1。

.space : 'world' | 'local'

定义应在哪个坐标空间中执行变换。

默认值为 'world'。

.translationSnap : number

默认情况下，3D 对象会连续平移。若将此属性设为数值 （世界单位），则可定义 3D 对象平移的步进。

默认值为 null。

.viewport : Vector4

视口矩形，以逻辑（CSS）像素为单位，原点位于画布左下角。 当渲染器使用子画布视口时请设置此项，以便指针坐标映射到正确区域。 若为 null，则使用整个画布。

默认值为 null。

方法

.attach( object : Object3D ) : TransformControls

设置应被变换的 3D 对象，并确保控制器 UI 可见。

object

应被变换的 3D 对象。
返回值： 对此控制器的引用。

.detach() : TransformControls

从控制器中移除当前 3D 对象，并使辅助 UI 不可见。

返回值： 对此控制器的引用。

.getHelper() : TransformControlsRoot

返回控制器的可视化表示。将辅助对象添加到场景中，即可 可视化地变换所附加的 3D 对象。

返回值： 辅助对象。

.getMode() : 'translate' | 'rotate' | 'scale'

返回变换模式。

返回值： 变换模式。

.getRaycaster() : Raycaster

返回用于用户交互的射线投射器。该对象在所有 TransformControls 实例之间共享。

返回值： 内部射线投射器。

.reset()

将对象的位置、旋转和缩放重置为当前变换开始时的状态。

.setColors( xAxis : number | Color | string, yAxis : number | Color | string, zAxis : number | Color | string, active : number | Color | string )

设置控制器 gizmo 的颜色。

xAxis

x 轴颜色。
yAxis

y 轴颜色。
zAxis

z 轴颜色。
active

活动元素的颜色。

.setMode( mode : 'translate' | 'rotate' | 'scale' )

设置给定的变换模式。

mode

要设置的变换模式。

.setRotationSnap( rotationSnap : number )

设置旋转吸附。

rotationSnap

要设置的旋转吸附值。

.setScaleSnap( scaleSnap : number )

设置缩放吸附。

scaleSnap

要设置的缩放吸附值。

.setSize( size : number )

设置辅助 UI 的尺寸。

size

要设置的尺寸。

.setSpace( space : 'world' | 'local' )

设置应用变换时所使用的坐标空间。

space

要设置的坐标空间。

.setTranslationSnap( translationSnap : number )

设置平移吸附。

translationSnap

要设置的平移吸附值。

事件

.change

当发生任何类型的变更（对象或属性变更）时触发。属性变更 是独立的事件，你可以为其添加事件监听器。事件类型为 "propertyname-changed"。

类型：
  • Object

.mouseDown

当指针（鼠标/触摸）变为活动状态时触发。

类型：
  • Object

.mouseUp

当指针（鼠标/触摸）不再活动时触发。

类型：
  • Object

.objectChange

当受控的 3D 对象发生变更时触发。

类型：
  • Object
