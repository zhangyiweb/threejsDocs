NodeMaterialObserver

此类由 WebGPURenderer 用作管理组件。 其主要目的是在即将渲染渲染对象之前， 判断它们是否需要刷新。

构造函数

new NodeMaterialObserver( builder : NodeBuilder )

构造一个新的节点材质观察者。

builder

节点构建器。

属性

.hasAnimation : boolean

节点构建器的 3D 对象是否处于动画状态。

.hasNode : boolean

材质是否使用节点对象。

.refreshUniforms : Array.<string>

所有可能的材质 uniform 列表。

.renderId : number

保存来自节点帧的当前渲染 ID。

默认值为 0。

.renderObjects : WeakMap.<RenderObject, Object>

一个节点材质可能被多个渲染对象使用，因此 监视器必须维护渲染对象列表。

方法

.containsNode( builder : NodeBuilder ) : boolean

如果节点构建器的材质使用 节点属性，则返回 true。

builder

当前节点构建器。
返回值： 节点构建器的材质是否使用节点属性。

.equals( renderObject : RenderObject, lightsData : Array.<Light>, renderId : number ) : boolean

如果给定渲染对象的状态未发生变化，则返回 true。

renderObject

渲染对象。
lightsData

当前材质的灯光。
renderId

当前渲染 ID。
返回值： 给定渲染对象是否与其缓存状态相等。

.firstInitialization( renderObject : RenderObject ) : boolean

如果给定渲染对象是该观察者首次验证，则返回 true。

renderObject

渲染对象。
返回值： 给定渲染对象是否是该观察者首次验证。

.getAttributesData( attributes : Object ) : Object

返回包含属性版本的属性数据结构， 用于监视。

attributes

几何体属性。
返回值： 用于监视属性版本的对象。

.getGeometryData( geometry : BufferGeometry ) : Object

返回包含几何体属性值的几何体数据结构， 用于监视。

geometry

几何体。
返回值： 用于监视几何体属性的对象。

.getLights( lightsNode : LightsNode, renderId : number ) : Array.<Object>

返回给定灯光节点和渲染 ID 的灯光。

lightsNode

灯光节点。
renderId

渲染 ID。
返回值： 给定灯光节点和渲染 ID 的灯光。

.getLightsData( materialLights : Array.<Light> ) : Array.<Object>

返回给定材质灯光的灯光数据。

materialLights

材质灯光。
返回值： 给定材质灯光的灯光数据。

.getMaterialData( material : Material ) : Object

返回包含材质属性值的材质数据结构， 用于监视。

material

材质。
返回值： 用于监视材质属性的对象。

.getRenderObjectData( renderObject : RenderObject ) : Object

返回给定渲染对象的监视数据。

renderObject

渲染对象。
返回值： 监视数据。

.needsRefresh( renderObject : RenderObject, nodeFrame : NodeFrame ) : boolean

检查给定渲染对象是否需要刷新。

renderObject

渲染对象。
nodeFrame

当前节点帧。
返回值： 给定渲染对象是否需要刷新。

.needsVelocity( renderer : Renderer ) : boolean

如果当前渲染会生成运动向量，则返回 true。

renderer

渲染器。
返回值： 当前渲染是否会生成运动向量。
