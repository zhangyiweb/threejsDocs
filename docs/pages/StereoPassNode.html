StereoPassNode

一种特殊的渲染通道节点，将场景渲染为立体图像。

导入

StereoPassNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { stereoPass } from 'three/addons/tsl/display/StereoPassNode.js';

构造函数

new StereoPassNode( scene : Scene, camera : Camera )

构造一个新的立体通道节点。

scene

要渲染的场景。
camera

用于渲染场景的相机。

属性

.isStereoPassNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.stereo : StereoCamera

用于渲染场景的内部立体相机。

方法

.updateBefore( frame : NodeFrame )

此方法用于每帧渲染一次立体效果。

frame

当前的节点帧。
重写： PassNode#updateBefore
