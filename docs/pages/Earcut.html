Earcut

耳切（earcut）多边形三角剖分算法的实现。代码移植自 mapbox/earcut。

构造函数

new Earcut()

参见：
  • https://github.com/mapbox/earcut

静态方法

.triangulate( data : Array.<number>, holeIndices : Array.<number>, dim : number ) : Array.<number>

对给定的形状定义进行三角剖分，并返回三角形数组。

data

包含二维点的数组。
holeIndices

定义孔洞的索引数组。
dim

输入数组中每个顶点的坐标数量。

默认值为 2。
返回值： 表示三角化面的数组。每个面由三个连续数字定义，分别表示顶点索引。
