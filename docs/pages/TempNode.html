TempNode

此模块使用缓存管理来创建临时变量： 若节点被多次使用，可避免重复计算。

该类是许多其他节点类型的基类。

构造函数

new TempNode( nodeType : string )

构造一个临时节点。

nodeType

节点类型。

默认值为 null。

属性

.isTempNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

方法

.hasDependencies( builder : NodeBuilder ) : boolean

在其他节点的上下文中，此节点是否被使用超过一次。

builder

节点构建器。
返回值： 指示是否对其他节点存在超过一处依赖的标志。
