FunctionNode

此类表示原生着色器函数。可用于通过原生着色器代码实现 节点材质的某些方面。有两个预定义的 TSL 函数以便于使用。

  • wgslFn：创建 WGSL 函数节点。
  • glslFn：创建 GLSL 函数节点。

一个带有单个 include 的基本示例如下：

代码示例

const desaturateWGSLFn = wgslFn( `
	fn desaturate( color:vec3<f32> ) -> vec3<f32> {
		let lum = vec3<f32>( 0.299, 0.587, 0.114 );
		return vec3<f32>( dot( lum, color ) );
	}`
);
const someWGSLFn = wgslFn( `
	fn someFn( color:vec3<f32> ) -> vec3<f32> {
		return desaturate( color );
	}
`, [ desaturateWGSLFn ] );
material.colorNode = someWGSLFn( { color: texture( map ) } );

构造函数

new FunctionNode( code : string, includes : Array.<Node>, language : 'js' | 'wgsl' | 'glsl' )

构造一个新的函数节点。

code

原生代码。

默认值为 ''。
includes

include 数组。

默认值为 []。
language

所使用的语言。

默认值为 ''。
