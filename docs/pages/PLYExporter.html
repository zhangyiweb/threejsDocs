PLYExporter

PLY 导出器。

PLY（Polygon 或 Stanford Triangle Format）是一种用于高效传输和加载简单静态 3D 内容的密集格式文件格式。同时支持二进制和 ASCII 格式。PLY 可存储顶点位置、颜色、法线和 UV 坐标，不保存纹理或纹理引用。

代码示例

const exporter = new PLYExporter();
const data = exporter.parse( scene, options );

导入

PLYExporter 是一个插件，必须显式导入，参见 Installation#Addons。

import { PLYExporter } from 'three/addons/exporters/PLYExporter.js';

构造函数

new PLYExporter()

方法

.parse( object : Object3D, onDone : PLYExporter~OnDone, options : PLYExporter~Options ) : string | ArrayBuffer

解析给定的 3D 对象并生成 PLY 输出。

若该 3D 对象由多个子对象和几何体组成，它们会在文件中合并为单个网格。

object

要导出的 3D 对象。
onDone

导出完成时执行的回调函数。
options

导出选项。
返回值： 导出的 PLY。

类型定义

.OnDone( result : string | ArrayBuffer )

PLYExporter 的 onDone 回调。

result

生成的 PLY ASCII 或二进制数据。

.Options

PLYExporter 的导出选项。

binary
boolean

是以二进制格式还是 ASCII 格式导出。

默认值为 false。
excludeAttributes
Array.<string>

要从导出的 PLY 文件中显式排除的属性。有效值为 'color'、'normal'、'uv' 和 'index'。若排除三角形索引，则导出点云。
littleEndian
boolean

二进制导出是否使用小端序。

默认值为 false。
customPropertyMapping
Object.<string, Array.<string>>

允许将自定义 buffer 属性导出为 PLY 顶点属性的映射。每个条目将 buffer 属性名称映射到 PLY 属性名称数组。属性名称的数量必须与 buffer 属性的 item size 一致。这是 PLYLoader.setCustomPropertyNameMapping() 的逆操作。
