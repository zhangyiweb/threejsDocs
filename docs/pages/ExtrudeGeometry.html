ExtrudeGeometry

根据路径形状创建拉伸几何体。

代码示例

const length = 12, width = 8;
const shape = new THREE.Shape();
shape.moveTo( 0,0 );
shape.lineTo( 0, width );
shape.lineTo( length, width );
shape.lineTo( length, 0 );
shape.lineTo( 0, 0 );
const geometry = new THREE.ExtrudeGeometry( shape );
const material = new THREE.MeshBasicMaterial( { color: 0x00ff00 } );
const mesh = new THREE.Mesh( geometry, material ) ;
scene.add( mesh );

构造函数

new ExtrudeGeometry( shapes : Shape | Array.<Shape>, options : ExtrudeGeometry~Options )

构造一个新的拉伸几何体。

shapes

一个形状或形状数组。
options

拉伸设置。

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

静态方法

.fromJSON( data : Object, shapes : Array.<Shape> ) : ExtrudeGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
shapes

形状数组。
返回值： 新实例。

类型定义

.Options

表示该几何体构造函数的 options 类型。

curveSegments
number

曲线上的点数。

默认值为 12.
steps
number

沿拉伸样条深度方向细分分段所用的点数。

默认值为 1.
depth
number

拉伸形状的深度。

默认值为 1.
bevelEnabled
boolean

是否对形状进行斜角处理。

默认值为 true.
bevelThickness
number

斜角切入原始形状的深度。

默认值为 0.2.
bevelSize
number

斜角从形状轮廓向外延伸的距离。

默认值为 bevelThickness-0.1.
bevelOffset
number

斜角从形状轮廓开始的距离。

默认值为 0.
bevelSegments
number

斜角层数。

默认值为 3.
extrudePath
Curve

形状应沿其拉伸的三维样条路径。路径拉伸不支持斜角。

默认值为 null.
UVGenerator
Object

提供 UV 生成函数以进行自定义 UV 生成的对象。
