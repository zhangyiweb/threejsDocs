LensflareMesh

创建一个跟随光源的模拟镜头光晕。

注意：此类只能与 WebGPURenderer 一起使用。 使用 WebGLRenderer 时，请使用 Lensflare。

代码示例

const light = new THREE.PointLight( 0xffffff, 1.5, 2000 );
const lensflare = new LensflareMesh();
lensflare.addElement( new LensflareElement( textureFlare0, 512, 0 ) );
lensflare.addElement( new LensflareElement( textureFlare1, 512, 0 ) );
lensflare.addElement( new LensflareElement( textureFlare2, 60, 0.6 ) );
light.add( lensflare );

导入

LensflareMesh 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { LensflareMesh } from 'three/addons/objects/LensflareMesh.js';

构造函数

new LensflareMesh()

构造一个新的镜头光晕网格。

属性

.frustumCulled : boolean

已重写，默认禁用视锥体裁剪。

默认值为 false。

重写： Mesh#frustumCulled

.isLensflareMesh : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.renderOrder : number

已重写，确保镜头光晕最后渲染。

默认值为 Infinity。

重写： Mesh#renderOrder

方法

.addElement( element : LensflareElement )

将给定的镜头光晕元素添加到此实例。

element

要添加的元素。

.dispose()

释放此实例分配的 GPU 相关资源。当此实例 不再在应用中使用时，请调用此方法。
