MeshToonNodeMaterial

MeshToonMaterial 的节点材质版本。

构造函数

new MeshToonNodeMaterial( parameters : Object )

构造一个新的网格卡通节点材质。

parameters

配置参数。

属性

.isMeshToonNodeMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.lights : boolean

设为 true，因为卡通材质会对光照做出响应。

默认值为 true。

重写： NodeMaterial#lights

方法

.setupLightingModel() : ToonLightingModel

设置光照模型。

重写： NodeMaterial#setupLightingModel
返回值： 光照模型。
