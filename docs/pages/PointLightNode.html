PointLightNode

用于将点光源表示为节点的模块。

构造函数

new PointLightNode( light : PointLight )

构造一个新的点光源节点。

light

点光源。

默认值为 null。

属性

.cutoffDistanceNode : UniformNode.<float>

表示截止距离的 uniform 节点。

.decayExponentNode : UniformNode.<float>

表示衰减指数的 uniform 节点。

方法

.setupShadowNode() : PointShadowNode

重写以设置点光源特定的阴影。

重写： AnalyticLightNode#setupShadowNode

.update( frame : NodeFrame )

重写以更新点光源特定的 uniform。

frame

当前节点帧的引用。
重写： AnalyticLightNode#update
