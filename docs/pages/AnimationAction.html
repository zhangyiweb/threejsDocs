AnimationAction

AnimationAction 的实例用于调度存储在 AnimationClip 中的动画播放。

构造函数

new AnimationAction( mixer : AnimationMixer, clip : AnimationClip, localRoot : Object3D, blendMode : NormalAnimationBlendMode | AdditiveAnimationBlendMode )

构造一个新的动画动作。

mixer

由该动作控制的混合器。
clip

包含实际关键帧的动画片段。
localRoot

执行该动作的根对象。

默认值为 null。
blendMode

混合模式。

属性

.blendMode : NormalAnimationBlendMode | AdditiveAnimationBlendMode

定义当两个或多个动画同时播放时，动画如何混合/组合。

.clampWhenFinished : boolean

若设置为 true，动画将在最后一帧自动暂停。

若设置为 false，当动作的最后一次循环结束时，AnimationAction#enabled 将自动切换为 false，从而使该动作不再产生影响。

注意：如果动作被中断，此成员不会产生任何影响（仅在其最后一次循环真正完成时才会生效）。

默认值为 false。

.enabled : boolean

若设置为 false，该动作将被禁用，不再产生影响。

当动作重新启用时，动画会从当前时间继续播放（将 enabled 设置为 false 不会重置该动作）。

默认值为 true。

.loop : LoopRepeat | LoopOnce | LoopPingPong

循环模式，通过 AnimationAction#setLoop 设置。

默认值为 LoopRepeat。

.paused : boolean

若设置为 true，该动作的播放将被暂停。

默认值为 false。

.repetitions : number

在该动作执行期间，所播放片段的重复次数。 可通过 AnimationAction#setLoop 设置。

若 AnimationAction#loop 设置为 THREE:LoopOnce，则设置此数值无效。

默认值为 Infinity。

.time : number

该动作的本地时间（以秒为单位，从 0 开始）。

该值会根据循环状态被钳制或包裹到 [0,clip.duration] 范围内。

默认值为 Infinity。

.timeScale : number

AnimationAction#time 的缩放因子。值为 0 时动画暂停，负值时动画倒放。

默认值为 1。

.weight : number

该动作的影响程度（取值区间为 [0, 1]）。可使用 0（无影响）到 1（完全影响）之间的值在多个动作之间进行混合。

默认值为 1。

.zeroSlopeAtEnd : boolean

无需为开始、循环和结束分别准备独立片段，即可实现平滑插值。

默认值为 true。

.zeroSlopeAtStart : boolean

无需为开始、循环和结束分别准备独立片段，即可实现平滑插值。

默认值为 true。

方法

.crossFadeFrom( fadeOutAction : AnimationAction, duration : number, warp : boolean ) : AnimationAction

在指定的时间间隔内，使该动作淡入，并使给定动作淡出。

fadeOutAction

要淡出的动画动作。
duration

淡入淡出的持续时间。
warp

是否使用变形（warping）。

默认值为 false。
返回值： 对该动画动作的引用。

.crossFadeTo( fadeInAction : AnimationAction, duration : number, warp : boolean ) : AnimationAction

在指定的时间间隔内，使该动作淡出，并使给定动作淡入。

fadeInAction

要淡入的动画动作。
duration

淡入淡出的持续时间。
warp

是否使用变形（warping）。

默认值为 false。
返回值： 对该动画动作的引用。

.fadeIn( duration : number ) : AnimationAction

在指定的时间间隔内，通过将权重从 0 逐渐增加到 1 来实现动画淡入。

duration

淡入的持续时间。
返回值： 对该动画动作的引用。

.fadeOut( duration : number ) : AnimationAction

在指定的时间间隔内，通过将权重从 1 逐渐降低到 0 来实现动画淡出。

duration

淡出的持续时间。
返回值： 对该动画动作的引用。

.getClip() : AnimationClip

返回该动画动作关联的动画片段。

返回值： 动画片段。

.getEffectiveTimeScale() : number

返回该动作的有效时间缩放值。

返回值： 有效时间缩放值。

.getEffectiveWeight() : number

返回该动作的有效权重。

返回值： 有效权重。

.getMixer() : AnimationMixer

返回该动画动作关联的动画混合器。

返回值： 动画混合器。

.getRoot() : Object3D

返回该动画动作的根对象。

返回值： 根对象。

.halt( duration : number ) : AnimationAction

在指定的时间间隔内，将该动画的速度减速至 0。

duration

持续时间。
返回值： 对该动画动作的引用。

.isRunning() : boolean

若动画正在运行，则返回 true。

返回值： 动画是否正在运行。

.isScheduled() : boolean

当已调用 AnimationAction#play 时，返回 true。

返回值： 动画是否已调度。

.play() : AnimationAction

开始播放动画。

返回值： 对该动画动作的引用。

.reset() : AnimationAction

重置动画播放。

返回值： 对该动画动作的引用。

.setDuration( duration : number ) : AnimationAction

设置该动作单次循环的持续时间。

duration

要设置的持续时间。
返回值： 对该动画动作的引用。

.setEffectiveTimeScale( timeScale : number ) : AnimationAction

设置该动作的有效时间缩放值。

当动作暂停时，它将不再产生影响，因此有效时间缩放值为零。

timeScale

要设置的时间缩放值。
返回值： 对该动画动作的引用。

.setEffectiveWeight( weight : number ) : AnimationAction

设置该动作的有效权重。

当动作被禁用时，它将不再产生影响，因此有效权重为零。

weight

要设置的权重。
返回值： 对该动画动作的引用。

.setLoop( mode : LoopRepeat | LoopOnce | LoopPingPong, repetitions : number ) : AnimationAction

配置该动作的循环设置。

mode

循环模式。
repetitions

重复次数。
返回值： 对该动画动作的引用。

.startAt( time : number ) : AnimationAction

定义动画应开始播放的时间。

time

开始时间（秒）。
返回值： 对该动画动作的引用。

.stop() : AnimationAction

停止播放动画。

返回值： 对该动画动作的引用。

.stopFading() : AnimationAction

停止应用于该动作的任何淡入淡出效果。

返回值： 对该动画动作的引用。

.stopWarping() : AnimationAction

停止应用于该动作的任何已调度变形（warping）。

返回值： 对该动画动作的引用。

.syncWith( action : AnimationAction ) : AnimationAction

将该动作与传入的另一个动作同步。

action

要同步的动作。
返回值： 对该动画动作的引用。

.warp( startTimeScale : number, endTimeScale : number, duration : number ) : AnimationAction

在指定的时间间隔内，通过将 AnimationAction#timeScale 从 startTimeScale 逐渐变化到 endTimeScale 来改变播放速度。

startTimeScale

起始时间缩放值。
endTimeScale

结束时间缩放值。
duration

持续时间。
返回值： 对该动画动作的引用。
