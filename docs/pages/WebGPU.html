WebGPU

用于进行基本 WebGPU 能力检测的工具模块。

导入

WebGPU 是一个插件，必须显式导入，参见 Installation#Addons。

import WebGPU from 'three/addons/capabilities/WebGPU.js';

静态方法

.getErrorMessage() : HTMLDivElement

返回一个表示格式化错误信息的 div 元素，可在不支持 WebGPU 时追加到网站中。

返回值： 表示不支持 WebGPU 的格式化错误信息的 div 元素。

.isAvailable() : boolean

若 WebGPU 可用则返回 true。

返回值： WebGPU 是否可用。
