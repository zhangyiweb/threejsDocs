Euler

表示欧拉角的类。

欧拉角通过在各轴上按指定角度旋转对象，并按指定的轴顺序 来描述旋转变换。

遍历实例将按对应顺序依次得到其分量（x、y、z、 order）。

代码示例

const a = new THREE.Euler( 0, 1, 1.57, 'XYZ' );
const b = new THREE.Vector3( 1, 0, 1 );
b.applyEuler(a);

构造函数

new Euler( x : number, y : number, z : number, order : string )

构造一个新的欧拉角实例。

x

x 轴的角度，单位为弧度。

默认值为 0。
y

y 轴的角度，单位为弧度。

默认值为 0。
z

z 轴的角度，单位为弧度。

默认值为 0。
order

表示旋转应用顺序的字符串。

默认值为 Euler.DEFAULT_ORDER。

属性

.isEuler : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.order : string

表示旋转应用顺序的字符串。

默认值为 'XYZ'。

.x : number

x 轴的角度，单位为弧度。

默认值为 0。

.y : number

y 轴的角度，单位为弧度。

默认值为 0。

.z : number

z 轴的角度，单位为弧度。

默认值为 0。

.DEFAULT_ORDER : string

默认的欧拉角顺序。

默认值为 'XYZ'。

方法

.clone() : Euler

返回一个新的欧拉角实例，其值从本实例复制而来。

返回值： 本实例的克隆。

.copy( euler : Euler ) : Euler

将给定欧拉角实例的值复制到本实例。

euler

要复制的欧拉角实例。
返回值： 对本欧拉角实例的引用。

.equals( euler : Euler ) : boolean

若本欧拉角实例与给定实例相等，则返回 true。

euler

用于相等性测试的欧拉角实例。
返回值： 本欧拉角实例是否与给定实例相等。

.fromArray( array : Array.<number, number, number, ?string> ) : Euler

将本欧拉角实例的分量设为给定数组中的值。数组的前三个 条目分别赋给 x、y 和 z 分量。可选的第四个条目 定义欧拉角顺序。

array

包含欧拉角分量值的数组。
返回值： 对本欧拉角实例的引用。

.reorder( newOrder : string ) : Euler

通过从当前欧拉角创建四元数，再用该四元数与 新顺序设置本欧拉角，从而以新顺序重置欧拉角。

警告：此操作会丢弃旋转圈数信息。

newOrder

表示旋转新应用顺序的字符串。
返回值： 对本欧拉角实例的引用。

.set( x : number, y : number, z : number, order : string ) : Euler

设置欧拉角各分量。

x

x 轴的角度，单位为弧度。
y

y 轴的角度，单位为弧度。
z

z 轴的角度，单位为弧度。
order

表示旋转应用顺序的字符串。
返回值： 对本欧拉角实例的引用。

.setFromQuaternion( q : Quaternion, order : string, update : boolean ) : Euler

根据归一化的四元数设置本欧拉角实例的各角度。

q

归一化的四元数。
order

表示旋转应用顺序的字符串。
update

是否应执行内部的 onChange 回调。

默认值为 true。
返回值： 对本欧拉角实例的引用。

.setFromRotationMatrix( m : Matrix4, order : string, update : boolean ) : Euler

根据纯旋转矩阵设置本欧拉角实例的各角度。

m

一个 4x4 矩阵，其上 3x3 部分为纯旋转矩阵（即未缩放）。
order

表示旋转应用顺序的字符串。
update

是否应执行内部的 onChange 回调。

默认值为 true。
返回值： 对本欧拉角实例的引用。

.setFromVector3( v : Vector3, order : string ) : Euler

根据给定向量设置本欧拉角实例的各角度。

v

该向量。
order

表示旋转应用顺序的字符串。
返回值： 对本欧拉角实例的引用。

.toArray( array : Array.<number, number, number, string>, offset : number ) : Array.<number, number, number, string>

将本欧拉角实例的分量写入给定数组。若未提供数组， 则方法会返回一个新实例。

array

用于存放欧拉角分量的目标数组。

默认值为 []。
offset

数组中第一个元素的索引。

默认值为 0。
返回值： 欧拉角各分量。
