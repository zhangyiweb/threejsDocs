NodeBuilder

用于根据 3D 对象及其节点材质定义生成着色器程序的构建器基类。

构造函数

new NodeBuilder( object : Object3D, renderer : Renderer, parser : NodeParser )

构造一个新的节点构建器。

object

3D 对象。
renderer

当前渲染器。
parser

对节点解析器的引用。

属性

.activeStacks : Array.<StackNode>

活动的栈节点。

.attributes : Array.<NodeAttribute>

此数组保存此构建器通过 AttributeNode 创建的节点属性。

.bindGroups : Array.<BindGroup>

对绑定组数组的引用。

.bindings : Object

此字典保存每个着色器阶段的绑定。

.bindingsIndexes : Object

此字典维护每个绑定组的绑定索引。

.bufferAttributes : Array.<NodeAttribute>

此数组保存此构建器通过 BufferAttributeNode 创建的节点属性。

.buildStage : 'setup' | 'analyze' | 'generate'

当前构建阶段。

.cache : NodeCache

构建器的缓存。

.camera : Camera

用于渲染该 3D 对象的相机。

.flowBuildStage( node : Node, buildStage : string, output : Node | string ) : Node | string

在特定构建阶段执行节点。

此函数可用于在标准构建过程之外任意执行指定的构建阶段。 例如，如果节点的类型依赖于 'setup' 阶段创建的属性， 则可以使用 flowBuildStage(node, 'setup') 在标准构建过程开始之前 执行 setup 构建阶段并访问其生成的节点。

node

要执行的节点。
buildStage

执行该节点的构建阶段。
output

期望的输出类型。例如 'vec3'。

默认值为 null.
返回值： 节点构建的结果。
