ClippingGroup

在较早的 three.js 版本中，裁剪是在渲染器或材质级别 全局定义的。这一特殊版本的 THREE.Group 允许将裁剪状态编码到场景 图中。也就是说，如果你创建该组的实例，所有 后代 3D 对象都会受到相应裁剪 平面的影响。

注意：ClippingGroup 只能与 WebGPURenderer 一起使用。

构造函数

new ClippingGroup()

构造一个新的裁剪组。

属性

.clipIntersection : boolean

是否使用裁剪平面的交集来裁剪对象，而不是它们的并集。

默认值为 false。

.clipShadows : boolean

是否应对阴影进行裁剪。

默认值为 false。

.clippingPlanes : Array.<Plane>

裁剪平面数组。

.enabled : boolean

是否应启用裁剪。

默认值为 true。

.isClippingGroup : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
