IndexNode

此类表示不同类型的着色器索引。以下预定义节点 对象覆盖了常见用例：

  • vertexIndex：网格中顶点的索引。
  • instanceIndex：网格实例或计算着色器调用的索引。
  • drawIndex：绘制调用的索引。
  • invocationLocalIndex：工作组负载范围内计算调用的索引。
  • invocationSubgroupIndex：子组范围内计算调用的索引。
  • subgroupIndex：计算调用的子组在其工作组内的索引。

构造函数

new IndexNode( scope : 'vertex' | 'instance' | 'subgroup' | 'invocationLocal' | 'invocationGlobal' | 'invocationSubgroup' | 'draw' )

构造一个新的索引节点。

scope

索引节点的作用域。

属性

.isIndexNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.scope : string

索引节点的作用域。
