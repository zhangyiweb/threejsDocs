FlyControls

此类可实现类似 Blender 等 DCC 工具中飞行模式的导航。你可以在 3D 空间中任意变换相机，不受任何限制（例如无需聚焦于特定目标）。

导入

FlyControls 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { FlyControls } from 'three/addons/controls/FlyControls.js';

构造函数

new FlyControls( object : Object3D, domElement : HTMLElement )

构造一个新的控制器实例。

object

由控制器管理的对象。
domElement

用于事件监听的 HTML 元素。

默认值为 null。

属性

.autoForward : boolean

若设为 true，相机在首次平移后会自动持续向前移动（不会停止）。

默认值为 false。

.dragToLook : boolean

若设为 true，只能通过拖拽交互来环顾四周。

默认值为 false。

.movementSpeed : number

移动速度。

默认值为 1。

.rollSpeed : number

旋转速度。

默认值为 0.005。

事件

.change

当相机被控制器变换时触发。

类型：
  • Object
