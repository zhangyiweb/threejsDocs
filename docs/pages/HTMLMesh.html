HTMLMesh

此类可将 DOM 元素渲染到画布上，并将其用作平面网格的纹理。

典型用例是将 lil-gui 的 GUI 渲染为纹理，以便兼容 VR。

代码示例

const gui = new GUI( { width: 300 } ); // create lil-gui instance
const mesh = new HTMLMesh( gui.domElement );
scene.add( mesh );

导入

HTMLMesh 是一个插件，必须显式导入，参见 Installation#Addons。

import { HTMLMesh } from 'three/addons/interactive/HTMLMesh.js';

构造函数

new HTMLMesh( dom : HTMLElement )

构造一个新的 HTML 网格。

dom

要显示为平面网格的 DOM 元素。

方法

.dispose()

释放该实例分配的 GPU 相关资源，并移除所有事件监听器。当应用中不再使用此实例时，请调用此方法。
