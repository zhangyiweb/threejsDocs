PropertyMixer

带缓冲的场景图属性，支持加权累加，供内部使用。

构造函数

new PropertyMixer( binding : PropertyBinding, typeName : string, valueSize : number )

构造一个新的属性混合器。

binding

属性绑定。
typeName

关键帧轨道类型名称。
valueSize

关键帧轨道值大小。

属性

.binding : PropertyBinding

属性绑定。

.cumulativeWeight : number

属性绑定的累计权重。

默认值为 0。

.cumulativeWeightAdditive : number

属性绑定的累计叠加权重。

默认值为 0。

.referenceCount : number

引用该属性绑定的关键帧轨道数量。

默认值为 0。

.useCount : number

当前正在使用该属性绑定的活动关键帧轨道数量。

默认值为 0。

.valueSize : number

关键帧轨道值大小。

方法

.accumulate( accuIndex : number, weight : number )

将 incoming 区域的数据累加到 accu<i>。

accuIndex

累加索引。
weight

权重。

.accumulateAdditive( weight : number )

将 incoming 区域的数据累加到 add。

weight

权重。

.apply( accuIndex : number )

当累加器不同时，将 accu<i> 的状态应用到绑定。

accuIndex

累加索引。

.restoreOriginalState()

将通过 PropertyMixer#saveOriginalState 保存的状态应用到绑定。

.saveOriginalState()

记录绑定属性的状态，并将其复制到两个累加器。
