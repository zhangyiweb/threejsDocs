RenderTarget

渲染目标（Render Target）是一块缓冲区，用于在后端渲染场景时由显卡写入像素。它常用于多种效果，例如在将渲染结果显示到屏幕之前对图像进行后处理。

构造函数

new RenderTarget( width : number, height : number, options : RenderTarget~Options )

创建一个渲染目标。

width

渲染目标的宽度。

默认值为 1。
height

渲染目标的高度。

默认值为 1。
options

配置对象。

属性

.depth : number

渲染目标的深度。

默认值为 1。

.depthBuffer : boolean

是否分配深度缓冲区。

默认值为 true。

.depthTexture : DepthTexture

不将深度保存在 renderbuffer 中，而改用纹理保存，便于后续处理，例如在后处理流程中使用。

默认值为 null。

.height : number

渲染目标的高度。

默认值为 1。

.isRenderTarget : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.multiview : boolean

该目标是否用于多视图渲染。

默认值为 false。

.resolveDepthBuffer : boolean

是否解析深度缓冲区。

默认值为 true。

.resolveStencilBuffer : boolean

是否解析模板缓冲区。

默认值为 true。

.samples : number

MSAA 采样数。

值为 0 时禁用 MSAA。

默认值为 0。

.scissor : Vector4

渲染目标视口内的矩形区域。该区域之外的片元将被丢弃。

默认值为 (0,0,width,height)。

.scissorTest : boolean

渲染到此渲染目标时是否启用剪刀测试。

默认值为 false。

.stencilBuffer : boolean

是否分配模板缓冲区。

默认值为 false。

.texture : Texture

表示默认颜色附着点的纹理。

.textures : Array.<Texture>

纹理数组。每个颜色附着点对应独立的纹理。至少包含一个用于默认颜色附着点的条目。

.useArrayDepthTexture : boolean

是否将深度纹理创建为数组纹理，以支持逐层深度测试。这与 multiview 独立，使分层渲染目标可在不使用 multiview 扩展的情况下使用数组深度。

默认值为 false。

.viewport : Vector4

表示渲染目标视口的矩形区域。

默认值为 (0,0,width,height)。

.width : number

渲染目标的宽度。

默认值为 1。

方法

.clone() : RenderTarget

返回一个拷贝了本实例值的渲染目标。

返回值： 当前实例的克隆。

.copy( source : RenderTarget ) : RenderTarget

复制指定渲染目标的设置。这是结构性拷贝，拷贝后两个渲染目标之间不共享资源，包括所有 MRT 纹理和深度纹理。

source

要复制的渲染目标。
返回值： 对该实例的引用。

.dispose()

释放本实例占用的 GPU 相关资源。在应用中不再使用该实例时应调用此方法。

触发：
  • RenderTarget#event:dispose

.setSize( width : number, height : number, depth : number )

设置此渲染目标的大小。

width

宽度。
height

高度。
depth

深度。

默认值为 1。

number
