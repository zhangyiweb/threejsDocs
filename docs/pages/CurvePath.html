CurvePath

扩展 Curve 的基类。CurvePath 本质上是一组首尾相连的曲线数组，但仍保留曲线的 API。

构造函数

new CurvePath()

构造一条新的曲线路径。

属性

.autoClose : boolean

路径是否应通过一条直线曲线自动闭合。

默认值为 false。

.curves : Array.<Curve>

定义该路径的曲线数组。

方法

.add( curve : Curve )

向该曲线路径添加一条曲线。

curve

要添加的曲线。

.closePath() : CurvePath

添加一条直线曲线以闭合路径。

返回值： 对该曲线路径的引用。

.getCurveLengths() : Array.<number>

返回已定义各曲线的累计长度列表。

返回值： 曲线长度。

.getPoint( t : number, optionalTarget : Vector2 | Vector3 ) : Vector2 | Vector3

根据给定的插值因子，返回二维或三维空间中的向量（取决于曲线定义）。

t

表示曲线上位置的插值因子，必须在 [0,1] 范围内。
optionalTarget

可选的目标向量，结果将写入其中。
重写： Curve#getPoint
返回值： 曲线上的位置。根据曲线定义，可以是二维或三维向量。
