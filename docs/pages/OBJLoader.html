OBJLoader

用于加载 OBJ 格式的加载器。

OBJ 格式是一种简单的数据格式， 以人类可读的方式表示 3D 几何体，包括每个顶点的位置、每个纹理坐标顶点的 UV 位置、 顶点法线，以及定义多边形的面（以顶点列表和纹理顶点列表表示）。

代码示例

const loader = new OBJLoader();
const object = await loader.loadAsync( 'models/monster.obj' );
scene.add( object );

导入

OBJLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { OBJLoader } from 'three/addons/loaders/OBJLoader.js';

构造函数

new OBJLoader( manager : LoadingManager )

构造一个新的 OBJ 加载器。

manager

加载管理器。

属性

.materials : MaterialCreator

对材质创建器的引用。

默认值为 null。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 OBJ 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( text : string ) : Group

解析给定的 OBJ 数据并返回生成的组。

text

原始 OBJ 数据字符串。
重写： Loader#parse
返回值： 解析后的 OBJ。

.setMaterials( materials : MaterialCreator ) : OBJLoader

为此 OBJ 设置材质创建器。该对象通过 MTLLoader 加载。

materials

为此 OBJ 创建材质的对象。
返回值： 对此加载器的引用。
