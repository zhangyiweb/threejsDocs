VertexNormalsHelper

可视化对象的顶点法线。

要求几何体中已将法线指定为缓冲区属性，或已使用 BufferGeometry#computeVertexNormals 计算过法线。

代码示例

const geometry = new THREE.BoxGeometry( 10, 10, 10, 2, 2, 2 );
const material = new THREE.MeshStandardMaterial();
const mesh = new THREE.Mesh( geometry, material );
scene.add( mesh );
const helper = new VertexNormalsHelper( mesh, 1, 0xff0000 );
scene.add( helper );

导入

VertexNormalsHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { VertexNormalsHelper } from 'three/addons/helpers/VertexNormalsHelper.js';

构造函数

new VertexNormalsHelper( object : Object3D, size : number, color : number | Color | string )

构造一个新的顶点法线辅助对象。

object

要可视化其顶点法线的对象。
size

辅助对象的大小。

默认值为 1。
color

辅助对象的颜色。

默认值为 0xff0000。

属性

.isVertexNormalsHelper : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.matrixAutoUpdate : boolean

已重写并设为 false，因为对象的世界变换 已编码在辅助对象的几何体数据中。

默认值为 false。

重写： LineSegments#matrixAutoUpdate

.object : Object3D

要可视化其顶点法线的对象。

.size : number

辅助对象的大小。

默认值为 1。

方法

.dispose()

释放此实例分配的 GPU 相关资源。 当应用中不再使用此实例时，请调用此方法。

.update()

根据对象的世界变换更新顶点法线预览。
