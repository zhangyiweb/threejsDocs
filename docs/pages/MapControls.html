MapControls

此类用于从鸟瞰视角在地图上变换相机。实现与 OrbitControls 共享，但使用特定的鼠标/触摸交互预设，并且默认禁用屏幕空间平移。

  • 旋转：右键，或左键 + ctrl/meta/shiftKey / 触摸：双指旋转。
  • 缩放：中键，或鼠标滚轮 / 触摸：双指张开或捏合。
  • 平移：左键，或方向键 / 触摸：单指移动。

导入

MapControls 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { MapControls } from 'three/addons/controls/MapControls.js';

构造函数

new MapControls()

属性

.mouseButtons : Object

此对象包含控制器所用鼠标操作的引用。

controls.mouseButtons = {
	LEFT: THREE.MOUSE.PAN,
	MIDDLE: THREE.MOUSE.DOLLY,
	RIGHT: THREE.MOUSE.ROTATE
}
重写： OrbitControls#mouseButtons

.screenSpacePanning : boolean

已重写并设为 false，以便垂直于世界空间方向 camera.up 进行平移。

默认值为 false。

重写： OrbitControls#screenSpacePanning

.touches : Object

此对象包含控制器所用触摸操作的引用。

controls.mouseButtons = {
	ONE: THREE.TOUCH.PAN,
	TWO: THREE.TOUCH.DOLLY_ROTATE
}
重写： OrbitControls#touches
