SpriteMaterial

用于渲染 Sprite 实例的材质。

代码示例

const map = new THREE.TextureLoader().load( 'textures/sprite.png' );
const material = new THREE.SpriteMaterial( { map: map, color: 0xffffff } );
const sprite = new THREE.Sprite( material );
sprite.scale.set(200, 200, 1)
scene.add( sprite );

构造函数

new SpriteMaterial( parameters : Object )

构造一个新的精灵材质。

parameters

包含一个或多个属性的对象， 用于定义材质的外观。材质的任何属性 （包括从继承材质中获得的任何属性）都可以传入 此处。颜色值可以传入 Color#set 所接受的任意类型的值。

属性

.alphaMap : Texture

Alpha 贴图是一张灰度纹理，用于控制表面各处的不透明度 （黑色：完全透明；白色：完全不透明）。

仅使用纹理的颜色，若存在 alpha 通道则会被 忽略。对于 RGB 和 RGBA 纹理，渲染器在采样此纹理时会使用绿色通道， 因为在 DXT 压缩格式以及未压缩的 RGB 565 格式中， 绿色通道具有额外的精度位。仅亮度以及 亮度/alpha 纹理仍可按预期工作。

alphaMap 表示非颜色数据。所指定的任何纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.color : Color

材质的颜色。

默认值为 (1,1,1)。

.fog : boolean

材质是否受雾效影响。

默认值为 true。

.isSpriteMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.map : Texture

颜色贴图。可选地包含 alpha 通道，通常与 Material#transparent 或 Material#alphaTest 结合使用。纹理贴图 颜色会受漫反射 color 调制。

map 表示颜色数据，纹理必须指定 Texture#colorSpace。大多数 map 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。

.rotation : number

精灵的旋转角度（以弧度为单位）。

默认值为 0。

.sizeAttenuation : boolean

指定精灵大小是否会随相机深度衰减（仅适用于透视相机）。

默认值为 true。

.transparent : boolean

已重写，因为精灵材质默认 为透明。

默认值为 true。

重写： Material#transparent
