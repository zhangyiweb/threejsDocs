AnimationObjectGroup

共享同一动画状态的一组对象。

用法：

  • 将原本作为 'root' 传给 AnimationMixer 构造函数或 .clipAction 方法的对象添加进来。
  • 改为将此对象作为 'root' 传入。
  • 混合器运行期间也可以后续添加和移除对象。

注意：

  • 对此混合器而言，该类的对象表现为单个对象，因此各对象的缓存控制必须在组级别进行。

限制：

  • 组内所有对象的动画属性必须兼容。
  • 单个属性只能通过目标组或直接控制，不能同时使用两种方式。

构造函数

new AnimationObjectGroup( …arguments : Object3D )

构造一个新的动画组。

arguments

共享同一动画状态的任意数量 3D 对象。

属性

.isAnimationObjectGroup : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.uuid : string (readonly)

3D 对象的 UUID。

方法

.add( …arguments : Object3D )

向该动画组添加任意数量的对象。

arguments

要添加的 3D 对象。

.remove( …arguments : Object3D )

从该动画组移除任意数量的对象。

arguments

要移除的 3D 对象。

.uncache( …arguments : Object3D )

释放该动画组中传入 3D 对象占用的所有内存资源。

arguments

要清除缓存的 3D 对象。
