Spherical

此类可用于将三维空间中的点表示为 球坐标。

构造函数

new Spherical( radius : number, phi : number, theta : number )

构造一个新的球坐标。

radius

半径，即从点到原点的欧几里得距离（直线距离）。

默认值为 1。
phi

从 y（上）轴起算的极角，单位为弧度。

默认值为 0。
theta

绕 y（上）轴的赤道角/方位角，单位为弧度。

默认值为 0。

属性

.phi : number

从 y（上）轴起算的极角，单位为弧度。

默认值为 0。

.radius : number

半径，即从点到原点的欧几里得距离（直线距离）。

默认值为 1。

.theta : number

绕 y（上）轴的赤道角/方位角，单位为弧度。

默认值为 0。

方法

.clone() : Spherical

返回一个新球坐标，其值从本实例复制而来。

返回值： 本实例的克隆。

.copy( other : Spherical ) : Spherical

将给定球坐标的值复制到本实例。

other

要复制的球坐标。
返回值： 对此球坐标的引用。

.makeSafe() : Spherical

将极角 [page:.phi phi] 限制在 0.000001 与 π - 0.000001 之间。

返回值： 对此球坐标的引用。

.set( radius : number, phi : number, theta : number ) : Spherical

通过复制给定值来设置球坐标的分量。

radius

半径。
phi

极角。
theta

方位角。
返回值： 对此球坐标的引用。

.setFromCartesianCoords( x : number, y : number, z : number ) : Spherical

根据给定的笛卡尔坐标设置球坐标分量。

x

x 值。
y

y 值。
z

z 值。
返回值： 对此球坐标的引用。

.setFromVector3( v : Vector3 ) : Spherical

根据给定向量设置球坐标分量，该向量假定保存的是 笛卡尔坐标。

v

要设置的向量。
返回值： 对此球坐标的引用。
