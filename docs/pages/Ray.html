Ray

从原点沿某一方向发射的射线。该类由 Raycaster 用于辅助光线投射。光线投射可用于 鼠标拾取（判断鼠标悬停在三维空间中的哪些物体上）等用途。

构造函数

new Ray( origin : Vector3, direction : Vector3 )

构造一条新射线。

origin

射线的原点。

默认值为 (0,0,0)。
direction

射线的（归一化）方向。

默认值为 (0,0,-1)。

属性

.direction : Vector3

射线的（归一化）方向。

.origin : Vector3

射线的原点。

方法

.applyMatrix4( matrix4 : Matrix4 ) : Ray

用给定的 4x4 变换矩阵变换此射线。

matrix4

变换矩阵。
返回值： 此射线的引用。

.at( t : number, target : Vector3 ) : Vector3

返回沿此射线给定距离处的向量。

t

沿射线检索位置的距离。
target

用于存储方法结果的目标向量。
返回值： 射线上的一个位置。

.clone() : Ray

返回一个从此实例复制值的新射线。

返回值： 此实例的克隆。

.closestPointToPoint( point : Vector3, target : Vector3 ) : Vector3

返回此射线上距给定点最近的点。

point

三维空间中的一个点，用于获取射线上最近的位置。
target

用于存储方法结果的目标向量。
返回值： 此射线上最近的点。

.copy( ray : Ray ) : Ray

将给定射线的值复制到此实例。

ray

要复制的射线。
返回值： 此射线的引用。

.distanceSqToPoint( point : Vector3 ) : number

返回此射线与给定点最近接近处的距离平方。

point

三维空间中要计算距离的点。
返回值： 距离的平方。

.distanceSqToSegment( v0 : Vector3, v1 : Vector3, optionalPointOnRay : Vector3, optionalPointOnSegment : Vector3 ) : number

返回此射线与给定线段之间的距离平方。

v0

线段的起点。
v1

线段的终点。
optionalPointOnRay

若提供，将接收此射线上距线段最近的点。
optionalPointOnSegment

若提供，将接收线段上距此射线最近的点。
返回值： 距离的平方。

.distanceToPlane( plane : Plane ) : number

计算从射线原点到给定平面的距离。若射线与平面不相交，则返回 null。

plane

要计算距离的平面。
返回值： 此射线是否与给定球体相交。

.distanceToPoint( point : Vector3 ) : number

返回此射线与给定点最近接近处的距离。

point

三维空间中要计算距离的点。
返回值： 距离。

.equals( ray : Ray ) : boolean

若此射线与给定射线相等，则返回 true。

ray

用于相等性测试的射线。
返回值： 此射线是否与给定射线相等。

.intersectBox( box : Box3, target : Vector3 ) : Vector3

计算此射线与给定包围盒的交点；若不相交则返回 null。

box

要相交的盒子。
target

用于存储方法结果的目标向量。
返回值： 交点。

.intersectPlane( plane : Plane, target : Vector3 ) : Vector3

计算此射线与给定平面的交点；若不相交则返回 null。

plane

要相交的平面。
target

用于存储方法结果的目标向量。
返回值： 交点。

.intersectSphere( sphere : Sphere, target : Vector3 ) : Vector3

计算此射线与给定球体的交点；若不相交则返回 null。

sphere

要相交的球体。
target

用于存储方法结果的目标向量。
返回值： 交点。

.intersectTriangle( a : Vector3, b : Vector3, c : Vector3, backfaceCulling : boolean, target : Vector3 ) : Vector3

计算此射线与给定三角形的交点；若不相交则返回 null。

a

三角形的第一个顶点。
b

三角形的第二个顶点。
c

三角形的第三个顶点。
backfaceCulling

是否使用背面剔除。
target

用于存储方法结果的目标向量。
返回值： 交点。

.intersectsBox( box : Box3 ) : boolean

若此射线与给定盒子相交，则返回 true。

box

要相交的盒子。
返回值： 此射线是否与给定盒子相交。

.intersectsPlane( plane : Plane ) : boolean

若此射线与给定平面相交，则返回 true。

plane

要相交的平面。
返回值： 此射线是否与给定平面相交。

.intersectsSphere( sphere : Sphere ) : boolean

若此射线与给定球体相交，则返回 true。

sphere

要相交的球体。
返回值： 此射线是否与给定球体相交。

.lookAt( v : Vector3 ) : Ray

调整射线方向，使其指向世界空间中的给定向量。

v

目标位置。
返回值： 此射线的引用。

.recast( t : number ) : Ray

沿射线方向将原点平移给定距离。

t

沿射线插值的距离。
返回值： 此射线的引用。

.set( origin : Vector3, direction : Vector3 ) : Ray

通过复制给定值设置射线分量。

origin

原点。
direction

方向。
返回值： 此射线的引用。
