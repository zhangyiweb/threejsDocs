CameraHelper

用于可视化相机视锥体所包含的内容。它使用线段可视化相机的视锥体。

基于 lightgl.js shadowmap example 中的视锥体可视化。

CameraHelper 必须是场景的子对象。

当相机发生变换或其投影矩阵更改时，需要调用相应辅助对象的 update() 方法。

代码示例

const camera = new THREE.PerspectiveCamera( 75, window.innerWidth / window.innerHeight, 0.1, 1000 );
const helper = new THREE.CameraHelper( camera );
scene.add( helper );

构造函数

new CameraHelper( camera : Camera )

构造一个新的相机辅助对象。

camera

要可视化的相机。

属性

.camera : Camera

正在可视化的相机。

.pointMap : Object.<string, Array.<number>>

包含用于可视化相机的点。

方法

.dispose()

释放该实例分配的 GPU 相关资源。当应用中不再使用该实例时，应调用此方法。

.setColors( frustum : Color, cone : Color, up : Color, target : Color, cross : Color ) : CameraHelper

定义辅助对象的颜色。

frustum

视锥体线段颜色。
cone

圆锥线段颜色。
up

上方向线段颜色。
target

目标线段颜色。
cross

十字线段颜色。
返回值： 对该辅助对象的引用。

.update()

根据相机的投影矩阵更新辅助对象。
