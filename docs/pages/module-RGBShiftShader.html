RGBShiftShader

导入

RGBShiftShader 是一个插件，必须显式导入，参见 Installation#Addons。

import { RGBShiftShader } from 'three/addons/shaders/RGBShiftShader.js';

属性

.RGBShiftShader : ShaderMaterial~Shader (inner, constant)

RGB 偏移着色器 将红色和蓝色通道从中心向相反方向偏移 移植自 https://web.archive.org/web/20090820185047/http://kriss.cx/tom/2009/05/rgb-shift/ 作者 Tom Butterworth / https://web.archive.org/web/20090810054752/http://kriss.cx/tom/

amount：偏移距离（1 为输入宽度） angle：偏移角度（弧度）
