Layers

图层对象可将 3D 对象分配到编号为 0 到 31 的 32 个图层中的一个或多个——内部以位掩码形式存储图层，默认情况下所有 3D 对象都属于图层 0。

这可用于控制可见性——对象必须与相机共享至少一个图层，才能在该相机视图渲染时可见。

所有继承自 Object3D 的类都有一个 layers 属性，它是该类的实例。

构造函数

new Layers()

构造一个新的图层实例，初始成员关系设置为图层 0。

属性

.mask : number

位掩码，存储该图层对象当前所属的 32 个图层。

方法

.disable( layer : number )

移除给定图层的成员关系。

layer

要禁用的图层。

.disableAll()

移除所有图层的成员关系。

.enable( layer : number )

添加给定图层的成员关系。

layer

要启用的图层。

.enableAll()

添加所有图层的成员关系。

.isEnabled( layer : number ) : boolean

若给定图层已启用，则返回 true。

layer

要测试的图层。
返回值： 给定图层是否已启用。

.set( layer : number )

将成员关系设置为给定图层，并移除所有其他图层的成员关系。

layer

要设置的图层。

.test( layers : Layers ) : boolean

若该对象与给定图层对象至少共享一个图层，则返回 true。

layers

要测试的图层对象。
返回值： 该对象与给定图层对象是否至少共享一个图层。

.toggle( layer : number )

切换给定图层的成员关系。

layer

要切换的图层。
