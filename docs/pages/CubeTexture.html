CubeTexture

创建由六张图像组成的立方体纹理。

代码示例

const loader = new THREE.CubeTextureLoader();
loader.setPath( 'textures/cube/pisa/' );
const textureCube = loader.load( [
	'px.png', 'nx.png', 'py.png', 'ny.png', 'pz.png', 'nz.png'
] );
const material = new THREE.MeshBasicMaterial( { color: 0xffffff, envMap: textureCube } );

构造函数

new CubeTexture( images : Array.<Image>, mapping : number, wrapS : number, wrapT : number, magFilter : number, minFilter : number, format : number, type : number, anisotropy : number, colorSpace : string )

构造一个新的立方体纹理。

images

保存立方体每个面图像的数组。

默认值为 []。
mapping

纹理映射方式。

默认值为 CubeReflectionMapping。
wrapS

wrapS 值。

默认值为 ClampToEdgeWrapping。
wrapT

wrapT 值。

默认值为 ClampToEdgeWrapping。
magFilter

放大过滤器值。

默认值为 LinearFilter。
minFilter

缩小过滤器值。

默认值为 LinearMipmapLinearFilter。
format

纹理格式。

默认值为 RGBAFormat。
type

纹理类型。

默认值为 UnsignedByteType。
anisotropy

各向异性值。

默认值为 Texture.DEFAULT_ANISOTROPY。
colorSpace

颜色空间值。

默认值为 NoColorSpace。

属性

.flipY : boolean

若设为 true，纹理在上传到 GPU 时会沿垂直轴翻转。

已覆盖并默认设为 false。

默认值为 false。

重写： Texture#flipY

.images : Array.<Image>

CubeTexture#image 的别名。

.isCubeTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
