ParameterNode

PropertyNode 的特殊版本，用于表示参数。

构造函数

new ParameterNode( nodeType : string, name : string )

构造一个新的参数节点。

nodeType

节点的类型。
name

着色器中参数的名称。

默认值为 null。

属性

.isParameterNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

方法

.getMemberType( builder : NodeBuilder, name : string ) : string

获取参数节点中成员变量的类型。

builder

节点构建器。
name

成员变量的名称。
重写： PropertyNode#getMemberType
