Bayer

导入

Bayer 是一个插件，必须显式导入，参见 Installation#Addons。

import { bayer16 } from 'three/addons/tsl/math/Bayer.js';

静态方法

.bayer16( uv : Node.<vec2> ) : Node.<vec4>

此 TSL 函数可用于采样 Bayer16 纹理，该纹理是一张 16x16 的 Bayer 矩阵图案纹理。 可用于抖动效果，也可作为蓝噪声的替代方案。在与光线步进（Ray Marching）配合使用时， 特别是在 VolumeNodeMaterial#offsetNode 中，可减轻条带问题， 从而在不显著影响视觉效果的情况下使用更少的步数。

uv

用于采样 bayer16 纹理的 uv。
返回值： 采样得到的 Bayer 值。

.bayerDither( color : Node.<vec3>, steps : Node.<float> ) : Node.<vec3>

此 TSL 函数对颜色输入应用 Bayer 抖动。它在颜色量化之前使用 4x4 Bayer 矩阵 图案添加结构化噪声，有助于在限制颜色深度时减少可见的颜色条带。

color

要应用抖动的输入颜色。
steps

每个通道的颜色步数。

默认值为 32。
返回值： 准备进行量化的抖动后颜色。
