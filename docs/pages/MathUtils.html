MathUtils

数学工具函数集合。

静态方法

.ceilPowerOfTwo( value : number ) : number

返回大于或等于给定数字的最小的 2 的幂。

value

要查找其 2 的幂的值。
返回值： 大于或等于给定数字的最小的 2 的幂。

.clamp( value : number, min : number, max : number ) : number

将给定值限制在 min 与 max 之间。

value

要限制的值。
min

最小值。
max

最大值。
返回值： 限制后的值。

.damp( x : number, y : number, lambda : number, dt : number ) : number

以类似弹簧的方式，使用增量时间将数值从 x 平滑插值到 y， 以保持与帧率无关的运动。详情参见 Frame rate independent damping using lerp。

x

当前点。
y

目标点。
lambda

较高的 lambda 值会使运动更突然， 较低的值会使运动更平缓。
dt

增量时间，单位为秒。
返回值： 插值后的值。

.degToRad( degrees : number ) : number

将角度转换为弧度。

degrees

以度为单位的值。
返回值： 转换后以弧度为单位的值。

.denormalize( value : number, array : TypedArray ) : number

根据给定的类型化数组对给定值进行反归一化。

value

要反归一化的值。
array

定义该值数据类型的类型化数组。
返回值： 反归一化后的浮点值，范围为 [0,1]。

.euclideanModulo( n : number, m : number ) : number

计算给定参数的欧几里得取模，即 ( ( n % m ) + m ) % m。

n

第一个参数。
m

第二个参数。
返回值： 欧几里得取模结果。

x

起点。

终点。

.isPowerOfTwo( value : number ) : boolean

.lerp( x : number, y : number, t : number ) : number

根据给定区间，返回在两个已知点之间线性插值的值—— t = 0 将返回 x，t = 1 将返回 y。

x

起点。
y

终点。
t

闭区间 [0, 1] 内的插值因子。
返回值： 插值后的值。

.mapLinear( x : number, a1 : number, a2 : number, b1 : number, b2 : number ) : number

对给定值执行从范围 <a1, a2> 到范围 <b1, b2> 的线性映射。

x

要映射的值。
a1

范围 A 的最小值。
a2

范围 A 的最大值。
b1

范围 B 的最小值。
b2

范围 B 的最大值。
返回值： 映射后的值。

x

.setQuaternionFromProperEuler( q : Quaternion, a : number, b : number, c : number, order : 'XYX' | 'XZX' | 'YXY' | 'YZY' | 'ZXZ' | 'ZYZ' )

根据给定角度与顺序所定义的 Intrinsic Proper Euler Angles 设置给定四元数。

旋转按 order 指定的轴顺序应用： 先应用角度 a 的旋转，再应用角度 b，最后应用角度 c。

q

要设置的四元数。
a

应用于第一轴的旋转，单位为弧度。
b

应用于第二轴的旋转，单位为弧度。
c

应用于第三轴的旋转，单位为弧度。
order

指定轴顺序的字符串。

.smootherstep( x : number, min : number, max : number ) : number

smoothstep 的变体， 在 x=0 和 x=1 处其一阶与二阶导数均为零。

x

根据其在 min 与 max 之间的位置进行求值的值。
min

最小值。任何低于 min 的 x 值将为 0。
max

最大值。任何高于 max 的 x 值将为 1。
返回值： 计算后的值。

.smoothstep( x : number, min : number, max : number ) : number

返回范围 [0,1] 内的值，表示 x 在 min 与 max 之间移动的百分比，但当 x 越接近 min 和 max 时， 会被平滑或放缓。

参见 Smoothstep 了解更多详情。

x

根据其在 min 与 max 之间的位置进行求值的值。
min

最小值。任何低于 min 的 x 值将为 0。
max

最大值。任何高于 max 的 x 值将为 1。
返回值： 计算后的值。
