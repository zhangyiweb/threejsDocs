ToonOutlinePassNode

表示用于在兼容对象上生成卡通轮廓效果的渲染通道。 仅材质类型为 MeshToonMaterial 和 MeshToonNodeMaterial 的 3D 对象会接收轮廓。

代码示例

const postProcessing = new RenderPipeline( renderer );
const scenePass = toonOutlinePass( scene, camera );
postProcessing.outputNode = scenePass;

构造函数

new ToonOutlinePassNode( scene : Scene, camera : Camera, colorNode : Node, thicknessNode : Node, alphaNode : Node )

构造一个新的轮廓通道节点。

scene

对场景的引用。
camera

对相机的引用。
colorNode

定义轮廓颜色。
thicknessNode

定义轮廓厚度。
alphaNode

定义轮廓透明度。

属性

.alphaNode : Node

定义轮廓透明度。

.colorNode : Node

定义轮廓颜色。

.name : string

此通道的名称。

默认值为 'Outline Pass'。

重写： PassNode#name

.thicknessNode : Node

定义轮廓厚度。
