BasicLightMapNode

仅与 MeshBasicNodeMaterial 相关的特殊版 IrradianceNode。由于该材质为无光照材质， 光照贴图需要使用特殊的缩放因子。

构造函数

new BasicLightMapNode( lightMapNode : Node.<vec3> )

构造一个新的基础光照贴图节点。

lightMapNode

光照贴图节点。

默认值为 null。

属性

.lightMapNode : Node.<vec3>

光照贴图节点。
