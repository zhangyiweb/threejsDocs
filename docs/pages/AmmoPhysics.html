AmmoPhysics

可将 Ammo.js 作为物理引擎集成到 three.js 应用中。可通过以下方式初始化 API：

该组件会自动从 CDN 导入 Ammo.js，因此使用时需要保持网络连接。

代码示例

const physics = await AmmoPhysics();

导入

AmmoPhysics 属于附加组件，需要显式导入，请参阅 Installation#Addons。

import { AmmoPhysics } from 'three/addons/physics/AmmoPhysics.js';

方法

.addMesh( mesh : Mesh, mass : number, restitution : number )

将给定网格添加到物理模拟中。

mesh

要添加的网格。
mass

网格的质量（千克）。

默认值为 0。
restitution

网格的恢复系数，通常取值范围为 0 到 1。表示物体碰撞时的弹性程度。

默认值为 0。

.addScene( scene : Object3D )

将给定场景添加到物理模拟中。仅处理在 Object3D#userData 字段中包含 physics 对象的网格。 该对象可用于存储网格的质量，例如：

box.userData.physics = { mass: 1 };
scene

要添加的场景或任意 3D 对象。

.setMeshPosition( mesh : Mesh, position : Vector3, index : number )

设置参与物理模拟的给定网格的位置。调用此方法会重置该网格当前模拟的速度。

mesh

要更新位置的网格。
position

新位置。
index

若网格为实例化网格，索引表示实例 ID。

默认值为 0。
