HalftoneShader

导入

HalftoneShader 是一个插件，必须显式导入，参见 Installation#Addons。

import { HalftoneShader } from 'three/addons/shaders/HalftoneShader.js';

属性

.HalftoneShader : ShaderMaterial~Shader (inner, constant)

RGB 半色调着色器。

由 HalftonePass 使用。

形状（1 = 圆点，2 = 椭圆，3 = 线条，4 = 正方形，5 = 菱形） 混合模式（1 = 线性，2 = 正片叠底，3 = 相加，4 = 变亮，5 = 变暗）
