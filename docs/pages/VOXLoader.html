VOXLoader

用于加载 VOX 格式的加载器。

代码示例

const loader = new VOXLoader();
const result = await loader.loadAsync( 'models/vox/monu10.vox' );
scene.add( result.scene.children[ 0 ] );

导入

VOXLoader 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { VOXLoader } from 'three/addons/loaders/VOXLoader.js';

构造函数

new VOXLoader()

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 VOX 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( buffer : ArrayBuffer ) : Object

解析给定的 VOX 数据并返回结果对象。

buffer

原始 VOX 数据，以数组缓冲区形式提供。
重写： Loader#parse
返回值： 解析后的 VOX 数据，包含属性：chunks、scene。
