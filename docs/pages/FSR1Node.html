FSR1Node

用于应用 AMD FidelityFX Super Resolution 1（FSR 1）的后处理节点。

包含两个通道：

  • EASU（Edge-Adaptive Spatial Upsampling，边缘自适应空间上采样）：以十字形采样 12 个纹理样本检测局部边缘方向，再将近似 Lanczos2 核塑造成与检测到的边缘对齐的椭圆。
  • RCAS（Robust Contrast-Adaptive Sharpening，鲁棒对比度自适应锐化）：使用 5 点十字形采样进行对比度感知锐化，并自动按像素限制强度以避免伪影。

注意：仅当你的应用受片段着色器性能限制、无法负担全分辨率渲染时才应使用 FSR 1。FSR 1 本身也会带来额外开销，因此着色简单的场景以原生分辨率渲染反而更快。此外，FSR 1 应始终配合已抗锯齿的源图像使用。

参考：https://gpuopen.com/fidelityfx-superresolution/。

导入

FSR1Node 是一个插件，必须显式导入，参见 Installation#Addons。

import { fsr1 } from 'three/addons/tsl/display/fsr1/FSR1Node.js';

构造函数

new FSR1Node( textureNode : TextureNode, sharpness : Node.<float>, denoise : Node.<bool> )

构造一个新的 FSR 1 节点。

textureNode

表示效果输入的纹理节点。
sharpness

RCAS 锐化强度。0 表示最大锐化，2 表示不锐化。

默认值为 0.2。
denoise

是否在噪声区域减弱 RCAS 锐化。

默认值为 false。

属性

.denoise : Node.<bool>

是否在噪声区域减弱 RCAS 锐化。

.sharpness : Node.<float>

RCAS 锐化强度。0 表示最大锐化，2 表示不锐化。

.textureNode : TextureNode

表示效果输入的纹理节点。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.FRAME，因为该节点会在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

方法

.dispose()

释放内部资源。当不再需要该效果时应调用此方法。

重写： TempNode#dispose

.getTextureNode() : PassTextureNode

将效果结果作为纹理节点返回。

返回值： 表示效果结果的纹理节点。

.setSize( width : number, height : number )

设置效果的输出尺寸。

width

宽度（像素）。
height

高度（像素）。

.setup( builder : NodeBuilder ) : PassTextureNode

用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
