StackNode

Stack 是为需要生成基于栈的代码（而非连续流）的节点提供的辅助工具。 通常在诸如 If、Else 这类场景中需要使用。

构造函数

new StackNode( parent : StackNode )

构造一个新的栈节点。

parent

父栈节点。

默认值为 null。

属性

.isStackNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.nodes : Array.<Node>

节点列表。

.outputNode : Node

输出节点。

默认值为 null。

.parent : StackNode

父栈节点。

默认值为 null。

方法

.Case( …params : any ) : StackNode

表示 TSL 中的 case 语句。TSL 版本接受任意数量的值。 最后一个参数必须是在 true 情况下应执行的回调方法。

params

Case() 语句的值以及回调方法。
返回值： 对此栈节点的引用。

.Default( method : function ) : StackNode

表示 Switch/Case 语句的默认代码块。

method

在 else 情况下执行的 TSL 代码。
返回值： 对此栈节点的引用。

.Else( method : function ) : StackNode

表示 TSL 中的 else 语句。

method

在 else 情况下执行的 TSL 代码。
返回值： 对此栈节点的引用。

.ElseIf( boolNode : Node, method : function ) : StackNode

表示 TSL 中的 elseif 语句。

boolNode

表示条件。
method

当条件求值为 true 时执行的 TSL 代码。
返回值： 对此栈节点的引用。

.If( boolNode : Node, method : function ) : StackNode

表示 TSL 中的 if 语句。

boolNode

表示条件。
method

当条件求值为 true 时执行的 TSL 代码。
返回值： 对此栈节点的引用。

.Switch( expression : any, method : function ) : StackNode

表示 TSL 中的 switch 语句。

expression

表示表达式。
method

当条件求值为 true 时执行的 TSL 代码。
返回值： 对此栈节点的引用。

.addToStack( node : Node, index : number ) : StackNode

将节点添加到此栈中。

node

要添加的节点。
index

节点的索引。若未指定，节点将被添加到栈的末尾。

默认值为 -1。
返回值： 对此栈节点的引用。

.addToStackBefore( node : Node ) : StackNode

在当前节点之前将节点添加到栈中。

node

要添加的节点。
返回值： 对此栈节点的引用。
