MaterialXLoader

用于加载 MaterialX 格式的加载器。

通过此加载器加载的节点材质只能与 WebGPURenderer 一起使用。

代码示例

const loader = new MaterialXLoader().setPath( SAMPLE_PATH );
const materials = await loader.loadAsync( 'standard_surface_brass_tiled.mtlx' );

导入

MaterialXLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { MaterialXLoader } from 'three/addons/loaders/MaterialXLoader.js';

构造函数

new MaterialXLoader( manager : LoadingManager )

构造一个新的 MaterialX 加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback ) : MaterialXLoader

从指定 URL 开始加载，并将加载完成的 MaterialX 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load
返回值： 对此加载器的引用。

.parse( text : string ) : Object.<string, NodeMaterial>

解析给定的 MaterialX 数据并返回生成的材质。

支持的 standard_surface 输入：

  • base、base_color：基础色/反照率
  • opacity：Alpha/透明度
  • specular_roughness：表面粗糙度
  • metalness：金属度
  • specular：镜面反射强度
  • specular_color：镜面反射颜色
  • ior：折射率
  • specular_anisotropy、specular_rotation：各向异性反射
  • transmission、transmission_color：透射属性
  • thin_film_thickness、thin_film_ior：薄膜干涉
  • sheen、sheen_color、sheen_roughness：光泽属性
  • normal：法线贴图
  • coat、coat_roughness、coat_color：清漆属性
  • emission、emissionColor：自发光属性
text

原始 MaterialX 数据字符串。
重写： Loader#parse
返回值： 包含解析后节点材质的字典。
