InputNode

表示数据输入节点的基类。

构造函数

new InputNode( value : any, nodeType : string )

构造一个新的输入节点。

value

此节点的值。可以是任意 JS 原始类型、函数、数组缓冲区，甚至是 three.js 对象（向量、矩阵、颜色）。
nodeType

节点类型。若未显式定义类型，节点会尝试从其值推导类型。

默认值为 null。

属性

.isInputNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.precision : 'low' | 'medium' | 'high'

着色器中该值的精度。

默认值为 null。

.value : any

此节点的值。可以是任意 JS 原始类型、函数、数组缓冲区，甚至是 three.js 对象（向量、矩阵、颜色）。

方法

.getInputType( builder : NodeBuilder ) : string

返回节点的输入类型，默认即为节点类型。派生模块 可能会重写此方法，并使用固定类型或通过分析计算类型。

输入类型与节点类型不同的典型例子是纹理。普通 RGBA 纹理的输入类型是 texture， 而其节点类型是 vec4。

builder

当前节点构建器。
返回值： 输入类型。

.setPrecision( precision : 'low' | 'medium' | 'high' ) : InputNode

将精度设置为给定值。如果最终精度必须通过分析计算得出， 该方法可在派生类中被重写。

precision

着色器中输入值的精度。
返回值： 对此节点的引用。
