OctreeHelper

用于可视化八叉树的辅助对象。

代码示例

const helper = new OctreeHelper( octree );
scene.add( helper );

导入

OctreeHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { OctreeHelper } from 'three/addons/helpers/OctreeHelper.js';

构造函数

new OctreeHelper( octree : Octree, color : number | Color | string )

构造一个新的八叉树辅助对象。

octree

要可视化的八叉树。
color

辅助对象的颜色。

默认值为 0xffff00。

属性

.color : number | Color | string

辅助对象的颜色。

.octree : Octree

要可视化的八叉树。

方法

.dispose()

释放此实例分配的 GPU 相关资源。 当应用中不再使用此实例时，请调用此方法。

.update()

更新辅助对象。每当八叉树的结构发生变化时，必须调用此方法。
