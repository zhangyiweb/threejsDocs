StorageTextureNode

此特殊版本的纹理节点可用于 通过计算着色器向存储纹理写入数据。

此节点只能与 WebGPU 后端一起使用。

代码示例

const storageTexture = new THREE.StorageTexture( width, height );
const computeTexture = Fn( ( { storageTexture } ) => {
	const posX = instanceIndex.mod( width );
	const posY = instanceIndex.div( width );
	const indexUV = uvec2( posX, posY );
	// generate RGB values
	const r = 1;
	const g = 1;
	const b = 1;
	textureStore( storageTexture, indexUV, vec4( r, g, b, 1 ) ).toWriteOnly();
} );
const computeNode = computeTexture( { storageTexture } ).compute( width * height );
renderer.computeAsync( computeNode );

构造函数

new StorageTextureNode( value : StorageTexture, uvNode : Node.<(vec2|vec3)>, storeNode : Node )

构造一个新的存储纹理节点。

value

存储纹理。
uvNode

uv 节点。
storeNode

应存储到纹理中的值节点。

默认值为 null。

属性

.access : string

纹理节点的访问类型。

默认值为 'writeOnly'。

.isStorageTextureNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.mipLevel : number

存储纹理要写入的 mip 级别。

默认值为 0。

.storeNode : Node

应存储到纹理中的值节点。

默认值为 null。

方法

.generate( builder : NodeBuilder, output : string ) : string

生成存储节点的代码片段。如果未定义 storeNode， 则按普通纹理生成纹理节点。

builder

当前节点构建器。
output

当前输出。
重写： TextureNode#generate
返回值： 生成的代码片段。

.generateSnippet( builder : NodeBuilder, textureProperty : string, uvSnippet : string, levelSnippet : string, biasSnippet : string, depthSnippet : string, compareSnippet : string, gradSnippet : Array.<string>, offsetSnippet : string ) : string

生成存储纹理的代码片段。

builder

当前节点构建器。
textureProperty

纹理属性。
uvSnippet

uv 片段。
levelSnippet

级别片段。
biasSnippet

偏置片段。
depthSnippet

深度片段。
compareSnippet

比较片段。
gradSnippet

梯度片段。
offsetSnippet

偏移片段。
重写： TextureNode#generateSnippet
返回值： 生成的代码片段。

.generateStore( builder : NodeBuilder )

生成存储纹理节点的代码片段。

builder

当前节点构建器。

.getInputType( builder : NodeBuilder ) : string

重写默认实现，返回固定值 'storageTexture'。

builder

当前节点构建器。
重写： TextureNode#getInputType
返回值： 输入类型。

.getTransformedUV( uvNode : Node ) : Node

重写默认实现，因为存储纹理 坐标是纹素坐标，不应被纹理 uv 矩阵变换。

uvNode

uv 节点。
重写： TextureNode#getTransformedUV
返回值： 未经修改的 uv 节点。

.setAccess( value : string ) : StorageTextureNode

定义节点的访问方式。

value

节点访问方式。
返回值： 对此节点的引用。

.setMipLevel( level : number ) : StorageTextureNode

设置要写入的 mip 级别。

level

mip 级别。
返回值： 对此节点的引用。

.store( uvNode : Node.<(vec2|vec3)>, storeNode : Node ) : StorageTextureNode

在给定坐标处将值存储到此存储纹理中。

uvNode

存储纹理坐标。
storeNode

应存储到纹理中的值节点。

默认值为 null。
返回值： 表示存储操作的存储纹理节点。

.toReadOnly() : StorageTextureNode

配置只读节点访问的便捷方法。

返回值： 对此节点的引用。

.toReadWrite() : StorageTextureNode

配置读写节点访问的便捷方法。

返回值： 对此节点的引用。

.toWriteOnly() : StorageTextureNode

配置只写节点访问的便捷方法。

返回值： 对此节点的引用。
