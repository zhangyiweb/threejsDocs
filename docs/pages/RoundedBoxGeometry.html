RoundedBoxGeometry

具有圆角和圆边的特殊长方体几何体。

代码示例

const geometry = new THREE.RoundedBoxGeometry();
const material = new THREE.MeshStandardMaterial( { color: 0x00ff00 } );
const cube = new THREE.Mesh( geometry, material );
scene.add( cube );

导入

RoundedBoxGeometry 是一个插件，必须显式导入，参见 Installation#Addons。

import { RoundedBoxGeometry } from 'three/addons/geometries/RoundedBoxGeometry.js';

构造函数

new RoundedBoxGeometry( width : number, height : number, depth : number, segments : number, radius : number )

构造一个新的圆角长方体几何体。

width

宽度。即平行于 X 轴的边的长度。

默认值为 1。
height

高度。即平行于 Y 轴的边的长度。

默认值为 1。
depth

深度。即平行于 Z 轴的边的长度。

默认值为 1。
segments

构成圆角的分段数量。

默认值为 2。
radius

圆角半径。

默认值为 0.1。

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

重写： BoxGeometry#parameters

静态方法

.fromJSON( data : Object ) : RoundedBoxGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
