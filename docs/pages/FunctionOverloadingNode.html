FunctionOverloadingNode

此类允许为同一函数定义多个重载版本。 根据函数调用的参数， 节点会选择最匹配的重载版本。

构造函数

new FunctionOverloadingNode( functionNodes : Array.<function()>, …parametersNodes : Node )

构造一个新的函数重载节点。

functionNodes

Fn 函数定义数组。
parametersNodes

参数节点列表。

属性

.functionNodes : Array.<function()>

Fn 函数定义数组。

.global : boolean

此节点被标记为全局。

默认值为 true。

重写： Node#global

.parametersNodes : Array.<Node>

参数节点列表。

方法

.generateNodeType( builder : NodeBuilder ) : string

此方法被重写，因为节点类型是从 函数的返回类型推断的。

builder

当前节点构建器。
重写： Node#generateNodeType
返回值： 节点类型。

.getCandidateFn( builder : NodeBuilder ) : FunctionNode

返回当前参数对应的候选函数。

builder

当前节点构建器。
返回值： 候选函数。

.setup( builder : NodeBuilder ) : Node

根据当前参数设置节点。

builder

当前节点构建器。
重写： Node#setup
返回值： 设置节点。
