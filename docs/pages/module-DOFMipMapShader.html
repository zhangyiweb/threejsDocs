DOFMipMapShader

导入

DOFMipMapShader 是一个插件，必须显式导入，参见 Installation#Addons。

import { DOFMipMapShader } from 'three/addons/shaders/DOFMipMapShader.js';

属性

.DOFMipMapShader : ShaderMaterial~Shader (inner, constant)

Matt Handley @applmak 使用 mipmaps 的景深着色器。

需要启用 mipmaps 的 2 的幂次方大小的渲染目标。
