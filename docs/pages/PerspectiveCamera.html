PerspectiveCamera

使用透视投影的相机。

此投影模式旨在模拟人眼的观察方式，是渲染 3D 场景时最常用的投影模式。

代码示例

const camera = new THREE.PerspectiveCamera( 45, width / height, 1, 1000 );
scene.add( camera );

构造函数

new PerspectiveCamera( fov : number, aspect : number, near : number, far : number )

构造一个新的透视相机。

fov

垂直视场角。

默认值为 50。
aspect

宽高比。

默认值为 1。
near

相机的近裁剪面。

默认值为 0.1。
far

相机的远裁剪面。

默认值为 2000。

属性

.aspect : number

宽高比，通常为画布宽度 / 画布高度。

默认值为 1。

.far : number

相机的远裁剪面。必须大于 PerspectiveCamera#near 的当前值。

默认值为 2000。

.filmGauge : number

用于较长轴的胶片尺寸。默认值为 35（毫米）。除非将 PerspectiveCamera#filmOffset 设置为非零值，否则此参数不会影响投影矩阵。

默认值为 35。

.filmOffset : number

水平偏心偏移，单位与 PerspectiveCamera#filmGauge 相同。

默认值为 0。

.focus : number

用于立体视觉和景深效果的物体距离。除非使用 StereoCamera，否则此参数不会影响投影矩阵。

默认值为 10。

.fov : number

垂直视场角，从视野底部到顶部，单位为度。

默认值为 50。

.isPerspectiveCamera : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.near : number

相机的近裁剪面。有效范围大于 0 且小于 PerspectiveCamera#far 的当前值。

请注意，与 OrthographicCamera 不同，0 不是 透视相机近裁剪面的有效值。

默认值为 0.1。

.view : Object

表示视锥体窗口规格。不应直接编辑此属性，而应通过 PerspectiveCamera#setViewOffset 和 PerspectiveCamera#clearViewOffset 进行设置。

默认值为 null。

.zoom : number

相机的缩放系数。

默认值为 1。

方法

.clearViewOffset()

从投影矩阵中移除视图偏移。

.getEffectiveFOV() : number

返回考虑 PerspectiveCamera#zoom 后的当前垂直视场角（度）。

返回值： 有效视场角。

.getFilmHeight() : number

返回胶片上图像的高度。若 PerspectiveCamera#aspect 大于或等于 1（横向格式），则结果等于 PerspectiveCamera#filmGauge。

返回值： 胶片高度。

.getFilmWidth() : number

返回胶片上图像的宽度。若 PerspectiveCamera#aspect 大于或等于 1（横向格式），则结果等于 PerspectiveCamera#filmGauge。

返回值： 胶片宽度。

.getFocalLength() : number

根据当前 PerspectiveCamera#fov 和 PerspectiveCamera#filmGauge 返回焦距。

返回值： 计算得到的焦距。

.getViewBounds( distance : number, minTarget : Vector2, maxTarget : Vector2 )

计算相机沿观察方向在指定距离处可见矩形的 2D 边界。 将 minTarget 和 maxTarget 设置为可见矩形左下角和右上角的坐标。

distance

观察距离。
minTarget

可见矩形左下角的坐标将写入此向量。
maxTarget

可见矩形右上角的坐标将写入此向量。

.getViewSize( distance : number, target : Vector2 ) : Vector2

计算相机沿观察方向在指定距离处可见矩形的宽度和高度。

distance

观察距离。
target

用于存储结果的目标向量，其中 x 为宽度，y 为高度。
返回值： 可见区域尺寸。

.setFocalLength( focalLength : number )

根据当前 PerspectiveCamera#filmGauge 以焦距设置视场角。

默认胶片尺寸为 35，因此可以为 35mm（全画幅）相机指定焦距。

focalLength

焦距和胶片尺寸必须使用相同单位。

.setViewOffset( fullWidth : number, fullHeight : number, x : number, y : number, width : number, height : number )

在更大的视锥体中设置偏移。这对于多窗口或多显示器/多机器配置非常有用。

例如，如果你有 3×2 台显示器，每台分辨率为 1920×1080，且显示器排列如下：

  +---+---+---+
  | A | B | C |
  +---+---+---+
  | D | E | F |
  +---+---+---+

则对每台显示器应这样调用：

const w = 1920;
const h = 1080;
const fullWidth = w * 3;
const fullHeight = h * 2;
// --A--
camera.setViewOffset( fullWidth, fullHeight, w * 0, h * 0, w, h );
// --B--
camera.setViewOffset( fullWidth, fullHeight, w * 1, h * 0, w, h );
// --C--
camera.setViewOffset( fullWidth, fullHeight, w * 2, h * 0, w, h );
// --D--
camera.setViewOffset( fullWidth, fullHeight, w * 0, h * 1, w, h );
// --E--
camera.setViewOffset( fullWidth, fullHeight, w * 1, h * 1, w, h );
// --F--
camera.setViewOffset( fullWidth, fullHeight, w * 2, h * 1, w, h );

请注意，显示器不必尺寸相同，也不必排列成网格。

fullWidth

多视图配置的完整宽度。
fullHeight

多视图配置的完整高度。
x

子相机的水平偏移。
y

子相机的垂直偏移。
width

子相机的宽度。
height

子相机的高度。

.updateProjectionMatrix()

更新相机的投影矩阵。在更改任何相机属性后必须调用。
