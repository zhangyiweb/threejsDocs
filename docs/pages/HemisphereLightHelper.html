HemisphereLightHelper

为给定的 HemisphereLight 创建由球形网格组成的可视化辅助。

当半球光发生变换或其灯光属性更改时，需要调用相应辅助对象的 update() 方法。

代码示例

const light = new THREE.HemisphereLight( 0xffffbb, 0x080820, 1 );
const helper = new THREE.HemisphereLightHelper( light, 5 );
scene.add( helper );

构造函数

new HemisphereLightHelper( light : HemisphereLight, size : number, color : number | Color | string )

构造一个新的半球光辅助对象。

light

要可视化的灯光。
size

用于可视化灯光的网格尺寸。

默认值为 1.
color

辅助对象的颜色。若未设置，辅助对象将采用灯光的颜色。

属性

.color : number | Color | string

构造函数中传入的颜色参数。若未设置，辅助对象将采用灯光的颜色。

.light : HemisphereLight

正在可视化的灯光。

方法

.dispose()

释放该实例分配的 GPU 相关资源。当应用中不再使用该实例时，应调用此方法。

.update()

更新辅助对象，使其与正在可视化的灯光的位置和方向相匹配。
