IsolateNode

此节点可用作另一个节点的缓存管理组件。 缓存通常在 NodeBuilder 中默认使用，但此节点 允许在构建过程中使用共享的父级缓存。

构造函数

new IsolateNode( node : Node, parent : boolean )

构造一个新的缓存节点。

node

应被缓存的节点。
parent

此节点是否引用共享的父级缓存。

默认值为 true。

属性

.isIsolateNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.node : Node

应被缓存的节点。

.parent : boolean

此节点是否引用共享的父级缓存。

默认值为 true。
