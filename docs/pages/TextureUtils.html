TextureUtils

包含纹理相关实用函数的类。

静态方法

.contain( texture : Texture, aspect : number ) : Texture

在不裁剪或拉伸纹理的前提下，将纹理在其表面内尽可能放大。该方法会保留纹理的原始宽高比。类似于 CSS object-fit: contain。

texture

纹理。
aspect

纹理的宽高比。
返回值： 更新后的纹理。

.cover( texture : Texture, aspect : number ) : Texture

将纹理缩放到能填满表面且不留空白的最小尺寸。该方法会保留纹理的原始宽高比。类似于 CSS object-fit: cover。

texture

纹理。
aspect

纹理的宽高比。
返回值： 更新后的纹理。

.fill( texture : Texture ) : Texture

将纹理配置为默认变换。类似于 CSS object-fit: fill。

texture

纹理。
返回值： 更新后的纹理。

.getByteLength( width : number, height : number, format : number, type : number ) : number

确定表示该纹理需要使用多少字节。

width

纹理宽度。
height

纹理高度。
format

纹理格式。
type

纹理类型。
返回值： 字节长度。
