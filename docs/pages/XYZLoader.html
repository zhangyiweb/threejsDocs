XYZLoader

用于加载 XYZ 格式的加载器。

XYZ 是一种用于存储点云的非常简单的格式。支持 XYZ（点）和 XYZRGB（点 + 颜色）布局。

代码示例

const loader = new XYZLoader();
const geometry = await loader.loadAsync( 'models/xyz/helix_201.xyz' );
geometry.center();
const vertexColors = ( geometry.hasAttribute( 'color' ) === true );
const material = new THREE.PointsMaterial( { size: 0.1, vertexColors: vertexColors } );
const points = new THREE.Points( geometry, material );
scene.add( points );

导入

XYZLoader 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { XYZLoader } from 'three/addons/loaders/XYZLoader.js';

构造函数

new XYZLoader()

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 XYZ 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( text : string ) : BufferGeometry

解析给定的 XYZ 数据并返回结果几何体。

text

原始 XYZ 数据，以字符串形式提供。
重写： Loader#parse
返回值： 表示点云的几何体。
