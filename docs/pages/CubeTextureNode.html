CubeTextureNode

此类均匀节点表示立方体纹理。

构造函数

new CubeTextureNode( value : CubeTexture, uvNode : Node.<vec3>, levelNode : Node.<int>, biasNode : Node.<float> )

构造一个新的立方体纹理节点。

value

立方体纹理。
uvNode

uv 节点。

默认值为 null。
levelNode

level 节点。

默认值为 null。
biasNode

bias 节点。

默认值为 null。

属性

.isCubeTextureNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.generateUV( builder : NodeBuilder, cubeUV : Node ) : string

生成 uv 代码片段。

builder

当前节点构建器。
cubeUV

要为其生成代码的 uv 节点。
重写： TextureNode#generateUV
返回值： 生成的代码片段。

.getDefaultUV() : Node.<vec3>

根据立方体纹理的映射类型返回默认 uv。

重写： TextureNode#getDefaultUV
返回值： 默认 uv 属性。

.getInputType( builder : NodeBuilder ) : string

覆盖默认实现，以返回合适的立方体纹理类型。

builder

当前节点构建器。
重写： TextureNode#getInputType
返回值： 输入类型。

.setUpdateMatrix( value : boolean )

以空实现覆盖，因为 updateMatrix 标志对立方体纹理无效。 uv 变换矩阵不会应用于立方体纹理。

value

更新开关。
重写： TextureNode#setUpdateMatrix

.setupUV( builder : NodeBuilder, uvNode : Node ) : Node

设置 uv 节点。根据后端以及纹理类型，可能需要 修改 uv 节点以保证正确采样。

builder

当前节点构建器。
uvNode

要设置的 uv 节点。
重写： TextureNode#setupUV
返回值： 更新后的 uv 节点。
