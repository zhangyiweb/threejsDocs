PassMultipleTextureNode

PassTextureNode 的扩展，允许管理多个内部纹理。与 getPreviousTexture() 相关 API 有关。

构造函数

new PassMultipleTextureNode( passNode : PassNode, textureName : string, previousTexture : boolean )

构造一个新的 pass texture 节点。

passNode

pass 节点。
textureName

输出纹理名称。
previousTexture

是否应使用上一帧数据。

默认值为 false。

属性

.isPassMultipleTextureNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.previousTexture : boolean

是否应使用上一帧数据。

.textureName : string

输出纹理名称。

方法

.updateTexture()

更新此节点的纹理引用。
