XREstimatedLight

此类可用于表示 XR 会话的环境光照。它依赖 WebXR Lighting Estimation API。

导入

XREstimatedLight 是一个插件，必须显式导入，参见 Installation#Addons。

import { XREstimatedLight } from 'three/addons/webxr/XREstimatedLight.js';

构造函数

new XREstimatedLight( renderer : WebGLRenderer, environmentEstimation : boolean )

构造一个新的光照。

renderer

渲染器。
environmentEstimation

是否使用环境光照估计。

默认值为 true。

类

XREstimatedLight

属性

.directionalLight : DirectionalLight

表示来自 XR 环境的主光源。

.environment : Texture

若环境光照估计可用且已请求，则会在 SessionLightProbe 中设置为立方体贴图。

默认值为 null。

.lightProbe : LightProbe

表示估计光照的光照探针。

方法

.dispose()

释放此实例分配的 GPU 相关资源。当此实例在应用中不再使用时，请调用此方法。
