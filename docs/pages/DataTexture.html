DataTexture

直接从原始缓冲区数据创建纹理。

数据的解释取决于 type 与 format：若类型为 UnsignedByteType，则 Uint8Array 适合用于寻址 纹素数据。若格式为 RGBAFormat，则每个纹素需要四个值： 红、绿、蓝与 Alpha（通常表示不透明度）。

构造函数

new DataTexture( data : TypedArray, width : number, height : number, format : number, type : number, mapping : number, wrapS : number, wrapT : number, magFilter : number, minFilter : number, anisotropy : number, colorSpace : string )

构造一个新的数据纹理。

data

缓冲区数据。

默认值为 null。
width

纹理的宽度。

默认值为 1。
height

纹理的高度。

默认值为 1。
format

纹理格式。

默认值为 RGBAFormat。
type

纹理类型。

默认值为 UnsignedByteType。
mapping

纹理映射方式。

默认值为 Texture.DEFAULT_MAPPING。
wrapS

wrapS 值。

默认值为 ClampToEdgeWrapping。
wrapT

wrapT 值。

默认值为 ClampToEdgeWrapping。
magFilter

放大过滤器值。

默认值为 NearestFilter。
minFilter

缩小过滤器值。

默认值为 NearestFilter。
anisotropy

各向异性值。

默认值为 Texture.DEFAULT_ANISOTROPY。
colorSpace

颜色空间。

默认值为 NoColorSpace。

属性

.flipY : boolean

若设为 true，纹理在上传到 GPU 时会沿垂直轴翻转。

已覆盖并默认设为 false。

默认值为 false。

重写： Texture#flipY

.generateMipmaps : boolean

是否为纹理生成 mipmap（若可能）。

已覆盖并默认设为 false。

默认值为 false。

重写： Texture#generateMipmaps

.image : Object

数据纹理的图像定义。

重写： Texture#image

.isDataTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.unpackAlignment : boolean

指定内存中每行像素起始位置的对齐要求。

已覆盖并默认设为 1。

默认值为 1。

重写： Texture#unpackAlignment
