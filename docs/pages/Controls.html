Controls

控制器的抽象基类。

构造函数

new Controls( object : Object3D, domElement : HTMLElement ) (abstract)

构造一个新的控制器实例。

object

由控制器管理的对象。
domElement

用于事件监听的 HTML 元素。

默认值为 null。

属性

.domElement : HTMLElement

用于事件监听的 HTML 元素。

默认值为 null。

.enabled : boolean

控制器是否响应用户输入。

默认值为 true。

.keys : Object

定义控制器键盘输入的对象。

.mouseButtons : Object

定义将何种操作分配给可用鼠标按钮的对象。具体支持哪些鼠标按钮和操作取决于控制器的实现。

.object : Object3D

由控制器管理的对象。

.state : number

控制器的内部状态。

默认值为 -1。

.touches : Object

定义将何种操作分配给何种触摸交互的对象。具体支持哪些触摸交互和操作取决于控制器的实现。

方法

.connect( element : HTMLElement )

将控制器连接到 DOM。此方法具有所谓的“副作用”，因为它会将模块的事件监听器添加到 DOM。

element

要连接的 DOM 元素。

.disconnect()

将控制器从 DOM 断开连接。

.dispose()

若不再需要使用该控制器，请调用此方法。它会释放所有内部资源并移除所有事件监听器。

.update( delta : number )

若控制器需要在每个模拟步骤更新其内部状态，应实现此方法。

delta

时间增量（秒）。
