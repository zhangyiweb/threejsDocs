ClusteredLighting

基于 Forward+ 分簇着色（Clustered Shading）的自定义光照实现， 会覆盖 WebGPURenderer 中的默认光照系统。适合 拥有大量点光源且具有真实深度复杂度的 3D 场景——视锥体被划分为 三维簇网格，因此每个片元只会计算实际影响到它的光源。

代码示例

const lighting = new ClusteredLighting();
renderer.lighting = lighting; // set lighting system

导入

ClusteredLighting 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { ClusteredLighting } from 'three/addons/lighting/ClusteredLighting.js';

构造函数

new ClusteredLighting( maxLights : number, tileSize : number, zSlices : number, maxLightsPerCluster : number )

构造一个新的分簇光照系统。

maxLights

点光源的最大数量。

默认值为 1024。
tileSize

屏幕分块大小（像素），即簇的 XY 尺寸。

默认值为 32。
zSlices

指数深度切片的数量。

默认值为 24。
maxLightsPerCluster

每个簇的光源列表容量。

默认值为 64。

类

ClusteredLighting

方法

.createNode( lights : Array.<Light> ) : ClusteredLightsNode

为给定的光源数组创建一个新的分簇光源节点。

此方法由渲染器内部调用，所有自定义光照实现都必须重写该方法。

lights

光源列表。
重写： Lighting#createNode
返回值： 分簇光源节点。
