PolyhedronGeometry

多面体是具有平面面的三维立体。该类会接收顶点数组，将其投影到球面上，再按所需细节级别进行细分。

构造函数

new PolyhedronGeometry( vertices : Array.<number>, indices : Array.<number>, radius : number, detail : number )

构造一个新的多面体几何体。

vertices

描述基础形状的扁平顶点数组。
indices

描述基础形状的扁平索引数组。
radius

形状的半径。

默认值为 1.
detail

几何体细分的层级数。细节越多，形状越平滑。

默认值为 0.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

静态方法

.fromJSON( data : Object ) : PolyhedronGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
