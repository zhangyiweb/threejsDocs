StructTypeNode

表示基于节点的系统中的结构体类型节点。 此类用于定义和管理结构体成员的布局与类型。 它扩展了基础 Node 类，并提供获取结构体长度、 检索成员类型，以及为构建器生成结构体类型的方法。

构造函数

new StructTypeNode( membersLayout : Object, name : string )

创建 StructTypeNode 的实例。

membersLayout

结构体成员的布局。
name

结构体的可选名称。

默认值为 null。

属性

.isStructTypeNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.membersLayout : Array.<{name: string, type: string, atomic: boolean}>

结构体成员的布局

.name : string

结构体的名称。

默认值为 null。

重写： Node#name

方法

.getLength() : number

返回结构体以 4 字节元素为单位的长度（例如 float 或 int 分量）。 长度通过对结构体成员长度求和并考虑内存对齐来计算。 若要得到以字节为单位的大小，将返回值乘以 4。

返回值： 结构体以 4 字节元素为单位的长度。
