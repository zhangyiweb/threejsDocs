Line3

由起点和终点表示的三维空间解析线段。

构造函数

new Line3( start : Vector3, end : Vector3 )

构造一个新的线段。

start

线段的起点。

默认值为 (0,0,0)。
end

线段的终点。

默认值为 (0,0,0)。

属性

.end : Vector3

线段的终点。

.start : Vector3

线段的起点。

方法

.applyMatrix4( matrix : Matrix4 ) : Line3

将 4x4 变换矩阵应用于本线段。

matrix

变换矩阵。
返回值： 对本线段的引用。

.at( t : number, target : Vector3 ) : Vector3

返回线段上某一位置处的向量。

t

介于 [0,1] 之间的值，表示线段上的位置。
target

用于存储方法结果的目标向量。
返回值： 差值向量。

.clone() : Line3

返回一个新的线段，其值从本实例复制而来。

返回值： 本实例的克隆。

.closestPointToPoint( point : Vector3, clampToLine : boolean, target : Vector3 ) : Vector3

返回给定点在线上的最近点。

point

要计算其在线上最近点的点。
clampToLine

是否将结果限制在 [0,1] 范围内。
target

用于存储方法结果的目标向量。
返回值： 线上的最近点。

.closestPointToPointParameter( point : Vector3, clampToLine : boolean ) : number

根据投影到线段上的最近点，返回点参数。

point

要返回其点参数的点。
clampToLine

是否将结果限制在 [0,1] 范围内。
返回值： 点参数。

.copy( line : Line3 ) : Line3

将给定线段的值复制到本实例。

line

要复制的线段。
返回值： 对本线段的引用。

.delta( target : Vector3 ) : Vector3

返回线段起点与终点的差值向量。

target

用于存储方法结果的目标向量。
返回值： 差值向量。

.distance() : number

返回线段起点与终点之间的欧几里得距离。

返回值： 欧几里得距离。

.distanceSq() : number

返回线段起点与终点之间的平方欧几里得距离。

返回值： 平方欧几里得距离。

.distanceSqToLine3( line : Line3, c1 : Vector3, c2 : Vector3 ) : number

返回本线段与给定线段之间的最近平方距离。

line

要计算最近平方距离的线段。
c1

本线段上的最近点。
c2

给定线段上的最近点。
返回值： 本线段与给定线段之间的平方距离。

.equals( line : Line3 ) : boolean

若本线段与给定线段相等，则返回 true。

line

用于相等性测试的线段。
返回值： 本线段是否与给定线段相等。

.getCenter( target : Vector3 ) : Vector3

返回线段的中心。

target

用于存储方法结果的目标向量。
返回值： 中心点。

.set( start : Vector3, end : Vector3 ) : Line3

通过复制给定向量来设置起点和终点的值。

start

起点。
end

终点。
返回值： 对本线段的引用。
