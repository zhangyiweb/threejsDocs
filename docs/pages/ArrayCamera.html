ArrayCamera

此类相机可用于通过预定义的一组相机高效渲染场景。这对渲染 VR 场景的性能至关重要。

ArrayCamera 实例始终包含一组子相机。必须为每个子相机定义 viewport 属性，以确定该相机渲染视口的哪一部分。

构造函数

new ArrayCamera( array : Array.<PerspectiveCamera> )

构造一个新的数组相机。

array

透视子相机数组。

默认值为 []。

属性

.cameras : Array.<PerspectiveCamera>

透视子相机数组。

.isArrayCamera : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.isMultiViewCamera : boolean (readonly)

该相机是否用于多视图渲染。

默认值为 false。
