CubeDepthTexture

此类可用于将立方体渲染的深度信息自动保存到 具有深度格式的立方体纹理中。用于 PointLight 阴影。

构造函数

new CubeDepthTexture( size : number, type : number, mapping : number, wrapS : number, wrapT : number, magFilter : number, minFilter : number, anisotropy : number, format : number )

构造一个新的立方体深度纹理。

size

每个立方体面的尺寸（宽度与高度）。
type

纹理类型。

默认值为 UnsignedIntType。
mapping

纹理映射方式。

默认值为 CubeReflectionMapping。
wrapS

wrapS 值。

默认值为 ClampToEdgeWrapping。
wrapT

wrapT 值。

默认值为 ClampToEdgeWrapping。
magFilter

放大过滤器值。

默认值为 NearestFilter。
minFilter

缩小过滤器值。

默认值为 NearestFilter。
anisotropy

各向异性值。

默认值为 Texture.DEFAULT_ANISOTROPY。
format

纹理格式。

默认值为 DepthFormat。

属性

.images : Array.<Image>

CubeDepthTexture#image 的别名。

.isCubeDepthTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.isCubeTexture : boolean (readonly)

在 WebGLTextures 中设为 true，以便按立方体纹理处理。

默认值为 true。
