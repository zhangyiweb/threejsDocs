SkeletonHelper

用于辅助可视化 Skeleton 的辅助对象。

代码示例

const helper = new THREE.SkeletonHelper( skinnedMesh );
scene.add( helper );

构造函数

new SkeletonHelper( object : Object3D )

构造一个新的骨架辅助对象。

object

通常是 SkinnedMesh 的实例。不过，只要表示骨骼层级（参见 Bone），也可使用任意三维对象。

属性

.bones : Array.<Bone>

辅助对象所可视化的骨骼列表。

.isSkeletonHelper : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.root : Object3D

正在可视化的对象。

方法

.dispose()

释放该实例分配的 GPU 相关资源。当应用中不再使用该实例时，应调用此方法。

.setColors( color1 : Color, color2 : Color ) : SkeletonHelper

定义辅助对象的颜色。

color1

每根骨骼的第一种线段颜色。
color2

每根骨骼的第二种线段颜色。
返回值： 对该辅助对象的引用。
