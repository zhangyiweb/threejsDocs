LightProbeNode

用于将光照探针表示为节点的模块。

构造函数

new LightProbeNode( light : LightProbe )

构造一个新的光照探针节点。

light

光照探针。

默认值为 null。

属性

.lightProbe : UniformArrayNode

以球谐函数 uniform 表示的光照探针。

方法

.update( frame : NodeFrame )

被重写以更新光照探针特定的 uniform。

frame

对当前节点帧的引用。
重写： AnalyticLightNode#update
