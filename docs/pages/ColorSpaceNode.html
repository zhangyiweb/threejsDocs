ColorSpaceNode

此节点表示颜色空间转换，即将颜色值从源颜色空间转换到目标颜色空间。

构造函数

new ColorSpaceNode( colorNode : Node, source : string, target : string )

构造一个新的颜色空间节点。

colorNode

表示要转换的颜色。
source

源颜色空间。
target

目标颜色空间。

属性

.colorNode : Node

表示要转换的颜色。

.source : string

源颜色空间。

.target : string

目标颜色空间。

方法

.resolveColorSpace( builder : NodeBuilder, colorSpace : string ) : string

此方法根据当前颜色管理与渲染器的配置，解析常量 WORKING_COLOR_SPACE 和 OUTPUT_COLOR_SPACE。

builder

当前的节点构建器。
colorSpace

要解析的颜色空间。
返回值： 解析后的颜色空间。
