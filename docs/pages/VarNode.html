VarNode

用于将着色器变量表示为节点的类。变量可由 现有节点创建，示例如下：

代码示例

const depth = sampleDepth( uvNode ).toVar( 'depth' );

构造函数

new VarNode( node : Node, name : string, readOnly : boolean )

构造一个新的变量节点。

node

应为其创建变量的节点。
name

着色器中的变量名称。

默认值为 null。
readOnly

只读标志。

默认值为 false。

属性

.global : boolean

VarNode 默认将此属性设为 true。

默认值为 true。

重写： Node#global

.intent : boolean

此标志用于表明该节点用于意图（intent）。

默认值为 false。

.isVarNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.name : string

着色器中的变量名称。若未定义名称， 节点系统会自动生成一个。

默认值为 null。

重写： Node#name

.node : Node

应为其创建变量的节点。

.parents : boolean

向节点系统添加此标志，以表明该节点需要父节点。

默认值为 true。

重写： Node#parents

.readOnly : boolean

只读标志。

默认值为 false。

方法

.getIntent() : boolean

返回此节点的意图标志。

返回值： 意图标志。

.isIntent( builder : NodeBuilder ) : boolean

检查此节点是否用于意图。

builder

节点构建器。
返回值： 此节点是否用于意图。

.setIntent( value : boolean ) : VarNode

设置此节点的意图标志。

此标志用于表明该节点用于意图， 不应直接构建。相反，它用于表明该节点 应被当作变量意图处理。

在无需创建新变量节点的情况下分配变量时很有用。

value

要为意图标志设置的值。
返回值： 此节点。
