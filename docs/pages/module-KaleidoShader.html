KaleidoShader

导入

KaleidoShader 是一个插件，必须显式导入，参见 Installation#Addons。

import { KaleidoShader } from 'three/addons/shaders/KaleidoShader.js';

属性

.KaleidoShader : ShaderMaterial~Shader (inner, constant)

万花筒着色器。 围绕中心点的径向反射 移植自：http://pixelshaders.com/editor/ 作者 Toby Schachman

sides：反射次数 angle：初始角度（弧度）
