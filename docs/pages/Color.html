Color

Color 实例由其在线性工作色彩空间中的 RGB 分量表示， 默认工作色彩空间为 LinearSRGBColorSpace。习惯上使用 SRGBColorSpace 的输入（如十六进制值与 CSS 字符串） 会自动转换到工作色彩空间。

// converted automatically from SRGBColorSpace to LinearSRGBColorSpace
const color = new THREE.Color().setHex( 0x112233 );

也可显式指定源色彩空间，以确保正确转换。

// assumed already LinearSRGBColorSpace; no conversion
const color = new THREE.Color().setRGB( 0.5, 0.5, 0.5 );
// converted explicitly from SRGBColorSpace to LinearSRGBColorSpace
const color = new THREE.Color().setRGB( 0.5, 0.5, 0.5, SRGBColorSpace );

若禁用 THREE.ColorManagement，则不会发生转换。详情参见 色彩管理。遍历 Color 实例将按对应顺序产出其分量 （r、g、b）。可用以下任一方式初始化 Color：

//empty constructor - will default white
const color1 = new THREE.Color();
//Hexadecimal color (recommended)
const color2 = new THREE.Color( 0xff0000 );
//RGB string
const color3 = new THREE.Color("rgb(255, 0, 0)");
const color4 = new THREE.Color("rgb(100%, 0%, 0%)");
//X11 color name - all 140 color names are supported.
//Note the lack of CamelCase in the name
const color5 = new THREE.Color( 'skyblue' );
//HSL string
const color6 = new THREE.Color("hsl(0, 100%, 50%)");
//Separate RGB values between 0 and 1
const color7 = new THREE.Color( 1, 0, 0 );

构造函数

new Color( r : number | string | Color, g : number, b : number )

构造一个新颜色。

注意：在 three.js 中指定颜色的标准方式是使用十六进制三元组， 文档其余部分也采用该方法。

r

颜色的红色分量。若未提供 g 和 b， 则可以是十六进制三元组、CSS 风格字符串或另一个 Color 实例。
g

绿色分量。
b

蓝色分量。

属性

.b : number

蓝色分量。

默认值为 1。

.g : number

绿色分量。

默认值为 1。

.isColor : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.r : number

红色分量。

默认值为 1。

.NAMES : Object

包含 X11 颜色名称的字典。

注意：像 Dark Orange 这样的多词名称会变为字符串 'darkorange'。

.lerp( color : Color, alpha : number ) : Color

将此颜色的 RGB 值向给定颜色的 RGB 值线性插值。 可将 alpha 参数理解为两种颜色之间的比例，其中 0.0 表示此颜色，1.0 表示第一个参数。

color

要趋近的颜色。
alpha

闭区间 [0,1] 内的插值因子。
返回值： 对此颜色的引用。

.lerpColors( color1 : Color, color2 : Color, alpha : number ) : Color

在给定颜色之间线性插值，并将结果存储在本实例中。 可将 alpha 参数理解为两种颜色之间的比例，其中 0.0 表示第一种颜色，1.0 表示第二种颜色。

color1

第一种颜色。
color2

第二种颜色。
alpha

闭区间 [0,1] 内的插值因子。
返回值： 对此颜色的引用。

.lerpHSL( color : Color, alpha : number ) : Color

将此颜色的 HSL 值向给定颜色的 HSL 值线性插值。它与 Color#lerp 的不同之处在于：不是直接从一种颜色插值到另一种， 而是经过两种颜色之间的所有色相。可将 alpha 参数理解为 两种颜色之间的比例，其中 0.0 表示此颜色，1.0 表示第一个参数。

color

要趋近的颜色。
alpha

闭区间 [0,1] 内的插值因子。
返回值： 对此颜色的引用。
