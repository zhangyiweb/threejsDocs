VelocityNode

用于表示运动或速度向量的节点。是运动模糊或 TRAA 等高级后期处理效果的基础。

该节点会跟踪上一帧的模型、视图和投影矩阵， 并用它们计算 NDC 空间中的偏移量。 这些偏移量即表示最终的速度。

构造函数

new VelocityNode()

构造一个新的速度节点。

属性

.previousCameraViewMatrix : UniformNode.<mat4>

表示上一帧视图矩阵的 uniform 节点。

默认值为 null。

.previousModelWorldMatrix : UniformNode.<mat4>

表示上一帧世界空间模型矩阵的 uniform 节点。

默认值为 null。

.previousProjectionMatrix : UniformNode.<mat4>

表示上一帧投影矩阵的 uniform 节点。

默认值为 null。

.projectionMatrix : Matrix4

当前投影矩阵。

默认值为 null。

.updateAfterType : string

已重写，因为速度节点在更新之后保存数据。

默认值为 'object'。

重写： TempNode#updateAfterType

.updateType : string

已重写，因为速度节点按对象更新。

默认值为 'object'。

重写： TempNode#updateType

方法

.setProjectionMatrix( projectionMatrix : Matrix4 )

设置给定的投影矩阵。

projectionMatrix

要设置的投影矩阵。

.setup( builder : NodeBuilder ) : Node.<vec2>

基于上一帧与当前顶点数据实现速度计算。

builder

对当前节点构建器的引用。
重写： TempNode#setup
返回值： 运动向量。

.update( frame : NodeFrame )

更新速度相关的 uniform。

frame

对当前节点帧的引用。
重写： TempNode#update

.updateAfter( frame : NodeFrame )

已重写以更新速度相关的 uniform。

frame

对当前节点帧的引用。
重写： TempNode#updateAfter
