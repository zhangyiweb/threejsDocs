LineSegmentsGeometry

由一系列顶点对组成的线段。

在 LineSegments2 中用于描述形状。

导入

LineSegmentsGeometry 是一个插件，必须显式导入，参见 Installation#Addons。

import { LineSegmentsGeometry } from 'three/addons/lines/LineSegmentsGeometry.js';

构造函数

new LineSegmentsGeometry()

构造一个新的线段几何体。

属性

.isLineSegmentsGeometry : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.applyMatrix4( matrix : Matrix4 ) : LineSegmentsGeometry

将给定的 4x4 变换矩阵应用到此几何体。

matrix

要应用的矩阵。
返回值： 对此实例的引用。

.fromEdgesGeometry( geometry : EdgesGeometry ) : LineSegmentsGeometry

根据给定的边缘几何体设置此线段几何体。

geometry

应作为此几何体数据源的几何体。
返回值： 对此几何体的引用。

.fromLineSegments( lineSegments : LineSegments ) : LineSegmentsGeometry

根据给定的线段设置此线段几何体。

lineSegments

应作为此几何体数据源的线段。 假定源几何体未使用索引。
返回值： 对此几何体的引用。

.fromMesh( mesh : Mesh ) : LineSegmentsGeometry

根据给定的网格设置此线段几何体。

mesh

应作为此几何体数据源的网格几何体。
返回值： 对此几何体的引用。

.fromWireframeGeometry( geometry : WireframeGeometry ) : LineSegmentsGeometry

根据给定的线框几何体设置此线段几何体。

geometry

应作为此几何体数据源的几何体。
返回值： 对此几何体的引用。

.setColors( array : Float32Array | Array.<number> ) : LineSegmentsGeometry

为此几何体设置给定的线颜色。长度必须是六的倍数，因为 每条线段按 (rgb rgb) 模式由起点与终点颜色定义。

array

要设置的位置数据。
返回值： 对此几何体的引用。

.setPositions( array : Float32Array | Array.<number> ) : LineSegmentsGeometry

为此几何体设置给定的线位置。长度必须是六的倍数，因为 每条线段按 (xyz xyz) 模式由起点与终点顶点定义。

array

要设置的位置数据。
返回值： 对此几何体的引用。
