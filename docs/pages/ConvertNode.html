ConvertNode

此模块属于 TSL 核心，通常不在应用层代码中使用。 它表示着色器生成过程中的转换操作， 即将节点的数据类型转换为目标数据类型。

构造函数

new ConvertNode( node : Node, convertTo : string )

构造一个新的转换节点。

node

应转换其类型的节点。
convertTo

目标节点类型。可通过 | 符号分隔以定义多种类型。

属性

.convertTo : string

目标节点类型。可通过 | 符号分隔以定义多种类型。

.node : Node

应转换其类型的节点。

方法

.generateNodeType( builder : NodeBuilder ) : string

此方法被重写，因为实现会尝试从 ConvertNode#convertTo 属性推断最匹配的类型。

builder

当前的节点构建器。
重写： Node#generateNodeType
返回值： 节点类型。
