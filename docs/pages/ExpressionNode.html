ExpressionNode

此类可用于在着色器代码中实现基本表达式。 典型示例包括 return、continue 或 discard 语句。

构造函数

new ExpressionNode( snippet : string, nodeType : string )

构造一个新的表达式节点。

snippet

原生代码片段。

默认值为 ''。
nodeType

节点类型。

默认值为 'void'。

属性

.snippet : string

原生代码片段。

默认值为 ''。
