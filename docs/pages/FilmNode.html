FilmNode

用于创建胶片颗粒效果的后处理节点。

导入

FilmNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { film } from 'three/addons/tsl/display/FilmNode.js';

构造函数

new FilmNode( inputNode : Node, intensityNode : Node.<float>, uvNode : Node.<vec2> )

构造一个新的胶片节点。

inputNode

表示效果输入的节点。
intensityNode

表示效果强度的节点。

默认值为 null。
uvNode

用于传入自定义（例如动画）UV 数据的节点。

默认值为 null。

属性

.inputNode : Node

表示效果输入的节点。

.intensityNode : Node.<float>

表示效果强度的节点。

默认值为 null。

.uvNode : Node.<vec2>

用于传入自定义（例如动画）UV 数据的节点。

默认值为 null。

方法

.setup( builder : NodeBuilder ) : ShaderCallNodeInternal

用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup
