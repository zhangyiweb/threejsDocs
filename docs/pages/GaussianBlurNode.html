GaussianBlurNode

用于创建高斯模糊效果的后处理节点。

导入

GaussianBlurNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { gaussianBlur, premultipliedGaussianBlur } from 'three/addons/tsl/display/GaussianBlurNode.js';

构造函数

new GaussianBlurNode( textureNode : TextureNode, directionNode : Node.<(vec2|float)>, sigma : number, options : Object )

构造一个新的高斯模糊节点。

textureNode

表示效果输入的纹理节点。
directionNode

定义模糊的方向与半径。

默认值为 null。
sigma

控制模糊滤波器的核。数值越大，模糊半径越宽。

默认值为 4。
options

高斯模糊效果的附加选项。

默认值为 {}。

premultipliedAlpha

是否对模糊效果使用预乘 alpha。

默认值为 false。
resolutionScale

效果的分辨率缩放。0.5 表示纹理节点分辨率的一半。

默认值为 1。

属性

.directionNode : Node.<(vec2|float)>

定义模糊的方向与半径。

.isGaussianBlurNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.premultipliedAlpha : boolean

效果是否使用预乘 alpha。若要模糊带透明度的纹理输入，应设为 true。

默认值为 false。

.resolution : Vector2

分辨率缩放。

默认值为 {(1,1)}。

已弃用： 是

.resolutionScale : number

分辨率缩放。

默认值为 (1)。

.sigma : number

控制模糊滤波器的核。数值越大，模糊半径越宽。

.textureNode : TextureNode

表示效果输入的纹理节点。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.FRAME，因为该节点会在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

方法

.dispose()

释放内部资源。当不再需要该效果时应调用此方法。

重写： TempNode#dispose

.getTextureNode() : PassTextureNode

将效果结果作为纹理节点返回。

返回值： 表示效果结果的纹理节点。

.setSize( width : number, height : number )

设置效果的尺寸。

width

效果的宽度。
height

效果的高度。

.setup( builder : NodeBuilder ) : PassTextureNode

用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
