AnimationLoader

用于加载 JSON 格式动画剪辑的类。文件在内部通过 FileLoader 加载。

代码示例

const loader = new THREE.AnimationLoader();
const animations = await loader.loadAsync( 'animations/animation.js' );

构造函数

new AnimationLoader( manager : LoadingManager )

构造一个新的动画加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的动画以包含 AnimationClip 实例的数组形式传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( json : Object ) : Array.<AnimationClip>

解析给定的 JSON 对象并返回动画剪辑数组。

json

序列化后的动画剪辑。
重写： Loader#parse
返回值： 解析后的动画剪辑。
