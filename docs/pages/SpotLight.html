SpotLight

该光源从单个点沿一个方向发射，形成锥形光柱， 且离光源越远锥体越大。

此光源可投射阴影——详情参见 SpotLightShadow。

代码示例

// white spotlight shining from the side, modulated by a texture
const spotLight = new THREE.SpotLight( 0xffffff );
spotLight.position.set( 100, 1000, 100 );
spotLight.map = new THREE.TextureLoader().load( url );
spotLight.castShadow = true;
spotLight.shadow.mapSize.width = 1024;
spotLight.shadow.mapSize.height = 1024;
spotLight.shadow.camera.near = 500;
spotLight.shadow.camera.far = 4000;
spotLight.shadow.camera.fov = 30;s

构造函数

new SpotLight( color : number | Color | string, intensity : number, distance : number, angle : number, penumbra : number, decay : number )

构造一个新的聚光灯。

color

光源颜色。

默认值为 0xffffff。
intensity

光源强度，单位为坎德拉 (cd)。

默认值为 1。
distance

灯光的最大照射范围。0 表示无限制。

默认值为 0。
angle

光线相对其方向的最大散射角度，上限为 Math.PI/2。

默认值为 Math.PI/3。
penumbra

聚光灯锥体中因半影而衰减的百分比。取值范围为 [0,1]。

默认值为 0。
decay

灯光沿距离衰减的程度。

默认值为 2。

属性

.angle : number

光线相对其方向的最大散射角度，上限为 Math.PI/2。

默认值为 Math.PI/3。

.decay : number

灯光沿距离衰减的程度。在物理正确渲染的语境下， 不应更改默认值。

默认值为 2。

.distance : number

灯光的最大照射范围。0 表示无限制。

默认值为 0。

.isSpotLight : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.map : Texture

用于调制灯光颜色的纹理。聚光灯颜色 会与该纹理的 RGB 值混合，混合比例 对应该纹理的 alpha 值。类似 cookie 的遮罩效果 通过像素值 (0, 0, 0, 1-cookie_value) 再现。

警告：若 Object3D#castShadow 设为 false，此属性将被禁用。

默认值为 null。

.penumbra : number

聚光灯锥体中因半影而衰减的百分比。 取值范围为 [0,1]。

默认值为 0。

.power : number

灯光功率。功率是以流明 (lm) 为单位的光通量。 更改功率也会更改灯光强度。

.shadow : SpotLightShadow

此属性保存灯光的阴影配置。

.target : Object3D

聚光灯从其位置指向 目标的位置。

要将目标位置改为默认值以外的值， 必须将其添加到场景中。

也可以将目标设为场景中的另一个 3D 物体， 灯光将跟踪该目标物体。
