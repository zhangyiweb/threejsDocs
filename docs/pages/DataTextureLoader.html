DataTextureLoader

用于加载二进制纹理格式（RGBE、EXR 或 TGA）的抽象基类。 纹理在内部通过 FileLoader 加载。

派生类必须实现 parse() 方法，其中包含各自格式的 解析逻辑。

构造函数

new DataTextureLoader( manager : LoadingManager ) (abstract)

构造一个新的数据纹理加载器。

manager

加载管理器。

方法

.createDataTexture( buffer : ArrayBuffer ) : DataTexture

解析给定的缓冲区并返回已配置的数据纹理。当纹理数据已在内存中 （例如通过拖放或从服务器加载），且无需经过 DataTextureLoader#load 时，可使用此方法解析。

buffer

原始纹理数据。
返回值： 数据纹理。

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback ) : DataTexture

从指定 URL 开始加载，并将加载完成的数据纹理 传递给 onLoad() 回调。该方法还会返回一个新的纹理对象，可直接 用于创建材质。若采用这种方式，当相应加载过程完成时， 纹理可能会出现在场景中。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load
返回值： 数据纹理。
