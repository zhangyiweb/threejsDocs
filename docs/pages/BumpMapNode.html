BumpMapNode

该类可用于向材质应用凹凸贴图。

代码示例

material.normalNode = bumpMap( texture( bumpTex ) );

构造函数

new BumpMapNode( textureNode : Node.<float>, scaleNode : Node.<float> )

构造一个新的凹凸贴图节点。

textureNode

表示凹凸贴图数据。
scaleNode

控制凹凸效果的强度。

默认值为 null。

属性

.scaleNode : Node.<float>

控制凹凸效果的强度。

默认值为 null。

.textureNode : Node.<float>

表示凹凸贴图数据。
