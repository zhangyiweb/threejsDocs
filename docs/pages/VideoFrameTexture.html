VideoFrameTexture

此类可作为定义视频数据的另一种方式。与 VideoTexture 使用 HTMLVideoElement 实例不同，VideoFrameTexture 期望通过 VideoFrameTexture#setFrame 手动定义每一帧。该模块的典型用例是 使用 WebCodecs API 解码视频帧时。

代码示例

const texture = new THREE.VideoFrameTexture();
texture.setFrame( frame );

构造函数

new VideoFrameTexture( mapping : number, wrapS : number, wrapT : number, magFilter : number, minFilter : number, format : number, type : number, anisotropy : number )

构造一个新的视频帧纹理。

mapping

纹理映射方式。

默认值为 Texture.DEFAULT_MAPPING。
wrapS

wrapS 值。

默认值为 ClampToEdgeWrapping。
wrapT

wrapT 值。

默认值为 ClampToEdgeWrapping。
magFilter

放大过滤器值。

默认值为 LinearFilter。
minFilter

缩小过滤器值。

默认值为 LinearFilter。
format

纹理格式。

默认值为 RGBAFormat。
type

纹理类型。

默认值为 UnsignedByteType。
anisotropy

各向异性值。

默认值为 Texture.DEFAULT_ANISOTROPY。

属性

.isVideoFrameTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.setFrame( frame : VideoFrame )

设置视频的当前帧。这会自动更新纹理， 以便数据可用于渲染。

frame

视频帧。

.update()

此方法被覆盖为空实现，因为 此类纹理通过 setFrame() 更新。

重写： VideoTexture#update
