Raycaster

此类用于辅助光线投射。光线投射可用于鼠标拾取（判断鼠标在三维空间中指向哪些对象）等场景。

构造函数

new Raycaster( origin : Vector3, direction : Vector3, near : number, far : number )

构造一个新的 Raycaster。

origin

光线投射的起点向量。
direction

为光线指定方向的（已归一化）方向向量。
near

返回的所有结果距离都大于 near。near 不能为负数。

默认值为 0。
far

返回的所有结果距离都小于 far。far 不能小于 near。

默认值为 Infinity。

属性

.camera : Camera

对精灵图等面向摄像机的 Billboard 对象进行光线投射时使用的相机。该字段可手动设置，或在调用 setFromCamera() 时自动设置。

默认值为 null。

.far : number

返回的所有结果距离都小于 far。far 不能小于 near。

默认值为 Infinity。

.layers : Layers

在执行相交检测时，可以有选择地忽略三维对象。以下代码示例确保只有位于图层 1 的三维对象会被 raycaster 检测。

raycaster.layers.set( 1 );
object.layers.enable( 1 );

.near : number

返回的所有结果距离都大于 near。near 不能为负数。

默认值为 0。

.params : Object

用于配置光线投射的参数对象。其结构如下：

{
	Mesh: {},
	Line: { threshold: 1 },
	LOD: {},
	Points: { threshold: 1 },
	Sprite: {}
}

其中 threshold 表示光线投射与对象相交时的精度，单位为世界坐标。

.ray : Ray

用于光线投射的光线。

方法

.intersectObject( object : Object3D, recursive : boolean, intersects : Array.<Raycaster~Intersection> ) : Array.<Raycaster~Intersection>

检测光线与对象（可选择是否包含子对象）之间的所有相交。返回的相交结果按距离排序，最近的在前。

Raycaster 在判断光线是否与对象相交时，会委托给所传入三维对象的 raycast() 方法。这使得网格、线条和点可以对光线投射做出不同的响应。

注意：对于网格，面的法线必须朝向光线起点才能被检测到；光线从面的背面穿过时的相交不会被检测到。若要对对象的两面都进行光线投射，请将 Material#side 设置为 THREE.DoubleSide。

object

要与光线检测相交的三维对象。
recursive

若为 true，还会检测所有子对象。否则仅检测与该对象的相交。

默认值为 true。
intersects

用于存放方法返回结果的目标数组。

默认值为 []。
返回值： 存放相交点的数组。

.intersectObjects( objects : Array.<Object3D>, recursive : boolean, intersects : Array.<Raycaster~Intersection> ) : Array.<Raycaster~Intersection>

检测光线与多个对象（可选择是否包含子对象）之间的所有相交。返回的相交结果按距离排序，最近的在前。

objects

要与光线检测相交的三维对象数组。
recursive

若为 true，还会检测所有子对象。否则仅检测与该对象的相交。

默认值为 true。
intersects

用于存放方法返回结果的目标数组。

默认值为 []。
返回值： 存放相交点的数组。

.set( origin : Vector3, direction : Vector3 )

从参数复制值，用新的起点和方向更新光线。

origin

光线投射的起点向量。
direction

为光线指定方向的（已归一化）方向向量。

.setFromCamera( coords : Vector2, camera : Camera )

使用给定的坐标和相机，为内部光线计算新的起点和方向。

coords

鼠标的二维坐标，采用归一化设备坐标（NDC）。X 和 Y 分量应在 -1 与 1 之间。
camera

光线应从其发出的相机。

.setFromXRController( controller : WebXRController ) : Raycaster

使用给定的 WebXR 控制器，为内部光线计算新的起点和方向。

controller

要从中复制位置与方向的控制器。
返回值： 对该实例的引用。

类型定义

.Intersection

光线投射相交检测的交点。

distance
number

从光线起点到交点的距离。
distanceToRay
number

部分三维对象（例如 Points）会提供交点到光线上最近点的距离。对于其他对象，该值为 undefined。
point
Vector3

交点，采用世界坐标。
face
Object

被相交的面。
faceIndex
number

面的索引。
object
Object3D

被相交的三维对象。
uv
Vector2

交点处的 U、V 坐标。
uv1
Vector2

交点处的第二组 U、V 坐标。
normal
Vector3

交点处的插值法线向量。
instanceId
number

光线与 InstancedMesh 相交处的实例索引。
