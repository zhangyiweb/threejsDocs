FunctionCallNode

此模块表示对 FunctionNode 的调用。开发者通常不会直接接触 此模块，因为他们使用预定义的 TSL 语法 wgslFn 和 glslFn，这些语法封装了 该逻辑。

构造函数

new FunctionCallNode( functionNode : FunctionNode, parameters : Object.<string, Node> )

构造一个新的函数调用节点。

functionNode

函数节点。

默认值为 null。
parameters

函数调用的参数。

默认值为 {}。

属性

.functionNode : FunctionNode

函数节点。

默认值为 null。

.parameters : Object.<string, Node>

函数调用的参数。

默认值为 {}。

方法

.generateNodeType( builder : NodeBuilder ) : string

返回此函数调用节点的类型。

builder

当前节点构建器。
重写： TempNode#generateNodeType
返回值： 此节点的类型。

.getMemberType( builder : NodeBuilder, name : string ) : string

返回此函数调用节点的函数节点。

builder

当前节点构建器。
name

成员名称。
重写： TempNode#getMemberType
返回值： 成员的类型。

.getParameters() : Object.<string, Node>

返回函数调用节点的参数。

返回值： 此节点的参数。

.setParameters( parameters : Object.<string, Node> ) : FunctionCallNode

设置函数调用节点的参数。

parameters

要设置的参数。
返回值： 对此节点的引用。
