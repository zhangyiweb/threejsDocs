Matrix4

表示一个 4x4 矩阵。

在三维计算机图形中，4x4 矩阵最常见的用途是作为变换矩阵。 关于 WebGL 中所用变换矩阵的入门介绍，可参阅本教程。

这使得表示三维空间中一点的三维向量可以通过与矩阵相乘， 经历平移、旋转、剪切、缩放、反射、正交或透视投影等变换。 这称为将矩阵应用到向量上。

关于行主序与列主序的说明：

构造函数和 Matrix3#set 方法按 行主序 接收参数，而内部则按列主序存储在 Matrix3#elements 数组中。 这意味着调用：

const m = new THREE.Matrix4();
m.set( 11, 12, 13, 14,
       21, 22, 23, 24,
       31, 32, 33, 34,
       41, 42, 43, 44 );

会使 elements 数组包含：

m.elements = [ 11, 21, 31, 41,
               12, 22, 32, 42,
               13, 23, 33, 43,
               14, 24, 34, 44 ];

并且内部所有计算都使用列主序进行。 不过，由于实际排序在数学上并无差异，且大多数人习惯按行主序思考矩阵， three.js 文档以行主序展示矩阵。请记住：若阅读源码，需要将此处列出的矩阵取转置，才能对应理解计算过程。

返回值： 此矩阵的引用。

.decompose( position : Vector3, quaternion : Quaternion, scale : Vector3 ) : Matrix4

将此矩阵分解为位置、旋转和缩放分量，并将结果写入给定对象。

注意：并非所有矩阵都能以这种方式分解。例如，若对象的父级具有非均匀缩放， 则该对象的世界矩阵可能无法分解，此方法可能不适用。

position

位置向量。
quaternion

作为四元数的旋转。
scale

缩放向量。
返回值： 此矩阵的引用。

.determinant() : number

计算并返回此矩阵的行列式。

基于此处所述方法。

返回值： 行列式。

.determinantAffine() : number

计算并返回 4x4 矩阵的行列式，但假定矩阵为仿射矩阵，从而节省部分计算。

对于仿射矩阵（如对象的世界矩阵），此值等于完整的 4x4 Matrix4#determinant，但计算成本更低。

假定底行为 [0, 0, 0, 1]。

返回值： 矩阵的行列式。

.equals( matrix : Matrix4 ) : boolean

matrix

用于相等性测试的矩阵。
返回值： 此矩阵是否与给定矩阵相等。
