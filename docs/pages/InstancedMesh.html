InstancedMesh

支持实例化渲染的特殊网格版本。如果你需要渲染大量 具有相同几何体和材质、但世界变换不同的对象，请使用 此类。使用 InstancedMesh 有助于减少绘制调用次数，从而 提升应用的整体渲染性能。

构造函数

new InstancedMesh( geometry : BufferGeometry, material : Material | Array.<Material>, count : number )

构造一个新的实例化网格。

geometry

网格几何体。
material

网格材质。
count

实例数量。

属性

.boundingBox : Box3

实例化网格的包围盒。可通过 InstancedMesh#computeBoundingBox 计算。

默认值为 null。

.boundingSphere : Sphere

实例化网格的包围球。可通过 InstancedMesh#computeBoundingSphere 计算。

默认值为 null。

.count : number

实例数量。

重写： Mesh#count

.instanceColor : InstancedBufferAttribute

表示所有实例的颜色。如果你通过 InstancedMesh#setColorAt 修改了实例数据，必须将其 BufferAttribute#needsUpdate 标志设为 true。

默认值为 null。

.instanceMatrix : InstancedBufferAttribute

表示所有实例的局部变换。如果你通过 InstancedMesh#setMatrixAt 修改了实例数据，必须将其 BufferAttribute#needsUpdate 标志设为 true。

.isInstancedMesh : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.morphTexture : DataTexture

表示所有实例的变形目标权重。如果你通过 InstancedMesh#setMorphAt 修改了实例数据，必须将其 Texture#needsUpdate 标志设为 true。

默认值为 null。

方法

.computeBoundingBox()

计算实例化网格的包围盒，并更新 InstancedMesh#boundingBox。 引擎不会自动计算包围盒；你的应用必须调用此方法。 如果通过 InstancedMesh#setMatrixAt 变换了某个实例，可能需要重新计算包围盒。

.computeBoundingSphere()

计算实例化网格的包围球，并更新 InstancedMesh#boundingSphere。 当需要时（例如用于光线投射或视锥体剔除），引擎会自动计算包围球。 如果通过 InstancedMesh#setMatrixAt 变换了某个实例，可能需要重新计算包围球。

.dispose()

释放此实例所分配的 GPU 相关资源。当你的应用不再使用 此实例时，请调用此方法。

.getColorAt( index : number, color : Color ) : Color

获取指定实例的颜色。

color

返回值： 目标颜色的引用。

.getMatrixAt( index : number, matrix : Matrix4 ) : Matrix4

获取指定实例的局部变换矩阵。

index

实例索引。
matrix

用于存储方法结果的目标对象。
返回值： 目标矩阵的引用。

.getMorphAt( index : number, object : Mesh )

获取指定实例的变形目标权重。

index

实例索引。
object

用于存储方法结果的目标对象。

.setColorAt( index : number, color : Color ) : InstancedMesh

将给定颜色设置到指定实例。更新完所有颜色后，请确保将 InstancedMesh#instanceColor 的 needsUpdate 标志设为 true。

index

实例索引。
color

实例颜色。
返回值： 此实例化网格的引用。

.setMatrixAt( index : number, matrix : Matrix4 ) : InstancedMesh

将给定局部变换矩阵设置到指定实例。更新完所有矩阵后，请确保将 InstancedMesh#instanceMatrix 的 needsUpdate 标志设为 true。

index

实例索引。
matrix

局部变换。
返回值： 此实例化网格的引用。

.setMorphAt( index : number, object : Mesh ) : InstancedMesh

将变形目标权重设置到指定实例。更新完所有影响权重后，请确保将 InstancedMesh#morphTexture 的 needsUpdate 标志设为 true。

index

实例索引。
object

一个网格，其 morphTargetInfluences 属性包含单个实例的变形目标权重。
返回值： 此实例化网格的引用。
