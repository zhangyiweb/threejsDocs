LOD

用于提供基本细节层次（LOD）机制的组件。

每个 LOD 级别都关联一个对象，可在指定距离处在它们之间切换渲染。 通常你会创建例如三个网格：一个用于远处（低细节）、一个用于中距离（中等细节）、 一个用于近距离（高细节）。

代码示例

const lod = new THREE.LOD();
const material = new THREE.MeshBasicMaterial( { color: 0xffff00 } );
//Create spheres with 3 levels of detail and create new LOD levels for them
for( let i = 0; i < 3; i++ ) {
	const geometry = new THREE.IcosahedronGeometry( 10, 3 - i );
	const mesh = new THREE.Mesh( geometry, material );
	lod.addLevel( mesh, i * 75 );
}
scene.add( lod );

构造函数

new LOD()

构造一个新的 LOD。

属性

.autoUpdate : boolean

LOD 对象是否由渲染器每帧自动更新。 若设为 false，则必须在渲染循环中自行调用 LOD#update。

默认值为 true。

.isLOD : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.levels : Array.<{object:Object3D, distance:number, hysteresis:number}>

此数组保存 LOD 级别。

方法

.addLevel( object : Object3D, distance : number, hysteresis : number ) : LOD

添加一个将在特定距离及更远处显示的网格。通常距离越远， 网格的细节越低。

object

要在此级别显示的 3D 对象。
distance

显示此细节层次的距离。

默认值为 0。
hysteresis

用于避免 LOD 边界闪烁的阈值，为距离的分数。

默认值为 0。
返回值： 此实例的引用。

.getCurrentLevel() : number

返回当前激活的 LOD 级别索引。

返回值： 当前激活的 LOD 级别索引。

.getObjectForDistance( distance : number ) : Object3D

返回第一个大于给定距离的 3D 对象的引用。

distance

LOD 距离。
返回值： 找到的 3D 对象。若未找到则为 null。

.raycast( raycaster : Raycaster, intersects : Array.<Object> )

计算投射光线与此 LOD 之间的交点。

raycaster

光线投射器。
intersects

保存交点的目标数组。
重写： Object3D#raycast

.removeLevel( distance : number ) : boolean

根据与相机的距离移除现有级别。 成功移除时返回 true，否则返回 false。

distance

要移除的级别的距离。
返回值： 该级别是否已被移除。

.update( camera : Camera )

通过根据给定相机的当前距离计算应可见的 LOD 级别来更新 LOD。

camera

用于渲染场景的相机。
