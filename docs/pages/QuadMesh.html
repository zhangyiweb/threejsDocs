QuadMesh

此模块是需要渲染全屏效果的通道的辅助工具， 这在后期处理场景中相当常见。

预期用法是复用单个四边形网格来渲染 后续通道，只需重新指定 material 引用即可。

注意：此模块只能与 WebGPURenderer 一起使用。

构造函数

new QuadMesh( material : Material )

构造一个新的四边形网格。

material

用于渲染四边形网格的材质。

默认值为 null。

属性

.camera : OrthographicCamera (readonly)

用于渲染四边形网格的相机。

.isQuadMesh : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.render( renderer : Renderer )

渲染四边形网格

renderer

渲染器。

.renderAsync( renderer : Renderer ) : Promise (async)

render() 的异步版本。

renderer

渲染器。
已弃用： 是
返回值： 在渲染完成后 resolve 的 Promise。
