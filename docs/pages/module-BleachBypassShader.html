BleachBypassShader

导入

BleachBypassShader 是一个插件，必须显式导入，参见 Installation#Addons。

import { BleachBypassShader } from 'three/addons/shaders/BleachBypassShader.js';

属性

.BleachBypassShader : ShaderMaterial~Shader (inner, constant)

漂白旁路着色器 [http://en.wikipedia.org/wiki/Bleach_bypass]，基于 Nvidia Shader library。
