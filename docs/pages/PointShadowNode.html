PointShadowNode

表示点光源节点对应的阴影实现。

构造函数

new PointShadowNode( light : PointLight, shadow : PointLightShadow )

构造一个新的点光源阴影节点。

light

投射阴影的点光源。
shadow

可选的点光源阴影。

默认值为 null。

方法

.getShadowFilterFn( type : number ) : function

重写默认实现，以返回点光源阴影专用的 过滤函数。

type

阴影类型。
重写： ShadowNode#getShadowFilterFn
返回值： 过滤函数。

.renderShadow( frame : NodeFrame )

重写默认实现，使用点光源专用的 渲染代码。

frame

对当前节点帧的引用。
重写： ShadowNode#renderShadow

.setupRenderTarget( shadow : LightShadow, builder : NodeBuilder ) : Object

重写默认实现，以创建带有 CubeDepthTexture 的 CubeRenderTarget。

shadow

光源阴影对象。
builder

对当前节点构建器的引用。
返回值： 包含阴影贴图和深度纹理的对象。

.setupShadowCoord( builder : NodeBuilder, shadowPosition : Node.<vec3> ) : Node.<vec3>

重写默认实现，直接使用未经修改的阴影位置。

builder

对当前节点构建器的引用。
shadowPosition

表示阴影位置的节点。
重写： ShadowNode#setupShadowCoord
返回值： 阴影坐标。

.setupShadowFilter( builder : NodeBuilder, inputs : Object ) : Node.<float>

重写默认实现，仅使用点光源专用的 阴影过滤函数。

builder

对当前节点构建器的引用。
inputs

定义阴影过滤的配置对象。

filterFn

此函数定义阴影贴图的过滤类型，例如 PCF。
depthTexture

对阴影贴图深度纹理的引用。
shadowCoord

用于从阴影贴图采样的阴影坐标。
shadow

光源阴影。
重写： ShadowNode#setupShadowFilter
返回值： 阴影过滤的结果节点。
