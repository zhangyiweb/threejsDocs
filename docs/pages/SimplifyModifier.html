SimplifyModifier

此类可通过简化几何体来修改几何体。此类修改器的典型用例是自动生成 LOD。

实现基于 Stan Melax 于 1998 年提出的 Progressive Mesh type Polygon Reduction Algorithm。

代码示例

const modifier = new SimplifyModifier();
geometry = modifier.modify( geometry );

导入

SimplifyModifier 是一个插件，必须显式导入，参见 Installation#Addons。

import { SimplifyModifier } from 'three/addons/modifiers/SimplifyModifier.js';

构造函数

new SimplifyModifier()

方法

.modify( geometry : BufferGeometry, count : number ) : BufferGeometry

通过对给定几何体应用简化，返回一个新的、已修改的几何体版本。 请注意，结果几何体始终是非索引几何体。

geometry

要修改的几何体。
count

要移除的顶点数量。
返回值： 一个新的、已修改的几何体。
