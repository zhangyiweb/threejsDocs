XRPlanes

WebXR 平面检测模块的实用工具类。若 WebXR 检测到平面，在使用以下代码片段时会自动将其作为薄盒网格添加到场景中。

代码示例

const planes = new XRPlanes( renderer );
scene.add( planes );

导入

XRPlanes 是一个插件，必须显式导入，参见 Installation#Addons。

import { XRPlanes } from 'three/addons/webxr/XRPlanes.js';

构造函数

new XRPlanes( renderer : WebGLRenderer | WebGPURenderer )

构造一个新的 XR 平面容器。

renderer

渲染器。
