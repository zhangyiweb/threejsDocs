LineSegments2

在成对顶点之间绘制的一系列线段。

相较于 LineSegments，它提供了更多功能，例如任意线宽，以及将线宽 改为世界单位。Line2 扩展了此对象，形成折线而非独立线段。

此模块只能与 WebGLRenderer 一起使用。使用 WebGPURenderer 时， 请从 lines/webgpu/LineSegments2.js 导入该类。

代码示例

const geometry = new LineSegmentsGeometry();
geometry.setPositions( positions );
geometry.setColors( colors );
const material = new LineMaterial( { linewidth: 5, vertexColors: true } );
const lineSegments = new LineSegments2( geometry, material );
scene.add( lineSegments );

导入

LineSegments2 是一个插件，必须显式导入，参见 Installation#Addons。

import { LineSegments2 } from 'three/addons/lines/LineSegments2.js';

构造函数

new LineSegments2( geometry : LineSegmentsGeometry, material : LineMaterial )

构造一条新的宽线。

geometry

线几何体。
material

线材质。

属性

.isLineSegments2 : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.computeLineDistances() : LineSegments2

计算渲染虚线所需的距离值数组。 对于几何体中的每个顶点，该方法会计算从当前点 到线起点的累积长度。

返回值： 对此实例的引用。

.raycast( raycaster : Raycaster, intersects : Array.<Object> )

计算投射光线与此实例之间的交点。

raycaster

射线投射器。
intersects

用于保存交点的目标数组。
重写： Mesh#raycast
