NodeLoader

用于以 three.js JSON Object/Scene 格式加载节点对象的加载器。

构造函数

new NodeLoader( manager : LoadingManager )

构造一个新的节点加载器。

manager

加载管理器的引用。

属性

.nodes : Object.<string, Node.constructor>

表示节点类型的字典。

.textures : Object.<string, Texture>

表示纹理的字典。

方法

.createNodeFromType( type : string ) : Node

根据给定类型创建节点对象。

type

节点类型。
返回值： 创建的节点实例。

.load( url : string, onLoad : function, onProgress : function, onError : function )

从指定 URL 加载节点定义。

url

要加载的文件路径/URL。
onLoad

加载完成时调用。
onProgress

加载进行中时调用。
onError

加载过程中抛出错误时调用。
重写： Loader#load

.parse( json : Object ) : Node

从给定的 JSON 解析节点。

json

JSON 定义

type

节点类型。
uuid

节点 UUID。
nodes

节点依赖项。
meta

元数据。
重写： Loader#parse
返回值： 解析后的节点。

.parseNodes( json : Array.<Object> ) : Object.<string, Node>

解析已加载节点的节点依赖项。

json

JSON 定义
返回值： 包含节点依赖项的字典。

.setNodes( value : Object.<string, Node.constructor> ) : NodeLoader

定义节点类型的字典。

value

以 <classname,class> 形式定义的节点库。
返回值： 此加载器的引用。

.setTextures( value : Object.<string, Texture> ) : NodeLoader

定义纹理的字典。

value

以 <uuid,texture> 形式定义的纹理库。
返回值： 此加载器的引用。
