ParametricGeometry

此类可用于基于参数曲面生成几何体。

参考：Mesh Generation with Python

代码示例

const geometry = new THREE.ParametricGeometry( klein, 25, 25 );
const material = new THREE.MeshBasicMaterial( { color: 0x00ff00 } );
const klein = new THREE.Mesh( geometry, material );
scene.add( klein );

导入

ParametricGeometry 是一个插件，必须显式导入，参见 Installation#Addons。

import { ParametricGeometry } from 'three/addons/geometries/ParametricGeometry.js';

构造函数

new ParametricGeometry( func : ParametricGeometry~Func, slices : number, stacks : number )

构造一个新的参数化几何体。

func

参数函数。默认为生成弯曲平面的函数。
slices

参数函数使用的切片数量。

默认值为 8。
stacks

参数函数使用的堆叠数量。

默认值为 8。

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

类型定义

.Func( u : number, v : number, target : Vector3 )

ParametricGeometry 的参数函数定义。

u

曲面上范围为 [0,1] 的 u 坐标。
v

曲面上范围为 [0,1] 的 v 坐标。
target

用于存储方法结果的目标向量。
