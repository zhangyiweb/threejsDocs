TextureLoader

用于加载纹理的类。图像在内部通过 ImageLoader 加载。

请注意，TextureLoader 在 r84 中已取消对进度 事件的支持。若需要支持进度事件的 TextureLoader，请参见 此讨论。

代码示例

const loader = new THREE.TextureLoader();
const texture = await loader.loadAsync( 'textures/land_ocean_ice_cloud_2048.jpg' );
const material = new THREE.MeshBasicMaterial( { map:texture } );

构造函数

new TextureLoader( manager : LoadingManager )

构造一个新的纹理加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback ) : Texture

从指定 URL 开始加载，并将完全加载的纹理 传递给 onLoad() 回调。该方法还会返回一个新的纹理对象，可直接 用于创建材质。如果采用这种方式，纹理可能会在 相应加载过程完成时突然出现在场景中。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

此加载器不支持。
onError

发生错误时执行。
重写： Loader#load
返回值： 纹理。
