AnaglyphEffect

使用物理正确的离轴立体投影创建红蓝立体（anaglyph）效果的类。

该实现使用 CameraUtils.frameCorners() 将立体相机视锥体对齐到虚拟屏幕平面， 在平面距离处提供准确的深度感知和零视差。

注意：此类仅可与 WebGLRenderer 配合使用。 使用 WebGPURenderer 时，请使用 AnaglyphPassNode。

导入

AnaglyphEffect 属于附加组件，需要显式导入，请参阅 Installation#Addons。

import { AnaglyphEffect } from 'three/addons/effects/AnaglyphEffect.js';

构造函数

new AnaglyphEffect( renderer : WebGLRenderer, width : number, height : number )

构造一个新的红蓝立体效果。

renderer

渲染器。
width

效果的宽度（物理像素）。

默认值为 512。
height

效果的高度（物理像素）。

默认值为 512。

属性

.eyeSep : number

瞳距（双眼间距），单位为世界坐标。人类典型瞳距为 0.064 米（64 毫米）。

默认值为 0.064。

.planeDistance : number

从观察者到虚拟屏幕平面的距离（世界坐标单位），在此距离处视差为零（屏幕深度）。 处于此距离的物体看起来位于屏幕表面。 更近的物体看起来在屏幕前方（负视差）。 更远的物体看起来在屏幕后方（正视差）。

屏幕尺寸根据相机在此距离处的 FOV 和宽高比计算， 确保立体视图与相机的视野匹配。

默认值为 0.5。

方法

.dispose()

释放内部资源。当不再需要该效果时应调用此方法。

.render( scene : Object3D, camera : Camera )

使用此效果时，应调用此方法而非默认的 WebGLRenderer#render。

scene

要渲染的场景。
camera

相机。

.setSize( width : number, height : number )

调整效果尺寸。

width

效果的宽度（逻辑像素）。
height

效果的高度（逻辑像素）。
