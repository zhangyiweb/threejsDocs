SSGINode

用于对场景应用屏幕空间全局光照（SSGI）的后处理节点。

参考：

  • https://github.com/cdrinmatane/SSRT3。
  • https://cdrinmatane.github.io/posts/ssaovb-code/。
  • https://cdrinmatane.github.io/cgspotlight-slides/ssilvb_slides.pdf。

效果的质量和性能主要取决于 sliceCount 和 stepCount。 每个像素采样的总数为 sliceCount * stepCount * 2。以下是根据是否使用时序滤波 推荐的一些预设。

使用时序滤波（推荐）：

  • 低：sliceCount 为 1，stepCount 为 12。
  • 中：sliceCount 为 2，stepCount 为 8。
  • 高：sliceCount 为 3，stepCount 为 16。

若出现闪烁等时序不稳定现象，可适当提高 slice 数量，并相应减少采样数以缓解性能损失。

不使用时序滤波：

  • 低：sliceCount 为 2，stepCount 为 6。
  • 中：sliceCount 为 3，stepCount 为 8。
  • 高：sliceCount 为 4，stepCount 为 12。

导入

SSGINode 是一个插件，必须显式导入，参见 Installation#Addons。

import { ssgi } from 'three/addons/tsl/display/SSGINode.js';

构造函数

new SSGINode( beautyNode : TextureNode, depthNode : TextureNode, normalNode : TextureNode, camera : PerspectiveCamera )

构造一个新的 SSGI 节点。

beautyNode

表示 beauty 或场景通道的纹理节点。
depthNode

表示场景深度的纹理节点。
normalNode

表示场景法线的纹理节点。
camera

用于渲染场景的相机。

属性

.aoIntensity : UniformNode.<float>

应用于 AO 的幂函数，用于使其看起来更暗或更亮。应在 [0, 4] 范围内。

默认值为 1。

.backfaceLighting : UniformNode.<float>

背面表面发射的光量。 应在 [0, 1] 范围内。

默认值为 0。

.beautyNode : TextureNode

表示 beauty 或场景通道的纹理节点。

.depthNode : TextureNode

表示场景深度的节点。

.expFactor : UniformNode.<float>

控制采样分布。在每个步进上应用的指数，使步长随距离递增。 应在 [1, 3] 范围内。

默认值为 2。

.giIntensity : UniformNode.<float>

间接漫反射光的强度。应在 [0, 100] 范围内。

默认值为 10。

.normalNode : TextureNode

表示场景法线的节点。若未向构造函数传入法线（因 MRT 不可用）， 可在着色器中从深度值自动重建法线。

.radius : UniformNode.<float>

世界空间中的有效采样半径。AO 和 GI 仅在该半径内产生影响。 应在 [1, 25] 范围内。

默认值为 12。

.sliceCount : UniformNode.<uint>

每个像素的半球 slice 数量。这会显著影响性能，应尽可能保持较低。 应在 [1, 4] 范围内。

默认值为 1。

.stepCount : UniformNode.<uint>

沿给定半球 slice 一侧采样的数量。这会显著影响性能，应尽可能保持较低。应在 [1, 32] 范围内。

默认值为 12。

.thickness : UniformNode.<float>

屏幕上物体在世界单位下的恒定厚度值。允许光线穿过超过该厚度值的表面后方。 应在 [0.01, 10] 范围内。

默认值为 1。

.updateBeforeType : string

updateBeforeType 被设置为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

.useLinearThickness : UniformNode.<bool>

是否随距离线性增加厚度（避免远距离丢失细节）。

默认值为 false。

.useScreenSpaceSampling : UniformNode.<bool>

使采样距离在屏幕空间而非世界空间（有助于近距离获得更多细节）。

默认值为 true。

.useTemporalFiltering : boolean

是否使用时序滤波。将此属性设为 true 需要使用 TRAANode。这有助于减少噪点， 但也会引入典型的 TAA 伪影，如拖影和时序不稳定。

若将此属性设为 false，则需要通过 DenoiseNode 手动降噪。

默认值为 true。

方法

.dispose()

释放内部资源。当效果不再使用时应调用此方法。

重写： TempNode#dispose

.getAONode() : PassTextureNode

以纹理节点形式返回效果的 AO 结果。

返回值： 表示效果 AO 结果的纹理节点。

.getGINode() : PassTextureNode

以纹理节点形式返回效果的 GI 结果。

返回值： 表示效果 GI 结果的纹理节点。

.setSize( width : number, height : number )

设置效果的尺寸。

width

效果的宽度。
height

效果的高度。

.setup( builder : NodeBuilder ) : PassTextureNode

此方法用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

此方法用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
