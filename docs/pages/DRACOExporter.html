DRACOExporter

使用 Draco 库压缩几何体的导出器。

Draco 是一个用于压缩与解压 3D 网格和点云的开源库。压缩后的几何体体积可显著减小，但客户端设备需要额外的解码时间。

独立的 Draco 文件扩展名为 .drc，包含顶点位置、法线、颜色及其他属性。Draco 文件不包含材质、纹理、动画或节点层级——若要使用这些功能，请将 Draco 几何体嵌入 glTF 文件。可使用 glTF-Pipeline 将普通 glTF 文件转换为 Draco 压缩的 glTF 文件。

该导出器需要事先将 Draco 编码器作为全局脚本加载：

<script src="https://cdn.jsdelivr.net/gh/google/draco@1.5.7/javascript/draco_encoder.js"></script>

const exporter = new DRACOExporter();
const data = await exporter.parseAsync( mesh, options );

导入

DRACOExporter 是一个插件，必须显式导入，参见 Installation#Addons。

import { DRACOExporter } from 'three/addons/exporters/DRACOExporter.js';

构造函数

new DRACOExporter()

属性

.MESH_EDGEBREAKER_ENCODING : number (constant)

Edgebreaker 编码。

默认值为 1。

.MESH_SEQUENTIAL_ENCODING : number (constant)

顺序编码。

默认值为 0。

方法

.parse()

已弃用： 请改用 DRACOExporter#parseAsync。

.parseAsync( object : Mesh | Points, options : DRACOExporter~Options ) : Promise.<Int8Array> (async)

解析给定的网格或点云并生成 Draco 输出。

object

要导出的网格或点云。
options

导出选项。
返回值： 一个 Promise，解析后返回导出的 Draco 数据。

类型定义

.Options

DRACOExporter 的导出选项。

decodeSpeed
number

指示如何针对解码速度调整编码器（0 表示更快的解码速度，但质量更差）。

默认值为 5。
encodeSpeed
number

指示如何调整编码器参数（0 表示更快的编码速度，但质量更差）。

默认值为 5。
encoderMethod
number

顺序编码（压缩率很低）或 Edgebreaker。Edgebreaker 以确定性的螺旋方式遍历网格三角形，这是该数据格式的主要优势来源。

默认值为 1。
quantization
Array.<number>

指示 Draco 文件中各类数据的精度，顺序为（POSITION、NORMAL、COLOR、TEX_COORD、GENERIC）。

默认值为 [ 16, 8, 8, 8, 8 ]。
exportUvs
boolean

是否导出 UV。

默认值为 true。
exportNormals
boolean

是否导出法线。

默认值为 true。
exportColor
boolean

是否导出颜色。

默认值为 false。
