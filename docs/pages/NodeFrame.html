NodeFrame

用于更新节点的管理类。该模块跟踪诸如 已经过时间、增量时间、渲染 ID 和帧 ID 等度量， 以便根据节点的配置正确地调用节点更新方法 Node#updateBefore、Node#update 和 Node#updateAfter。

构造函数

new NodeFrame()

构造一个新的节点帧。

属性

.camera : Camera

对当前相机的引用。

默认值为 null。

.deltaTime : number

增量时间（以秒为单位）。

默认值为 0。

.frameId : number

帧 ID。

默认值为 0。

.material : Material

对当前材质的引用。

默认值为 null。

.object : Object3D

对当前 3D 对象的引用。

默认值为 null。

.renderId : number

渲染 ID。

默认值为 0。

.renderer : Renderer

对当前渲染器的引用。

默认值为 null。

.scene : Scene

对当前场景的引用。

默认值为 null。

.time : number

已经过时间（以秒为单位）。

默认值为 0。

.updateAfterMap : WeakMap.<Node, Object>

用于控制 Node#updateAfter 的调用。

.updateBeforeMap : WeakMap.<Node, Object>

用于控制 Node#updateBefore 的调用。

.updateMap : WeakMap.<Node, Object>

用于控制 Node#update 的调用。

方法

.update()

更新节点帧的内部状态。此方法由渲染器 在其内部动画循环中调用。

.updateAfterNode( node : Node )

此方法对给定节点执行 Node#updateAfter。 它确保遵守 Node#updateAfterType，即更新 仅根据更新类型在每帧、每次渲染或每个对象上执行一次。

node

应被更新的节点。

.updateBeforeNode( node : Node )

此方法对给定节点执行 Node#updateBefore。 它确保遵守 Node#updateBeforeType，即更新 仅根据更新类型在每帧、每次渲染或每个对象上执行一次。

node

应被更新的节点。

.updateNode( node : Node )

此方法对给定节点执行 Node#update。 它确保遵守 Node#updateType，即更新 仅根据更新类型在每帧、每次渲染或每个对象上执行一次。

node

应被更新的节点。
