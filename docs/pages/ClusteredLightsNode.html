ClusteredLightsNode

LightsNode 的自定义版本，实现 Forward+ 集群着色： 视锥体被细分为 3D 集群网格（X × Y 屏幕图块 乘以指数间隔的 Z 深度切片），每个集群仅包含 与其相交的点光源球体。在着色时，每个片段 查找其所属集群并仅遍历该集群的光源。与 2D 平铺 光照不同，集群着色会剔除共享屏幕像素但位于 不同深度的光源——适用于具有真实深度复杂度的 3D 场景。

导入

ClusteredLightsNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { clusteredLights } from 'three/addons/tsl/lighting/ClusteredLightsNode.js';

构造函数

new ClusteredLightsNode( maxLights : number, tileSize : number, zSlices : number, maxLightsPerCluster : number )

构造一个新的集群光源节点。

maxLights

点光源的最大数量。

默认值为 1024。
tileSize

屏幕图块大小（像素），即集群 XY 尺寸。

默认值为 32。
zSlices

指数深度切片数量。

默认值为 24。
maxLightsPerCluster

每个集群的光源列表容量。

默认值为 64。
