TrackballControls

此类与 OrbitControls 类似。但它不会保持恒定的相机 up 向量。这意味着当相机绕过「北极」和「南极」时，不会翻转以保持「正立」。

导入

TrackballControls 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { TrackballControls } from 'three/addons/controls/TrackballControls.js';

构造函数

new TrackballControls( object : Object3D, domElement : HTMLElement )

构造一个新的控制器实例。

object

由控制器管理的对象。
domElement

用于事件监听的 HTML 元素。

默认值为 null。

属性

.dynamicDampingFactor : number

定义阻尼强度。仅在 staticMoving 设为 false 时生效。

默认值为 0.2。

.keys : Array.<string>

此数组保存用于控制交互的键码。

  • 按下第一个定义的键时，所有鼠标交互（左、中、右）执行旋转。
  • 按下第二个定义的键时，所有鼠标交互（左、中、右）执行缩放。
  • 按下第三个定义的键时，所有鼠标交互（左、中、右）执行平移。

默认值为 KeyA, KeyS, KeyD，分别对应 A、S、D。

重写： Controls#keys

.maxDistance : number

你能向外推拉多远（仅透视相机）。

默认值为 Infinity。

.maxZoom : number

你能缩小多远（仅正交相机）。

默认值为 Infinity。

.minDistance : number

你能向内推拉多近（仅透视相机）。

默认值为 0。

.minZoom : number

你能放大多近（仅正交相机）。

默认值为 0。

.mouseButtons : Object

此对象包含控制器所用鼠标操作的引用。

controls.mouseButtons = {
	LEFT: THREE.MOUSE.ROTATE,
	MIDDLE: THREE.MOUSE.DOLLY,
	RIGHT: THREE.MOUSE.PAN
}
重写： Controls#mouseButtons

.noPan : boolean

是否禁用平移。

默认值为 false。

.noRotate : boolean

是否禁用旋转。

默认值为 false。

.noZoom : boolean

是否禁用缩放。

默认值为 false。

.panSpeed : number

平移速度。

默认值为 0.3。

.rotateSpeed : number

旋转速度。

默认值为 1。

.screen : Object (readonly)

表示屏幕属性。在调用 handleResize() 时自动设置。

.staticMoving : boolean

是否禁用阻尼。

默认值为 false。

.target : Vector3

控制器的焦点。

.zoomSpeed : number

缩放速度。

默认值为 1.2。

方法

.handleResize()

应用程序窗口调整大小时必须调用。

.reset()

将控制器重置为初始状态。

事件

.change

当相机被控制器变换时触发。

类型：
  • Object

.end

当交互结束时触发。

类型：
  • Object

.start

当交互开始时触发。

类型：
  • Object
