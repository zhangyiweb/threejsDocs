InterleavedBuffer

「交错」（Interleaved）表示多个属性（可能类型不同，例如 position、normal、uv、color）被打包到单个 ArrayBuffer 中。

关于交错数组的介绍可参阅：Interleaved array basics

构造函数

new InterleavedBuffer( array : TypedArray, stride : number )

构造一个新的交错缓冲。

array

用于存储属性数据的共享缓冲类型化数组。
stride

每个顶点的类型化数组元素数量。

属性

.array : TypedArray

用于存储属性数据的共享缓冲类型化数组。

.count : number (readonly)

数组中的元素总数。

.isInterleavedBuffer : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.needsUpdate : number

指示该属性已更改并应重新发送到 GPU 的标志。当你修改数组的值时，将此属性设置为 true。

默认值为 false。

.stride : number

每个顶点的类型化数组元素数量。

.updateRanges : Array.<Object>

可用于仅更新所存储向量的部分分量（例如，仅与颜色相关的分量）。使用 addUpdateRange() 函数向该数组添加更新范围。

.usage : StaticDrawUsage | DynamicDrawUsage | StreamDrawUsage | StaticReadUsage | DynamicReadUsage | StreamReadUsage | StaticCopyUsage | DynamicCopyUsage | StreamCopyUsage

定义数据存储的预期使用模式，以进行优化。

注意：缓冲在首次使用后，其 usage 无法更改。应改为实例化一个新的缓冲，并在下次渲染前设置所需的 usage。

默认值为 StaticDrawUsage。

.uuid : string (readonly)

该交错缓冲的 UUID。

.version : number

版本号，每次将 needsUpdate 设置为 true 时递增。

方法

.addUpdateRange( start : number, count : number )

添加数据数组中需要在 GPU 上更新的数据范围。

start

开始更新的位置。
count

要更新的分量数量。

.clearUpdateRanges()

清除更新范围。

.clone( data : Object ) : InterleavedBuffer

返回一个从当前实例复制值的新交错缓冲。

data

包含共享 ArrayBuffer 的对象，用于保留共享结构。
返回值： 当前实例的克隆。

.copy( source : InterleavedBuffer ) : InterleavedBuffer

将给定交错缓冲的值复制到当前实例。

source

要复制的交错缓冲。
返回值： 对该实例的引用。

.copyAt( index1 : number, interleavedBuffer : InterleavedBuffer, index2 : number ) : InterleavedBuffer

从给定交错缓冲复制一个向量到当前缓冲。属性缓冲中的起始位置和目标位置由给定索引表示。

index1

当前交错缓冲中的目标索引。
interleavedBuffer

要复制的源交错缓冲。
index2

给定交错缓冲中的源索引。
返回值： 对该实例的引用。

.onUpload( callback : function ) : InterleavedBuffer

设置给定回调函数，该函数在 Renderer 将数组数据传输到 GPU 后执行。可在上传完成后、CPU 端不再需要数据时执行清理操作。

callback

onUpload() 回调。
返回值： 对该实例的引用。

.onUploadCallback()

在渲染器将属性数组数据传输到 GPU 后执行的回调函数。

.set( value : TypedArray | Array, offset : number ) : InterleavedBuffer

在交错缓冲中设置给定数组数据。

value

要设置的数组数据。
offset

当前交错缓冲数组中的偏移量。

默认值为 0。
返回值： 对该实例的引用。

.setUsage( value : StaticDrawUsage | DynamicDrawUsage | StreamDrawUsage | StaticReadUsage | DynamicReadUsage | StreamReadUsage | StaticCopyUsage | DynamicCopyUsage | StreamCopyUsage ) : InterleavedBuffer

设置当前交错缓冲的 usage。

value

要设置的 usage。
返回值： 对该交错缓冲的引用。

.toJSON( data : Object ) : Object

将交错缓冲序列化为 JSON。

data

可选值，用于保存有关序列化的元信息。
返回值： 表示已序列化交错缓冲的 JSON 对象。
