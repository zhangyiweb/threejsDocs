LightProbeGridHelper

通过在每个探针位置渲染一个球体来可视化 LightProbeGrid， 球体使用探针的 L2 球谐函数进行着色。

对所有探针使用单次 InstancedMesh 绘制调用。

代码示例

const helper = new LightProbeGridHelper( probes );
scene.add( helper );

导入

LightProbeGridHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { LightProbeGridHelper } from 'three/addons/helpers/LightProbeGridHelper.js';

构造函数

new LightProbeGridHelper( probes : LightProbeGrid, sphereSize : number )

构造一个新的辐照度探针网格辅助对象。

probes

要可视化的探针网格。
sphereSize

每个探针球体的半径。

默认值为 0.12。

属性

.probes : LightProbeGrid

要可视化的探针网格。

方法

.dispose()

释放此实例分配的 GPU 相关资源。 当应用中不再使用此实例时，请调用此方法。

重写： InstancedMesh#dispose

.update()

根据当前探针网格重建实例矩阵和 UVW 属性。 在更改 probes 或重新烘焙后调用此方法。
