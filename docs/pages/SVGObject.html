SVGObject

可用于将 SVG 元素包装为 3D 对象。

导入

SVGObject 是一个插件，必须显式导入，参见 Installation#Addons。

import { SVGObject } from 'three/addons/renderers/SVGRenderer.js';

构造函数

new SVGObject( node : SVGElement )

构造一个新的 SVG 对象。

node

SVG 元素。

属性

.isSVGObject : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.node : SVGElement

此 SVG 元素。
