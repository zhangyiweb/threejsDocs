TDSLoader

基于 lib3ds 的 3DS 格式加载器。

加载带 UV 的几何体，以及支持纹理的基础材质属性。

代码示例

const loader = new TDSLoader();
loader.setResourcePath( 'models/3ds/portalgun/textures/' );
const object = await loader.loadAsync( 'models/3ds/portalgun/portalgun.3ds' );
scene.add( object );

导入

TDSLoader 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { TDSLoader } from 'three/addons/loaders/TDSLoader.js';

构造函数

new TDSLoader( manager : LoadingManager )

构造一个新的 3DS 加载器。

manager

加载管理器。

属性

.debug : boolean

是否应启用调试模式。

默认值为 false。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 3DS 资源传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( arraybuffer : ArrayBuffer, path : string ) : Group

解析给定的 3DS 数据并返回结果数据。

arraybuffer

原始 3DS 数据（ArrayBuffer 格式）。
path

资源路径。
重写： Loader#parse
返回值： 以组（Group）形式表示的已解析资源。
