Box2

表示二维空间中的轴对齐包围盒（AABB）。

构造函数

new Box2( min : Vector2, max : Vector2 )

构造一个新的包围盒。

min

表示盒子下边界的向量。

默认值为 (Infinity,Infinity)。
max

表示盒子上边界的向量。

默认值为 (-Infinity,-Infinity)。

属性

.isBox2 : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.max : Vector2

盒子的上边界。

.min : Vector2

盒子的下边界。

方法

.clampPoint( point : Vector2, target : Vector2 ) : Vector2

将给定点钳制在此盒子的边界内。

point

要钳制的点。
target

用于存储方法结果的目标向量。
返回值： 钳制后的点。

.clone() : Box2

返回一个新盒子，其值从本实例复制而来。

返回值： 此实例的克隆。

.containsBox( box : Box2 ) : boolean

若此包围盒完全包含给定包围盒，则返回 true。 若此盒子与给定盒子相同，该函数也返回 true。

box

要测试的包围盒。
返回值： 包围盒是否包含给定包围盒。

.containsPoint( point : Vector2 ) : boolean

若给定点位于此盒子内部或边界上，则返回 true。

point

要测试的点。
返回值： 包围盒是否包含给定点。

.copy( box : Box2 ) : Box2

将给定盒子的值复制到本实例。

box

要复制的盒子。
返回值： 对此包围盒的引用。

.distanceToPoint( point : Vector2 ) : number

返回从此盒子任意边到指定点的欧几里得距离。若 给定点位于此盒子内部，则距离为 0。

point

要计算距离的点。
返回值： 欧几里得距离。

.equals( box : Box2 ) : boolean

若此包围盒与给定包围盒相等，则返回 true。

box

要测试相等性的盒子。
返回值： 此包围盒是否与给定包围盒相等。

.expandByPoint( point : Vector2 ) : Box2

扩展此盒子的边界以包含给定点。

point

应由包围盒包含的点。
返回值： 对此包围盒的引用。

.expandByScalar( scalar : number ) : Box2

按给定标量扩展盒子的每个维度。若为负数，则 盒子的维度会被收缩。

scalar

应用于扩展包围盒的标量值。
返回值： 对此包围盒的引用。

.expandByVector( vector : Vector2 ) : Box2

按给定向量对此盒子进行等量扩展。此盒子的宽度 会在两个方向上按向量的 x 分量扩展。此盒子的高度 会在两个方向上按向量的 y 分量扩展。

vector

应用于扩展包围盒的向量。
返回值： 对此包围盒的引用。

.getCenter( target : Vector2 ) : Vector2

返回此盒子的中心点。

target

用于存储方法结果的目标向量。
返回值： 中心点。

.getParameter( point : Vector2, target : Vector2 ) : Vector2

返回一个点，表示该点相对此盒子宽度与高度的比例。

point

二维空间中的一个点。
target

用于存储方法结果的目标向量。
返回值： 相对此盒子宽度与高度的比例点。

.getSize( target : Vector2 ) : Vector2

返回此盒子的尺寸。

target

用于存储方法结果的目标向量。
返回值： 尺寸。

.intersect( box : Box2 ) : Box2

计算此包围盒与给定包围盒的交集，将此盒子的上边界 设为两个盒子上边界中较小者，将此盒子的下边界 设为两个盒子下边界中较大者。若 没有重叠，则使此盒子为空。

box

要与之求交的包围盒。
返回值： 对此包围盒的引用。

.intersectsBox( box : Box2 ) : boolean

若给定包围盒与此包围盒相交，则返回 true。

box

要测试的包围盒。
返回值： 给定包围盒是否与此包围盒相交。

.isEmpty() : boolean

若此盒子的边界内不包含任何点，则返回 true。 注意：下边界与上边界相等的盒子仍包含一个点， 即两个边界共享的那个点。

返回值： 此盒子是否为空。

.makeEmpty() : Box2

使此盒子为空，即在二维空间中不包围任何区域。

返回值： 对此包围盒的引用。

.set( min : Vector2, max : Vector2 ) : Box2

设置此盒子的下边界与上边界。 请注意，此方法仅从给定对象复制数值。

min

盒子的下边界。
max

盒子的上边界。
返回值： 对此包围盒的引用。

.setFromCenterAndSize( center : Vector2, size : Vector2 ) : Box2

将此盒子的中心设为给定中心向量，并将此盒子的宽度、高度 设为给定尺寸值。

center

盒子的中心。
size

盒子的 x 与 y 尺寸。
返回值： 对此包围盒的引用。

.setFromPoints( points : Array.<Vector2> ) : Box2

设置此盒子的上、下边界，使其包围给定数组中的位置数据。

points

保存二维位置数据的数组，元素为 Vector2 实例。
返回值： 对此包围盒的引用。

.translate( offset : Vector2 ) : Box2

将给定偏移量加到此包围盒的上、下边界上， 从而在二维空间中平移该盒子。

offset

用于平移包围盒的偏移量。
返回值： 对此包围盒的引用。

.union( box : Box2 ) : Box2

计算此盒子与给定盒子的并集，将此盒子的上边界 设为两个盒子上边界中较大者，将此盒子的下边界 设为两个盒子下边界中较小者。

box

将与本实例求并集的包围盒。
返回值： 对此包围盒的引用。
