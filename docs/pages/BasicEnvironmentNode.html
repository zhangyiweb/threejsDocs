BasicEnvironmentNode

表示基于图像的光照（IBL）的基础模型。环境通过等距柱状投影或立方体贴图格式的环境贴图定义。 BasicEnvironmentNode 面向非 PBR 材质，例如 MeshBasicNodeMaterial 或 MeshPhongNodeMaterial。

构造函数

new BasicEnvironmentNode( envNode : Node )

构造一个新的基础环境节点。

envNode

表示环境的节点。

默认值为 null。

属性

.envNode : Node

表示环境的节点。

默认值为 null。
