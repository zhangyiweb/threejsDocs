NodeCode

NodeBuilder 在构建节点的过程中会创建此类的实例。 它们表示用户定义的、原生着色器代码片段，将由构建器注入。 为此，节点代码字典维护在 NodeBuilder#codes 中。

构造函数

new NodeCode( name : string, type : string, code : string )

构造一个新的代码节点。

name

代码的名称。
type

节点类型。
code

原生着色器代码。

默认值为 ''。

属性

.code : string

原生着色器代码。

默认值为 ''。

.name : string

代码的名称。

.type : string

节点类型。
