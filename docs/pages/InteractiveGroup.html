InteractiveGroup

此类可用于将 3D 对象组织到一个交互组中。组本身可监听 Pointer、Mouse 或 XR 控制器事件，以检测对子孙 3D 对象的选择。若某个 3D 对象被选中，相应事件将派发给该对象。

代码示例

const group = new InteractiveGroup();
group.listenToPointerEvents( renderer, camera );
group.listenToXRControllerEvents( controller1 );
group.listenToXRControllerEvents( controller2 );
scene.add( group );
// now add objects that should be interactive
group.add( mesh1, mesh2, mesh3 );

导入

InteractiveGroup 是一个插件，必须显式导入，参见 Installation#Addons。

import { InteractiveGroup } from 'three/addons/interactive/InteractiveGroup.js';

构造函数

new InteractiveGroup()

属性

.camera : Camera

用于射线投射的相机。

默认值为 null。

.controllers : Array.<Group>

XR 控制器数组。

.element : HTMLElement

内部使用的 DOM 元素。

默认值为 null。

.raycaster : Raycaster

内部射线投射器。

方法

.disconnect()

将此交互组从 DOM 以及所有 XR 控制器断开连接。

.disconnectXrControllerEvents()

将此交互组从所有 XR 控制器断开连接。

.disconnectionPointerEvents()

将此交互组从所有 Pointer 和 Mouse 事件断开连接。

.listenToPointerEvents( renderer : WebGPURenderer | WebGLRenderer, camera : Camera )

调用此方法可确保交互组监听 Pointer 和 Mouse 事件。目标为给定渲染器的 domElement。相机用于内部射线投射，以便根据事件检测 3D 对象。

renderer

渲染器。
camera

相机。

.listenToXRControllerEvents( controller : Group )

调用此方法可确保交互组监听给定 XR 控制器的事件。

controller

XR 控制器。
