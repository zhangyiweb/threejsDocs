LineMaterial

用于绘制线框风格几何体的材质。

与 LineBasicMaterial 不同，它支持任意线宽，并允许使用世界单位 而非屏幕空间单位。此材质与 LineSegments2 和 Line2 一起使用。

此模块只能与 WebGLRenderer 一起使用。使用 WebGPURenderer 时， 请使用 Line2NodeMaterial。

导入

LineMaterial 是一个插件，必须显式导入，参见 Installation#Addons。

import { LineMaterial } from 'three/addons/lines/LineMaterial.js';

构造函数

new LineMaterial( parameters : Object )

构造一个新的线段几何体。

parameters

一个包含一个或多个属性的对象， 用于定义材质的外观。材质的任何属性 （包括从继承材质中获得的任何属性）都可以在此传入。 颜色值可以传入 Color#set 所接受的任意类型的值。

属性

.alphaToCoverage : boolean

是否使用 alphaToCoverage。启用后，在使用 MSAA 时 可以改善线边缘的抗锯齿效果。

重写： ShaderMaterial#alphaToCoverage

.color : Color

材质的颜色。

默认值为 (1,1,1)。

.dashOffset : number

虚线周期中虚线的起始位置。

默认值为 0。

.dashScale : number

虚线和间隙的缩放比例。

默认值为 1。

.dashSize : number

虚线段的大小。

默认值为 1。

.dashed : boolean

线条是虚线还是实线。

默认值为 false。

.gapSize : number

间隙的大小。

默认值为 0。

.isLineMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.linewidth : number

当 worldUnits 为 false（默认）时，以 CSS 像素单位控制线宽； 当 worldUnits 为 true 时，以世界单位控制线宽。

默认值为 1。

重写： ShaderMaterial#linewidth

.opacity : number

不透明度。

默认值为 1。

重写： ShaderMaterial#opacity

.resolution : Vector2

视口大小，以屏幕像素为单位。必须保持更新，才能使 屏幕空间渲染准确。LineSegments2.onBeforeRender 回调 会对可见对象执行该更新。

.worldUnits : boolean

材质的尺寸（宽度、虚线间隙）是否以世界单位表示。

默认值为 false。
