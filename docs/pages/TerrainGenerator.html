TerrainGenerator

将程序化山脉烘焙进单个 THREE.BufferGeometry， 并返回可直接加入场景的 THREE.Group。

高度场是导数阻尼的分形求和（Quilez 的假侵蚀）：每一 倍频在当前坡度已陡峭处被抑制，细节集中到风化的脊线上， 低频域扭曲则让这些脊线蜿蜒。再经过若干次热力（岩屑）侵蚀， 松弛超过休止角的坡度，把分形的针状尖峰沉降成真实山脊。

网格用交替的四边形对角线三角化（菱形图案），使粗糙网格 也能保持轮廓而不出现单向纹理感。表面在 TSL 中按海拔与坡度自着色—— 草地、森林、岩石、岩屑与积雪，并带细节法线与大气透视——因此无需材质或纹理。

烘焙后的高度网格通过 TerrainGenerator#sampleHeight 暴露， 以便散布的森林（或其他对象）能精确坐落在表面上。

代码示例

const terrain = new TerrainGenerator( { seed: 1 } );
scene.add( terrain.build() );

构造函数

new TerrainGenerator()
