OBJExporter

OBJ 导出器。

OBJExporter 无法将材质数据导出为 MTL 文件，因此仅支持几何数据。

代码示例

const exporter = new OBJExporter();
const data = exporter.parse( scene );

导入

OBJExporter 是一个插件，必须显式导入，参见 Installation#Addons。

import { OBJExporter } from 'three/addons/exporters/OBJExporter.js';

构造函数

new OBJExporter()

方法

.parse( object : Object3D ) : string

解析给定的 3D 对象并生成 OBJ 输出。

若该 3D 对象由多个子对象和几何体组成，它们会在文件中合并为单个网格。

object

要导出的 3D 对象。
返回值： 导出的 OBJ。
