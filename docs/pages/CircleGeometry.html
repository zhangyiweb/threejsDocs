CircleGeometry

欧几里得几何中的简单形状。由若干围绕中心点、向外延伸至给定半径的三角形分段构成。从起始角度和给定圆心角按逆时针方向构建。也可用于创建正多边形，此时分段数决定边数。

代码示例

const geometry = new THREE.CircleGeometry( 5, 32 );
const material = new THREE.MeshBasicMaterial( { color: 0xffff00 } );
const circle = new THREE.Mesh( geometry, material );
scene.add( circle )

构造函数

new CircleGeometry( radius : number, segments : number, thetaStart : number, thetaLength : number )

构造一个新的圆形几何体。

radius

圆的半径。

默认值为 1.
segments

分段数（三角形数量），最小值为 3。

默认值为 32.
thetaStart

第一段的起始角度（弧度）。

默认值为 0.
thetaLength

圆扇形的圆心角（常称为 theta），单位为弧度。默认值会生成完整的圆。

默认值为 Math.PI*2.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

静态方法

.fromJSON( data : Object ) : CircleGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
