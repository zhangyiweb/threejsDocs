EXRLoader

OpenEXR 纹理格式的加载器。

EXRLoader 目前支持未压缩、ZIP(S)、RLE、PIZ、B44/A 和 DWA/B 压缩。 支持以 UnsignedByte、HalfFloat 和 Float 类型读取数据纹理。

代码示例

const loader = new EXRLoader();
const texture = await loader.loadAsync( 'textures/memorial.exr' );

导入

EXRLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { EXRLoader } from 'three/addons/loaders/EXRLoader.js';

构造函数

new EXRLoader( manager : LoadingManager )

构造一个新的 EXR 加载器。

manager

加载管理器。

属性

.outputFormat : RGBAFormat | RGFormat | RedFormat

纹理输出格式。

默认值为 RGBAFormat。

.part : number

对于多部分 EXR 文件，要加载的部分索引。

默认值为 0。

.type : HalfFloatType | FloatType

纹理类型。

默认值为 HalfFloatType。

方法

.parse( buffer : ArrayBuffer ) : DataTextureLoader~TexData

解析给定的 EXR 纹理数据。

buffer

原始纹理数据。
重写： DataTextureLoader#parse
返回值： 表示已解析纹理数据的对象。

.setDataType( value : HalfFloatType | FloatType ) : EXRLoader

设置纹理类型。

value

要设置的纹理类型。
返回值： 对此加载器的引用。

.setOutputFormat( value : RGBAFormat | RGFormat | RedFormat ) : EXRLoader

设置纹理输出格式。默认为 RGBAFormat。

value

纹理输出格式。
返回值： 对此加载器的引用。

.setPart( value : number ) : EXRLoader

对于多部分 EXR 文件，设置要加载的部分。

value

要加载的部分索引。
返回值： 对此加载器的引用。
