ComputeNode

表示计算着色器节点。

构造函数

new ComputeNode( computeNode : Node, workgroupSize : Array.<number> )

构造一个新的计算节点。

computeNode

定义计算着色器逻辑的节点。
workgroupSize

定义计算着色器执行时工作组在 X、Y、Z 维度上尺寸的数组。

属性

.computeNode : Node

定义计算着色器逻辑的节点。

.count : number | Array.<number>

要执行的线程（调用）总数。若为数字，将用于自动生成针对 instanceIndex 的边界检查。

.countNode : UniformNode

保存用于边界检查的调度计数的 uniform 节点。 当 count 为数字时自动创建。

.dispatchSize : number | Array.<number>

工作组在 X、Y、Z 轴上的调度尺寸。 在未提供 count 时直接使用。

.isComputeNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.name : string

uniform 的名称或标签。

默认值为 ''。

重写： Node#name

.onInitFunction : function

计算节点完成初始化时执行的回调。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.OBJECT，因为默认情况下 ComputeNode#updateBefore 会对每个对象执行一次。

默认值为 'object'。

重写： Node#updateBeforeType

.version : number

节点的版本。

重写： Node#version

.workgroupSize : Array.<number>

定义计算着色器执行时工作组在 X、Y、Z 维度上尺寸的数组。

默认值为 [ 64 ]。

方法

.dispose()

为此节点触发 dispose 事件。

重写： Node#dispose

.label( name : string ) : ComputeNode

设置 ComputeNode#name 属性。

name

uniform 的名称。
已弃用： 是
返回值： 对此节点的引用。

.onInit( callback : function ) : ComputeNode

设置在初始化期间运行的回调。

callback

回调函数。
返回值： 对此节点的引用。

.setName( name : string ) : ComputeNode

设置 ComputeNode#name 属性。

name

uniform 的名称。
返回值： 对此节点的引用。

.updateBefore( frame : NodeFrame )

此方法执行该节点的计算。

frame

对当前节点帧的引用。
重写： Node#updateBefore
