WoodNodeMaterial

使用 TSL（Three.js Shading Language）的程序化木质材质。

用法示例：

// 使用预设（推荐用于常见木材类型） const material = WoodNodeMaterial.fromPreset('walnut', 'gloss');

// 使用自定义参数（用于高级定制） const material = new WoodNodeMaterial({ centerSize: 1.2, ringThickness: 1/40, darkGrainColor: new THREE.Color('#2a1a0a'), lightGrainColor: new THREE.Color('#8b4513'), clearcoat: 1, clearcoatRoughness: 0.3 });

// 混合预设与自定义覆盖 const walnutParams = GetWoodPreset('walnut', 'raw'); const material = new WoodNodeMaterial({ ...walnutParams, ringThickness: 1/50, // 覆盖特定参数 clearcoat: 1 // 添加表面处理 });

构造函数

new WoodNodeMaterial()

类

WoodNodeMaterial
