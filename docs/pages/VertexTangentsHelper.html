VertexTangentsHelper

可视化对象的顶点切线。

要求几何体中已将切线指定为缓冲区属性，或已使用 BufferGeometry#computeTangents 计算过切线。

代码示例

const helper = new VertexTangentsHelper( mesh, 1, 0xff0000 );
scene.add( helper );

导入

VertexTangentsHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { VertexTangentsHelper } from 'three/addons/helpers/VertexTangentsHelper.js';

构造函数

new VertexTangentsHelper( object : Object3D, size : number, color : number | Color | string )

构造一个新的顶点切线辅助对象。

object

要可视化其顶点切线的对象。
size

辅助对象的大小。

默认值为 1。
color

辅助对象的颜色。

默认值为 0xff0000。

属性

.matrixAutoUpdate : boolean

已重写并设为 false，因为对象的世界变换 已编码在辅助对象的几何体数据中。

默认值为 false。

重写： LineSegments#matrixAutoUpdate

.object : Object3D

要可视化其顶点切线的对象。

辅助对象的大小。

方法

.dispose()

释放此实例分配的 GPU 相关资源。 当应用中不再使用此实例时，请调用此方法。

.update()

根据对象的世界变换更新顶点切线预览。
