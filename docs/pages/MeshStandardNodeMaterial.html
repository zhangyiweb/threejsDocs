MeshStandardNodeMaterial

MeshStandardMaterial 的节点材质版本。

构造函数

new MeshStandardNodeMaterial( parameters : Object )

构造一个新的网格标准节点材质。

parameters

配置参数。

属性

.emissiveNode : Node.<vec3>

标准材质的自发光颜色默认由 emissive、 emissiveIntensity 和 emissiveMap 属性推断。此节点属性允许 覆盖默认值，改用节点定义自发光颜色。

若不想覆盖自发光颜色，而只想修改现有 值，请使用 materialEmissive。

默认值为 null。

.isMeshStandardNodeMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.lights : boolean

设为 true，因为标准材质会对光照做出响应。

默认值为 true。

重写： NodeMaterial#lights

.metalnessNode : Node.<float>

标准材质的金属度默认由 metalness 和 metalnessMap 属性推断。此节点属性允许 覆盖默认值，改用节点定义金属度。

若不想覆盖金属度，而只想修改现有 值，请使用 materialMetalness。

默认值为 null。

.roughnessNode : Node.<float>

标准材质的粗糙度默认由 roughness 和 roughnessMap 属性推断。此节点属性允许 覆盖默认值，改用节点定义粗糙度。

若不想覆盖粗糙度，而只想修改现有 值，请使用 materialRoughness。

默认值为 null。

方法

.setupEnvironment( builder : NodeBuilder ) : EnvironmentNode.<vec3>

已重写，因为此类材质使用 EnvironmentNode 实现基于 PBR（PMREM）的环境贴图。此外， 该方法还会遵循 Scene.environment。

builder

当前节点构建器。
重写： NodeMaterial#setupEnvironment
返回值： 环境节点。

.setupLightingModel() : PhysicalLightingModel

设置光照模型。

重写： NodeMaterial#setupLightingModel
返回值： 光照模型。

.setupSpecular()

设置与高光相关的节点变量。

.setupVariants( builder : NodeBuilder )

设置标准材质特有的节点变量。

builder

当前节点构建器。
重写： NodeMaterial#setupVariants
