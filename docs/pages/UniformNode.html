UniformNode

用于表示 uniform 的类。

构造函数

new UniformNode( value : any, nodeType : string )

构造一个新的 uniform 节点。

value

此节点的值。通常为 JS 基本类型或 three.js 对象（向量、矩阵、颜色、纹理）。
nodeType

节点类型。若未显式定义类型，节点会尝试从其值推导类型。

默认值为 null。

属性

.groupNode : UniformGroupNode

此 uniform 所属的 uniform 组。默认情况下，uniform 按对象管理，但它们也可能属于按帧或按渲染调用 更新的共享组。

.isUniformNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.name : string

uniform 的名称或标签。

默认值为 ''。

重写： InputNode#name

方法

.getGroup() : UniformGroupNode

返回 UniformNode#groupNode。

返回值： uniform 组。

.getUniformHash( builder : NodeBuilder ) : string

默认情况下，此方法返回 Node#getHash 的结果，但派生 类可能会用不同的实现重写此方法。

builder

当前节点构建器。
返回值： uniform 哈希值。

.label( name : string ) : UniformNode

设置 UniformNode#name 属性。

name

uniform 的名称。
已弃用： 是
返回值： 对此节点的引用。

.setGroup( group : UniformGroupNode ) : UniformNode

设置 UniformNode#groupNode 属性。

group

uniform 组。
返回值： 对此节点的引用。

.setName( name : string ) : UniformNode

设置 UniformNode#name 属性。

name

uniform 的名称。
返回值： 对此节点的引用。
