UniformArrayElementNode

表示对 uniform 数组节点的元素访问。

构造函数

new UniformArrayElementNode( uniformArrayNode : UniformArrayNode, indexNode : IndexNode )

构造一个新的缓冲区节点。

uniformArrayNode

要访问的 uniform 数组节点。
indexNode

定义所访问数组元素位置的索引数据。

属性

.isArrayBufferElementNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。
