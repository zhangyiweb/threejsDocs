Volume

此类用于处理 NRRDLoader 的输出。 它包含体数据及其相关信息。目前仅支持三维数据。

导入

Volume 是一个插件，必须显式导入，参见 Installation#Addons。

import { Volume } from 'three/addons/misc/Volume.js';

构造函数

new Volume( xLength : number, yLength : number, zLength : number, type : string, arrayBuffer : ArrayBuffer )

构造一个新的体数据对象。

xLength

体数据的宽度。
yLength

体数据的长度。
zLength

体数据的深度。
type

数据类型（uint8、uint16 等）。
arrayBuffer

包含体数据的缓冲区。

属性

.RASDimensions : Array.<number>

此数组保存体数据在 RAS 空间中的尺寸。

.axisOrder : Array.<string>

由 NRRD 头文件指定的轴顺序。

.data : TypedArray

体数据。

.inverseMatrix : Martrix3

RAS 到 IJK 的变换矩阵。

.lowerThreshold : number

值低于此阈值的体素将不会出现在切片中。 若更改此值，与此体数据关联的所有切片的 geometryNeedsUpdate 将自动设为 true。

.matrix : Martrix3

IJK 到 RAS 的变换矩阵。

.offset : Array.<number>

体数据在 RAS 坐标系中的偏移量。

.segmentation : boolean

是否使用分割模式。 该模式可以正确加载 16 位 nrrd 文件。

默认值为 false。

.sliceList : Array.<VolumeSlice>

与此体数据关联的所有切片列表。

.spacing : Array.<number>

从 IJK 到 RAS 坐标系应用于体数据的间距。

.upperThreshold : number

值高于此阈值的体素将不会出现在切片中。 若更改此值，与此体数据关联的所有切片的 geometryNeedsUpdate 将自动设为 true。

.xLength : number

体数据在 IJK 坐标系中的宽度。

默认值为 1。

.yLength : number

体数据在 IJK 坐标系中的高度。

默认值为 1。

.zLength : number

体数据在 IJK 坐标系中的深度。

默认值为 1。

方法

.access( i : number, j : number, k : number ) : number

计算 IJK 坐标系中给定坐标在数据数组中对应的索引。

i

第一个坐标。
j

第二个坐标。
k

第三个坐标。
返回值： 索引。

.computeMinMax() : Array.<number>

计算体数据中的最小值和最大值。

返回值： 最小/最大值，格式为 [min,max]。

.extractPerpendicularPlane( axis : 'x' | 'y' | 'z', RASIndex : number ) : Object

计算切片的朝向，并返回与几何体相关的全部信息，例如 sliceAccess、 平面矩阵（在 RAS 坐标系中的朝向与位置）以及平面在两种坐标系中的尺寸。

axis

切片的法线轴。
RASIndex

切片的索引。
返回值： 包含切片几何体全部有用信息的对象。

.extractSlice( axis : 'x' | 'y' | 'z', index : number ) : VolumeSlice

返回与给定轴和索引对应的切片。 坐标采用 Right Anterior Superior（RAS）格式。

axis

切片的法线轴。
index

切片的索引。
返回值： 提取的切片。

.getData( i : number, j : number, k : number ) : number

data[access(i,j,k)] 的快捷方式。

i

第一个坐标。
j

第二个坐标。
k

第三个坐标。
返回值： 数据数组中的值。

.map( functionToMap : function, context : Object ) : Volume

对所有体素应用函数，请注意，体素值将被替换。

functionToMap

应用于每个体素的函数，将以下列参数调用： 体素值、体素索引、数据（TypedArray）。
context

可指定调用函数时的上下文，默认为此 Volume 实例。
返回值： 对此实例的引用。

.repaintAllSlices() : Volume

对此体数据提取的所有切片调用 repaint。

参见：
  • VolumeSlice#repaint
返回值： 对此体数据实例的引用。

.reverseAccess( index : number ) : Array.<number>

根据数据中的给定索引，检索对应体素的 IJK 坐标。

index

体素索引。
返回值： IJK 坐标，格式为 [x,y,z]。
