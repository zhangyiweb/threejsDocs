SobelOperatorNode

用于通过 Sobel 滤波器检测边缘的后处理节点。 Sobel 滤波器应在色调映射和输出色彩空间转换之后应用。

导入

SobelOperatorNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { sobel } from 'three/addons/tsl/display/SobelOperatorNode.js';

构造函数

new SobelOperatorNode( textureNode : TextureNode )

构造一个新的 Sobel 算子节点。

textureNode

表示效果输入的纹理节点。

属性

.textureNode : TextureNode

表示效果输入的纹理节点。

.updateBeforeType : string

updateBeforeType 被设置为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧更新一次内部 uniform。

默认值为 'frame'。

重写： TempNode#updateBeforeType

方法

.setup( builder : NodeBuilder ) : ShaderCallNodeInternal

此方法用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

此方法用于每帧更新效果的 uniform。

frame

当前的节点帧。
重写： TempNode#updateBefore
