TextGeometry

用于将文本生成为单一几何体的类。通过提供文本字符串以及一组参数（包括已加载的字体和拉伸设置）来构造。

更多详情请参阅 FontLoader 页面。

TextGeometry 使用 typeface.json 生成的字体。部分现有字体位于 /examples/fonts。

代码示例

const loader = new FontLoader();
const font = await loader.loadAsync( 'fonts/helvetiker_regular.typeface.json' );
const geometry = new TextGeometry( 'Hello three.js!', {
	font: font,
	size: 80,
	depth: 5,
	curveSegments: 12
} );

导入

TextGeometry 是一个插件，必须显式导入，参见 Installation#Addons。

import { TextGeometry } from 'three/addons/geometries/TextGeometry.js';

构造函数

new TextGeometry( text : string, parameters : TextGeometry~Options )

构造一个新的文本几何体。

text

应转换为几何体的文本。
parameters

文本设置。

类型定义

.Options

表示该几何体构造函数的 options 类型。

font
Font

字体。
size
number

文字大小。

默认值为 100。
depth
number

拉伸形状的深度。

默认值为 50。
curveSegments
number

曲线上的点数。

默认值为 12。
steps
number

沿拉伸样条深度方向细分分段所用的点数。

默认值为 1。
bevelEnabled
boolean

是否对形状进行斜角处理。

默认值为 false。
bevelThickness
number

斜角切入原始形状的深度。

默认值为 10。
bevelSize
number

斜角从形状轮廓向外延伸的距离。

默认值为 8。
bevelOffset
number

斜角从形状轮廓开始的距离。

默认值为 0。
bevelSegments
number

斜角层数。

默认值为 3。
direction
string

字符方向：ltr（从左到右）、rtl（从右到左）和 tb（从上到下）。

默认值为 'ltr'。
extrudePath
Curve

形状应沿其拉伸的三维样条路径。路径拉伸不支持斜角。

默认值为 null。
UVGenerator
Object

提供 UV 生成函数以进行自定义 UV 生成的对象。
