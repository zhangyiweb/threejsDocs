RectAreaLightNode

将矩形区域光表示为节点的模块。

构造函数

new RectAreaLightNode( light : RectAreaLight )

构造一个新的矩形区域光节点。

light

矩形区域光源。

默认值为 null。

属性

.halfHeight : UniformNode.<vec3>

表示区域光半高的 uniform 节点。

.halfWidth : UniformNode.<vec3>

表示区域光半宽的 uniform 节点。

.updateType : string

updateType 设置为 NodeUpdateType.RENDER，因为该光源 依赖 viewMatrix，而它在每次渲染调用时可能不同。

默认值为 'render'。

重写： AnalyticLightNode#updateType

方法

.update( frame : NodeFrame )

重写此方法以更新矩形区域光专用的 uniform。

frame

对当前节点帧的引用。
重写： AnalyticLightNode#update

静态方法

.setLTC( ltc : RectAreaLightTexturesLib )

用于配置内部 BRDF 近似纹理数据。

ltc

BRDF 近似纹理数据。
