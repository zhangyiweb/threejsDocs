LightingContextNode

LightingContextNode 表示对 ContextNode 模块的扩展， 通过添加光照相关的上下文数据。它表示 LightsNode 的运行时上下文。

构造函数

new LightingContextNode( lightsNode : LightsNode, lightingModel : LightingModel, materialLightings : Array.<LightingNode>, backdropNode : Node.<vec3>, backdropAlphaNode : Node.<float> )

构造一个新的光照上下文节点。

lightsNode

灯光节点。
lightingModel

当前的光照模型。

默认值为 null。
materialLightings

材质光照节点。
backdropNode

背景节点。

默认值为 null。
backdropAlphaNode

背景透明度节点。

默认值为 null。

属性

.backdropAlphaNode : Node.<float>

背景透明度节点。

默认值为 null。

.backdropNode : Node.<vec3>

背景节点。

默认值为 null。

.lightingModel : LightingModel

当前的光照模型。

默认值为 null。

.materialLightings : Array.<LightingNode>

方法

.getContext() : Object

返回光照上下文对象。

返回值： 光照上下文对象。
