GLTFExporter

glTF 2.0 导出器。

glTF（GL Transmission Format）是一种开放格式规范， 用于高效传输和加载 3D 内容。资源可以以 JSON（.gltf）或二进制（.glb）格式提供。外部文件存储纹理（.jpg、.png）及附加二进制数据（.bin）。glTF 资源可包含一个或多个场景，包括网格、材质、纹理、蒙皮、骨骼、形变目标、动画、灯光和/或相机。

GLTFExporter 支持以下glTF 2.0 扩展：

  • KHR_lights_punctual
  • KHR_materials_clearcoat
  • KHR_materials_dispersion
  • KHR_materials_emissive_strength
  • KHR_materials_ior
  • KHR_materials_iridescence
  • KHR_materials_specular
  • KHR_materials_sheen
  • KHR_materials_transmission
  • KHR_materials_unlit
  • KHR_materials_volume
  • KHR_mesh_quantization
  • KHR_texture_transform
  • EXT_materials_bump
  • EXT_mesh_gpu_instancing
  • EXT_texture_webp

以下 glTF 2.0 扩展由外部用户插件提供支持：

  • KHR_materials_variants

代码示例

const exporter = new GLTFExporter();
const data = await exporter.parseAsync( scene, options );

导入

GLTFExporter 是一个插件，必须显式导入，参见 Installation#Addons。

import { GLTFExporter } from 'three/addons/exporters/GLTFExporter.js';

构造函数

new GLTFExporter()

构造一个新的 glTF 导出器。

属性

.textureUtils : WebGLTextureUtils | WebGPUTextureUtils

纹理工具模块的引用。

默认值为 null。

方法

.parse( input : Scene | Array.<Scene>, onDone : GLTFExporter~OnDone, onError : GLTFExporter~OnError, options : GLTFExporter~Options )

解析给定场景并生成 glTF 输出。

input

一个场景或场景数组。
onDone

导出完成时执行的回调函数。
onError

发生错误时执行的回调函数。
options

选项。

.parseAsync( input : Scene | Array.<Scene>, options : GLTFExporter~Options ) : Promise.<(ArrayBuffer|string)>

GLTFExporter#parse 的异步版本。

input

一个场景或场景数组。
options

选项。
返回值： 一个 Promise，解析后返回导出的 glTF 数据。

.register( callback : function ) : GLTFExporter

注册插件回调。此 API 在内部用于实现各种 glTF 扩展，第三方代码也可使用它为导出器添加额外逻辑。

callback

要注册的回调函数。
返回值： 对此导出器的引用。

.setTextureUtils( utils : WebGLTextureUtils | WebGPUTextureUtils ) : GLTFExporter

设置此导出器的纹理工具。仅在需要导出压缩纹理时相关。

根据使用的是 WebGLRenderer 还是 WebGPURenderer，必须注入对应的纹理工具 WebGLTextureUtils 或 WebGPUTextureUtils。

utils

纹理工具。
返回值： 对此导出器的引用。

.unregister( callback : function ) : GLTFExporter

注销插件回调。

callback

要注销的回调函数。
返回值： 对此导出器的引用。

类型定义

.OnDone( result : ArrayBuffer | string )

GLTFExporter 的 onDone 回调。

result

生成的 .gltf（JSON）或 .glb（二进制）数据。

.OnError( error : Error )

GLTFExporter 的 onError 回调。

error

错误对象。

.Options

GLTFExporter 的导出选项。

trs
boolean

为每个节点导出位置、旋转和缩放，而非矩阵。

默认值为 false。
onlyVisible
boolean

仅导出可见的 3D 对象。

默认值为 true。
binary
boolean

以二进制（.glb）格式导出，返回 ArrayBuffer。

默认值为 false。
maxTextureSize
number

将图像最大尺寸（宽度和高度）限制为给定值。

默认值为 Infinity。
animations
Array.<AnimationClip> | Array.<Array.<AnimationClip>>

要包含在导出中的动画列表。导出单个 3D 对象或场景时，这是一个扁平的剪辑列表。 导出多个场景数组时，必须是嵌套数组，每个场景对应一个剪辑列表，按索引与输入匹配。

默认值为 []。
includeCustomExtensions
boolean

导出对象 userData.gltfExtensions 属性上定义的自定义 glTF 扩展。

默认值为 false。
