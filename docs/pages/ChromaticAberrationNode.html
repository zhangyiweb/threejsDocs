ChromaticAberrationNode

用于应用色差效果的后处理节点。 该效果通过分离并偏移红、绿、蓝通道，模拟真实相机镜头中出现的色边现象。

导入

ChromaticAberrationNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { chromaticAberration } from 'three/addons/tsl/display/ChromaticAberrationNode.js';

构造函数

new ChromaticAberrationNode( textureNode : TextureNode, strengthNode : Node, centerNode : Node, scaleNode : Node )

构造一个新的色差节点。

textureNode

表示效果输入的纹理节点。
strengthNode

色差效果强度，以节点形式表示。
centerNode

效果的中心点，以节点形式表示。
scaleNode

从中心点进行阶梯式缩放的缩放因子，以节点形式表示。

属性

.centerNode : Node

保存效果中心点的节点。

.scaleNode : Node

保存阶梯式缩放因子的节点。

.strengthNode : Node

保存效果强度的节点。

.textureNode : texture

表示效果输入的纹理节点。

方法

.setup( builder : NodeBuilder ) : ShaderCallNodeInternal

用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup
