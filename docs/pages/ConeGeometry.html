ConeGeometry

用于表示圆锥体的几何体类。

代码示例

const geometry = new THREE.ConeGeometry( 5, 20, 32 );
const material = new THREE.MeshBasicMaterial( { color: 0xffff00 } );
const cone = new THREE.Mesh(geometry, material );
scene.add( cone );

构造函数

new ConeGeometry( radius : number, height : number, radialSegments : number, heightSegments : number, openEnded : boolean, thetaStart : number, thetaLength : number )

构造一个新的圆锥体几何体。

radius

圆锥底面的半径。

默认值为 1.
height

圆锥的高度。

默认值为 1.
radialSegments

沿圆锥圆周方向的分段面数量。

默认值为 32.
heightSegments

沿圆锥高度方向的面行数。

默认值为 1.
openEnded

圆锥底面是开放还是封闭。

默认值为 false.
thetaStart

第一段的起始角度（弧度）。

默认值为 0.
thetaLength

圆扇形的圆心角（常称为 theta），单位为弧度。默认值会生成完整的圆锥。

默认值为 Math.PI*2.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

重写： CylinderGeometry#parameters

静态方法

.fromJSON( data : Object ) : ConeGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
