PointsMaterial

用于渲染点图元的材质。

材质定义了可渲染 3D 对象的外观。

代码示例

const vertices = [];
for ( let i = 0; i < 10000; i ++ ) {
	const x = THREE.MathUtils.randFloatSpread( 2000 );
	const y = THREE.MathUtils.randFloatSpread( 2000 );
	const z = THREE.MathUtils.randFloatSpread( 2000 );
	vertices.push( x, y, z );
}
const geometry = new THREE.BufferGeometry();
geometry.setAttribute( 'position', new THREE.Float32BufferAttribute( vertices, 3 ) );
const material = new THREE.PointsMaterial( { color: 0x888888 } );
const points = new THREE.Points( geometry, material );
scene.add( points );

构造函数

new PointsMaterial( parameters : Object )

构造一个新的点材质。

parameters

包含一个或多个属性的对象， 用于定义材质外观。材质的任意属性 （包括从继承材质获得的任意属性）均可在此传入。 颜色值可以是 Color#set 所接受的任意类型。

属性

.alphaMap : Texture

Alpha 贴图是一张灰度纹理，用于控制表面各处的不透明度 （黑色：完全透明；白色：完全不透明）。

仅使用纹理的颜色，若存在 alpha 通道则忽略。 对于 RGB 和 RGBA 纹理，渲染器在采样时会使用绿色通道， 因为 DXT 压缩以及未压缩的 RGB 565 格式为绿色提供了额外的精度位。 仅亮度以及亮度/alpha 纹理也可按预期工作。

alphaMap 表示非颜色数据。所指定的任意纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.color : Color

材质的颜色。

默认值为 (1,1,1)。

.fog : boolean

材质是否受雾影响。

默认值为 true。

.isPointsMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.map : Texture

颜色贴图。可选择包含 alpha 通道，通常与 Material#transparent 或 Material#alphaTest 配合使用。贴图颜色 会受漫反射 color 调制。

map 表示颜色数据，纹理必须指定 Texture#colorSpace。多数 map 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。

.size : number

以像素为单位定义点的大小。

若值超过硬件相关参数（如 gl.ALIASED_POINT_SIZE_RANGE），可能会被限制。

默认值为 1。

.sizeAttenuation : boolean

指定单个点的大小是否会随相机深度衰减（仅透视相机）。

默认值为 true。
