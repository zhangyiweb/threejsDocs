PolarGridHelper

该辅助对象用于定义极坐标网格。网格是由线段组成的二维阵列。

代码示例

const radius = 10;
const sectors = 16;
const rings = 8;
const divisions = 64;
const helper = new THREE.PolarGridHelper( radius, sectors, rings, divisions );
scene.add( helper );

构造函数

new PolarGridHelper( radius : number, sectors : number, rings : number, divisions : number, color1 : number | Color | string, color2 : number | Color | string )

构造一个新的极坐标网格辅助对象。

radius

极坐标网格的半径。可为任意正数。

默认值为 10。
sectors

网格将被划分的扇区数量。可为任意正整数。

默认值为 16。
rings

环的数量。可为任意正整数。

默认值为 16。
divisions

每个圆所使用的线段分段数。可为任意正整数。

默认值为 64。
color1

用于网格元素的第一种颜色。

默认值为 0x444444。
color2

用于网格元素的第二种颜色。

默认值为 0x888888。

方法

.dispose()

释放该实例分配的 GPU 相关资源。当应用中不再使用该实例时，应调用此方法。
