Transpiler

用于将着色器代码从一种语言转译为另一种语言的类。

目前 Transpiler 仅支持将 GLSL 转换为 TSL。它旨在帮助开发者将自定义材质从当前材质系统迁移到新的基于节点的材质系统。

导入

Transpiler 是一个插件，必须显式导入，参见 Installation#Addons。

import Transpiler from 'three/addons/transpiler/Transpiler.js';

构造函数

new Transpiler( decoder : GLSLDecoder, encoder : TSLEncoder )

构造一个新的转译器。

decoder

GLSL 解码器。
encoder

TSL 编码器。

属性

.decoder : GLSLDecoder

GLSL 解码器。该组件解析 GLSL 并生成与语言无关的 AST，供后续处理使用。

.encoder : TSLEncoder

TSL 编码器。它接收 AST 并输出 TSL 代码。

.linker : Linker

链接器。它处理 AST 并解析变量与函数引用，确保所有依赖项正确链接。

方法

.parse( source : string ) : string

解析给定的 GLSL 源码并返回 TSL 语法。

source

GLSL 源码。
返回值： TSL 代码。
