ThreeMFLoader

3D Manufacturing Format (3MF) 格式加载器。

支持核心规范中的以下特性：

  • 3D Models
  • Object Resources (Meshes and Components)
  • Material Resources (Base Materials)

对 3MF Materials and Properties Extension 仅部分支持。

  • Texture 2D
  • Texture 2D Groups
  • Color Groups (Vertex Colors)
  • Metallic Display Properties (PBR)

代码示例

const loader = new ThreeMFLoader();
const object = await loader.loadAsync( './models/3mf/truck.3mf' );
object.rotation.set( - Math.PI / 2, 0, 0 ); // z-up conversion
scene.add( object );

导入

ThreeMFLoader 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { ThreeMFLoader } from 'three/addons/loaders/3MFLoader.js';

构造函数

new ThreeMFLoader( manager : LoadingManager )

构造一个新的 3MF 加载器。

manager

加载管理器。

属性

.availableExtensions : Array.<Object>

可用扩展的数组。

方法

.addExtension( extension : Object )

添加一个 3MF 扩展。

extension

要添加的扩展。

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 3MF 资源传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( data : ArrayBuffer ) : Group

解析给定的 3MF 数据并返回生成的组（Group）。

data

原始 3MF 资源数据（ArrayBuffer 格式）。
重写： Loader#parse
返回值： 表示已解析资源的组（Group）。
