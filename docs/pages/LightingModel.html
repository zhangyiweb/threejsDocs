LightingModel

用于实现光照模型的抽象类。该模块定义了 多个方法，具体的光照模型可以实现这些方法。这些 方法在光照求值过程的不同阶段执行。

构造函数

new LightingModel()

方法

.ambientOcclusion( builder : NodeBuilder ) (abstract)

此方法用于实现环境光遮蔽（ambient occlusion）项。 与其他方法不同，此方法必须由光照模型在其间接光照项中手动调用。

builder

当前的节点构建器。

.direct( lightData : Object, builder : NodeBuilder ) (abstract)

此方法用于实现直接光照项， 在方向光、点光源和聚光灯节点的构建过程中执行。

lightData

灯光数据。
builder

当前的节点构建器。

.directRectArea( lightData : Object, builder : NodeBuilder ) (abstract)

此方法用于为矩形区域光节点实现直接光照项。

lightData

灯光数据。
builder

当前的节点构建器。

.finish( builder : NodeBuilder ) (abstract)

此方法用于执行最终任务，例如对出射光进行最终更新。

builder

当前的节点构建器。

.indirect( builder : NodeBuilder ) (abstract)

此方法用于实现间接光照项。

builder

当前的节点构建器。

.start( builder : NodeBuilder ) (abstract)

此方法用于设置光照模型和上下文数据， 这些数据随后在求值过程中使用。

builder

当前的节点构建器。
