MeshNormalMaterial

将法线向量映射为 RGB 颜色的材质。

构造函数

new MeshNormalMaterial( parameters : Object )

构造一个新的 mesh normal 材质。

parameters

包含一个或多个定义材质外观属性的对象。材质的任何属性（包括继承材质的任何属性）都可在此传入。颜色值可以是 Color#set 接受的任意类型。

属性

.bumpMap : Texture

用于创建凹凸贴图的纹理。黑白值映射为相对于光照的感知深度。凹凸实际上不会改变物体的几何形状，只影响光照。若已定义法线贴图，则此贴图会被忽略。

bumpMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.bumpScale : number

凹凸贴图对材质的影响程度。典型范围为 [0,1]。

默认值为 1。

.displacementBias : number

位移贴图像素值在网格顶点上的偏移量。 该偏置会加到经缩放的位移贴图采样值上。 未设置位移贴图时，此值不会生效。

默认值为 0。

.displacementMap : Texture

位移贴图会影响网格顶点的位置。与仅影响材质明暗的其他贴图不同，被位移的顶点可以投射阴影、遮挡其他物体，并在其他方面表现得像真实几何体。位移纹理是一幅图像，其中每个像素的值（白色最高）会映射并重新定位网格顶点。为获得最佳效果，请将位移贴图与匹配的法线贴图配对使用，因为渲染器无法根据位移后的顶点重新计算表面法线。

默认值为 null。

.displacementScale : number

位移贴图对网格的影响程度（黑色表示无位移，白色表示最大位移）。未设置位移贴图时，此值不会生效。

默认值为 0。

.flatShading : boolean

材质是否以平面着色方式渲染。

默认值为 false。

.isMeshNormalMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.normalMap : Texture

用于创建法线贴图的纹理。RGB 值会影响每个像素片段的表面法线，并改变颜色的受光方式。法线贴图不会改变表面的实际形状，只影响光照。若材质的法线贴图按左手坐标系编写，则应将 normalScale 的 y 分量取反，以补偿手性差异。

normalMap 表示非颜色数据。所赋纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.normalMapType : TangentSpaceNormalMap | ObjectSpaceNormalMap

法线贴图的类型。

默认值为 TangentSpaceNormalMap。

.normalScale : Vector2

法线贴图对材质的影响程度。典型取值范围为 [0,1]。

默认值为 (1,1)。

.wireframe : boolean

将几何体渲染为线框。

默认值为 false。

.wireframeLinewidth : number

控制线框的粗细。

WebGL 和 WebGPU 会忽略此属性，始终渲染 1 像素宽的线条。

默认值为 1。
