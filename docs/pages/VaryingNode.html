VaryingNode

用于将着色器 varying 表示为节点的类。Varying 可由 现有节点创建，示例如下：

代码示例

const positionLocal = positionGeometry.toVarying( 'vPositionLocal' );

构造函数

new VaryingNode( node : Node, name : string )

构造一个新的 varying 节点。

node

应为其创建 varying 的节点。
name

着色器中的 varying 名称。

默认值为 null。

属性

.global : boolean

此标志用于全局缓存。

默认值为 true。

重写： Node#global

.interpolationSampling : string

varying 数据的插值采样类型。

默认值为 null。

.interpolationType : string

varying 数据的插值类型。

默认值为 null。

.isVaryingNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.name : string

着色器中的 varying 名称。若未定义名称， 节点系统会自动生成一个。

默认值为 null。

重写： Node#name

.node : Node

应为其创建 varying 的节点。

方法

.setInterpolation( type : string, sampling : string ) : VaryingNode

定义该 varying 的插值类型。

type

插值类型。
sampling

插值采样类型。

默认值为 null。
返回值： 对此节点的引用。

.setupVarying( builder : NodeBuilder ) : NodeVarying

此方法使用当前节点构建器执行 varying 节点的设置。

builder

当前节点构建器。
返回值： 来自节点构建器的节点 varying。
