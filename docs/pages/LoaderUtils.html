LoaderUtils

提供加载器工具函数的类。

构造函数

new LoaderUtils()

静态方法

.extractUrlBase( url : string ) : string

从给定 URL 中提取基础 URL。

url

要从中提取基础 URL 的 URL。
返回值： 提取出的基础 URL。

.resolveURL( url : string, path : string ) : string

相对于给定路径解析相对 URL。绝对路径、data URL 和 blob URL 将原样返回。无效 URL 将返回空字符串。

url

要解析的 URL。
path

相对 URL 要相对于其解析的基础路径。
返回值： 解析后的 URL。
