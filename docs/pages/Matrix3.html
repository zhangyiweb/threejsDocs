Matrix3

表示一个 3x3 矩阵。

关于行主序与列主序的说明：

构造函数和 Matrix3#set 方法按 行主序 接收参数，而内部则按列主序存储在 Matrix3#elements 数组中。 这意味着调用：

const m = new THREE.Matrix();
m.set( 11, 12, 13,
       21, 22, 23,
       31, 32, 33 );

会使 elements 数组包含：

m.elements = [ 11, 21, 31,
               12, 22, 32,
               13, 23, 33 ];

并且内部所有计算都使用列主序进行。 不过，由于实际排序在数学上并无差异，且大多数人习惯按行主序思考矩阵， three.js 文档以行主序展示矩阵。请记住：若阅读源码，需要将此处列出的矩阵取转置，才能对应理解计算过程。

构造函数

new Matrix3( n11 : number, n12 : number, n13 : number, n21 : number, n22 : number, n23 : number, n31 : number, n32 : number, n33 : number )

构造一个新的 3x3 矩阵。参数应按行主序提供。若未提供参数，构造函数将矩阵初始化为单位矩阵。

n11

1-1 矩阵元素。
n12

1-2 矩阵元素。
n13

1-3 矩阵元素。
n21

2-1 矩阵元素。
n22

2-2 矩阵元素。
n23

2-3 矩阵元素。
n31

3-1 矩阵元素。
n32

3-2 矩阵元素。
n33

3-3 矩阵元素。

属性

.elements : Array.<number>

按列主序排列的矩阵值列表。

方法

.clone() : Matrix3

返回一个从此实例复制值的矩阵。

返回值： 此实例的克隆。

.copy( m : Matrix3 ) : Matrix3

将给定矩阵的值复制到此实例。

m

要复制的矩阵。
返回值： 此矩阵的引用。

.determinant() : number

计算并返回此矩阵的行列式。

返回值： 行列式。

.equals( matrix : Matrix3 ) : boolean

若此矩阵与给定矩阵相等，则返回 true。

matrix

用于相等性测试的矩阵。
返回值： 此矩阵是否与给定矩阵相等。

.extractBasis( xAxis : Vector3, yAxis : Vector3, zAxis : Vector3 ) : Matrix3

将此矩阵的基提取到所提供的三个轴向量中。

xAxis

基的 x 轴。
yAxis

基的 y 轴。
zAxis

基的 z 轴。
返回值： 此矩阵的引用。

.fromArray( array : Array.<number>, offset : number ) : Matrix3

根据给定数组设置矩阵元素。

array

offset

默认值为 0。
返回值： 此矩阵的引用。

.getNormalMatrix( matrix4 : Matrix4 ) : Matrix3

计算法线矩阵，即给定 4x4 矩阵左上角 3x3 部分的逆转置。

返回值： 此矩阵的引用。

.identity() : Matrix3

将此矩阵设为 3x3 单位矩阵。

返回值： 此矩阵的引用。

.invert() : Matrix3

使用解析方法求此矩阵的逆。 行列式为零时无法求逆。若尝试求逆，该方法会生成零矩阵。

返回值： 此矩阵的引用。
