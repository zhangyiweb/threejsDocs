TetrahedronGeometry

用于表示四面体的几何体类。

代码示例

const geometry = new THREE.TetrahedronGeometry();
const material = new THREE.MeshBasicMaterial( { color: 0xffff00 } );
const tetrahedron = new THREE.Mesh( geometry, material );
scene.add( tetrahedron );

构造函数

new TetrahedronGeometry( radius : number, detail : number )

构造一个新的四面体几何体。

radius

四面体的半径。

默认值为 1.
detail

设为大于 0 的值会增加顶点，使其不再是四面体。

默认值为 0.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

重写： PolyhedronGeometry#parameters

静态方法

.fromJSON( data : Object ) : TetrahedronGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
