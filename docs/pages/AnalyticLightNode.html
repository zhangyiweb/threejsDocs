AnalyticLightNode

解析光节点的基类。

构造函数

new AnalyticLightNode( light : Light )

构造一个新的解析光节点。

light

光源。

默认值为 null。

属性

.baseColorNode : Node

该属性用于保留 AnalyticLightNode#colorNode 的原始值引用。 在使用阴影时，最终颜色节点由另一个不同的节点表示。

默认值为 null。

.color : Color

灯光的颜色值。

.colorNode : Node

灯光的颜色节点。若光源设置了 colorNode，则指向该节点；否则 基于 AnalyticLightNode#color 创建一个 uniform 节点。

.isAnalyticLightNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.light : Light

光源。

默认值为 null。

.shadowColorNode : Node

表示灯光的阴影颜色。

默认值为 null。

.shadowNode : ShadowNode

表示灯光的阴影。

默认值为 null。

.updateType : string

由于解析光节点每帧更新一次，因此覆盖了该属性。

默认值为 'frame'。

重写： LightingNode#updateType

方法

.disposeShadow()

释放与阴影相关的内部资源。

.getLightVector( builder : NodeBuilder ) : Node.<vec3>

返回一个表示方向向量的节点，该向量从视图空间中的当前位置 指向视图空间中灯光的位置。

builder

用于设置灯光的构建器对象。
返回值： 灯光向量节点。

.setup( builder : NodeBuilder )

与大多数其他节点不同，光照节点在 Node#setup 中不返回输出节点。 光照节点的主要目的是配置当前的 LightingModel 和/或 调用相应的接口方法。

builder

当前的节点构建器。
重写： LightingNode#setup

.setupDirect( builder : NodeBuilder ) : Object | undefined (abstract)

为解析光节点设置直接光照。

builder

用于设置灯光的构建器对象。
返回值： 直接光数据（颜色与方向）。

.setupDirectRectArea( builder : NodeBuilder ) : Object | undefined (abstract)

为解析光节点设置直接矩形区域光照。

builder

用于设置灯光的构建器对象。
返回值： 直接矩形区域光数据。

.setupShadow( builder : NodeBuilder )

为此灯光设置阴影。仅当灯光投射阴影且当前构建对象接收阴影时， 才会执行此方法。它将阴影纳入光照计算。

builder

当前的节点构建器。

.setupShadowNode() : ShadowNode

为此灯光设置阴影节点。存在此方法是为了让具体的灯光类 可以设置不同类型的阴影节点。

返回值： 创建的阴影节点。

.update( frame : NodeFrame )

更新方法用于每帧更新灯光的 uniform。 在具体的灯光节点中可能会被覆盖，以更新灯光特有的 uniform。

frame

对当前节点帧的引用。
重写： LightingNode#update
