InspectorBase

new InspectorBase()

InspectorBase 是所有检查器的基类。

属性

.currentFrame : Object

当前正在处理的帧。

.nodeFrame

返回当前渲染器的节点帧。

方法

.begin()

在帧开始时调用。

.beginCompute( uid : string, computeNode : ComputeNode )

在计算操作开始时调用。

uid

渲染上下文的唯一标识符。
computeNode

正在执行的计算节点。

.beginRender( uid : string, scene : Scene, camera : Camera, renderTarget : WebGLRenderTarget )

在渲染操作开始时调用。

uid

渲染上下文的唯一标识符。
scene

正在渲染的场景。
camera

用于渲染的相机。
renderTarget

渲染目标（若有）。

.computeAsync( computeNode : ComputeNode, dispatchSizeOrCount : number | Array.<number> )

当执行计算操作时调用。

computeNode

正在执行的计算节点。
dispatchSizeOrCount

调度大小或数量。

.copyFramebufferToTexture( framebufferTexture : Texture )

在执行帧缓冲复制操作时调用。

framebufferTexture

与帧缓冲关联的纹理。

.copyTextureToTexture( srcTexture : Texture, dstTexture : Texture )

在执行纹理复制操作时调用。

srcTexture

源纹理。
dstTexture

目标纹理。

.finish()

在帧结束时调用。

.finishCompute( uid : string, computeNode : ComputeNode )

在计算操作结束时调用。

uid

渲染上下文的唯一标识符。
computeNode

正在执行的计算节点。

.finishRender( uid : string )

在动画循环结束时调用。

uid

渲染上下文的唯一标识符。

.getRenderer() : WebGLRenderer

返回与此检查器关联的渲染器。

返回值： 关联的渲染器。

.init()

初始化检查器。

.inspect( node : Node )

检查一个节点。

node

要检查的节点。

.setRenderer( renderer : WebGLRenderer ) : InspectorBase

为此检查器设置渲染器。

renderer

要与此检查器关联的渲染器。
返回值： 此检查器实例。
