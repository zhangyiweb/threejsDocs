InspectorNode

InspectorNode 是一个包装节点，允许在渲染过程中检查节点的值。 可用于调试或分析渲染管线中的节点输出。

构造函数

new InspectorNode( node : Node, name : string, callback : function | null )

创建一个 InspectorNode。

node

要检查的节点。
name

检查器节点的可选名称。

默认值为 ''。
callback

可选回调，用于在 setup 期间修改节点。

默认值为 null。

属性

.type

返回节点的类型。

方法

.generateNodeType( builder : NodeBuilder ) : string

返回被包装节点的类型。

builder

节点构建器。
重写： Node#generateNodeType

.getName() : string

返回检查器节点的名称。

.setup( builder : NodeBuilder ) : Node

设置检查器节点。

builder

节点构建器。
重写： Node#setup
返回值： 设置后的节点。

.update( frame : NodeFrame )

更新检查器节点，允许检查被包装的节点。

frame

对当前节点帧的引用。
重写： Node#update
