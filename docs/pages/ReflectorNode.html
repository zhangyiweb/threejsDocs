ReflectorNode

此节点可用于实现类似镜面的平面反射表面。

代码示例

const groundReflector = reflector();
material.colorNode = groundReflector;
const plane = new Mesh( geometry, material );
plane.add( groundReflector.target );

构造函数

new ReflectorNode( parameters : Object )

构造一个新的反射器节点。

parameters

包含配置参数的对象。

默认值为 {}。

target

反射器所关联的 3D 对象。

默认值为 new Object3D()。
resolutionScale

分辨率缩放。

默认值为 1。
generateMipmaps

是否生成 mipmaps。

默认值为 false。
bounces

反射器是否可以渲染其他反射器节点。

默认值为 true。
depth

是否生成深度数据。

默认值为 false。
samples

内部渲染目标的抗锯齿采样数。
defaultTexture

默认纹理节点。
reflector

反射器基类节点。

属性

.reflector : ReflectorBaseNode

对内部反射器节点的引用。

.target : Object3D

对反射器所关联 3D 对象的引用。

方法

.dispose()

释放内部资源。当节点不再使用时应调用此方法。

重写： TextureNode#dispose

.getDepthNode() : Node

返回表示镜面深度的节点。可用于实现 更高级的反射效果，如距离衰减。

返回值： 深度节点。
