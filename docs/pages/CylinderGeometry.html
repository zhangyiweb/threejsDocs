CylinderGeometry

用于表示圆柱体的几何体类。

代码示例

const geometry = new THREE.CylinderGeometry( 5, 5, 20, 32 );
const material = new THREE.MeshBasicMaterial( { color: 0xffff00 } );
const cylinder = new THREE.Mesh( geometry, material );
scene.add( cylinder );

构造函数

new CylinderGeometry( radiusTop : number, radiusBottom : number, height : number, radialSegments : number, heightSegments : number, openEnded : boolean, thetaStart : number, thetaLength : number )

构造一个新的圆柱体几何体。

radiusTop

圆柱顶部的半径。

默认值为 1.
radiusBottom

圆柱底部的半径。

默认值为 1.
height

圆柱的高度。

默认值为 1.
radialSegments

沿圆柱圆周方向的分段面数量。

默认值为 32.
heightSegments

沿圆柱高度方向的面行数。

默认值为 1.
openEnded

圆柱底面是开放还是封闭。

默认值为 false.
thetaStart

第一段的起始角度（弧度）。

默认值为 0.
thetaLength

圆扇形的圆心角（常称为 theta），单位为弧度。默认值会生成完整的圆柱。

默认值为 Math.PI*2.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

静态方法

.fromJSON( data : Object ) : CylinderGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
