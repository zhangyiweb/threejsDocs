InterleavedBufferAttribute

一种采用交错数据存储的缓冲属性替代实现。交错属性共享同一交错数据存储（InterleavedBuffer），并通过不同的偏移量引用缓冲区中的数据。

构造函数

new InterleavedBufferAttribute( interleavedBuffer : InterleavedBuffer, itemSize : number, offset : number, normalized : boolean )

构造一个新的交错缓冲属性。

interleavedBuffer

保存交错数据的缓冲区。
itemSize

单个条目的大小。
offset

属性在缓冲区中的偏移量。
normalized

数据是否进行归一化。

默认值为 false。

属性

.array : TypedArray

保存交错缓冲属性数据的数组。

.count : number (readonly)

此缓冲属性的条目数量。

.data : InterleavedBuffer

保存交错数据的缓冲区。

.isInterleavedBufferAttribute : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.itemSize : number

单个条目的大小，参见 BufferAttribute#itemSize。

.name : string

缓冲属性的名称。

.needsUpdate : number

标记此属性已更改、需要重新传输到 GPU。修改数组中的值时，请将其设为 true。

默认值为 false。

.normalized : InterleavedBuffer

数据是否进行归一化，参见 BufferAttribute#normalized

.offset : number

属性在缓冲区中的偏移量。

.clone( data : Object ) : BufferAttribute | InterleavedBufferAttribute

返回一个新缓冲属性，其值从本实例复制。

若未提供参数，克隆交错缓冲属性将对缓冲区数据执行解交错。

data

返回值： 当前实例的克隆。
