ShadowNode

表示光照节点的默认阴影实现。

构造函数

new ShadowNode( light : Light, shadow : LightShadow )

构造一个新的阴影节点。

light

投射阴影的灯光。
shadow

可选的灯光阴影。

默认值为 null。

属性

.depthLayer : number (readonly)

当使用渲染目标数组重写 setupRenderTarget 时，可用此索引指定深度层。

默认值为 true。

.isShadowNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.shadow : LightShadow

定义该灯光阴影属性的灯光阴影对象。

默认值为 null。

.shadowMap : RenderTarget

阴影贴图的引用，它是一个渲染目标。

默认值为 null。

.vsmMaterialHorizontal : NodeMaterial

仅与 VSM 阴影相关。用于渲染第二个 VSM 通道的节点材质。

默认值为 null。

.vsmMaterialVertical : NodeMaterial

仅与 VSM 阴影相关。用于渲染第一个 VSM 通道的节点材质。

默认值为 null。

.vsmShadowMapHorizontal : RenderTarget

仅与 VSM 阴影相关。第二个 VSM 渲染通道的渲染目标。

默认值为 null。

.vsmShadowMapVertical : RenderTarget

仅与 VSM 阴影相关。第一个 VSM 渲染通道的渲染目标。

默认值为 null。

方法

.dispose()

释放该阴影节点的内部资源。

重写： ShadowBaseNode#dispose

.getShadowFilterFn( type : number ) : function

返回给定阴影类型的阴影过滤函数。

type

阴影类型。
返回值： 过滤函数。

.renderShadow( frame : NodeFrame )

渲染阴影。此函数的逻辑本可包含在 ShadowNode#updateShadow 中，但更专用的阴影节点可能需要 自定义阴影贴图渲染。通过提供独立方法，更容易重写默认行为。

frame

当前节点帧的引用。

.setup( builder : NodeBuilder ) : ShaderCallNodeInternal

该实现执行输出节点的设置。仅当渲染器全局启用了阴影贴图时才会产生输出。

builder

当前节点构建器的引用。
重写： ShadowBaseNode#setup
返回值： 输出节点。

.setupShadow( builder : NodeBuilder ) : Node.<vec3>

设置阴影输出节点。

builder

当前节点构建器的引用。
返回值： 阴影输出节点。

.setupShadowCoord( builder : NodeBuilder, shadowPosition : Node.<vec3> ) : Node.<vec3>

设置阴影坐标。

builder

当前节点构建器的引用。
shadowPosition

表示阴影位置的节点。
返回值： 阴影坐标。

.setupShadowFilter( builder : NodeBuilder, inputs : Object ) : Node.<float>

设置阴影过滤。

builder

当前节点构建器的引用。
inputs

定义阴影过滤的配置对象。

filterFn

此函数定义阴影贴图的过滤类型，例如 PCF。
depthTexture

阴影贴图纹理数据的引用。
shadowCoord

用于从阴影贴图采样的阴影坐标。
shadow

灯光阴影。
返回值： 阴影过滤的结果节点。

.updateBefore( frame : NodeFrame )

该实现在必要时执行阴影贴图的更新。

frame

当前节点帧的引用。
重写： ShadowBaseNode#updateBefore

.updateShadow( frame : NodeFrame )

更新阴影。

frame

当前节点帧的引用。

.vsmPass( renderer : Renderer )

对于 VSM，需要额外的渲染通道。

renderer

当前渲染器的引用。
