SkinnedMesh

拥有 Skeleton 的网格，可用于通过蒙皮/骨骼动画 来驱动几何体顶点。

除了有效的骨架外，蒙皮网格还要求其几何体中包含 作为缓冲属性的皮肤索引和权重。这些属性定义了哪些骨骼 以何种程度影响单个顶点。

通常蒙皮网格不会手动创建，而是由 GLTFLoader 或 FBXLoader 等加载器导入相应模型。

构造函数

new SkinnedMesh( geometry : BufferGeometry, material : Material | Array.<Material> )

构造一个新的蒙皮网格。

geometry

网格几何体。
material

网格材质。

属性

.bindMatrix : Matrix4

用于绑定骨骼变换的基础矩阵。

.bindMatrixInverse : Matrix4

用于重置绑定骨骼变换的基础矩阵。

.bindMode : AttachedBindMode | DetachedBindMode

AttachedBindMode 表示蒙皮网格与骨架共享同一世界空间。 使用 DetachedBindMode 时则并非如此，该模式在多个蒙皮网格 共享同一骨架时很有用。

默认值为 AttachedBindMode。

.boundingBox : Box3

蒙皮网格的包围盒。可通过 SkinnedMesh#computeBoundingBox 计算。

默认值为 null。

.boundingSphere : Sphere

蒙皮网格的包围球。可通过 SkinnedMesh#computeBoundingSphere 计算。

默认值为 null。

.isSkinnedMesh : boolean (readonly)

该标志可用于类型检测。

默认值为 true。

方法

.applyBoneTransform( index : number, target : Vector3 | Vector4 ) : Vector3 | Vector4

将与给定索引关联的骨骼变换应用到给定向量。 可通过提供 w 分量为 1 或 0 的 Vector4，分别用于变换位置或方向向量。 返回更新后的向量。

index

顶点索引。
target

用于存储方法结果的目标对象。
返回值： 更新后的顶点属性数据。

.bind( skeleton : Skeleton, bindMatrix : Matrix4 )

将给定骨架绑定到蒙皮网格。

skeleton

要绑定的骨架。
bindMatrix

绑定矩阵。若未提供绑定矩阵， 将改用蒙皮网格的世界矩阵。

.computeBoundingBox()

计算蒙皮网格的包围盒，并更新 SkinnedMesh#boundingBox。 引擎不会自动计算包围盒；必须由应用调用此方法。 若蒙皮网格处于动画状态，应每帧重新计算包围盒，以反映 当前动画状态。

.computeBoundingSphere()

计算蒙皮网格的包围球，并更新 SkinnedMesh#boundingSphere。 引擎会在需要时（例如用于光线投射和视锥体裁剪）自动计算一次包围球。 若蒙皮网格处于动画状态，应每帧重新计算包围球，以反映 当前动画状态。

.normalizeSkinWeights()

归一化蒙皮网格几何体中定义为缓冲属性的皮肤权重。

.pose()

此方法将蒙皮网格设置为休息姿势。
