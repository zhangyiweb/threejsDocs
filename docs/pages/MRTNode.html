MRTNode

此节点可用于为渲染设置 MRT 上下文。以下是典型的用于后期处理的 MRT 设置：

MRT 输出定义为字典。

代码示例

const mrtNode = mrt( {
  output: output,
  normal: normalView
} ) ;

构造函数

new MRTNode( outputNodes : Object.<string, Node> )

构造一个新的输出结构节点。

outputNodes

MRT 输出。

属性

.blendModes : Object.<string, BlendMode>

存储每个输出混合模式的字典。

.isMRTNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.outputNodes : Object.<string, Node>

表示 MRT 输出的字典。键是输出的名称， 值是产生输出结果的节点。

方法

.get( name : string ) : Node

返回给定名称的输出节点。

name

输出的名称。
返回值： 输出节点。

.getBlendMode( name : string ) : BlendMode

返回给定输出名称的混合模式。

name

输出的名称。
返回值： 混合模式。

.has( name : string ) : NodeBuilder

如果 MRT 节点具有给定名称的输出，则返回 true。

name

输出的名称。
返回值： MRT 节点是否具有给定名称的输出。

.merge( mrtNode : MRTNode ) : MRTNode

将给定 MRT 节点的输出与此节点的输出合并。

mrtNode

要合并的 MRT。
返回值： 具有合并输出的新 MRT 节点。

.setBlendMode( name : string, blend : BlendMode ) : MRTNode

为给定输出名称设置混合模式。

name

输出的名称。
blend

混合模式。
返回值： 当前的 MRT 节点。
