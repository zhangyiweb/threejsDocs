TileCreasedNormalsPlugin

3d-tiles-renderer 的插件，为每个已加载 tile 的几何体计算折痕顶点法线： 除面之间夹角大于折痕角的位置外，其余位置使用平滑法线。适用于 Google Photorealistic 3D Tiles 等 不含顶点法线的摄影测量 tile 集。

法线在 Web Worker 中计算，tile 处理不会阻塞主线程。tile 的法线就绪后才会显示。

代码示例

tiles.registerPlugin( new TileCreasedNormalsPlugin( { creaseAngle: Math.PI / 6 } ) );

导入

TileCreasedNormalsPlugin 是一个插件，必须显式导入，参见 Installation#Addons。

import { TileCreasedNormalsPlugin } from 'three/addons/misc/TileCreasedNormalsPlugin.js';

构造函数

new TileCreasedNormalsPlugin( options : Object )

构造一个新的插件。

options

配置选项。

creaseAngle

折痕角（弧度）。

默认值为 Math.PI/3。

属性

.creaseAngle : number

折痕角（弧度）。

方法

.dispose()

插件注销或 tiles 渲染器销毁时由 tiles 渲染器调用。

.processTileModel( scene : Object3D ) : Promise

tiles 渲染器为每个已加载的 tile 模型调用此方法。返回的 promise 解析后才会显示 tile。

scene

tile 模型。
返回值： 所有几何体完成折痕法线计算后解析的 promise。
