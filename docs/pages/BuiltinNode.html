BuiltinNode

该节点允许为内置着色器变量设置值。 这对于硬件加速顶点裁剪等功能是必需的。

构造函数

new BuiltinNode( name : string )

构造一个新的内置节点。

name

内置着色器变量的名称。

属性

.isBuiltinNode : boolean (readonly)

该标志可用于类型检测。

默认值为 true。

.name : string

内置着色器变量的名称。

重写： Node#name

方法

.generate( builder : NodeBuilder ) : string

生成内置节点的代码片段。

builder

当前的节点构建器。
重写： Node#generate
返回值： 生成的代码片段。
