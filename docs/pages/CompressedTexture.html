CompressedTexture

基于压缩形式的数据创建纹理。

此类纹理通常通过 CompressedTextureLoader 加载。

构造函数

new CompressedTexture( mipmaps : Array.<Object>, width : number, height : number, format : number, type : number, mapping : number, wrapS : number, wrapT : number, magFilter : number, minFilter : number, anisotropy : number, colorSpace : string )

构造一个新的压缩纹理。

mipmaps

该数组保存所有 mipmap（包括基础 mip） 的数据与尺寸。
width

纹理的宽度。
height

纹理的高度。
format

纹理格式。

默认值为 RGBAFormat。
type

纹理类型。

默认值为 UnsignedByteType。
mapping

纹理映射方式。

默认值为 Texture.DEFAULT_MAPPING。
wrapS

wrapS 值。

默认值为 ClampToEdgeWrapping。
wrapT

wrapT 值。

默认值为 ClampToEdgeWrapping。
magFilter

放大过滤器值。

默认值为 LinearFilter。
minFilter

缩小过滤器值。

默认值为 LinearMipmapLinearFilter。
anisotropy

各向异性值。

默认值为 Texture.DEFAULT_ANISOTROPY。
colorSpace

颜色空间。

默认值为 NoColorSpace。

属性

.flipY : boolean (readonly)

若设为 true，纹理在上传到 GPU 时会沿垂直轴翻转。

已覆盖并默认设为 false，因为无法翻转压缩纹理。

默认值为 false。

重写： Texture#flipY

.generateMipmaps : boolean (readonly)

是否为纹理生成 mipmap（若可能）。

已覆盖并默认设为 false，因为无法为压缩数据生成 mipmap。 Mipmap 必须嵌入在压缩纹理文件中。

默认值为 false。

重写： Texture#generateMipmaps

.image : Object

压缩纹理的 image 属性仅定义其尺寸。

重写： Texture#image

.isCompressedTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.mipmaps : Array.<Object>

该数组保存所有 mipmap（包括基础 mip）的数据与尺寸。

重写： Texture#mipmaps
