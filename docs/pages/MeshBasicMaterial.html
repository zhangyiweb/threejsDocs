MeshBasicMaterial

用于以简单着色（平面或线框）方式绘制几何体的材质。

此材质不受光照影响。

构造函数

new MeshBasicMaterial( parameters : Object )

构造一个新的基础网格材质。

parameters

包含一个或多个属性的对象，用于定义材质外观。 材质的任意属性（包括继承材质中的任意属性）都可在此传入。 颜色值可以是 Color#set 所接受的任意类型。

属性

.alphaMap : Texture

alpha 贴图是控制表面不透明度的灰度纹理 （黑色：完全透明；白色：完全不透明）。

仅使用纹理的颜色，若存在 alpha 通道则忽略。 对于 RGB 和 RGBA 纹理，渲染器在采样此纹理时会使用绿色通道， 因为 DXT 压缩格式以及未压缩的 RGB 565 格式为绿色提供了额外精度。 仅亮度以及亮度/alpha 纹理仍可按预期工作。

alphaMap 表示非颜色数据。所指定的任意纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.aoMap : Texture

此纹理的红色通道用作环境光遮蔽贴图。 需要第二套 UV。

aoMap 表示非颜色数据。所指定的任意纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.aoMapIntensity : number

环境光遮蔽效果的强度。范围为 [0,1]，其中 0 禁用环境光遮蔽。当强度为 1 且 AO 贴图的 红色通道也为 1 时，表面上的环境光会被完全遮蔽。

默认值为 1。

.color : Color

材质颜色。

默认值为 (1,1,1)。

.combine : MultiplyOperation | MixOperation | AddOperation

如何将表面颜色结果与环境贴图（若有）组合。

设为 MixOperation 时，会使用 MeshBasicMaterial#reflectivity 在两种颜色之间混合。

默认值为 MultiplyOperation。

.envMap : Texture

环境贴图。

envMap 表示亮度数据，纹理必须指定 Texture#colorSpace。大多数 envMap 纹理会设置 texture.colorSpace = LinearSRGBColorSpace，并使用浮点类型格式， 如 .exr 或 .hdr。

默认值为 null。

.envMapRotation : Euler

环境贴图的旋转（弧度）。

默认值为 (0,0,0)。

.fog : boolean

材质是否受雾影响。

默认值为 true。

.isMeshBasicMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.lightMap : Texture

光照贴图。需要第二套 UV。

lightMap 表示预烘焙照度数据，纹理必须指定 Texture#colorSpace。大多数 lightMap 纹理会设置 texture.colorSpace = LinearSRGBColorSpace，并使用浮点类型格式， 如 .exr 或 .hdr。

默认值为 null。

.lightMapIntensity : number

烘焙光照的强度。

默认值为 1。

.map : Texture

颜色贴图。可选择包含 alpha 通道，通常与 Material#transparent 或 Material#alphaTest 结合使用。纹理贴图 颜色会与漫反射 color 相乘调制。

map 表示颜色数据，纹理必须指定 Texture#colorSpace。大多数 map 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。

.reflectivity : number

环境贴图对表面的影响程度。 有效范围为 0（无反射）到 1（完全反射）。

默认值为 1。

.refractionRatio : number

空气的折射率（IOR，约等于 1）除以 材质折射率。与环境贴图模式 CubeRefractionMapping 和 EquirectangularRefractionMapping 一起使用。 折射率比值不应超过 1。

默认值为 0.98。

.specularMap : Texture

材质使用的高光贴图。

specularMap 表示颜色数据，纹理必须指定 Texture#colorSpace。大多数 specularMap 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。

.wireframe : boolean

将几何体渲染为线框。

默认值为 false。

.wireframeLinecap : 'round' | 'bevel' | 'miter'

定义线框端点外观。

仅可与 SVGRenderer 一起使用。

默认值为 'round'。

.wireframeLinejoin : 'round' | 'bevel' | 'miter'

定义线框连接处外观。

仅可与 SVGRenderer 一起使用。

默认值为 'round'。

.wireframeLinewidth : number

控制线框粗细。

仅可与 SVGRenderer 一起使用。

默认值为 1。
