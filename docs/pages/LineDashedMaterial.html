LineDashedMaterial

用于渲染线条图元的材质。

材质定义可渲染 3D 对象的外观。

代码示例

const material = new THREE.LineDashedMaterial( {
	color: 0xffffff,
	scale: 1,
	dashSize: 3,
	gapSize: 1,
} );

构造函数

new LineDashedMaterial( parameters : Object )

构造一个新的虚线材质。

parameters

包含一个或多个属性的对象，用于定义材质外观。 材质的任意属性（包括继承材质中的任意属性）都可在此传入。 颜色值可以是 Color#set 所接受的任意类型。

属性

.dashSize : number

虚线段的大小。这也是与描边之间的间隙相关尺寸。

默认值为 3。

.gapSize : number

间隙大小。

默认值为 1。

.isLineDashedMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.scale : number

线条虚线部分的缩放。

默认值为 1。
