CSS2DRenderer

此渲染器是 CSS3DRenderer 的简化版本。仅支持平移变换。

若希望将基于 HTML 的标签与 3D 对象结合使用，该渲染器非常有用。同样地， 相应的 DOM 元素会被包装为 CSS2DObject 实例并添加到 场景图中。所有其他类型的可渲染 3D 对象（如网格或点云）都会被忽略。

CSS2DRenderer 仅支持 100% 的浏览器缩放和显示缩放。

导入

CSS2DRenderer 是一个插件，必须显式导入，参见 Installation#Addons。

import { CSS2DRenderer } from 'three/addons/renderers/CSS2DRenderer.js';

构造函数

new CSS2DRenderer( parameters : CSS2DRenderer~Parameters )

构造一个新的 CSS2D 渲染器。

parameters

参数。

属性

.domElement : HTMLElement

渲染器将其子元素追加到的 DOM。

.sortObjects : boolean

控制渲染器是否为 CSS2DObject 的 DOM 元素分配 z-index 值。 若设为 true，则首先根据 renderOrder 分配 z-index， 其次根据到相机的距离分配。若设为 false，则不分配 z-index 值。

默认值为 true。

方法

.getSize() : Object

返回包含渲染器宽度和高度的对象。

返回值： 渲染器的尺寸。

.render( scene : Object3D, camera : Camera )

使用给定相机渲染给定场景。

scene

场景或其他类型的 3D 对象。
camera

相机。

.setSize( width : number, height : number )

将渲染器调整为给定的宽度和高度。

width

渲染器的宽度。
height

渲染器的高度。

类型定义

.Parameters

CSS2DRenderer 的构造函数参数。

element
HTMLElement

渲染器将其子元素追加到的 DOM 元素。 若未在此处传入，将创建一个新的 div 元素。
