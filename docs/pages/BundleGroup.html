BundleGroup

一个专用组，使应用能够访问 WebGPU 的 Render Bundle API。该组及其所有后代节点 被视为一个渲染包，并由渲染器按此方式处理。

此模块仅在使用 WebGPU 后端的 WebGPURenderer 中得到完整支持。 在 WebGL 后端下，该组在技术上仍可渲染，但不会获得 任何性能提升。

构造函数

new BundleGroup()

构造一个新的 bundle 组。

属性

.isBundleGroup : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.needsUpdate : boolean

当 bundle 组发生变化时，将此属性设为 true。

默认值为 false。

.static : boolean

bundle 是否为静态。设为 true 时，假定结构 是静态的且不会改变。例如，不会向组中 添加新对象。

若仍需更改，可通过将 needsUpdate 标志设为 true 来强制更新。

默认值为 true。

重写： Group#static

.type : string (readonly)

此属性仅用于在序列化/反序列化期间 检测类型。它应始终 与类名匹配。

默认值为 'BundleGroup'。

重写： Group#type

.version : number (readonly)

bundle 组的版本。

默认值为 0。
