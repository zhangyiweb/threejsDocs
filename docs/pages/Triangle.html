Triangle

由三个向量定义其三个顶点的几何三角形。

构造函数

new Triangle( a : Vector3, b : Vector3, c : Vector3 )

构造一个新的三角形。

a

三角形的第一个顶点。

默认值为 (0,0,0)。
b

三角形的第二个顶点。

默认值为 (0,0,0)。
c

三角形的第三个顶点。

默认值为 (0,0,0)。

属性

.a : Vector3

三角形的第一个顶点。

.b : Vector3

三角形的第二个顶点。

.c : Vector3

三角形的第三个顶点。

方法

.clone() : Triangle

返回一个新三角形，其值从本实例复制而来。

返回值： 本实例的克隆。

.closestPointToPoint( p : Vector3, target : Vector3 ) : Vector3

返回三角形上到给定点最近的点。

p

要计算最近点的目标点。
target

用于存储方法结果的目标向量。
返回值： 三角形上最近的点。

.containsPoint( point : Vector3 ) : boolean

如果给定点投影到三角形所在平面后落在三角形内，则返回 true。

point

要测试的三维空间中的点。
返回值： 给定点投影到三角形所在平面后是否落在三角形内。

.copy( triangle : Triangle ) : Triangle

将给定三角形的值复制到本实例。

triangle

要复制的三角形。
返回值： 对此三角形的引用。

.equals( triangle : Triangle ) : boolean

如果此三角形与给定三角形相等，则返回 true。

triangle

要测试相等性的三角形。
返回值： 此三角形是否与给定三角形相等。

.getArea() : number

计算三角形的面积。

返回值： 三角形的面积。

.getBarycoord( point : Vector3, target : Vector3 ) : Vector3

根据给定向量计算重心坐标。 如果三角形退化则返回 null。

point

三维空间中的点。
target

用于存储方法结果的目标向量。
返回值： 给定点的重心坐标

.getInterpolation( point : Vector3, v1 : Vector3, v2 : Vector3, v3 : Vector3, target : Vector3 ) : Vector3

计算给定点在三角形上的重心坐标插值。 如果三角形退化则返回 null。

point

插值点的位置。
v1

第一个顶点的插值量。
v2

第二个顶点的插值量。
v3

第三个顶点的插值量。
target

用于存储方法结果的目标向量。
返回值： 插值结果。

.getMidpoint( target : Vector3 ) : Vector3

计算三角形的中点。

target

用于存储方法结果的目标向量。
返回值： 三角形的中点。

.getNormal( target : Vector3 ) : Vector3

计算三角形的法线。

target

用于存储方法结果的目标向量。
返回值： 三角形的法线。

.getPlane( target : Plane ) : Plane

计算三角形所在的平面。

target

用于存储方法结果的目标向量。
返回值： 三角形所在的平面。

.intersectsBox( box : Box3 ) : boolean

如果此三角形与给定包围盒相交，则返回 true。

box

要相交检测的包围盒。
返回值： 此三角形是否与给定包围盒相交。

.isFrontFacing( direction : Vector3 ) : boolean

如果三角形朝向给定方向，则返回 true。

direction

（已归一化的）方向向量。
返回值： 三角形是否朝向给定方向。

.set( a : Vector3, b : Vector3, c : Vector3 ) : Triangle

通过复制给定值设置三角形的顶点。

a

三角形的第一个顶点。
b

三角形的第二个顶点。
c

三角形的第三个顶点。
返回值： 对此三角形的引用。

.setFromAttributeAndIndices( attribute : BufferAttribute, i0 : number, i1 : number, i2 : number ) : Triangle

通过复制给定属性值设置三角形的顶点。

attribute

包含三维点数据的缓冲属性。
i0

表示三角形第一个顶点的属性索引。
i1

表示三角形第二个顶点的属性索引。
i2

表示三角形第三个顶点的属性索引。
返回值： 对此三角形的引用。

.setFromPointsAndIndices( points : Array.<Vector3>, i0 : number, i1 : number, i2 : number ) : Triangle

通过复制给定数组值设置三角形的顶点。

points

包含三维点的数组。
i0

表示三角形第一个顶点的数组索引。
i1

表示三角形第二个顶点的数组索引。
i2

表示三角形第三个顶点的数组索引。
返回值： 对此三角形的引用。
