AudioListener

表示场景中所有空间音频和非空间音频效果的虚拟监听器。three.js 应用通常只创建一个监听器。它是 Audio 和 PositionalAudio 等音频实体的必需构造参数。

在大多数情况下，监听器对象是相机的子对象，因此相机的 3D 变换即代表监听器的 3D 变换。

构造函数

new AudioListener()

构造一个新的音频监听器。

属性

.context : AudioContext (readonly)

原生音频上下文。

.filter : AudioNode (readonly)

可选的滤波器。

通过 AudioListener#setFilter 设置。

默认值为 null。

.gain : GainNode (readonly)

用于音量控制的增益节点。

.timeDelta : number (readonly)

使用 linearRampToValueAtTime() 所需的时间增量值。

默认值为 0。

方法

.getFilter() : AudioNode

返回当前设置的滤波器。

返回值： 滤波器。

.getInput() : GainNode

返回监听器的输入节点。

其他音频节点通过此方法连接到该监听器。

返回值： 输入节点。

.getMasterVolume() : number

返回应用的主音量。

返回值： 主音量。

.removeFilter() : AudioListener

移除该监听器当前的滤波器。

返回值： 对该监听器的引用。

.setFilter( value : AudioNode ) : AudioListener

为监听器设置给定滤波器。

value

要设置的滤波器。
返回值： 对该监听器的引用。

.setMasterVolume( value : number ) : AudioListener

设置应用的主音量。该音量设置会影响场景中所有音频节点。

value

要设置的主音量。
返回值： 对该监听器的引用。
