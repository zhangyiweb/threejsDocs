ColladaLoader

Collada 格式的加载器。

Collada 格式非常复杂，因此该加载器仅支持 官方规范 中定义的一部分功能。

采用 Z-UP 坐标系的资源会通过简单旋转转换为 Y-UP。 顶点数据不会被转换。

代码示例

const loader = new ColladaLoader();
const result = await loader.loadAsync( './models/collada/elf/elf.dae' );
scene.add( result.scene );

导入

ColladaLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { ColladaLoader } from 'three/addons/loaders/ColladaLoader.js';

构造函数

new ColladaLoader()

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 Collada 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( text : string, path : string ) : Object

解析给定的 Collada 数据，并返回包含已解析场景、 动画剪辑数组和运动学的结果对象。

text

作为字符串的原始 Collada 数据。
path

资源路径。
重写： Loader#parse
返回值： 表示已解析资源的对象。
