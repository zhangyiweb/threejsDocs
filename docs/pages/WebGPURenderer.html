WebGPURenderer

此渲染器是 WebGLRenderer 的新替代方案。WebGPURenderer 能够 面向不同的后端。默认情况下，若浏览器支持 WebGPU，渲染器会尝试使用 WebGPU 后端。 否则，WebGPURenderer 会回退到 WebGL 2 后端。

构造函数

new WebGPURenderer( parameters : WebGPURenderer~Options )

构造一个新的 WebGPU 渲染器。

parameters

配置参数。

属性

.isWebGPURenderer : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.library : StandardNodeLibrary

通用默认值被覆盖为 用于类型映射的标准节点库。

重写： Renderer#library

类型定义

.Options

WebGPURenderer 选项。

logarithmicDepthBuffer
boolean

是否启用对数深度缓冲。

默认值为 false。
reversedDepthBuffer
boolean

是否启用反向深度缓冲。

默认值为 false。
alpha
boolean

默认帧缓冲（表示画布的最终内容）应为透明还是不透明。

默认值为 true。
depth
boolean

默认帧缓冲是否应具有深度缓冲。

默认值为 true。
stencil
boolean

默认帧缓冲是否应具有模板缓冲。

默认值为 false。
antialias
boolean

是否启用 MSAA 作为默认抗锯齿。

默认值为 false。
samples
number

当 antialias 为 true 时，默认使用 4 个采样。将此参数设为非 0 的任意整数值可覆盖默认值。

默认值为 0。
forceWebGL
boolean

若设为 true，无论是否支持 WebGPU，渲染器都将使用 WebGL 2 后端。

默认值为 false。
multiview
boolean

若设为 true，在支持的情况下，渲染器将在 WebXR 渲染期间使用多视图（multiview）。

默认值为 false。
outputType
number

输出到画布的纹理类型。默认使用设备首选格式；其他格式可能会带来额外开销。
outputBufferType
number

定义输出缓冲的类型。为获得最佳质量，推荐默认的 HalfFloatType。 为节省内存和带宽，也可使用 UnsignedByteType，但这会降低渲染质量。

默认值为 HalfFloatType。
