LUT3dlLoader

3DL LUT 格式的加载器。

参考：

  • 3D LUTs
  • Format Spec for .3dl

代码示例

const loader = new LUT3dlLoader();
const map = loader.loadAsync( 'luts/Presetpro-Cinematic.3dl' );

导入

LUT3dlLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { LUT3dlLoader } from 'three/addons/loaders/LUT3dlLoader.js';

构造函数

new LUT3dlLoader( manager : LoadingManager )

构造一个新的 3DL LUT 加载器。

manager

加载管理器。

类

LUT3dlLoader

属性

.type : UnsignedByteType | FloatType

纹理类型。

默认值为 UnsignedByteType。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载的 3DL LUT 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( input : string ) : Object

解析给定的 3DL LUT 数据并返回生成的 3D 数据纹理。

input

作为字符串的原始 3DL LUT 数据。
重写： Loader#parse
返回值： 已解析的 3DL LUT。

.setType( type : UnsignedByteType | FloatType ) : LUT3dlLoader

设置纹理类型。

type

要设置的纹理类型。
返回值： 对此加载器的引用。
