ScreenNode

此节点提供一系列与屏幕相关的度量值。 根据 ScreenNode#scope 的不同，节点可以表示 分辨率或视口数据，以及片元或 UV 坐标。

构造函数

new ScreenNode( scope : 'coordinate' | 'viewport' | 'size' | 'uv' | 'dpr' )

构造一个新的屏幕节点。

scope

节点的作用域。

属性

.isViewportNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.scope : 'coordinate' | 'viewport' | 'size' | 'uv' | 'dpr'

根据所选作用域，节点表示不同的度量值。

  • ScreenNode.COORDINATE：按 WebGPU 标准表示的当前片元相对于窗口的坐标。
  • ScreenNode.VIEWPORT：当前视口，表示为四维向量。
  • ScreenNode.SIZE：当前绑定帧缓冲区的尺寸。
  • ScreenNode.UV：归一化坐标。
  • ScreenNode.DPR：设备像素比。

方法

.generateNodeType() : 'float' | 'vec2' | 'vec4'

此方法被重写，因为节点类型取决于所选作用域。

重写： Node#generateNodeType
返回值： 节点类型。

.getUpdateType() : NodeUpdateType

此方法被重写，因为节点的更新类型取决于所选作用域。

重写： Node#getUpdateType
返回值： 更新类型。

.update( frame : NodeFrame )

ScreenNode 实现了 Node#update，以从当前渲染器 获取视口和尺寸信息。

frame

对当前节点帧的引用。
重写： Node#update
