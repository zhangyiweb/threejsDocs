WebGLRenderTarget

在 WebGLRenderer 上下文中使用的渲染目标。

构造函数

new WebGLRenderTarget( width : number, height : number, options : RenderTarget~Options )

构造一个新的渲染目标。

width

渲染目标的宽度。

默认值为 1。
height

渲染目标的高度。

默认值为 1。
options

配置对象。

属性

.isWebGLRenderTarget : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
