StorageInstancedBufferAttribute

这种特殊类型的实例化缓冲属性用于计算着色器。 在早期的 three.js 版本中，只能通过 JavaScript 在 CPU 上更新属性数据， 然后再将数据上传到 GPU。借助新的材质系统和渲染器，现在可以使用计算着色器 在 GPU 上更高效地计算属性数据。

用法是创建此类的实例，并将其作为输入提供给 StorageBufferNode。

注意：此类型缓冲属性只能与 WebGPURenderer 一起使用。

构造函数

new StorageInstancedBufferAttribute( count : number | TypedArray, itemSize : number, typeClass : TypedArray.constructor )

构造一个新的存储实例化缓冲属性。

count

元素数量。也可以将类型化数组作为参数传入。 此时后续参数将不再需要。
itemSize

每个元素的分量数。
typeClass

类型化数组构造函数。

默认值为 Float32Array。

属性

.isStorageInstancedBufferAttribute : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
