Font

表示字体的类。

构造函数

new Font( data : Object )

构造一个新的字体。

data

作为 JSON 的字体数据。

属性

.data : Object

作为 JSON 的字体数据。

.isFont : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.generateShapes( text : string, size : number, direction : string ) : Array.<Shape>

根据给定的文本和大小生成几何形状。此方法的返回结果 应与 ShapeGeometry 配合使用以生成实际的几何体数据。

text

文本。
size

文本大小。

默认值为 100。
direction

字符方向：ltr（从左到右）、rtl（从右到左）和 tb（从上到下）。

默认值为 'ltr'。
返回值： 表示文本的形状数组。
