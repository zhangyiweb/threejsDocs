NodeObjectLoader

用于使用节点材质加载 3D 对象的特殊类型对象加载器。

构造函数

new NodeObjectLoader( manager : LoadingManager )

构造一个新的节点对象加载器。

manager

加载管理器的引用。

属性

.nodeMaterials : Object.<string, NodeMaterial.constructor>

表示节点材质类型的字典。

.nodes : Object.<string, Node.constructor>

表示节点类型的字典。

方法
