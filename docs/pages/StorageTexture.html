StorageTexture

这种特殊类型的纹理用于计算着色器。 可用计算着色器来计算纹理的数据。

注意：此类型纹理只能与 WebGPURenderer 和 WebGPU 后端一起使用。

构造函数

new StorageTexture( width : number, height : number )

构造一个新的存储纹理。

width

存储纹理的宽度。

默认值为 1。
height

存储纹理的高度。

默认值为 1。

属性

.image : Object

仅表示纹理尺寸的图像对象。

重写： Texture#image

.isStorageTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.magFilter : number

存储纹理的默认 magFilter 为 THREE.LinearFilter。

重写： Texture#magFilter

.minFilter : number

存储纹理的默认 minFilter 为 THREE.LinearFilter。

重写： Texture#minFilter

.mipmapsAutoUpdate : boolean

当为 true 时，计算写入后将自动生成 mipmap。 当为 false 时，必须通过计算着色器手动写入 mipmap。

默认值为 true。

方法

.setSize( width : number, height : number )

设置存储纹理的尺寸。

width

存储纹理的新宽度。
height

存储纹理的新高度。
