MeshStandardMaterial

标准的基于物理的材质，使用金属度-粗糙度工作流。

基于物理的渲染（PBR）近来已成为许多 3D 应用的标准，例如 Unity、 Unreal 和 3D Studio Max。

这种方法与旧方法的不同之处在于：它不再使用近似来描述 光与表面的交互方式，而是采用 物理上正确的模型。其理念是：不必为了在特定光照下好看而反复调整 材质，而是可以创建一种在所有光照场景下都能“正确” 反应的材质。

在实践中，这比 MeshLambertMaterial 或 MeshPhongMaterial 能得到更准确、更真实的效果，代价是 计算开销相对更高。MeshStandardMaterial 使用逐片段 着色。

注意：使用此材质时，为获得最佳效果，应始终指定环境贴图。

关于 PBR 概念及其材质设置的非技术性介绍，可参阅 marmoset 团队的这些文章：

  • Basic Theory of Physically Based Rendering
  • Physically Based Rendering and You Can Too

three.js（以及大多数其他 PBR 系统）所用方法的技术细节，可参见 Brent Burley 的这篇 Disney 论文 （pdf）。

构造函数

new MeshStandardMaterial( parameters : Object )

构造一个新的网格标准材质。

parameters

包含一个或多个属性的对象， 用于定义材质外观。材质的任意属性 （包括从继承材质获得的任意属性）均可在此传入。 颜色值可以是 Color#set 所接受的任意类型。

属性

.alphaMap : Texture

Alpha 贴图是一张灰度纹理，用于控制表面各处的不透明度 （黑色：完全透明；白色：完全不透明）。

仅使用纹理的颜色，若存在 alpha 通道则忽略。 对于 RGB 和 RGBA 纹理，渲染器在采样时会使用绿色通道， 因为 DXT 压缩以及未压缩的 RGB 565 格式为绿色提供了额外的精度位。 仅亮度以及亮度/alpha 纹理也可按预期工作。

alphaMap 表示非颜色数据。所指定的任意纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.aoMap : Texture

该纹理的红色通道用作环境光遮蔽贴图。 需要第二套 UV。

aoMap 表示非颜色数据。所指定的任意纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.aoMapIntensity : number

环境光遮蔽效果的强度。范围为 [0,1]，其中 0 禁用环境光遮蔽。当强度为 1 且 AO 贴图的 红色通道也为 1 时，表面上的环境光会被完全遮蔽。

默认值为 1。

.bumpMap : Texture

用于创建凹凸贴图的纹理。黑白值映射到相对于光照的 感知深度。凹凸实际上不会影响 对象的几何体，只影响光照。若已定义法线贴图， 则会忽略此贴图。

bumpMap 表示非颜色数据。所指定的任意纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.bumpScale : number

凹凸贴图对材质的影响程度。典型范围为 [0,1]。

默认值为 1。

.color : Color

材质的颜色。

默认值为 (1,1,1)。

.displacementBias : number

位移贴图数值在网格顶点上的偏移量。 该偏置会加到位移贴图的缩放采样结果上。 未设置位移贴图时，此值不会生效。

默认值为 0。

.displacementMap : Texture

位移贴图会影响网格顶点的位置。与 其他仅影响材质明暗的贴图不同， 被位移的顶点可以投射阴影、阻挡其他对象，并以其他方式 充当真实几何体。位移纹理是一张图像，其中每个像素的值 （白色最高）会映射并 重新定位网格的顶点。为获得最佳效果，请将 位移贴图与匹配的法线贴图配对使用，因为渲染器无法 从位移后的顶点重新计算表面法线。

displacementMap 表示非颜色数据。所指定的任意纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.displacementScale : number

位移贴图对网格的影响程度（黑色表示无 位移，白色表示最大位移）。未设置位移 贴图时，此值不会生效。

默认值为 0。

.emissive : Color

材质的自发光（光）颜色，本质上是一种不受其他光照影响的纯色。

默认值为 (0,0,0)。

.emissiveIntensity : number

自发光强度。会调制自发光颜色。

默认值为 1。

.emissiveMap : Texture

设置自发光（辉光）贴图。自发光贴图颜色会受 自发光颜色和自发光强度调制。若使用了自发光贴图， 请务必将自发光颜色设置为非黑色。

emissiveMap 表示颜色数据，纹理必须指定 Texture#colorSpace。多数 emissiveMap 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。

.envMap : Texture

环境贴图。为保证物理上正确的渲染，环境贴图 会在内部通过 PMREMGenerator 进行预处理。

envMap 表示亮度数据，纹理必须指定 Texture#colorSpace。多数 envMap 纹理会设置 texture.colorSpace = LinearSRGBColorSpace，并使用浮点类型格式， 如 .exr 或 .hdr。

默认值为 null。

.envMapIntensity : number

通过乘以其颜色来缩放环境贴图的效果。

默认值为 1。

.envMapRotation : Euler

环境贴图的旋转（以弧度为单位）。

默认值为 (0,0,0)。

.flatShading : boolean

材质是否使用平面着色渲染。

默认值为 false。

.fog : boolean

材质是否受雾影响。

默认值为 true。

.isMeshStandardMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.lightMap : Texture

光照贴图。需要第二套 UV。

lightMap 表示预烘焙的照度数据，纹理必须指定 Texture#colorSpace。多数 lightMap 纹理会设置 texture.colorSpace = LinearSRGBColorSpace，并使用浮点类型格式， 如 .exr 或 .hdr。

默认值为 null。

.lightMapIntensity : number

烘焙光照的强度。

默认值为 1。

.map : Texture

颜色贴图。可选择包含 alpha 通道，通常与 Material#transparent 或 Material#alphaTest 配合使用。贴图颜色 会受漫反射 color 调制。

map 表示颜色数据，纹理必须指定 Texture#colorSpace。多数 map 纹理会设置 texture.colorSpace = SRGBColorSpace。

默认值为 null。

.metalness : number

材质类似金属的程度。木材或石材等非金属材质使用 0.0，金属使用 1.0，二者之间通常没有中间值。 0.0 到 1.0 之间的值可用于表现生锈金属的外观。 若同时提供了 metalnessMap，两个值会相乘。

默认值为 0。

.metalnessMap : Texture

该纹理的蓝色通道用于改变材质的金属度。

metalnessMap 表示非颜色数据。所指定的任意纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.normalMap : Texture

用于创建法线贴图的纹理。RGB 值会影响每个像素片段的表面 法线，从而改变颜色的受光方式。法线 贴图不会改变表面的实际形状，只影响光照。若 材质的法线贴图按左手坐标系约定制作， 应将 normalScale 的 y 分量取反以补偿 不同的手性。

normalMap 表示非颜色数据。所指定的任意纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.normalMapType : TangentSpaceNormalMap | ObjectSpaceNormalMap

法线贴图的类型。

默认值为 TangentSpaceNormalMap。

.normalScale : Vector2

法线贴图对材质的影响程度。典型取值范围为 [0,1]。

默认值为 (1,1)。

.roughness : number

材质表面的粗糙程度。0.0 表示光滑的镜面反射，1.0 表示完全漫反射。若同时提供了 roughnessMap， 两个值会相乘。

默认值为 1。

.roughnessMap : Texture

该纹理的绿色通道用于改变材质的粗糙度。

roughnessMap 表示非颜色数据。所指定的任意纹理必须设置 texture.colorSpace = NoColorSpace（默认）。

默认值为 null。

.wireframe : boolean

将几何体渲染为线框。

默认值为 false。

.wireframeLinecap : 'round' | 'bevel' | 'miter'

定义线框端点的外观。

仅可与 SVGRenderer 一起使用。

默认值为 'round'。

.wireframeLinejoin : 'round' | 'bevel' | 'miter'

定义线框接头的外观。

仅可与 SVGRenderer 一起使用。

默认值为 'round'。

.wireframeLinewidth : number

控制线框的粗细。

仅可与 SVGRenderer 一起使用。

默认值为 1。
