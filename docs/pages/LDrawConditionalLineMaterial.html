LDrawConditionalLineMaterial

专用于通过 LDrawLoader 加载的网格的特殊线条材质。

此模块只能与 WebGLRenderer 一起使用。使用 WebGPURenderer 时，请从 LDrawConditionalLineNodeMaterial.js 导入该类。

导入

LDrawConditionalLineMaterial 是一个插件，必须显式导入，参见 Installation#Addons。

import { LDrawConditionalLineMaterial } from 'three/addons/materials/LDrawConditionalLineMaterial.js';

构造函数

new LDrawConditionalLineMaterial( parameters : Object )

构造一个新的条件线条材质。

parameters

包含一个或多个定义材质外观属性的对象。材质的任何属性（包括继承材质的任何属性）都可在此传入。颜色值可以是 Color#set 接受的任意类型。

属性

.color : Color

材质的颜色。

默认值为 (1,1,1)。

.isLDrawConditionalLineMaterial : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

.opacity : number

材质的不透明度。

默认值为 1。

重写： ShaderMaterial#opacity
