TAAUNode

用于执行时域抗锯齿上采样（TAAU）的特殊节点。

与 TRAA 类似，该节点在多帧上累积抖动采样，并使用运动向量重投影历史帧。与 TRAA 不同的是，输入缓冲区（beauty、depth、velocity）应以低于渲染器绘制缓冲区分辨率的分辨率渲染——通常通过 PassNode#setResolutionScale 降低上游通道的分辨率——解析通道则对抖动输入采样使用 9 点 Blackman-Harris 滤波器（高斯近似）重建输出分辨率的图像。其结果可作为 FSR2/3 的替代方案，在单次通道中同时完成抗锯齿与上采样。

参考文献：

  • Karis，"High Quality Temporal Supersampling"，SIGGRAPH 2014，https://advances.realtimerendering.com/s2014/
  • Riley/Arcila，FidelityFX Super Resolution 2，GDC 2022，https://gpuopen.com/download/GDC_FidelityFX_Super_Resolution_2_0.pdf

注意：使用 TAAU 时必须禁用 MSAA。

导入

TAAUNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { taau } from 'three/addons/tsl/display/TAAUNode.js';

构造函数

new TAAUNode( beautyNode : TextureNode, depthNode : TextureNode, velocityNode : TextureNode, camera : Camera )

构造一个新的 TAAU 节点。

beautyNode

表示效果输入的纹理节点。
depthNode

表示场景深度的节点。
velocityNode

表示场景速度的节点。
camera

用于渲染场景的相机。

属性

.beautyNode : TextureNode

表示效果输入的纹理节点。

.camera : Camera

用于渲染场景的相机。

.currentFrameWeight : number

解析时应用于当前帧的基准权重。较低的值会产生更平滑的结果，累积时间更长，但在遮挡解除区域收敛较慢；运动因子会叠加在其上，因此快速移动的像素仍能快速响应。

默认值为 0.025。

.depthNode : TextureNode

表示场景深度的节点。

.depthThreshold : number

当当前深度与上一帧深度的差值超过此阈值时，历史帧被视为无效。

默认值为 0.0005。

.edgeDepthDiff : number

3×3 邻域内的深度差值，用于判定像素是否为边缘。

默认值为 0.001。

.isTAAUNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.maxVelocityLength : number

当速度的像素长度接近此值时，历史帧将变为无效。

默认值为 128。

.updateBeforeType : string

updateBeforeType 设为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

.velocityNode : TextureNode

表示场景速度的节点。

方法

.clearViewOffset()

清除场景相机的视图偏移。

.dispose()

释放内部资源。当效果不再使用时，应调用此方法。

重写： TempNode#dispose

.getTextureNode() : PassTextureNode

以纹理节点的形式返回效果结果。

返回值： 表示效果结果的纹理节点。

.setSize( outputWidth : number, outputHeight : number )

设置效果的输出尺寸（历史帧与解析目标）。上一帧深度纹理在 updateBefore() 中独立设置尺寸，以匹配场景当前的深度纹理。

outputWidth

输出宽度（绘制缓冲区宽度）。
outputHeight

输出高度（绘制缓冲区高度）。

.setViewOffset( inputWidth : number, inputHeight : number )

将 TAAU 当前的抖动定义为场景相机的视图偏移。抖动被缩小到一个输出像素（而非一个输入像素），以便 Halton 序列在多帧上逐渐填充输出亚像素网格。

inputWidth

相机渲染到的输入缓冲区宽度。
inputHeight

相机渲染到的输入缓冲区高度。

.setup( builder : NodeBuilder ) : PassTextureNode

此方法用于设置效果的渲染目标与 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

此方法用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
