XRHandModelFactory

与 XRControllerModelFactory 类似，此类允许为 WebXR 控制器创建手部模型，可作为视觉表示添加到场景中。

代码示例

const handModelFactory = new XRHandModelFactory();
const hand = renderer.xr.getHand( 0 );
hand.add( handModelFactory.createHandModel( hand ) );
scene.add( hand );

导入

XRHandModelFactory 是一个插件，必须显式导入，参见 Installation#Addons。

import { XRHandModelFactory } from 'three/addons/webxr/XRHandModelFactory.js';

构造函数

new XRHandModelFactory( gltfLoader : GLTFLoader, onLoad : function )

构造一个新的 XR 手部模型工厂。

gltfLoader

用于加载手部模型的 glTF 加载器。

默认值为 null。
onLoad

手部模型加载完成后执行的回调。

默认值为 null。

属性

.gltfLoader : GLTFLoader

用于加载手部模型的 glTF 加载器。

默认值为 null。

.onLoad : function

手部模型加载完成后执行的回调。

默认值为 null。

.path : string

模型资源库的路径。

默认值为 null。

方法

.createHandModel( controller : Group, profile : 'spheres' | 'boxes' | 'mesh' ) : XRHandModel

为指定的 WebXR 手部控制器创建控制器模型。

controller

手部控制器。
profile

定义模型类型的模型配置文件。
返回值： XR 手部模型。

.setPath( path : string ) : XRHandModelFactory

设置手部模型资源库的路径。

path

要设置的路径。
返回值： 对此实例的引用。
