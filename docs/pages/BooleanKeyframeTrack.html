BooleanKeyframeTrack

用于布尔类型关键帧值的轨道。

构造函数

new BooleanKeyframeTrack( name : string, times : Array.<number>, values : Array.<boolean> )

构造一个新的布尔关键帧轨道。

此关键帧轨道类型没有 interpolation 参数，因为插值始终为离散类型。

name

关键帧轨道名称。
times

关键帧时间列表。
values

关键帧值列表。

属性

.DefaultInterpolation : InterpolateLinear | InterpolateDiscrete | InterpolateSmooth

该关键帧轨道的默认插值类型。

默认值为 InterpolateDiscrete。

重写： KeyframeTrack#DefaultInterpolation

.ValueBufferType : TypedArray | Array

该关键帧轨道的值缓冲区类型。

默认值为 Array.constructor。

重写： KeyframeTrack#ValueBufferType

.ValueTypeName : string

值类型名称。

默认值为 'bool'。

重写： KeyframeTrack#ValueTypeName
