PixelationPassNode

以像素化效果渲染场景的特殊渲染通道节点。

导入

PixelationPassNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { pixelationPass } from 'three/addons/tsl/display/PixelationPassNode.js';

构造函数

new PixelationPassNode( scene : Scene, camera : Camera, pixelSize : Node.<float> | number, normalEdgeStrength : Node.<float> | number, depthEdgeStrength : Node.<float> | number )

构造一个新的像素化通道节点。

scene

要渲染的场景。
camera

用于渲染场景的相机。
pixelSize

像素大小。

默认值为 6。
normalEdgeStrength

法线边缘强度。

默认值为 0.3。
depthEdgeStrength

深度边缘强度。

默认值为 0.4。

属性

.depthEdgeStrength : number

深度边缘强度。

默认值为 0.4。

.isPixelationPassNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.normalEdgeStrength : number

法线边缘强度。

默认值为 0.3。

.pixelSize : number

像素大小。

默认值为 6。

方法

.setSize( width : number, height : number )

设置通道的尺寸。

width

通道的宽度。
height

通道的高度。
重写： PassNode#setSize

.setup( builder : NodeBuilder ) : PixelationNode

此方法用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： PassNode#setup
