Reflector

可用于创建平面反射表面，例如镜子。

注意：此类只能与 WebGLRenderer 一起使用。 使用 WebGPURenderer 时，请使用 ReflectorNode。

代码示例

const geometry = new THREE.PlaneGeometry( 100, 100 );
const reflector = new Reflector( geometry, {
	clipBias: 0.003,
	textureWidth: window.innerWidth * window.devicePixelRatio,
	textureHeight: window.innerHeight * window.devicePixelRatio,
	color: 0xc1cbcb
} );
scene.add( reflector );

导入

Reflector 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { Reflector } from 'three/addons/objects/Reflector.js';

构造函数

new Reflector( geometry : BufferGeometry, options : Reflector~Options )

构造一个新的反射器。

geometry

反射器的几何体。
options

配置选项。

属性

.forceUpdate : boolean

是否强制更新，无论反射器是否在视野内。

默认值为 false。

.isReflector : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

方法

.dispose()

释放此实例分配的 GPU 相关资源。 当应用中不再使用此实例时，请调用此方法。

.getReflectionCamera( camera : Camera ) : Camera

返回给定相机对应的反射相机。反射相机用于 从反射器的视角渲染场景，从而生成正确的反射效果。

camera

场景的相机。
返回值： 对应的反射相机。

.getRenderTarget() : WebGLRenderTarget

返回反射器的内部渲染目标。

返回值： 内部渲染目标

类型定义

.Options

Reflector 的构造函数选项。

color
number | Color | string

反射器的颜色。

默认值为 0x7F7F7F。
textureWidth
number

纹理宽度。值越高，反射越清晰，但开销也更大。

默认值为 512。
textureHeight
number

纹理高度。值越高，反射越清晰，但开销也更大。

默认值为 512。
clipBias
number

裁剪偏置。

默认值为 0。
shader
Object

可用于传入自定义着色器，以定义如何将反射视图投影到反射器的几何体上。
multisample
number

MSAA 使用的采样数。0 表示禁用 MSAA。

默认值为 4。
