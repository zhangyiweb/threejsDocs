SidewalkGenerator

为城市街区生成抬高的人行道：每个街区一块圆角混凝土板，边缘环绕突出于步行面并下落到路面的独立花岗岩路缘石。 按一组放置变换实例化，并用自带的程序化材质装扮（浇筑混凝土板、刻划伸缩缝、花岗岩路缘）。 返回包含两个实例化网格的 THREE.Group——步行板与路缘。

与建筑生成器不同，此类自带材质：板与路缘的几何体以及为其着色的 TSL 都在此一并管理。

代码示例

const sidewalk = new SidewalkGenerator( { width: 90, depth: 60, height: 0.5 } );
scene.add( sidewalk.build( placements ) ); // placements: Matrix4[]

构造函数

new SidewalkGenerator()
