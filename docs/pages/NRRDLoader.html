NRRDLoader

用于加载 NRRD 格式的加载器。

代码示例

const loader = new NRRDLoader();
const volume = await loader.loadAsync( 'models/nrrd/I.nrrd' );

导入

NRRDLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { NRRDLoader } from 'three/addons/loaders/NRRDLoader.js';

构造函数

new NRRDLoader( manager : LoadingManager )

构造一个新的 NRRD 加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 NRRD 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( data : ArrayBuffer ) : Volume

解析给定的 NRRD 数据并返回生成的体数据。

data

原始 NRRD 数据 ArrayBuffer。
重写： Loader#parse
返回值： 解析后的体数据。

.setSegmentation( segmentation : boolean )

切换分割模式。

segmentation

是否使用分割模式。
