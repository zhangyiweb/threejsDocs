PropertyNode

此类表示着色器属性。可用于 显式定义属性并为其赋值。

PropertyNode 被引擎用于为 TSL 代码 预定义常见的材质属性。

代码示例

const threshold = property( 'float', 'threshold' ).assign( THRESHOLD );

构造函数

new PropertyNode( nodeType : string, name : string, varying : boolean, placeholderNode : Node )

构造一个新的属性节点。

nodeType

节点的类型。
name

着色器中属性的名称。

默认值为 null。
varying

此属性是否为 varying。

默认值为 false。
placeholderNode

未赋值时使用的占位节点。

默认值为 null。

属性

.global : boolean

此标志用于全局缓存。

默认值为 true。

重写： Node#global

.isPropertyNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.name : string

着色器中属性的名称。若未定义名称， 节点系统会自动生成一个。

默认值为 null。

重写： Node#name

.placeholderNode : Node

属性未赋值时的占位节点。

默认值为 null。

.varying : boolean

此属性是否为 varying。

默认值为 false。
