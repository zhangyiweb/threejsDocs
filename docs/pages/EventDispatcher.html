EventDispatcher

该模块允许在自定义 JavaScript 对象上分发事件对象。

主仓库：eventdispatcher.js

代码示例：

代码示例

class Car extends EventDispatcher {
	start() {
		this.dispatchEvent( { type: 'start', message: 'vroom vroom!' } );
	}
};
// Using events with the custom object
const car = new Car();
car.addEventListener( 'start', function ( event ) {
	alert( event.message );
} );
car.start();

构造函数

new EventDispatcher()

方法

.addEventListener( type : string, listener : function )

将给定的事件监听器添加到给定的事件类型上。

type

要监听的事件类型。
listener

当事件触发时被调用的函数。

.dispatchEvent( event : Object )

分发一个事件对象。

event

要触发的事件。

.hasEventListener( type : string, listener : function ) : boolean

如果给定的事件监听器已添加到给定的事件类型上，则返回 true。

type

事件类型。
listener

要检查的监听器。
返回值：给定的事件监听器是否已添加到给定的事件类型上。

.removeEventListener( type : string, listener : function )

从给定的事件类型上移除给定的事件监听器。

type

事件类型。
listener

要移除的监听器。
