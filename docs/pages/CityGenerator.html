CityGenerator

排布城市街区网格，并用各自拥有独立种子、高度与占地的 SkyscraperGenerator 塔楼填充每个地块，可选放置在抬高的人行道板（路缘）上。返回一个可直接加入场景的 THREE.Group。

传入建筑材质来装扮塔楼；人行道则通过 SidewalkGenerator 自行装扮。布局通过 CityGenerator#layout 暴露，以便周围场景（道路标线等） 能对齐同一网格。

代码示例

const city = new CityGenerator( { seed: 1 } );
scene.add( city.build( materials ) );

构造函数

new CityGenerator()
