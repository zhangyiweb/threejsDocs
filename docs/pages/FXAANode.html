FXAANode

用于应用 FXAA 的后处理节点。该节点需要 sRGB 输入，因此色调映射与色彩空间转换必须在抗锯齿之前完成。

导入

FXAANode 是一个插件，必须显式导入，参见 Installation#Addons。

import { fxaa } from 'three/addons/tsl/display/FXAANode.js';

构造函数

new FXAANode( textureNode : TextureNode )

构造一个新的 FXAA 节点。

textureNode

表示效果输入的纹理节点。

属性

.textureNode : TextureNode

表示效果输入的纹理节点。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.FRAME，因为该节点会在 updateBefore() 中每帧更新一次内部 uniform。

默认值为 'frame'。

重写： TempNode#updateBeforeType

方法

.setup( builder : NodeBuilder ) : ShaderCallNodeInternal

用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

用于每帧更新一次效果的 uniform。

frame

当前的节点帧。
重写： TempNode#updateBefore
