NURBSCurve

此类表示一条 NURBS 曲线。

实现基于带权重 w=weight 的控制点 (x, y [, z=0 [, w=1]])。

导入

NURBSCurve 是一个插件，必须显式导入，参见 Installation#Addons。

import { NURBSCurve } from 'three/addons/curves/NURBSCurve.js';

构造函数

new NURBSCurve( degree : number, knots : Array.<number>, controlPoints : Array.<(Vector2|Vector3|Vector4)>, startKnot : number, endKnot : number )

构造一条新的 NURBS 曲线。

degree

NURBS 阶数。
knots

节点，以数字平铺数组表示。
controlPoints

保存控制点的数组。
startKnot

起始节点在 knots 数组中的索引。
endKnot

结束节点在 knots 数组中的索引。

属性

.controlPoints : Array.<Vector4>

控制点数组。

.degree : number

NURBS 阶数。

.endKnot : number

结束节点在 knots 数组中的索引。

.knots : Array.<number>

节点，以数字平铺数组表示。

.startKnot : number

起始节点在 knots 数组中的索引。

方法

.getPoint( t : number, optionalTarget : Vector3 ) : Vector3

此方法根据给定的插值因子返回三维空间中的向量。

t

表示曲线上位置的插值因子，必须在 [0,1] 范围内。
optionalTarget

可选的目标向量，结果将写入其中。
重写： Curve#getPoint
返回值： 曲线上的位置。

.getTangent( t : number, optionalTarget : Vector3 ) : Vector3

返回给定插值因子处的单位切向量。

t

插值因子。
optionalTarget

可选的目标向量，结果将写入其中。
重写： Curve#getTangent
返回值： 切向量。
