ContextNode

此节点可用作另一个节点的上下文管理组件。 NodeBuilder 在特定上下文中执行其节点构建过程， 而此节点允许修改该上下文。典型用例是覆盖 getUV()，例如：

代码示例

node.context( { getUV: () => customCoord } );
\// or
material.contextNode = context( { getUV: () => customCoord } );
\// or
renderer.contextNode = context( { getUV: () => customCoord } );
\// or
scenePass.contextNode = context( { getUV: () => customCoord } );

构造函数

new ContextNode( node : Node, value : Object )

构造一个新的上下文节点。

node

应修改其上下文的节点。

默认值为 null。
value

修改后的上下文数据。

默认值为 {}。

属性

.isContextNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.node : Node

应修改其上下文的节点。

.value : Object

修改后的上下文数据。

默认值为 {}。

方法

.generateNodeType( builder : NodeBuilder ) : string

此方法被重写，以确保返回 ContextNode#node 的类型。

builder

当前的节点构建器。
重写： Node#generateNodeType
返回值： 节点类型。

.getFlowContextData() : Object

从所有父级上下文节点收集上下文数据。

返回值： 收集到的上下文数据。

.getMemberType( builder : NodeBuilder, name : string ) : string

此方法被重写，以确保返回 ContextNode#node 的成员类型。

builder

当前的节点构建器。
name

成员名称。
重写： Node#getMemberType
返回值： 成员类型。

.getScope() : Node

此方法被重写，以确保返回对 ContextNode#node 的引用。

重写： Node#getScope
返回值： 对 ContextNode#node 的引用。
