BypassNode

该类会生成给定节点的代码，但在输出中返回另一个节点。 可用于调用不返回值（即类型为 void）的方法或节点， 而输入处又要求返回值的场景。例如：

代码示例

material.colorNode = myColor.bypass( runVoidFn() )

构造函数

new BypassNode( outputNode : Node, callNode : Node )

构造一个新的旁路节点。

outputNode

输出节点。
callNode

调用节点。

属性

.callNode : Node

调用节点。

.isBypassNode : boolean (readonly)

该标志可用于类型检测。

默认值为 true。

.outputNode : Node

输出节点。
