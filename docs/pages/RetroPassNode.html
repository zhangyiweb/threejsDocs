RetroPassNode

对场景应用复古 PS1 风格效果的后处理通道。

此节点以经典 PlayStation 1 视觉特征渲染场景：

  • 顶点吸附：顶点被吸附到屏幕像素，产生标志性的"抖动"几何体
  • 仿射纹理映射：纹理在不进行透视校正的情况下采样，产生扭曲效果
  • 低分辨率：默认 0.25 缩放（相当于典型的 320x240）
  • 最近邻过滤：清晰像素化纹理，无平滑处理

构造函数

new RetroPassNode( scene : Scene, camera : Camera, options : Object )

创建一个新的 RetroPassNode 实例。

scene

要渲染的场景。
camera

用于渲染的相机。
options

复古通道的附加选项。

默认值为 {}。

affineDistortion

可选节点，用于对 UV 应用仿射扭曲。

默认值为 null。
