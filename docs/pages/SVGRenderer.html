SVGRenderer

此渲染器可用于使用 SVG 渲染几何数据。生成的矢量 图形在以下用例中尤其有用：

  • 动画徽标或图标。
  • 交互式 2D/3D 图表或图形。
  • 交互式地图。
  • 复杂或动画化的用户界面。

SVGRenderer 有多种优势。它能生成清晰锐利的输出， 且与实际视口分辨率无关。SVG 元素可通过 CSS 设置样式。 并且具有良好的可访问性，因为可以添加标题或描述等元数据 （对搜索引擎或屏幕阅读器很有用）。

不过，存在一些重要限制：

  • 无高级着色。
  • 不支持纹理。
  • 不支持阴影。

导入

SVGRenderer 是一个插件，必须显式导入，参见 Installation#Addons。

import { SVGRenderer } from 'three/addons/renderers/SVGRenderer.js';

构造函数

new SVGRenderer()

构造一个新的 SVG 渲染器。

属性

.autoClear : boolean

是否在渲染调用前自动执行清除。

默认值为 true。

.domElement : SVGSVGElement

渲染器将其子元素追加到的 DOM。

.info : Object

提供已渲染顶点和面数量的 相关信息。

.outputColorSpace : SRGBColorSpace | LinearSRGBColorSpace

输出颜色空间。

默认值为 SRGBColorSpace。

.overdraw : number

为防止抗锯齿缝隙而放大多边形的小数像素数。 范围为 [0,1]。

默认值为 0.5。

.sortElements : boolean

是否对元素进行排序。

默认值为 true。

.sortObjects : boolean

是否对 3D 对象进行排序。

默认值为 true。

方法

.clear()

使用已定义的清除颜色执行手动清除。

.getSize() : Object

返回包含渲染器宽度和高度的对象。

返回值： 渲染器的尺寸。

.render( scene : Object3D, camera : Camera )

使用给定相机渲染给定场景。

scene

场景或其他类型的 3D 对象。
camera

相机。

.setClearColor( color : number | Color | string )

设置清除颜色。

color

要设置的清除颜色。

.setPrecision( precision : number )

设置用于创建路径的数据精度。

precision

要设置的精度。

.setQuality( quality : 'low' | 'high' )

设置渲染质量。设为 high 时，浏览器会优先提升 SVG 质量， 而非渲染速度和几何精度。

quality

质量。

.setSize( width : number, height : number )

将渲染器调整为给定的宽度和高度。

width

渲染器的宽度。
height

渲染器的高度。
