UniformsGroup

用于在单个组中管理多个 uniform 的类。渲染器会将此类定义作为单个 UBO 处理。

由于该类仅可在 ShaderMaterial 上下文中使用，因此仅支持 WebGLRenderer。

构造函数

new UniformsGroup()

构造一个新的 uniforms 组。

属性

.buffer : Float32Array

包含 uniform 值的 Float32 数组缓冲区。

重写： UniformBuffer#buffer

.byteLength : number

缓冲区的字节长度（已按正确的缓冲区对齐计算）。

重写： UniformBuffer#byteLength

.id : number (readonly)

3D 对象的 ID。

.isUniformsGroup : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.isUniformsGroup : boolean (readonly)

此标志可用于类型检测。

.name : string

重写： UniformBuffer#name

.uniforms : Array.<Uniform>

保存 uniform 的数组。

.uniforms : Array.<Uniform>

uniform 对象数组。

缓冲区的用途。

.update() : boolean

通过更新内部 uniform 列表中的每个 uniform 对象来更新该组。这些 uniform 对象会检查其值是否实际发生变化，因此仅在确有值变化时此方法才返回 true。

重写： UniformBuffer#update
返回值： uniforms 是否已更新且必须上传到 GPU。
