SubgroupFunctionNode

此类表示一组内置的 WGSL 着色器函数，可在工作组内 的计算或片元着色器调用的子组（subgroup，或称 'warp'）中 同步执行操作。通常，这些函数会使用子组内所有活动调用的数据 同步执行操作，然后将结果广播给所有活动调用。在其他图形 API 中， 子组函数也称为 wave 内建函数（DirectX/HLSL）或 warp 内建函数（CUDA）。

构造函数

new SubgroupFunctionNode( method : string, aNode : Node, bNode : Node )

构造一个新的函数节点。

method

要构造的子组/wave 内建方法。
aNode

该方法的第一个参数。

默认值为 null。
bNode

该方法的第二个参数。

默认值为 null。
