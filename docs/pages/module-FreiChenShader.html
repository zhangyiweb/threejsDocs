FreiChenShader

导入

FreiChenShader 是一个插件，必须显式导入，参见 Installation#Addons。

import { FreiChenShader } from 'three/addons/shaders/FreiChenShader.js';

属性

.FreiChenShader : ShaderMaterial~Shader (inner, constant)

使用 Frei-Chen 滤波器的边缘检测着色器。 基于 http://rastergrid.com/blog/2011/01/frei-chen-edge-detector。

aspect：vec2 类型，值为 (1/width, 1/height)
