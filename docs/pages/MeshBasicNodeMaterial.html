MeshBasicNodeMaterial

MeshBasicMaterial 的节点材质版本。

构造函数

new MeshBasicNodeMaterial( parameters : Object )

构造一个新的基础网格节点材质。

parameters

配置参数。

属性

.isMeshBasicNodeMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.lights : boolean

尽管基础材质从定义上不受光照影响，我们仍将 此属性设为 true，因为我们使用光照模型来计算 片元着色器的出射光。

默认值为 true。

重写： NodeMaterial#lights

方法

.setupEnvironment( builder : NodeBuilder ) : BasicEnvironmentNode.<vec3>

已重写，因为此类材质使用 BasicEnvironmentNode 实现默认环境贴图。

builder

当前节点构建器。
重写： NodeMaterial#setupEnvironment
返回值： 环境节点。

.setupLightMap( builder : NodeBuilder ) : BasicLightMapNode.<vec3>

必须重写此方法，因为基础材质的光照贴图 使用特殊的缩放因子进行求值。

builder

当前节点构建器。
重写： NodeMaterial#setupLightMap
返回值： 光照贴图节点。

.setupLightingModel() : BasicLightingModel

设置光照模型。

重写： NodeMaterial#setupLightingModel
返回值： 光照模型。

.setupNormal() : Node.<vec3>

基础材质不受法线贴图和凹凸贴图影响，因此我们 默认返回 normalViewGeometry。

重写： NodeMaterial#setupNormal
返回值： 法线节点。

.setupOutgoingLight() : Node.<vec3>

此材质重写了该方法，因为 lights 设为 true， 但我们仍希望将漫反射颜色作为出射光返回。

重写： NodeMaterial#setupOutgoingLight
返回值： 出射光节点。
