LightProbeGrid

由 L2 球谐辐照度探针组成的三维网格，用于提供 与位置相关的漫反射全局光照。

注意：此类仅可与 WebGLRenderer 配合使用。 适用于 WebGPURenderer 的版本将在稍后添加。

全部七个打包的 SH 子体积存储在单个 RGBA WebGL3DRenderTarget 中，沿 Z 轴采用纹理图集布局。 每个子体积占用 ( nz + 2 ) 个图集切片：两端各有一个填充切片 （复制最近的边缘数据切片），以防止硬件三线性过滤 跨子体积边界采样时出现颜色渗漏。

图集布局（nz = resolution.z，PADDING = 1）：

图集总深度 = 7 * ( nz + 2 )。

烘焙完全在 GPU 上完成：立方体贴图渲染、SH 投影以及 纹理打包均在 GPU 上执行，无需 CPU 回读。

代码示例

  slice   0              : padding  (copy of sub-volume 0, data slice 0)
  slices  1 … nz         : sub-volume 0 data
  slice   nz + 1         : padding  (copy of sub-volume 0, data slice nz-1)
  slice   nz + 2         : padding  (copy of sub-volume 1, data slice 0)
  slices  nz+3 … 2*nz+2  : sub-volume 1 data
  …

导入

LightProbeGrid 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { LightProbeGrid } from 'three/addons/lighting/LightProbeGrid.js';

构造函数

new LightProbeGrid( width : number, height : number, depth : number, widthProbes : number, heightProbes : number, depthProbes : number )

构造一个新的辐照度探针网格。

该体积以对象的位置为中心。

width

体积沿 X 轴的完整宽度。

默认值为 1。
height

体积沿 Y 轴的完整高度。

默认值为 1。
depth

体积沿 Z 轴的完整深度。

默认值为 1。
widthProbes

沿 X 轴的探针数量。默认为 Math.max( 2, Math.round( width ) + 1 )。
heightProbes

沿 Y 轴的探针数量。默认为 Math.max( 2, Math.round( height ) + 1 )。
depthProbes

沿 Z 轴的探针数量。默认为 Math.max( 2, Math.round( depth ) + 1 )。

属性

.boundingBox : Box3

网格的世界空间包围盒。由 LightProbeGrid#bake 自动更新。

.depth : number

体积沿 Z 轴的完整深度。

.height : number

体积沿 Y 轴的完整高度。

.isLightProbeGrid : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.resolution : Vector3

各轴上的探针数量。

.texture : Data3DTexture

存储全部七个打包 SH 子体积的单个 RGBA 图集三维纹理。

默认值为 null。

.width : number

体积沿 X 轴的完整宽度。

方法

.bake( renderer : WebGLRenderer, scene : Scene, options : Object )

通过在每个探针位置渲染立方体贴图并投影到 L2 SH 来烘焙所有探针。 可选地进行额外迭代以捕获间接反弹——每一额外遍次会采样上一遍次的 图集作为间接光，因此在烘焙前将网格加入场景时， 每多一遍就会累积一次反弹。

renderer

渲染器。
scene

要渲染的场景。
options

烘焙选项。

cubemapSize

每个立方体贴图面的分辨率。

默认值为 8。
near

立方体相机的近裁剪面。

默认值为 0.1。
far

立方体相机的远裁剪面。

默认值为 100。
bounces

初始直接光照遍次之后的额外反弹遍次。

默认值为 0。

.dispose()

释放 GPU 资源。

.getProbePosition( ix : number, iy : number, iz : number, target : Vector3 ) : Vector3

返回网格索引 (ix, iy, iz) 处探针的世界空间位置。

ix

X 索引。
iy

Y 索引。
iz

Z 索引。
target

目标向量。
返回值： 世界空间位置。

.updateBoundingBox()

根据当前位置和尺寸更新世界空间包围盒。
