RGBShiftNode

用于偏移/分离 RGB 颜色通道的后处理节点。该效果将颜色通道分离并相互偏移。

导入

RGBShiftNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { rgbShift } from 'three/addons/tsl/display/RGBShiftNode.js';

构造函数

new RGBShiftNode( textureNode : TextureNode, amount : number, angle : number )

构造一个新的 RGB 偏移节点。

textureNode

表示效果输入的纹理节点。
amount

RGB 偏移量。

默认值为 0.005。
angle

定义颜色偏移的方向。

默认值为 0。

属性

.amount : UniformNode.<float>

RGB 偏移量。

.angle : UniformNode.<float>

定义颜色偏移的方向。

.textureNode : TextureNode

表示效果输入的纹理节点。

方法

.setup( builder : NodeBuilder ) : ShaderCallNodeInternal

此方法用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup
