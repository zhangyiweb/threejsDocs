DenoiseNode

用于对原始屏幕空间环境光遮蔽（AO）输出等数据进行降噪的后处理节点。 降噪可以显著提升环境光遮蔽的质量，但也会给后处理管线带来相当大的 额外开销。最好将其设为可选功能（例如通过图形设置控制）。

参考：https://openaccess.thecvf.com/content/WACV2021/papers/Khademi_Self-Supervised_Poisson-Gaussian_Denoising_WACV_2021_paper.pdf。

导入

DenoiseNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { denoise } from 'three/addons/tsl/display/DenoiseNode.js';

构造函数

new DenoiseNode( textureNode : TextureNode, depthNode : Node.<float>, normalNode : Node.<vec3>, camera : Camera )

构造一个新的降噪节点。

textureNode

表示效果输入的纹理节点（例如 AO）。
depthNode

表示场景深度的节点。
normalNode

表示场景法线的节点。
camera

用于渲染场景的相机。

属性

.depthNode : Node.<float>

表示场景深度的节点。

.depthPhi : UniformNode.<float>

深度 Phi 值。

.index : UniformNode.<float>

索引。

.lumaPhi : UniformNode.<float>

亮度 Phi 值。

.noiseNode : TextureNode

表示内部噪声纹理的节点。

.normalNode : Node.<vec3>

表示场景法线的节点。若未向构造函数传入法线（因为 MRT 不可用）， 可在着色器中从深度值自动重建法线。

.normalPhi : UniformNode.<float>

法线 Phi 值。

.radius : UniformNode.<float>

半径。

.textureNode : TextureNode

表示效果输入的纹理节点（例如 AO）。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧更新一次内部 uniform。

默认值为 'frame'。

重写： TempNode#updateBeforeType

方法

.setup( builder : NodeBuilder ) : ShaderCallNodeInternal

用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

用于每帧更新一次内部 uniform。

frame

当前的节点帧。
重写： TempNode#updateBefore
