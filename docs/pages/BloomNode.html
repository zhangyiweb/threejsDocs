BloomNode

用于创建泛光（Bloom）效果的后处理节点。

const renderPipeline = new THREE.RenderPipeline( renderer );
const scenePass = pass( scene, camera );
const scenePassColor = scenePass.getTextureNode( 'output' );
const bloomPass = bloom( scenePassColor );
renderPipeline.outputNode = scenePassColor.add( bloomPass );

默认情况下，该节点会影响整个图像。如需选择性泛光， 可使用材质的 emissive 属性来控制哪些物体参与泛光。这可以通过 MRT 实现。

const renderPipeline = new THREE.RenderPipeline( renderer );
const scenePass = pass( scene, camera );
scenePass.setMRT( mrt( {
	output,
	emissive
} ) );
const scenePassColor = scenePass.getTextureNode( 'output' );
const emissivePass = scenePass.getTextureNode( 'emissive' );
const bloomPass = bloom( emissivePass );
renderPipeline.outputNode = scenePassColor.add( bloomPass );

导入

BloomNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { bloom } from 'three/addons/tsl/display/BloomNode.js';

构造函数

new BloomNode( inputNode : Node.<vec4>, strength : number, radius : number, threshold : number )

构造一个新的泛光节点。

inputNode

表示效果输入的节点。
strength

泛光强度。

默认值为 1。
radius

泛光半径。

默认值为 0。
threshold

亮度阈值，限制哪些明亮区域参与泛光效果。

默认值为 0。

属性

.highPassFn : function

可用于注入自定义高通滤波器（例如用于变形效果）。

.inputNode : Node.<vec4>

表示效果输入的节点。

.radius : UniformNode.<float>

泛光半径。必须在 [0,1] 范围内。

.smoothWidth : UniformNode.<float>

可用于微调从场景中提取的亮度。

.strength : UniformNode.<float>

泛光强度。

.threshold : UniformNode.<float>

亮度阈值，限制哪些明亮区域参与泛光效果。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

方法

.dispose()

释放内部资源。当不再需要该效果时应调用此方法。

重写： TempNode#dispose

.getResolutionScale() : number

获取当前通道的分辨率缩放比例。

返回值： 当前的分辨率缩放比例。1 表示全分辨率。

.getTextureNode() : PassTextureNode

以纹理节点的形式返回效果结果。

返回值： 表示效果结果的纹理节点。

.setResolutionScale( resolutionScale : number ) : BloomNode

设置通道的分辨率缩放比例。 分辨率缩放比例是一个与渲染器宽度和高度相乘的因子。

resolutionScale

要设置的分辨率缩放比例。1 表示全分辨率。
返回值： 对此节点的引用。

.setSize( width : number, height : number )

设置效果尺寸。

width

效果的宽度。
height

效果的高度。

.setup( builder : NodeBuilder ) : PassTextureNode

用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
