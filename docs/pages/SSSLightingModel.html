SSSLightingModel

表示 MeshSSSNodeMaterial 的光照模型。

构造函数

new SSSLightingModel( clearcoat : boolean, sheen : boolean, iridescence : boolean, anisotropy : boolean, transmission : boolean, dispersion : boolean, sss : boolean )

构造一个新的物理光照模型。

clearcoat

是否支持清漆（clearcoat）。

默认值为 false。
sheen

是否支持光泽（sheen）。

默认值为 false。
iridescence

是否支持虹彩（iridescence）。

默认值为 false。
anisotropy

是否支持各向异性（anisotropy）。

默认值为 false。
transmission

是否支持透射（transmission）。

默认值为 false。
dispersion

是否支持色散（dispersion）。

默认值为 false。
sss

是否支持 SSS。

默认值为 false。

属性

.useSSS : boolean

光照模型是否应使用 SSS。

默认值为 false。

方法

.direct( input : Object, builder : NodeBuilder )

在默认实现基础上扩展了 SSS 项。

参考：Approximating Translucency for a Fast, Cheap and Convincing Subsurface Scattering Look

input

输入数据。
builder

当前节点构建器。
重写： PhysicalLightingModel#direct
