TextureHelper

可用于显示任意类型纹理以进行调试的辅助对象。 根据纹理类型（2D、3D、Array），辅助对象会呈现为平面网格或盒体网格。

此辅助对象只能与 WebGLRenderer 一起使用。 使用 WebGPURenderer 时，请从 TextureHelperGPU.js 导入。

导入

TextureHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { TextureHelper } from 'three/addons/helpers/TextureHelper.js';

构造函数

new TextureHelper( texture : Texture, width : number, height : number, depth : number )

构造一个新的纹理辅助对象。

texture

要可视化的纹理。
width

辅助对象的宽度。

默认值为 1。
height

辅助对象的高度。

默认值为 1。
depth

辅助对象的深度。

默认值为 1。

属性

.texture : Texture

要可视化的纹理。

方法

.dispose()

释放此实例分配的 GPU 相关资源。 当应用中不再使用此实例时，请调用此方法。
