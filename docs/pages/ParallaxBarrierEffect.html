ParallaxBarrierEffect

用于创建视差屏障效果的类。

注意：此类仅可与 WebGLRenderer 配合使用。 使用 WebGPURenderer 时，请使用 ParallaxBarrierPassNode。

导入

ParallaxBarrierEffect 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { ParallaxBarrierEffect } from 'three/addons/effects/ParallaxBarrierEffect.js';

构造函数

new ParallaxBarrierEffect( renderer : WebGLRenderer )

构造一个新的视差屏障效果。

renderer

渲染器。

方法

.dispose()

释放内部资源。当不再需要该效果时应调用此方法。

.render( scene : Object3D, camera : Camera )

使用此效果时，应调用此方法而非默认的 WebGLRenderer#render。

scene

要渲染的场景。
camera

相机。

.setSize( width : number, height : number )

调整效果尺寸。

width

效果的宽度（逻辑像素）。
height

效果的高度（逻辑像素）。
