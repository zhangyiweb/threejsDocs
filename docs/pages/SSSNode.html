SSSNode

用于对场景应用屏幕空间阴影（SSS）的后处理节点。

屏幕空间阴影（也称为接触阴影）理想情况下应与传统阴影贴图配合使用。它们最适合 在较小尺度上渲染较小物体的精细阴影，例如高精度模型上的复杂阴影细节。换言之： 用阴影贴图作为基础，用屏幕空间阴影补充细节。

此实现产生的阴影边缘可能对某些用例来说过于生硬。 在与 beauty 通道合成之前，可使用 box、gaussian 或 hash 模糊来柔化边缘。代码示例：

限制：

  • 理想情况下最大阴影长度不应超过 1 米。否则，由于光线步进过程中需要评估更多采样， 效果在计算上会变得非常昂贵。可通过降低 quality 参数来缓解此问题。
  • 该效果只能与单个平行光（场景的主光源）配合使用。 此主光源通常代表太阳或日光。
  • 与其他屏幕空间技术一样，SSS 在阴影计算中只能考虑当前在相机视野内可见的物体。

参考：

  • https://panoskarabelas.com/posts/screen_space_shadows/。
  • https://www.bendstudio.com/blog/inside-bend-screen-space-shadows/。

代码示例

const sssPass = sss( scenePassDepth, camera, mainLight );
const sssBlur = boxBlur( sssPass.r, { size: 2, separation: 1 } ); // 可选模糊

导入

SSSNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { sss } from 'three/addons/tsl/display/SSSNode.js';

构造函数

new SSSNode( depthNode : TextureNode, camera : Camera, mainLight : DirectionalLight )

构造一个新的 SSS 节点。

depthNode

表示场景深度的纹理节点。
camera

用于渲染场景的相机。
mainLight

场景的主平行光。

属性

.depthNode : TextureNode

表示 beauty 通道深度的节点。

.maxDistance : UniformNode.<float>

世界单位下的最大阴影长度。更长的阴影会导致更多计算开销。

默认值为 0.1。

.quality : UniformNode.<float>

此参数控制光线步进的精细程度。 取值范围为 [0,1]，其中 1 表示最佳质量（最大光线步进迭代/采样数），0 表示完全不采样。

对于大多数用例，0.5 的质量通常已足够。请尽量保持此参数尽可能低。较大的值会导致明显更多的开销。

默认值为 0.5。

.resolutionScale : number

分辨率缩放。有效值在 [0,1] 范围内。1 表示最佳质量，但也会导致 更多计算开销。设为 0.5 表示以半分辨率计算效果。

默认值为 1。

.shadowIntensity : UniformNode.<float>

阴影强度。必须在 [0, 1] 范围内。

默认值为 1.0。

.thickness : UniformNode.<float>

深度测试厚度。

默认值为 0.01。

.updateBeforeType : string

updateBeforeType 被设置为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

.useTemporalFiltering : boolean

是否使用时序滤波。将此属性设为 true 需要使用 TRAANode。这有助于减少噪点， 但也会引入典型的 TAA 伪影，如拖影和时序不稳定。

默认值为 false。

方法

.dispose()

释放内部资源。当效果不再使用时应调用此方法。

重写： TempNode#dispose

.getTextureNode() : PassTextureNode

以纹理节点形式返回效果结果。

返回值： 表示效果结果的纹理节点。

.setSize( width : number, height : number )

设置效果的尺寸。

width

效果的宽度。
height

效果的高度。

.setup( builder : NodeBuilder ) : PassTextureNode

此方法用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

此方法用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
