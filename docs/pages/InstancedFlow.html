InstancedFlow

Flow 的实例化版本，用于使网格沿曲线弯曲，其中实例沿曲线放置。

此模块只能与 WebGLRenderer 一起使用。

导入

InstancedFlow 是一个插件，必须显式导入，参见 Installation#Addons。

import { InstancedFlow } from 'three/addons/modifiers/CurveModifier.js';

构造函数

new InstancedFlow( count : number, curveCount : number, geometry : Geometry, material : Material )

构造一个新的 InstancedFlow 实例。

count

实例化元素的数量。
curveCount

应预分配的曲线数量。
geometry

用于实例化网格的几何体。
material

用于实例化网格的材质。

类

InstancedFlow

方法

.moveIndividualAlongCurve( index : number, offset : number )

将单个元素沿曲线移动指定的量。

index

要更新的元素。
offset

偏移量。

.setCurve( index : number, curveNo : number )

选择某个元素应使用的曲线。

index

要更新的实例化元素的索引。
curveNo

该元素应使用的曲线索引。

.writeChanges( index : number )

关于使用哪条曲线以及在曲线上位置的额外信息，存储在实例化对象矩阵的平移分量中。 此方法将这些信息写入矩阵，并将其标记为需要更新。

index

要更新的实例化元素的索引。
