FogExp2

此类可用于定义二次指数雾：相机附近视野清晰， 随着距离增加，雾密度比指数增长更快。

代码示例

const scene = new THREE.Scene();
scene.fog = new THREE.FogExp2( 0xcccccc, 0.002 );

构造函数

new FogExp2( color : number | Color, density : number )

构造一个新的雾效果。

color

雾的颜色。
density

定义雾变浓的速度。

默认值为 0.00025。

属性

.color : Color

雾的颜色。

.density : number

定义雾变浓的速度。

默认值为 0.00025。

.isFogExp2 : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.name : string

雾的名称。

方法

.clone() : FogExp2

返回一个从此实例复制数值的新雾对象。

返回值： 此实例的克隆。

.toJSON( meta : Object | string ) : Object

将雾序列化为 JSON。

meta

可选值，保存有关序列化的元信息。
返回值： 表示序列化雾的 JSON 对象
