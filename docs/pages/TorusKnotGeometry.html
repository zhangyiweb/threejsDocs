TorusKnotGeometry

创建圆环结，其具体形状由一对互质整数 p 和 q 定义。若 p 与 q 不互质，结果将为圆环链。

代码示例

const geometry = new THREE.TorusKnotGeometry( 10, 3, 100, 16 );
const material = new THREE.MeshBasicMaterial( { color: 0xffff00 } );
const torusKnot = new THREE.Mesh( geometry, material );
scene.add( torusKnot );

构造函数

new TorusKnotGeometry( radius : number, tube : number, tubularSegments : number, radialSegments : number, p : number, q : number )

构造一个新的圆环结几何体。

radius

圆环结的半径。

默认值为 1.
tube

管道半径。

默认值为 0.4.
tubularSegments

管道方向分段数。

默认值为 64.
radialSegments

径向分段数。

默认值为 8.
p

该值决定几何体绕其旋转对称轴缠绕的次数。

默认值为 2.
q

该值决定几何体绕圆环内部一个圆缠绕的次数。

默认值为 3.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

静态方法

.fromJSON( data : Object ) : TorusKnotGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
