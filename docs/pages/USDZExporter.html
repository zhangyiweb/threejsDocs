USDZExporter

USDZ 导出器。

代码示例

const exporter = new USDZExporter();
const arraybuffer = await exporter.parseAsync( scene );

导入

USDZExporter 是一个插件，必须显式导入，参见 Installation#Addons。

import { USDZExporter } from 'three/addons/exporters/USDZExporter.js';

构造函数

new USDZExporter()

构造一个新的 USDZ 导出器。

属性

.textureUtils : WebGLTextureUtils | WebGPUTextureUtils

纹理工具模块的引用。

默认值为 null。

方法

.parse( scene : Object3D, onDone : USDZExporter~OnDone, onError : USDZExporter~OnError, options : USDZExporter~Options )

解析给定的 3D 对象并生成 USDZ 输出。

scene

要导出的 3D 对象。
onDone

导出完成时执行的回调函数。
onError

发生错误时执行的回调函数。
options

导出选项。

.parseAsync( scene : Object3D, options : USDZExporter~Options ) : Promise.<ArrayBuffer> (async)

USDZExporter#parse 的异步版本。

scene

要导出的 3D 对象。
options

导出选项。
返回值： 一个 Promise，解析后返回导出的 USDZ 数据。

.setTextureUtils( utils : WebGLTextureUtils | WebGPUTextureUtils )

设置此导出器的纹理工具。仅在需要导出压缩纹理时相关。

根据使用的是 WebGLRenderer 还是 WebGPURenderer，必须注入对应的纹理工具 WebGLTextureUtils 或 WebGPUTextureUtils。

utils

纹理工具。

类型定义

.OnDone( result : ArrayBuffer )

USDZExporter 的 onDone 回调。

result

生成的 USDZ 数据。

.OnError( error : Error )

USDZExporter 的 onError 回调。

error

错误对象。

.Options

USDZExporter 的导出选项。

maxTextureSize
number

导出时使用的最大纹理尺寸。

默认值为 1024。
includeAnchoringProperties
boolean

是否包含锚定属性。

默认值为 true。
onlyVisible
boolean

仅导出可见的 3D 对象。

默认值为 true。
ar
Object

若 includeAnchoringProperties 设为 true，可通过 ar.anchoring.type 和 ar.planeAnchoring.alignment 配置锚定类型和对齐方式。
quickLookCompatible
boolean

是否使导出的 USDZ 兼容 QuickLook，即修改资源以适配 Apple Feedback 中的 FB10036297 和 FB11442287 缺陷。

默认值为 false。
animations
Array.<AnimationClip>

要烘焙为目标对象上 xformOp 时间采样的动画剪辑。仅导出 position、quaternion 和 scale 轨道。

默认值为 []。
animationFrameRate
number

写入动画采样时使用的时间码（每秒帧数）。

默认值为 60。
