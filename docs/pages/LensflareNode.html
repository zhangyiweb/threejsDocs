LensflareNode

用于添加基于泛光的镜头光晕效果的后处理节点。此效果要求先通过泛光通道提取场景的泛光。

参考资料：

  • https://john-chapman-graphics.blogspot.com/2013/02/pseudo-lens-flare.html.
  • https://john-chapman.github.io/2017/11/05/pseudo-lens-flare.html.

导入

LensflareNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { lensflare } from 'three/addons/tsl/display/LensflareNode.js';

构造函数

new LensflareNode( textureNode : TextureNode, params : Object )

构造一个新的镜头光晕节点。

textureNode

表示场景泛光的纹理节点。
params

用于配置效果的参数对象。

ghostTint

定义光晕/鬼影的色调。

默认值为 vec3(1, 1, 1)。
threshold

控制效果的大小和强度。阈值越高，光晕越小。

默认值为 float(0.5)。
ghostSamples

表示每个亮斑围绕中心旋转的光晕/鬼影数量。

默认值为 float(4)。
ghostSpacing

定义光晕/鬼影的间距。

默认值为 float(0.25)。
ghostAttenuationFactor

定义光晕/鬼影的衰减因子。

默认值为 float(25)。
downSampleRatio

定义降采样比例，因为该效果通常不以全分辨率渲染。

默认值为 4。

属性

.downSampleRatio : number

定义降采样比例，因为该效果通常不以全分辨率渲染。

.ghostAttenuationFactorNode : Node.<float>

定义光晕/鬼影的衰减因子。

.ghostSamplesNode : Node.<float>

表示每个亮斑围绕中心旋转的光晕/鬼影数量。

.ghostSpacingNode : Node.<float>

定义光晕/鬼影的间距。

.ghostTintNode : Node.<vec3>

定义光晕/鬼影的色调。

.textureNode : TextureNode

表示场景泛光的纹理节点。

.thresholdNode : Node.<float>

控制效果的大小和强度。阈值越高，光晕越小。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧渲染一次效果。

默认值为 'frame'。

重写： TempNode#updateBeforeType

方法

.dispose()

释放内部资源。当不再需要该效果时应调用此方法。

重写： TempNode#dispose

.getTextureNode() : PassTextureNode

以纹理节点的形式返回效果结果。

返回值： 表示效果结果的纹理节点。

.setSize( width : number, height : number )

设置效果的尺寸。

width

效果的宽度。
height

效果的高度。

.setup( builder : NodeBuilder ) : PassTextureNode

此方法用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

此方法用于每帧渲染一次效果。

frame

当前的节点帧。
重写： TempNode#updateBefore
