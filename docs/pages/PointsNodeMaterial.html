PointsNodeMaterial

PointsMaterial 的节点材质版本。

该材质可以有两种用法：

  • 通过 Points 渲染点图元。由于 WebGPU 仅支持像素大小为 1 的点图元，因此无法定义尺寸。
const pointCloud = new THREE.Points( geometry, new THREE.PointsNodeMaterial() );
  • 通过精灵渲染点图元。在这种情况下会尊重尺寸， 参见 PointsNodeMaterial#sizeNode。
const instancedPoints = new THREE.Sprite( new THREE.PointsNodeMaterial( { positionNode: instancedBufferAttribute( positionAttribute ) } ) );

构造函数

new PointsNodeMaterial( parameters : Object )

构造一个新的点节点材质。

parameters

配置参数。

属性

.alphaToCoverage : boolean

是否应使用 alpha to coverage。

默认值为 true。

重写： SpriteNodeMaterial#alphaToCoverage

.isPointsNodeMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.sizeNode : Node.<vec2>

此节点属性提供了设置点大小的另一种方式。

注意：WebGPU 仅支持 1 像素大小的点图元。因此， 当该材质与 Points 和 WebGPU 后端一起使用时，此节点无效。若应用需要渲染大于 1 像素的点， 应将该材质与 Sprite 和实例化一起使用。

默认值为 null。
