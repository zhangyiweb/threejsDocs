MD2CharacterComplex

此类表示用于管理 MD2 动画角色资源的管理组件。与 MD2Character 相比，它提供了更丰富的 API。

导入

MD2CharacterComplex 是一个插件，必须显式导入，参见 Installation#Addons。

import { MD2CharacterComplex } from 'three/addons/misc/MD2CharacterComplex.js';

构造函数

new MD2CharacterComplex()

构造一个新的 MD2 角色。

属性

.angularSpeed : number

角色的角速度。

默认值为 2.5。

.animationFPS : number

帧率（FPS）。

默认值为 6。

.backAcceleration : number

角色的后退加速度。

默认值为 600。

.controls : Object

移动控制对象。

默认值为 null。

.currentSkin : Texture

当前皮肤纹理。

默认值为 undefined。

.frontAcceleration : number

角色的前进加速度。

默认值为 600。

.frontDeceleration : number

角色的前进减速度。

默认值为 600。

.maxReverseSpeed : number

角色的最大后退速度。

默认值为 - 275。

.maxSpeed : number

角色的最大前进速度。

默认值为 275。

.meshBody : Mesh

身体网格。

默认值为 null。

.meshWeapon : Mesh

武器网格。

默认值为 null。

.root : Object3D

根 3D 对象。

.scale : number

网格缩放比例。

默认值为 1。

.skinsBody : Array.<Texture>

身体皮肤纹理。

.skinsWeapon : Array.<Texture>

武器皮肤纹理。

.transitionFrames : number

过渡帧数。

默认值为 15。

.weapons : Array.<Mesh>

武器网格列表。

方法

.enableShadows( enable : boolean )

切换角色网格的阴影投射与接收。

enable

是否启用阴影。

.loadParts( config : Object )

根据给定配置加载角色模型。

config

定义模型和纹理路径的配置对象。

.setAnimation( animationName : string )

将指定动画片段设为当前活动动画。

animationName

动画片段名称。

.setPlaybackRate( rate : number )

设置动画播放速率。

rate

要设置的播放速率。

.setSkin( index : number )

根据给定皮肤索引设置皮肤。这将导致身体网格使用不同的纹理。

index

皮肤索引。

.setVisible( enable : boolean )

切换角色网格的可见性。

enable

角色是否可见。

.setWeapon( index : number )

根据给定武器索引设置武器。这将导致角色持有不同的武器。

index

武器索引。

.setWireframe( wireframeEnabled : boolean )

设置线框材质标志。

wireframeEnabled

是否启用线框渲染。

.shareParts( original : MD2CharacterComplex )

从另一个角色模型共享部分资源。

original

原始 MD2 角色。

.updateAnimations( delta : number )

更新网格动画。必须在动画循环中调用。

delta

增量时间（秒）。

.updateBehaviors()

根据控制输入更新动画状态。

.updateMovementModel( delta : number )

根据控制输入变换角色模型。

delta

增量时间（秒）。
