Storage3DTexture

这种特殊类型的纹理用于计算着色器。 可用计算着色器来计算纹理的数据。

注意：此类型纹理只能与 WebGPURenderer 和 WebGPU 后端一起使用。

构造函数

new Storage3DTexture( width : number, height : number, depth : number )

构造一个新的存储纹理。

width

存储纹理的宽度。

默认值为 1。
height

存储纹理的高度。

默认值为 1。
depth

存储纹理的深度。

默认值为 1。

属性

.image : Object

仅表示纹理尺寸的图像对象。

重写： Texture#image

.is3DTexture : boolean

指示此纹理是否为 3D 纹理。

.isStorageTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.magFilter : number

存储纹理的默认 magFilter 为 THREE.LinearFilter。

重写： Texture#magFilter

.minFilter : number

存储纹理的默认 minFilter 为 THREE.LinearFilter。

重写： Texture#minFilter

.wrapR : number

定义纹理在深度方向上的包裹方式，对应于 UVW 映射中的 W。

方法

.setSize( width : number, height : number, depth : number )

设置存储 3D 纹理的尺寸。

width

存储纹理的新宽度。
height

存储纹理的新高度。
depth

存储纹理的新深度。
