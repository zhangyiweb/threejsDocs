MaxMipLevelNode

一种特殊类型的 uniform 节点，用于计算给定纹理节点的最大 mipmap 级别。

代码示例

const level = maxMipLevel( textureNode );

构造函数

new MaxMipLevelNode( textureNode : TextureNode )

构造一个新的最大 mip 级别节点。

textureNode

要计算最大 mip 级别的纹理节点。

属性

.texture : Texture (readonly)

纹理。

.textureNode : TextureNode (readonly)

要计算最大 mip 级别的纹理节点。

.updateType : string

updateType 设置为 NodeUpdateType.FRAME，因为该节点在其 MaxMipLevelNode#update 方法中每帧更新一次纹理。

默认值为 'frame'。

重写： UniformNode#updateType
