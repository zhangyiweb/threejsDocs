VertexColorNode

用于表示顶点颜色的属性节点。

构造函数

new VertexColorNode( index : number )

构造一个新的顶点颜色节点。

index

属性索引。

属性

.index : number

属性索引，用于启用多套顶点颜色。

默认值为 0。

.isVertexColorNode : boolean (readonly)

此标志可用于类型测试。

默认值为 true。

方法

.getAttributeName( builder : NodeBuilder ) : string

重写默认实现，以考虑属性索引。

builder

当前节点构建器。
重写： AttributeNode#getAttributeName
返回值： 属性名称。
