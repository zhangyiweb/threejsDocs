BasicLightingModel

表示无光照材质的光照模型。唯一的光照贡献 是经过环境光遮蔽与材质漫反射颜色调制的烘焙间接光照。 支持环境贴图。用于 MeshBasicNodeMaterial。

构造函数

new BasicLightingModel()

构造一个新的基础光照模型。

方法

.finish( builder : NodeBuilder )

实现环境贴图。

builder

当前的节点构建器。
重写： LightingModel#finish

.indirect( builder : NodeBuilder )

实现带调制的烘焙间接光照。

builder

当前的节点构建器。
重写： LightingModel#indirect
