ArrayNode

ArrayNode 表示一组节点的集合，通常通过 array 函数创建。

代码示例

const colors = array( [
	vec3( 1, 0, 0 ),
	vec3( 0, 1, 0 ),
	vec3( 0, 0, 1 )
] );
const redColor = tintColors.element( 0 );

构造函数

new ArrayNode( nodeType : string, count : number, values : Array.<Node> )

构造一个新的数组节点。

nodeType

元素的数据类型。
count

数组大小。
values

数组的默认值。

默认值为 null。

属性

.count : number

数组大小。

.isArrayNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.values : Array.<Node>

数组的默认值。

方法

.generate( builder : NodeBuilder ) : string

此方法构建输出节点，并将结果数组作为着色器字符串返回。

builder

当前的节点构建器。
重写： TempNode#generate
返回值： 生成的着色器字符串。

.generateNodeType( builder : NodeBuilder ) : string

返回节点的类型。

builder

当前的节点构建器。
重写： TempNode#generateNodeType
返回值： 节点的类型。

.getArrayCount( builder : NodeBuilder ) : number

返回节点数组中的元素数量。

builder

当前的节点构建器。
重写： TempNode#getArrayCount
返回值： 节点数组中的元素数量。

.getElementType( builder : NodeBuilder ) : string

返回节点的类型。
