NodeMaterialLoader

用于加载节点材质的特殊类型材质加载器。

构造函数

new NodeMaterialLoader( manager : LoadingManager )

构造一个新的节点材质加载器。

manager

加载管理器的引用。

属性

.nodeMaterials : Object.<string, NodeMaterial.constructor>

表示节点材质类型的字典。

.nodes : Object.<string, Node.constructor>

表示节点类型的字典。

方法

.createMaterialFromType( type : string ) : Node

根据给定类型创建节点材质。

type

节点材质类型。
重写： MaterialLoader#createMaterialFromType
返回值： 创建的节点材质实例。

.parse( json : Object ) : NodeMaterial

从给定的 JSON 解析节点材质。

json

JSON 定义
重写： MaterialLoader#parse
返回值： 解析后的材质。

.setNodeMaterials( value : Object.<string, NodeMaterial.constructor> ) : NodeLoader

定义节点材质类型的字典。

value

以 <classname,class> 形式定义的节点材质库。
返回值： 此加载器的引用。

.setNodes( value : Object.<string, Node.constructor> ) : NodeLoader

定义节点类型的字典。

value

以 <classname,class> 形式定义的节点库。
返回值： 此加载器的引用。
