TileShadowNodeHelper

用于管理和显示 TileShadowNode 调试可视化的辅助类。

导入

TileShadowNodeHelper 是一个插件，必须显式导入，参见 Installation#Addons。

import { TileShadowNodeHelper } from 'three/addons/tsl/shadows/TileShadowNodeHelper.js';

构造函数

new TileShadowNodeHelper( tileShadowNode : TileShadowNode )

tileShadowNode

要调试的 TileShadowNode 实例。

方法

.dispose()

从场景中移除所有调试对象（平面和辅助对象）。

.init()

初始化调试显示（平面和相机辅助对象）。 应在 TileShadowNode 初始化其灯光和阴影节点之后调用。

.update()

更新调试可视化（特别是相机辅助对象）。 应在 TileShadowNode 的 update 方法中调用。
