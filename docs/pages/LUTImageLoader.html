LUTImageLoader

用于加载 LUT 图像的加载器。

代码示例

const loader = new LUTImageLoader();
const map = loader.loadAsync( 'luts/NeutralLUT.png' );

导入

LUTImageLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { LUTImageLoader } from 'three/addons/loaders/LUTImageLoader.js';

构造函数

new LUTImageLoader( manager : LoadingManager )

构造一个新的 LUT 加载器。

manager

加载管理器。

类

LUTImageLoader

属性

.flip : boolean

是否垂直翻转 LUT。

根据 LUT 的来源，纹理的绿色通道可能位于底部（例如 Unreal） 或位于顶部（例如 Unity URP Color Lookup）。如果你使用的是来自 Unity 管线的 LUT 图像条带，请将此属性设置为 true。

默认值为 false。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载的 LUT 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( dataArray : Uint8ClampedArray, size : number ) : Object

解析给定的 LUT 数据并返回生成的 3D 数据纹理。

dataArray

原始 LUT 数据。
size

LUT 尺寸。
重写： Loader#parse
返回值： 表示已解析 LUT 的对象。
