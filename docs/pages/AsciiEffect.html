AsciiEffect

用于创建 ASCII 效果的类。

ASCII 生成基于 jsascii。

导入

AsciiEffect 是一个附加组件，必须显式导入，参见 Installation#Addons。

import { AsciiEffect } from 'three/addons/effects/AsciiEffect.js';

构造函数

new AsciiEffect( renderer : WebGLRenderer, charSet : string, options : AsciiEffect~Options )

构造一个新的 ASCII 效果。

renderer

渲染器。
charSet

字符集。

默认值为 ' .:-=+*#%@'。
options

配置参数。

属性

.domElement : HTMLDivElement

该效果的 DOM 元素。必须使用此元素，而不是默认的 WebGLRenderer#domElement。

方法

.render( scene : Object3D, camera : Camera )

使用此效果时，应调用此方法而非默认的 WebGLRenderer#render。

scene

要渲染的场景。
camera

相机。

.setSize( w : number, h : number )

调整效果尺寸。

w

效果的宽度（逻辑像素）。
h

效果的高度（逻辑像素）。

类型定义

.Options

此类型表示 AsciiEffect 的配置设置。

resolution
number

值越高，细节越多。

默认值为 0.15。
scale
number

效果的缩放。

默认值为 1。
color
boolean

是否启用颜色。画质更好，但会降低渲染速度。

默认值为 false。
alpha
boolean

是否启用透明度。

默认值为 false。
block
boolean

是否启用块状字符。

默认值为 false。
invert
boolean

是否反转颜色。

默认值为 false。
strResolution
'low' | 'medium' | 'high'

字符串分辨率。

默认值为 'low'。
