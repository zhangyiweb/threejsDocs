DepthTexture

此类可用于将渲染的深度信息自动保存到纹理中。

构造函数

new DepthTexture( width : number, height : number, type : number, mapping : number, wrapS : number, wrapT : number, magFilter : number, minFilter : number, anisotropy : number, format : number, depth : number )

构造一个新的深度纹理。

width

纹理的宽度。
height

纹理的高度。
type

纹理类型。

默认值为 UnsignedIntType。
mapping

纹理映射方式。

默认值为 Texture.DEFAULT_MAPPING。
wrapS

wrapS 值。

默认值为 ClampToEdgeWrapping。
wrapT

wrapT 值。

默认值为 ClampToEdgeWrapping。
magFilter

放大过滤器值。

默认值为 LinearFilter。
minFilter

缩小过滤器值。

默认值为 LinearFilter。
anisotropy

各向异性值。

默认值为 Texture.DEFAULT_ANISOTROPY。
format

纹理格式。

默认值为 DepthFormat。
depth

纹理的深度。

默认值为 1。

属性

.compareFunction : NeverCompare | LessCompare | EqualCompare | LessEqualCompare | GreaterCompare | NotEqualCompare | GreaterEqualCompare | AlwaysCompare

对应深度比较函数的代码。

默认值为 null。

.flipY : boolean

若设为 true，纹理在上传到 GPU 时会沿垂直轴翻转。

已覆盖，默认设置为 false。

默认值为 false。

重写： Texture#flipY

.generateMipmaps : boolean

是否为纹理生成 mipmap（若可能）。

已覆盖，默认设置为 false。

默认值为 false。

重写： Texture#generateMipmaps

.isDepthTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。
