BufferNode

一种特殊类型的 uniform 节点，将类数组数据表示为 uniform 缓冲区。访问通常通过 element() 完成， 它会返回 ArrayElementNode 的实例。例如：

一般建议使用管理更完善的 UniformArrayNode， 因为它能处理更多输入类型，并自动处理缓冲区填充。

代码示例

const bufferNode = buffer( array, 'mat4', count );
const matrixNode = bufferNode.element( index ); // access a matrix from the buffer

构造函数

new BufferNode( value : Array.<number>, bufferType : string, bufferCount : number )

构造一个新的缓冲区节点。

value

类数组的缓冲区数据。
bufferType

缓冲区的数据类型。
bufferCount

缓冲区元素的数量。

默认值为 0。

属性

.bufferCount : number

保存引用节点值的 uniform 节点。

默认值为 0。

.bufferType : string

缓冲区的数据类型。

.isBufferNode : boolean (readonly)

该标志可用于类型检测。

默认值为 true。

.updateRanges : Array.<{start: number, count: number}>

更新范围的数组。

方法

.addUpdateRange( start : number, count : number )

添加数据数组中需要在 GPU 上更新的一段数据范围。

start

开始更新的位置。
count

要更新的组件数量。

.clearUpdateRanges()

清除更新范围。

.getElementType( builder : NodeBuilder ) : string

缓冲区元素的数据类型。

builder

当前的节点构建器。
重写： UniformNode#getElementType
返回值： 元素类型。

.getInputType( builder : NodeBuilder ) : string

重写默认实现，返回固定值 'buffer'。

builder

当前的节点构建器。
重写： UniformNode#getInputType
返回值： 输入类型。
