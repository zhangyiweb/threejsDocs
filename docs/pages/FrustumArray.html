FrustumArray

FrustumArray 用于判断对象是否至少在相机数组中的一台相机内可见。 这对多视图渲染器尤其有用。

构造函数

new FrustumArray()

构造一个新的视锥体数组。

属性

.coordinateSystem : WebGLCoordinateSystem | WebGPUCoordinateSystem

要使用的坐标系。

默认值为 WebGLCoordinateSystem。

方法

.clone() : FrustumArray

返回一个新的视锥体数组，其值从本实例复制而来。

返回值： 本实例的克隆。

.containsPoint( point : Vector3 ) : boolean

若给定点位于任意已缓存的视锥体内，则返回 true。

在调用此方法之前，必须在每次渲染时调用一次 FrustumArray#setFromArrayCamera。

point

要测试的点。
返回值： 该点是否在任意相机中可见。

.copy( frustumArray : FrustumArray ) : FrustumArray

将给定视锥体数组的值复制到本实例。

frustumArray

要复制的视锥体数组。
返回值： 对本视锥体数组的引用。

.intersectsBox( box : Box3 ) : boolean

若给定包围盒与任意已缓存的视锥体相交，则返回 true。

在调用此方法之前，必须在每次渲染时调用一次 FrustumArray#setFromArrayCamera。

box

要测试的包围盒。
返回值： 该包围盒是否在任意相机中可见。

.intersectsObject( object : Object3D ) : boolean

若三维对象的包围球与任意已缓存的视锥体相交，则返回 true。

在调用此方法之前，必须在每次渲染时调用一次 FrustumArray#setFromArrayCamera。

object

要测试的三维对象。
返回值： 该三维对象是否在任意相机中可见。

.intersectsSphere( sphere : Sphere ) : boolean

若给定包围球与任意已缓存的视锥体相交，则返回 true。

在调用此方法之前，必须在每次渲染时调用一次 FrustumArray#setFromArrayCamera。

sphere

要测试的包围球。
返回值： 该包围球是否在任意相机中可见。

.intersectsSprite( sprite : Sprite ) : boolean

若给定精灵与任意已缓存的视锥体相交，则返回 true。

在调用此方法之前，必须在每次渲染时调用一次 FrustumArray#setFromArrayCamera。

sprite

要测试的精灵。
返回值： 该精灵是否在任意相机中可见。

.setFromArrayCamera( cameraArray : ArrayCamera ) : FrustumArray

为给定阵列相机中的每台相机计算并缓存一个视锥体。

cameraArray

其子相机用于定义视锥体的阵列相机。
返回值： 对本视锥体数组的引用。
