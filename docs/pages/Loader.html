Loader

加载器的抽象基类。

构造函数

new Loader( manager : LoadingManager ) (abstract)

创建一个新的加载器。

manager

加载管理器。

属性

.crossOrigin : string

用于实现 CORS 的 crossOrigin 字符串，以便从允许 CORS 的不同域加载 URL。

默认值为 'anonymous'。

.manager : LoadingManager

加载管理器。

默认值为 DefaultLoadingManager。

.path : string

资源将从中加载的基础路径。

.requestHeader : Object.<string, any>

HTTP 请求中使用的 请求头。

.resourcePath : string

附加资源（如纹理）将从中加载的基础路径。

.withCredentials : boolean

XMLHttpRequest 是否使用凭据。

默认值为 false。

.DEFAULT_MATERIAL_NAME : string

加载器在为已加载的 3D 对象创建材质时使用的默认材质名称。

注意：并非所有加载器都会遵循此设置。

默认值为 '__DEFAULT'。

方法

.abort() : Loader (abstract)

可在加载器中实现此方法，以中止正在进行的请求。

返回值： 此实例的引用。

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback ) (abstract)

所有具体加载器都需要实现此方法。它包含从后端加载资源的逻辑。

url

要加载文件的路径/URL。
onLoad

加载过程完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。

.loadAsync( url : string, onProgress : onProgressCallback ) : Promise

Loader#load 的异步版本。

url

要加载文件的路径/URL。
onProgress

加载进行中时执行。
返回值： 资源加载完成后解析的 Promise。

.parse( data : any ) (abstract)

所有具体加载器都需要实现此方法。它包含将资源解析为 three.js 实体的逻辑。

data

要解析的数据。

.setCrossOrigin( crossOrigin : string ) : Loader

设置 crossOrigin 字符串，以便从允许 CORS 的不同域加载 URL。

crossOrigin

crossOrigin 值。
返回值： 此实例的引用。

.setPath( path : string ) : Loader

设置资源的基础路径。

path

基础路径。
返回值： 此实例的引用。

.setRequestHeader( requestHeader : Object ) : Loader

设置给定的请求头。

requestHeader

用于配置 HTTP 请求的 请求头。
返回值： 此实例的引用。

.setResourcePath( resourcePath : string ) : Loader

设置依赖资源（如纹理）的基础路径。

resourcePath

资源路径。
返回值： 此实例的引用。

.setWithCredentials( value : boolean ) : Loader

XMLHttpRequest 是否使用凭据（如 cookie、授权头或 TLS 客户端证书），参见 XMLHttpRequest.withCredentials。

注意：如果从本地或同一域加载文件，此设置无效。

value

withCredentials 值。
返回值： 此实例的引用。
