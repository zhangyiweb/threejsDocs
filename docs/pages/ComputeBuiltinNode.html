ComputeBuiltinNode

ComputeBuiltinNode 表示计算作用域中的内置值，用于暴露当前正在运行的 调度以及/或者运行设备的相关信息。

此节点只能与 WebGPU 后端一起使用。

构造函数

new ComputeBuiltinNode( builtinName : string, nodeType : string )

构造一个新的计算内置节点。

builtinName

内置名称。
nodeType

节点类型。

方法

.generateNodeType( builder : NodeBuilder ) : string

此方法被重写，因为节点类型直接由 nodeType 推导。

builder

当前的节点构建器。
重写： Node#generateNodeType
返回值： 节点类型。

.getBuiltinName( builder : NodeBuilder ) : string

返回内置名称。

builder

当前的节点构建器。
返回值： 内置名称。

.getHash( builder : NodeBuilder ) : string

此方法被重写，因为哈希值由内置名称推导。

builder

当前的节点构建器。
重写： Node#getHash
返回值： 哈希值。

.hasBuiltin( builder : NodeBuilder ) : boolean

当前节点构建器是否具有该内置项。

builder

当前的节点构建器。
返回值： 构建器是否具有该内置项。

.setBuiltinName( builtinName : string ) : ComputeBuiltinNode

设置内置名称。

builtinName

内置名称。
返回值： 对此节点的引用。
