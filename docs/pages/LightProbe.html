LightProbe

光照探针是向 3D 场景添加光照的另一种方式。与经典光源 （例如平行光、点光源或聚光灯）不同，光照探针本身不发光。 它们存储穿过 3D 空间的光照信息。在渲染过程中， 击中 3D 物体的光照会通过使用光照探针中的数据来近似计算。

光照探针通常由（辐射）环境贴图创建。 类 LightProbeGenerator 可用于从立方体贴图或渲染目标创建光照探针。 不过，光照估计数据也可以以其他形式提供，例如通过 WebXR。 这使得能够渲染对真实世界光照做出响应的增强现实内容。

three.js 中当前的探针实现支持所谓的漫反射光照探针。 这类光照探针在功能上等效于辐照度环境贴图。

构造函数

new LightProbe( sh : SphericalHarmonics3, intensity : number )

构造一个新的光照探针。

sh

表示编码光照信息的球谐函数。
intensity

光源强度。

默认值为 1。

属性

.isLightProbe : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.sh : SphericalHarmonics3

光照探针使用球谐函数来编码光照信息。
