NodeVar

NodeBuilder 在构建节点的过程中会创建此类的实例。 它们表示构建器最终将生成的着色器变量。为此，节点变量字典维护在 NodeBuilder#vars 中。

构造函数

new NodeVar( name : string, type : string, readOnly : boolean, count : number )

构造一个新的节点变量。

name

变量的名称。
type

变量的类型。
readOnly

只读标志。

默认值为 false。
count

大小。

默认值为 null。

属性

.count : number

大小。

.isNodeVar : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.name : string

变量的名称。

.readOnly : boolean

只读标志。

.type : string

变量的类型。
