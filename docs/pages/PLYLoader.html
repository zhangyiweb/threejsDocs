PLYLoader

用于加载 PLY 格式（又称 Polygon File Format 或 Stanford Triangle Format）的加载器。

限制：

  • ASCII 解码假定文件为 UTF-8 编码。

代码示例

const loader = new PLYLoader();
const geometry = await loader.loadAsync( './models/ply/ascii/dolphins.ply' );
scene.add( new THREE.Mesh( geometry ) );

导入

PLYLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { PLYLoader } from 'three/addons/loaders/PLYLoader.js';

构造函数

new PLYLoader( manager : LoadingManager )

构造一个新的 PLY 加载器。

manager

加载管理器。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载完成的 PLY 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( data : ArrayBuffer ) : BufferGeometry

解析给定的 PLY 数据并返回生成的几何体。

data

原始 PLY 数据 ArrayBuffer。
重写： Loader#parse
返回值： 解析后的几何体。

.setCustomPropertyNameMapping( mapping : Object )

position、uv、normal 和 color 属性以外的自定义属性 可通过 setCustomPropertyNameMapping 方法添加。 例如，以下代码将元素属性 "custom_property_a" 和 "custom_property_b" 映射到 item size 为 2 的属性 "customAttribute"。 属性 item size 由属性数组中元素属性的数量决定。

loader.setCustomPropertyNameMapping( {
	customAttribute: ['custom_property_a', 'custom_property_b'],
} );
mapping

映射字典。

.setPropertyNameMapping( mapping : Object )

设置属性名称映射，将默认属性名 映射到自定义名称。例如，以下代码将文件中的 "diffuse_(red|green|blue)" 属性映射到标准颜色名称。

loader.setPropertyNameMapping( {
	diffuse_red: 'red',
	diffuse_green: 'green',
	diffuse_blue: 'blue'
} );
mapping

映射字典。
