Material

材质的抽象基类。

材质定义可渲染 3D 对象的外观。

构造函数

new Material() (abstract)

构造一个新材质。

属性

.allowOverride : boolean

是否允许用 Scene#overrideMaterial 覆盖此材质。

默认值为 true。

.alphaHash : boolean

启用 alpha 哈希透明度，作为 Material#transparent 或 Material#alphaTest 的替代方案。若不透明度低于 随机阈值，则不会渲染该材质。随机化会引入一些颗粒或噪点，但可近似实现 alpha 混合，同时避免排序相关问题。使用 TAA 可减少由此产生的噪点。

默认值为 false。

.alphaTest : number (readonly)

设置运行 alpha 测试时使用的 alpha 值。若不透明度 低于此值，则不会渲染该材质。

默认值为 0。

.alphaToCoverage : boolean

是否应启用 alpha to coverage。仅可在启用 MSAA 的上下文中使用 （即创建渲染器时将 antialias 参数设为 true）。启用后 可平滑裁剪平面边缘以及经 alphaTest 裁剪的边缘锯齿。

默认值为 false。

.blendAlpha : number

表示常量混合颜色的 alpha 值。

仅在使用自定义混合且混合因子为 ConstantAlpha 或 OneMinusConstantAlpha 时生效。

默认值为 0。

.blendColor : Color

表示常量混合颜色的 RGB 值。

仅在使用自定义混合且混合因子为 ConstantColor 或 OneMinusConstantColor 时生效。

默认值为 (0,0,0)。

.blendDst : ZeroFactor | OneFactor | SrcColorFactor | OneMinusSrcColorFactor | SrcAlphaFactor | OneMinusSrcAlphaFactor | DstAlphaFactor | OneMinusDstAlphaFactor | DstColorFactor | OneMinusDstColorFactor | SrcAlphaSaturateFactor | ConstantColorFactor | OneMinusConstantColorFactor | ConstantAlphaFactor | OneMinusConstantAlphaFactor

定义混合目标因子。

默认值为 OneMinusSrcAlphaFactor。

.blendDstAlpha : ZeroFactor | OneFactor | SrcColorFactor | OneMinusSrcColorFactor | SrcAlphaFactor | OneMinusSrcAlphaFactor | DstAlphaFactor | OneMinusDstAlphaFactor | DstColorFactor | OneMinusDstColorFactor | SrcAlphaSaturateFactor | ConstantColorFactor | OneMinusConstantColorFactor | ConstantAlphaFactor | OneMinusConstantAlphaFactor

定义混合目标 alpha 因子。

默认值为 null。

.blendEquation : AddEquation | SubtractEquation | ReverseSubtractEquation | MinEquation | MaxEquation

定义混合方程。

默认值为 AddEquation。

.blendEquationAlpha : AddEquation | SubtractEquation | ReverseSubtractEquation | MinEquation | MaxEquation

定义 alpha 通道的混合方程。

默认值为 null。

.blendSrc : ZeroFactor | OneFactor | SrcColorFactor | OneMinusSrcColorFactor | SrcAlphaFactor | OneMinusSrcAlphaFactor | DstAlphaFactor | OneMinusDstAlphaFactor | DstColorFactor | OneMinusDstColorFactor | SrcAlphaSaturateFactor | ConstantColorFactor | OneMinusConstantColorFactor | ConstantAlphaFactor | OneMinusConstantAlphaFactor

定义混合源因子。

默认值为 SrcAlphaFactor。

.blendSrcAlpha : ZeroFactor | OneFactor | SrcColorFactor | OneMinusSrcColorFactor | SrcAlphaFactor | OneMinusSrcAlphaFactor | DstAlphaFactor | OneMinusDstAlphaFactor | DstColorFactor | OneMinusDstColorFactor | SrcAlphaSaturateFactor | ConstantColorFactor | OneMinusConstantColorFactor | ConstantAlphaFactor | OneMinusConstantAlphaFactor

定义混合源 alpha 因子。

默认值为 null。

.blending : NoBlending | NormalBlending | AdditiveBlending | SubtractiveBlending | MultiplyBlending | CustomBlending

定义材质的混合类型。

若要使 Material#blendSrc、Material#blendDst 或 Material#blendEquation 等自定义混合属性生效，必须将其设为 CustomBlending。

默认值为 NormalBlending。

.clipIntersection : boolean

改变裁剪平面的行为，使裁剪的是它们的交集， 而非并集。

默认值为 false。

.clipShadows : boolean

定义是否根据此材质上指定的裁剪平面 裁剪阴影。

默认值为 false。

.clippingPlanes : Array.<Plane>

用户定义的裁剪平面，以世界空间中的 THREE.Plane 对象指定。 这些平面作用于使用此材质的对象。 到平面的有符号距离为负的空间点会被裁剪 （不渲染）。这需要 WebGLRenderer#localClippingEnabled 为 true。

默认值为 null。

.colorWrite : boolean

是否渲染材质颜色。

可与 Object3D#renderOder 结合使用，以创建遮挡其他对象的不可见对象。

默认值为 true。

.depthFunc : NeverDepth | AlwaysDepth | LessDepth | LessEqualDepth | EqualDepth | GreaterEqualDepth | GreaterDepth | NotEqualDepth

定义深度函数。

默认值为 LessEqualDepth。

.depthTest : boolean

渲染此材质时是否启用深度测试。 禁用深度测试时，深度写入也会被隐式禁用。

默认值为 true。

.depthWrite : boolean

渲染此材质是否对深度缓冲产生影响。

绘制 2D 叠加层时，禁用深度写入可能很有用， 以便将多层内容叠在一起而不产生 z-index 伪影。

默认值为 true。

.dithering : boolean

是否对颜色应用抖动，以消除色带感。

默认值为 false。

.forceSinglePass : boolean

双面透明对象是否应单次通道渲染。

引擎默认以两次绘制调用渲染双面透明对象（先背面， 后正面），以减轻透明度伪影。但在某些场景下， 这种方法不会提升质量，却会使绘制调用翻倍，例如渲染扁平 植被（如草地精灵）时。在这些情况下，可将 forceSinglePass 标志设为 true， 以禁用双通道渲染，避免性能问题。

默认值为 false。

.id : number (readonly)

材质的 ID。

.isMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.name : string

材质名称。

.needsUpdate : boolean

将此属性设为 true 表示引擎需要重新编译该材质。

默认值为 false。

.opacity : number

定义材质的透明程度。 0.0 表示完全透明，1.0 表示完全不透明。

若 Material#transparent 未设为 true， 材质将保持完全不透明，此值仅影响其颜色。

默认值为 1。

.polygonOffset : boolean

是否使用多边形偏移。启用后，每个片元的深度值在从相应顶点的深度值插值后会被偏移。 偏移在执行深度测试之前、将值写入深度缓冲之前添加。

可用于渲染隐藏线图像、在表面贴贴花，以及渲染带高亮边缘的实体。

默认值为 false。

.polygonOffsetFactor : number

指定用于为每个多边形创建可变深度偏移的比例因子。

默认值为 0。

.polygonOffsetUnits : number

与实现相关的值相乘，以创建恒定深度偏移。

默认值为 0。

.precision : 'highp' | 'mediump' | 'lowp'

覆盖渲染器对此材质的默认精度。

默认值为 null。

.premultipliedAlpha : boolean

是否预乘 alpha（透明度）值。

默认值为 false。

.shadowSide : FrontSide | BackSide | DoubleSide

定义哪一面投射阴影。若为 null，投射阴影的面按如下规则确定：

  • 当 Material#side 设为 FrontSide 时，背面投射阴影。
  • 当 Material#side 设为 BackSide 时，正面投射阴影。
  • 当 Material#side 设为 DoubleSide 时，双面均投射阴影。

默认值为 null。

.side : FrontSide | BackSide | DoubleSide

定义渲染哪一面——正面、背面或双面。

默认值为 FrontSide。

.stencilFail : ZeroStencilOp | KeepStencilOp | ReplaceStencilOp | IncrementStencilOp | DecrementStencilOp | IncrementWrapStencilOp | DecrementWrapStencilOp | InvertStencilOp

当比较函数返回 false 时执行的模板操作。

默认值为 KeepStencilOp。

.stencilFunc : NeverStencilFunc | LessStencilFunc | EqualStencilFunc | LessEqualStencilFunc | GreaterStencilFunc | NotEqualStencilFunc | GreaterEqualStencilFunc | AlwaysStencilFunc

要使用的模板比较函数。

默认值为 AlwaysStencilFunc。

.stencilFuncMask : number

与模板缓冲比较时使用的位掩码。

默认值为 0xff。

.stencilRef : number

执行模板比较或模板操作时使用的值。

默认值为 0。

.stencilWrite : boolean

是否对模板缓冲执行模板操作。要对模板缓冲进行写入或比较， 此值必须为 true。

默认值为 false。

.stencilWriteMask : number

写入模板缓冲时使用的位掩码。

默认值为 0xff。

.stencilZFail : ZeroStencilOp | KeepStencilOp | ReplaceStencilOp | IncrementStencilOp | DecrementStencilOp | IncrementWrapStencilOp | DecrementWrapStencilOp | InvertStencilOp

当比较函数返回 true 但深度测试失败时执行的模板操作。

默认值为 KeepStencilOp。

.stencilZPass : ZeroStencilOp | KeepStencilOp | ReplaceStencilOp | IncrementStencilOp | DecrementStencilOp | IncrementWrapStencilOp | DecrementWrapStencilOp | InvertStencilOp

当比较函数返回 true 且深度测试通过时执行的模板操作。

默认值为 KeepStencilOp。

.toneMapped : boolean

定义此材质是否根据渲染器的色调映射设置进行色调映射。

渲染到渲染目标、使用后期处理或使用 WebGPURenderer 时会被忽略。在这些情况下，所有材质都会应用色调映射。

默认值为 true。

.transparent : boolean

定义此材质是否透明。这会影响渲染， 因为透明对象需要特殊处理，并在非透明对象之后渲染。

设为 true 时，材质的透明程度由 Material#opacity 控制。

默认值为 false。

.type : string (readonly)

type 属性用于在序列化/反序列化上下文中 检测对象类型。

.userData : Object

可用于存储关于材质的自定义数据的对象。 不应持有函数引用，因为这些不会被克隆。

.uuid : string (readonly)

材质的 UUID。

.version : number (readonly)

从 0 开始，统计 Material#needsUpdate 被设为 true 的次数。

默认值为 0。

.vertexColors : boolean

若设为 true，应使用顶点颜色。

引擎支持 RGB 和 RGBA 顶点颜色，取决于使用的是三分量（RGB）还是 四分量（RGBA）颜色缓冲属性。

默认值为 false。

.visible : boolean

定义使用此材质的 3D 对象是否可见。

默认值为 true。

方法

.clone() : Material

返回一个从此实例复制值的新材质。

返回值： 此实例的克隆。

.copy( source : Material ) : Material

将给定材质的值复制到此实例。

source

要复制的材质。
返回值： 此实例的引用。

.customProgramCacheKey() : string

若使用了 Material#onBeforeCompile，可用此回调标识 onBeforeCompile() 中所用的设置值，以便 three.js 能按需重用缓存的 着色器或为此材质重新编译着色器。

此方法仅可在使用 WebGLRenderer 渲染时使用。

返回值： 自定义程序缓存键。

.dispose()

释放此实例分配的 GPU 相关资源。当应用中 不再使用此实例时，请调用此方法。

触发：
  • Material#event:dispose

.fromJSON( json : Object, textures : Object.<string, Texture> ) : Material

从给定的 JSON 反序列化材质。

json

保存已序列化材质的 JSON。
textures

保存材质所引用纹理的字典。
返回值： 此材质的引用。

.onBeforeCompile( shaderobject : Object, renderer : WebGLRenderer )

可选回调，在着色器程序编译前立即执行。 该函数以着色器源代码为参数调用。适用于修改内置材质。

此方法仅可在使用 WebGLRenderer 渲染时使用。自定义材质的 推荐方式是使用 WebGPURenderer 配合新的 节点材质系统与 TSL。

shaderobject

该对象包含 uniform 以及顶点与片元着色器源代码。
renderer

渲染器的引用。

.onBeforeRender( renderer : WebGLRenderer, scene : Scene, camera : Camera, geometry : BufferGeometry, object : Object3D, group : Object )

可选回调，在材质用于渲染 3D 对象之前立即执行。

此方法仅可在使用 WebGLRenderer 渲染时使用。

renderer

渲染器。
scene

场景。
camera

用于渲染场景的相机。
geometry

3D 对象的几何体。
object

3D 对象。
group

几何体分组数据。

.setValues( values : Object )

此方法可用于从参数对象设置默认值。 它是通用实现，因此可与不同类型的材质一起使用。

values

要设置的材质值。

.toJSON( meta : Object | string ) : Object

将材质序列化为 JSON。

meta

可选值，保存关于序列化的元信息。
参见：
  • ObjectLoader#parse
返回值： 表示已序列化材质的 JSON 对象。
