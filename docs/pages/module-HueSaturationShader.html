HueSaturationShader

导入

HueSaturationShader 是一个插件，必须显式导入，参见 Installation#Addons。

import { HueSaturationShader } from 'three/addons/shaders/HueSaturationShader.js';

属性

.HueSaturationShader : ShaderMaterial~Shader (inner, constant)

色相和饱和度调整，https://github.com/evanw/glfx.js。

色相：-1 到 1（-1 为负方向旋转 180 度，0 为无变化，依此类推） 饱和度：-1 到 1（-1 为纯灰，0 为无变化，1 为最大对比度）
