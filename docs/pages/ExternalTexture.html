ExternalTexture

表示在相同渲染器上下文中外部创建的纹理。

这可能来自受保护的媒体流、设备摄像头画面， 或其他数据源（如深度传感器）。

构造函数

new ExternalTexture( sourceTexture : WebGLTexture | GPUTexture )

创建一个新的原始纹理。

sourceTexture

外部纹理。

默认值为 null。

属性

.isExternalTexture : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.sourceTexture : WebGLTexture | GPUTexture

外部源纹理。

默认值为 null。
