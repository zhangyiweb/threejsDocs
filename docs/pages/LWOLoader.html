LWOLoader

LWO 格式的加载器。

支持 LWO3 和 LWO2 格式。

参考：

  • LWO3 format specification
  • LWO2 format specification

代码示例

const loader = new LWOLoader();
const lwoData = await loader.loadAsync( 'models/lwo/Objects/LWO3/Demo.lwo' );
const mesh = object.meshes[ 0 ];
scene.add( mesh );

导入

LWOLoader 是一个插件，必须显式导入，参见 Installation#Addons。

import { LWOLoader } from 'three/addons/loaders/LWOLoader.js';

构造函数

new LWOLoader( manager : LoadingManager )

构造一个新的 LWO 加载器。

manager

加载管理器。
已弃用： 自 r185 起。

方法

.load( url : string, onLoad : function, onProgress : onProgressCallback, onError : onErrorCallback )

从指定 URL 开始加载，并将加载的 LWO 资源 传递给 onLoad() 回调。

url

要加载的文件路径/URL，也可以是 data URI。
onLoad

加载完成时执行。
onProgress

加载进行中时执行。
onError

发生错误时执行。
重写： Loader#load

.parse( iffBuffer : ArrayBuffer, path : string, modelName : string ) : Object

解析给定的 LWO 数据并返回生成的网格和材质。

iffBuffer

作为 ArrayBuffer 的原始 LWO 数据。
path

URL 基础路径。
modelName

模型名称。
重写： Loader#parse
返回值： 包含已解析网格和材质的对象。
