TileShadowNode

扩展 ShadowBaseNode 以实现瓦片化阴影贴图的类。 它允许将阴影贴图拆分为多个瓦片，每个瓦片拥有独立的光源和相机， 从而提升大型场景的阴影质量与性能。

注意：此类目前不支持 VSMShadowMap。

导入

TileShadowNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { TileShadowNode } from 'three/addons/tsl/shadows/TileShadowNode.js';

构造函数

new TileShadowNode( light : Light, options : Object )

创建 TileShadowNode 实例。

light

用于阴影贴图的原始光源。
options

瓦片化阴影节点的配置选项。

默认值为 {}。

tilesX

X 轴方向的瓦片数量。

默认值为 2。
tilesY

Y 轴方向的瓦片数量。

默认值为 2。
resolution

阴影贴图的分辨率。
debug

是否启用调试模式。

默认值为 false。

方法

.disposeLightsAndNodes()

辅助方法，用于移除光源及关联的节点/目标。 在 dispose 和可能的重新初始化过程中内部使用。

.generateTiles( tilesX : number, tilesY : number ) : Array.<Object>

根据指定的 X、Y 轴瓦片数量生成阴影贴图的瓦片。

tilesX

X 轴方向的瓦片数量。
tilesY

Y 轴方向的瓦片数量。
返回值： 瓦片对象数组，每个对象包含瓦片的边界与索引。

.init( builder : Builder )

初始化瓦片化阴影节点，为每个瓦片创建光源、相机和阴影贴图。

builder

用于创建渲染目标及其他资源的构建器。

.setup( builder : Builder ) : Node

设置阴影节点以进行渲染。

builder

用于设置阴影节点的构建器。
重写： ShadowBaseNode#setup
返回值： 表示阴影值的节点。

.syncLightTransformation( lwLight : LwLight, sourceLight : Light )

将瓦片光源的变换与源光源同步。

lwLight

要同步的瓦片光源。
sourceLight

要复制变换的源光源。

.update()

更新每个瓦片的光源变换与阴影相机。

重写： ShadowBaseNode#update

.updateBefore( frame : NodeFrame )

此实现在必要时更新阴影贴图。

frame

当前节点帧的引用。
重写： ShadowBaseNode#updateBefore

.updateLightDirection()

根据光源的目标位置更新初始光照方向。

.updateShadow( frame : NodeFrame )

更新阴影贴图渲染。

frame

当前节点帧的引用。
