ShapeGeometry

由一个或多个路径形状创建单面多边形几何体。

代码示例

const arcShape = new THREE.Shape()
	.moveTo( 5, 1 )
	.absarc( 1, 1, 4, 0, Math.PI * 2, false );
const geometry = new THREE.ShapeGeometry( arcShape );
const material = new THREE.MeshBasicMaterial( { color: 0x00ff00, side: THREE.DoubleSide } );
const mesh = new THREE.Mesh( geometry, material ) ;
scene.add( mesh );

构造函数

new ShapeGeometry( shapes : Shape | Array.<Shape>, curveSegments : number )

构造一个新的形状几何体。

shapes

一个形状或形状数组。
curveSegments

每个形状的分段数。

默认值为 12.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

静态方法

.fromJSON( data : Object, shapes : Array.<Shape> ) : ShapeGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
shapes

形状数组。
返回值： 新实例。
