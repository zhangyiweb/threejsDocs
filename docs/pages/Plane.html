Plane

在三维空间中无限延伸的二维平面，以 Hessian 法线形式 由单位长度法向量和常量表示。

构造函数

new Plane( normal : Vector3, constant : number )

构造一个新平面。

normal

定义平面法线的单位长度向量。

默认值为 (1,0,0)。
constant

从原点到平面的有符号距离。

默认值为 0。

属性

.constant : number

从原点到平面的有符号距离。

默认值为 0。

.isPlane : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.normal : Vector3

定义平面法线的单位长度向量。

方法

.applyMatrix4( matrix : Matrix4, optionalNormalMatrix : Matrix4 ) : Plane

对平面应用 4x4 矩阵。该矩阵必须是仿射齐次变换。

可选的法线矩阵可预先计算，例如：

const optionalNormalMatrix = new THREE.Matrix3().getNormalMatrix( matrix );
matrix

变换矩阵。
optionalNormalMatrix

预先计算的法线矩阵。
返回值： 此平面的引用。

.clone() : Plane

返回一个从此实例复制值的新平面。

返回值： 此实例的克隆。

.coplanarPoint( target : Vector3 ) : Vector3

返回与平面共面的向量，通过将原点处的法线投影到平面上得到。

target

用于存储方法结果的目标向量。
返回值： 共面点。

.copy( plane : Plane ) : Plane

将给定平面的值复制到此实例。

plane

要复制的平面。
返回值： 此平面的引用。

.distanceToPoint( point : Vector3 ) : number

返回给定点到此平面的有符号距离。

point

要计算距离的点。
返回值： 有符号距离。

.distanceToSphere( sphere : Sphere ) : number

返回给定球体到此平面的有符号距离。

sphere

要计算距离的球体。
返回值： 有符号距离。

.equals( plane : Plane ) : boolean

若此平面与给定平面相等，则返回 true。

plane

用于相等性测试的平面。
返回值： 此平面是否与给定平面相等。

.intersectLine( line : Line3, target : Vector3, clampToLine : boolean ) : Vector3

返回给定直线与平面的交点。若直线不相交则返回 null。若直线与平面共面，则返回直线的起点。

line

要计算相交的直线。
target

用于存储方法结果的目标向量。
clampToLine

是否将交点限制在线段范围内。

默认值为 true。
返回值： 交点。若未检测到相交则返回 null。

.intersectsBox( box : Box3 ) : boolean

若给定包围盒与平面相交，则返回 true。

box

要测试的包围盒。
返回值： 给定包围盒是否与平面相交。

.intersectsLine( line : Line3 ) : boolean

若给定线段与平面相交（穿过平面），则返回 true。

line

要测试的直线。
返回值： 给定线段是否与平面相交。

.intersectsSphere( sphere : Sphere ) : boolean

若给定包围球与平面相交，则返回 true。

sphere

要测试的包围球。
返回值： 给定包围球是否与平面相交。

.negate() : Plane

对平面法线和常量取反。

返回值： 此平面的引用。

.normalize() : Plane

归一化平面法线，并相应调整常量。

返回值： 此平面的引用。

.projectPoint( point : Vector3, target : Vector3 ) : Vector3

将给定点投影到平面上。

point

要投影的点。
target

用于存储方法结果的目标向量。
返回值： 平面上的投影点。

.set( normal : Vector3, constant : number ) : Plane

通过复制给定值设置平面分量。

normal

法线。
constant

常量。
返回值： 此平面的引用。

.setComponents( x : number, y : number, z : number, w : number ) : Plane

通过将 x、y、z 定义为平面法线、w 定义为常量来设置平面分量。

x

法线 x 分量的值。
y

法线 y 分量的值。
z

法线 z 分量的值。
w

常量值。
返回值： 此平面的引用。

.setFromCoplanarPoints( a : Vector3, b : Vector3, c : Vector3 ) : Plane

由三个共面点设置平面。假定绕序为逆时针，并由此决定平面法线方向。

a

第一个共面点。
b

第二个共面点。
c

第三个共面点。
返回值： 此平面的引用。

.setFromNormalAndCoplanarPoint( normal : Vector3, point : Vector3 ) : Plane

根据给定法线和共面点（即位于平面上的点）设置平面。

normal

法线。
point

共面点。
返回值： 此平面的引用。

.translate( offset : Vector3 ) : Plane

按给定偏移向量定义的距离平移平面。 注意：这只会影响平面常量，不会影响法向量。

offset

偏移向量。
返回值： 此平面的引用。
