MeshPhongNodeMaterial

MeshPhongMaterial 的节点材质版本。

构造函数

new MeshPhongNodeMaterial( parameters : Object )

构造一个新的 mesh phong 节点材质。

parameters

配置参数。

属性

.isMeshPhongNodeMaterial : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.lights : boolean

设为 true，因为 phong 材质会对光照产生响应。

默认值为 true。

重写： NodeMaterial#lights

.shininessNode : Node.<float>

phong 材质的光泽度默认从 shininess 属性推断。此节点属性允许覆盖默认值， 改用节点定义光泽度。

若不想覆盖光泽度，而只想修改现有值， 请使用 materialShininess。

默认值为 null。

.specularNode : Node.<vec3>

phong 材质的高光颜色默认从 specular 属性推断。此节点属性允许覆盖默认值， 改用节点定义高光颜色。

若不想覆盖高光颜色，而只想修改现有值， 请使用 materialSpecular。

默认值为 null。

方法

已重写，因为此类材质使用 BasicEnvironmentNode 来实现默认的环境贴图映射。

builder

.setupLightingModel() : PhongLightingModel

设置光照模型。

重写： NodeMaterial#setupLightingModel
返回值： 光照模型。

.setupVariants( builder : NodeBuilder )

设置 phong 专用的节点变量。

builder

重写： NodeMaterial#setupVariants
