TessellateModifier

此类可通过打断超过最大长度的边来修改几何体。

代码示例

const modifier = new TessellateModifier( 8, 6 );
geometry = modifier.modify( geometry );

导入

TessellateModifier 是一个插件，必须显式导入，参见 Installation#Addons。

import { TessellateModifier } from 'three/addons/modifiers/TessellateModifier.js';

构造函数

new TessellateModifier( maxEdgeLength : number, maxIterations : number )

构造一个新的细分修改器。

maxEdgeLength

最大边长。

默认值为 0.1。
maxIterations

迭代次数。

默认值为 6。

属性

.maxEdgeLength : number

最大边长。

默认值为 0.1。

.maxIterations : number

迭代次数。

默认值为 6。

方法

.modify( geometry : BufferGeometry ) : BufferGeometry

通过对给定几何体应用细分，返回一个新的、已修改的几何体版本。 请注意，结果几何体始终是非索引几何体。

geometry

要修改的几何体。
返回值： 一个新的、已修改的几何体。
