JoinNode

此模块是 TSL 核心的一部分，通常不在应用层代码中使用。 它表示着色器生成过程中的连接（join）操作。 例如，可以将两个单独的浮点数组合/连接成 vec2 类型。

构造函数

new JoinNode( nodes : Array.<Node>, nodeType : string )

构造一个新的连接节点。

nodes

应被连接的节点数组。
nodeType

节点类型。

默认值为 null。

属性

.nodes : Array.<Node>

应被连接的节点数组。

方法

.generateNodeType( builder : NodeBuilder ) : string

此方法被重写，因为若未显式定义，节点类型必须根据 连接后的数据长度推断。

builder

当前的节点构建器。
重写： TempNode#generateNodeType
返回值： 节点类型。
