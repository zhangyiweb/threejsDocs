NURBSSurface

此类表示一个 NURBS 曲面。

实现基于带权重 w=weight 的控制点 (x, y [, z=0 [, w=1]])。

导入

NURBSSurface 是一个插件，必须显式导入，参见 Installation#Addons。

import { NURBSSurface } from 'three/addons/curves/NURBSSurface.js';

构造函数

new NURBSSurface( degree1 : number, degree2 : number, knots1 : Array.<number>, knots2 : Array.<number>, controlPoints : Array.<Array.<(Vector2|Vector3|Vector4)>> )

构造一个新的 NURBS 曲面。

degree1

第一维 NURBS 阶数。
degree2

第二维 NURBS 阶数。
knots1

第一维节点，以数字平铺数组表示。
knots2

第二维节点，以数字平铺数组表示。
controlPoints

保存控制点的二维数组。

属性

.controlPoints : Array.<Array.<(Vector2|Vector3|Vector4)>>

保存控制点数组的数组。

.degree1 : number

第一维 NURBS 阶数。

.degree2 : number

第二维 NURBS 阶数。

.knots1 : Array.<number>

第一维节点，以数字平铺数组表示。

.knots2 : Array.<number>

第二维节点，以数字平铺数组表示。

方法

.getPoint( t1 : number, t2 : number, target : Vector3 )

此方法根据给定的插值因子返回三维空间中的向量。该向量位于 NURBS 曲面上。

t1

表示曲面上 u 位置的第一插值因子，必须在 [0,1] 范围内。
t2

表示曲面上 v 位置的第二插值因子，必须在 [0,1] 范围内。
target

结果写入的目标向量。
