FlipNode

此模块属于 TSL 核心，通常不在应用层代码中使用。 它表示着色器生成过程中的翻转操作， 即按以下公式翻转归一化值：

x = 1 - x;

FlipNode 在内部用于实现节点对象上任意 flipXYZW()、flipRGBA() 以及 flipSTPQ() 方法的调用。例如：

uvNode = uvNode.flipY();

构造函数

new FlipNode( sourceNode : Node, components : string )

构造一个新的翻转节点。

sourceNode

应翻转其分量的节点。
components

应翻转的分量，例如 'x' 或 'xy'。

属性

.components : string

应翻转的分量，例如 'x' 或 'xy'。

.sourceNode : Node

应翻转其分量的节点。
