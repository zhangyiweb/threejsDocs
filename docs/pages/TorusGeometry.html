TorusGeometry

用于表示圆环体的几何体类。

代码示例

const geometry = new THREE.TorusGeometry( 10, 3, 16, 100 );
const material = new THREE.MeshBasicMaterial( { color: 0xffff00 } );
const torus = new THREE.Mesh( geometry, material );
scene.add( torus );

构造函数

new TorusGeometry( radius : number, tube : number, radialSegments : number, tubularSegments : number, arc : number, thetaStart : number, thetaLength : number )

构造一个新的圆环体几何体。

radius

圆环半径，从圆环中心到管道中心的距离。

默认值为 1.
tube

管道半径。必须小于 radius。

默认值为 0.4.
radialSegments

径向分段数。

默认值为 12.
tubularSegments

管道方向分段数。

默认值为 48.
arc

圆心角（弧度）。

默认值为 Math.PI*2.
thetaStart

管道扫掠的起始角度（弧度）。

默认值为 0.
thetaLength

管道扫掠的长度（弧度）。

默认值为 Math.PI*2.

属性

.parameters : Object

保存用于生成该几何体的构造函数参数。实例化后的任何修改都不会改变几何体。

静态方法

.fromJSON( data : Object ) : TorusGeometry

工厂方法，根据给定的 JSON 对象创建该类的实例。

data

表示已序列化几何体的 JSON 对象。
返回值： 新实例。
