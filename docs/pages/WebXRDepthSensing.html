WebXRDepthSensing

管理 Depth Sensing API 访问的 XR 模块。

构造函数

new WebXRDepthSensing()

构造一个新的深度感知模块。

属性

.depthFar : number

深度远裁剪值。

.depthNear : number

深度近裁剪值。

.mesh : Mesh

用于可视化深度纹理的平面网格。

.texture : ExternalTexture

表示用户环境深度的不透明纹理。

方法

.getDepthTexture() : ExternalTexture

返回表示用户环境深度的纹理。

返回值： 深度纹理。

.getMesh( cameraXR : ArrayCamera ) : Mesh

返回用于可视化深度纹理的平面网格。

cameraXR

XR 相机。
返回值： 平面网格。

.init( depthData : XRWebGLDepthInformation, renderState : XRRenderState )

初始化深度感知模块。

depthData

XR 深度数据。
renderState

XR 渲染状态。

.reset()

重置该模块。
