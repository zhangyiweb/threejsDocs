FigureEightPolynomialKnot

8 字多项式结。

导入

FigureEightPolynomialKnot 是一个插件，必须显式导入，参见 Installation#Addons。

import { FigureEightPolynomialKnot } from 'three/addons/curves/CurveExtras.js';

构造函数

new FigureEightPolynomialKnot( scale : number )

构造一个新的 8 字多项式结。

scale

曲线的缩放比例。

默认值为 1。

属性

.scale : number

曲线的缩放比例。

默认值为 1。

方法

.getPoint( t : number, optionalTarget : Vector3 ) : Vector3

此方法根据给定的插值因子返回三维空间中的一个向量。

t

表示曲线上位置的插值因子，必须在 [0,1] 范围内。
optionalTarget

可选的目标向量，结果将写入其中。
重写： Curve#getPoint
返回值： 曲线上的位置。
