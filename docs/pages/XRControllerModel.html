XRControllerModel

表示 XR 控制器模型。

构造函数

new XRControllerModel()

构造一个新的 XR 控制器模型。

属性

.envMap : Texture

控制器的环境贴图。

默认值为 null。

.motionController : MotionController

运动控制器。

默认值为 null。

方法

.setEnvironmentMap( envMap : Texture ) : XRControllerModel

设置应用于控制器模型的环境贴图。

envMap

要应用的环境贴图。
返回值： 对本实例的引用。

.updateMatrixWorld( force : boolean )

已用自定义实现重写。从 XRInputSource 轮询数据，并更新模型的组件以匹配真实世界数据。

force

设为 true 时，即使 Object3D#matrixWorldAutoUpdate 设为 false，也会强制重新计算世界矩阵。

默认值为 false。
重写： Object3D#updateMatrixWorld
