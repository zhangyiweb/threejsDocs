ReferenceElementNode

此类仅在引用属性为类数组类型时相关。 在此情况下，ReferenceElementNode 允许通过索引 引用数据结构中的特定元素。

构造函数

new ReferenceElementNode( referenceNode : ReferenceBaseNode, indexNode : Node )

构造一个新的引用元素节点。

referenceNode

引用节点。
indexNode

定义元素访问的索引节点。

属性

.isReferenceElementNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.isReferenceElementNode : boolean (readonly)

此标志可用于类型检测。

默认值为 true。

.referenceNode : ReferenceBaseNode

类似于 ReferenceBaseNode#reference，额外的 属性指向当前节点。

默认值为 null。

.referenceNode : ReferenceNode

类似于 ReferenceNode#reference，额外的 属性指向当前节点。

默认值为 null。

方法

.generateNodeType() : string

此方法已重写，因为节点类型从 引用节点的 uniform 类型推断。

重写： ArrayElementNode#generateNodeType
返回值： 节点类型。

.generateNodeType() : string

此方法已重写，因为节点类型从 引用节点的 uniform 类型推断。

重写： ArrayElementNode#generateNodeType
返回值： 节点类型。
