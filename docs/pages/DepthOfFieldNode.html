DepthOfFieldNode

用于创建景深（DOF）效果的后处理节点。

参考：

  • https://pixelmischiefblog.wordpress.com/2016/11/25/bokeh-depth-of-field/
  • https://www.adriancourreges.com/blog/2016/09/09/doom-2016-graphics-study/

导入

DepthOfFieldNode 是一个插件，必须显式导入，参见 Installation#Addons。

import { dof } from 'three/addons/tsl/display/DepthOfFieldNode.js';

构造函数

new DepthOfFieldNode( textureNode : TextureNode, viewZNode : Node.<float>, focusDistanceNode : Node.<float>, focalLengthNode : Node.<float>, bokehScaleNode : Node.<float> )

构造一个新的 DOF 节点。

textureNode

表示效果输入的纹理节点。
viewZNode

表示场景 viewZ 深度值。
focusDistanceNode

定义效果的焦点，即沿相机观察方向的世界单位距离。
focalLengthNode

物体距离焦平面多远会完全失焦，以世界单位表示。
bokehScaleNode

用于艺术化调整散景大小的无量纲值。

属性

.bokehScaleNode : Node.<float>

用于艺术化调整散景大小的无量纲值。

.focalLengthNode : Node.<float>

物体距离焦平面多远会完全失焦，以世界单位表示。

.focusDistanceNode : Node.<float>

定义效果的焦点，即沿相机观察方向的世界单位距离。

.textureNode : TextureNode

表示效果输入的纹理节点。

.updateBeforeType : string

updateBeforeType 设置为 NodeUpdateType.FRAME，因为该节点在 updateBefore() 中每帧更新一次内部 uniform。

默认值为 'frame'。

重写： TempNode#updateBeforeType

.viewZNode : Node.<float>

表示场景 viewZ 深度值。

方法

.dispose()

释放内部资源。当不再需要该效果时应调用此方法。

重写： TempNode#dispose

.getTextureNode() : PassTextureNode

以纹理节点的形式返回效果结果。

返回值： 表示效果结果的纹理节点。

.setSize( width : number, height : number )

设置效果尺寸。

width

效果的宽度。
height

效果的高度。

.setup( builder : NodeBuilder ) : ShaderCallNodeInternal

用于设置效果的 TSL 代码。

builder

当前的节点构建器。
重写： TempNode#setup

.updateBefore( frame : NodeFrame )

用于每帧更新一次效果的 uniform。

frame

当前的节点帧。
重写： TempNode#updateBefore
